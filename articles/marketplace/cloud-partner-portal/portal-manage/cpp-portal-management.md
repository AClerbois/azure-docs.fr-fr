# <a name="cloud-partner-portal-management"></a>Gestion du portail des partenaires cloud

Cette section décrit les types d’opérations disponibles pour créer et gérer les comptes requis pour utiliser le [portail Microsoft Cloud Partner](https://cloudpartner.azure.com).  Ces comptes sont requis pour le [place de marché Azure](https://azuremarketplace.microsoft.com) et [AppSource](https://appsource.microsoft.com).  


## <a name="background-information"></a>Informations contextuelles

Avant d’utiliser le portail Cloud Partner, vous devez vous familiariser avec les articles suivants :

- [Stratégies de la place de marché Azure et les conditions](../../policies-terms.md) répertorie les différentes exigences juridiques et techniques et les règles qui s’appliquent aux utilisateurs d’et les éditeurs à la [place de marché Azure](https://azuremarketplace.microsoft.com) et [AppSource](https://appsource.microsoft.com).
- [Devenir un éditeur de place de marché Cloud](../../become-publisher.md) fournit une vue d’ensemble de toutes les étapes requises pour devenir un serveur de publication de cloud Microsoft officielle.


## <a name="account-creation-and-management"></a>La création du compte et gestion

Cette section contient les articles suivants qui expliquent comment gérer les comptes de serveur de publication de la place de marché.

- [Conditions préalables pour la gestion des comptes](./cpp-manage-prerequisites.md) explique comment devenir un serveur de publication cloud inscrit, la nécessité d’utiliser le portail CLoud Partner.
- [Gérer le profil d’éditeur Azure Marketplace et AppSource](./cpp-manage-publisher-profile.md) explique comment utiliser le portail Cloud Partner pour afficher et gérer votre profil d’éditeur place de marché Microsoft. 
- [La gestion des utilisateurs sur le portail Cloud Partner](./cpp-manage-users.md) ajouter des utilisateurs au cloud de votre organisation compte et comment gérer leur accès, pour permettre à plusieurs personnes de travailler sur les offres mêmes publication.


## <a name="next-steps"></a>Étapes suivantes

Vous devez respecter les [conditions préalables pour la gestion des comptes](./cpp-manage-prerequisites.md) avant de continuer.  Par la suite, vous devez également prendre la [visite du portail Cloud Partner](../portal-tour/cpp-portal-tour.md) pour vous familiariser avec l’interface utilisateur du portail.
