# Détails concernant le comportement du CDN Azure fourni par Akamai

Cet article décrit en détail le comportement autorisé spécifique du **CDN Azure fourni par Akamai**.

> [AZURE.IMPORTANT] Les informations contenues dans cet article ne s’appliquent pas au CDN Azure ayant pour origine des points de terminaison Verizon.

## Ports d’origine autorisés

Le **CDN Azure fourni par Akamai** permet d’accéder à un sous-ensemble de ports TCP sur le serveur d’origine. Voici la liste des plages de ports autorisées :
