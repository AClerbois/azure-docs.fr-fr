# <a name="stream-analytics-resource-health-page-in-the-azure-portal"></a>Page Resource Health de Stream Analytics du portail Azure

La page **Resource Health** du portail Azure affiche l’intégrité de vos travaux Azure Stream Analytics. Il affiche également des conseils actionnables qui peuvent vous aider à résoudre les problèmes. 

La page **Resource Health** vous permet de déterminer plus rapidement si la racine d’un problème se trouve dans l’application, ou si elle est provoquée par un événement dans Azure. Cela peut vous aider à déterminer quand ouvrir une demande de support pour recevoir de l’aide.

## <a name="troubleshooting-workflow"></a>Résolution de problèmes de flux de travail

Pour voir l’intégrité des ressources pour vos travaux Stream Analytics, sélectionnez la page **Resource Health**. 

![Flux de résolution des problèmes Stream Analytics - accès](media/stream-analytics-resource-health/stream-analytics-access-troubleshooting.png)

L’organigramme suivant peut vous aider à prendre des décisions sur les éléments à rechercher et à déterminer le moment auquel ouvrir une demande de support.

![Flux de résolution des problèmes Stream Analytics - mappage](media/stream-analytics-resource-health/stream-analytics-troubleshooting-map.png)
