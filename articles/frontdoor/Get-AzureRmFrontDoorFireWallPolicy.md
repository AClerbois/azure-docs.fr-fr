# Get-AzureRmFrontDoorFireWallPolicy

Obtenir la stratégie de pare-feu d’applications web (WAF)

## SYNTAXE

```
Get-AzureRmFrontDoorFireWallPolicy -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Description
L’applet de commande **Get-AzureRmFrontDoorFireWallPolicy** obtient la stratégie de pare-feu d’applications web (WAF) dans un groupe de ressources sous l’abonnement actuel

## EXEMPLES

### Exemple 1 : Obtenir une stratégie de pare-feu d’applications web (WAF) appelée $Name dans $resourceGroup
```powershell
PS C:\> Get-AzureRmFrontDoorFireWallPolicy -Name $Name -ResourceGroupName $resourceGroup

                     osoft.Network/frontdoorwebapplicationfirewallpolicies/{Name}

```
