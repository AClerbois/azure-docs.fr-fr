> [!NOTE]
> Avant de commencer ce didacticiel, terminez la [simulateur en ligne Raspberry Pi](../articles/iot-hub/iot-hub-raspberry-pi-web-simulator-get-started.md) didacticiel ou un des didacticiels appareil ; par exemple, [Raspberry Pi avec node.js](../articles/iot-hub/iot-hub-raspberry-pi-kit-node-get-started.md). Dans ces articles, vous configurez votre Azure IoT device et un IoT hub, et vous déployez un exemple d’application pour s’exécuter sur votre appareil. L’application envoie les données de capteur collectées à votre instance IoT Hub.
