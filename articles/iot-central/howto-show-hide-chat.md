# <a name="toggle-live-chat"></a>Activer/désactiver les conversation en direct

Cet article vous montre comment activer/désactiver la conversation en direct dans votre application IoT Central. Vous pouvez utiliser conversation en direct pour accéder au support technique.

> [!NOTE]
> L’option de conversation est disponible uniquement pour les abonnements d’essai.

## <a name="chat-with-us"></a>Discutez avec nous

Pour obtenir un support technique, ouvrez votre IoT Central application et sélectionnez **Discutez avec nous**.

![Discutez avec nous](media/howto-show-hide-chat/chat-with-us.png)

Vous pouvez taper une question comme indiqué dans la capture d’écran suivante :

![Formulaire « Configurer les détails de l’appareil » contenant les détails des paramètres et des propriétés](media/howto-show-hide-chat/sample-chat.png)

## <a name="hide-chat"></a>Masquer la conversation

Pour masquer la conversation, choisissez **conversation masquer** dans le **aide** panneau :

 ![Masquer la conversation](media/howto-show-hide-chat/hide-chat.png)

## <a name="enable-chat"></a>Activer la conversation

Pour afficher la conversation, choisissez **afficher le Chat** dans le **aide** panneau :

 ![Afficher la conversation](media/howto-show-hide-chat/show-chat.png)
