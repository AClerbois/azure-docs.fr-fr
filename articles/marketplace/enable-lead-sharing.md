# <a name="enable-lead-sharing"></a>Activer le partage des prospects
Assurez-vous que la gestion des prospects est activée pour votre annonce dans la [Place de marché Azure](https://azuremarketplace.microsoft.com) ou [Microsoft AppSource](https://appsource.microsoft.com) afin que vous puissiez recevoir les coordonnées des publics suivants :

| Vitrine | Public visé |
|:--- |:--- |
| AppSource | Clients professionnels |
| Place de marché Azure | Tous les clients |

Vos prospects constituent la base des campagnes répertoriées dans le tableau suivant, des méthodes de vente utilisées par vos commerciaux de terrain, et des informations concernant les performances de votre application ou offre :

| Vitrine | Campagne |
|:--- |:--- |
| AppSource | Nurture |
| Place de marché Azure | Génération dans la demande |

Utilisez les pratiques suivantes pour vos prospects :
*   Qualifier vos prospects et évaluer les prospects comme des opportunités de vente.
*   Renforcer votre relation avec les prospects pour les transformer en opportunités.
*   Aligner vos efforts sur la stratégie globale de votre campagne marketing.

Vos prospects présentent un intérêt client dans les annonces suivantes :

| Vitrine | Énumération |
|:--- |:--- |
| AppSource | <table> <tr><td>Application</td></tr> <tr><td>Services de conseil</td></tr> </table> |
| Place de marché Azure | <table> <tr><td>Offre</td></tr> <tr><td>Technology</td></tr> </table> |

Vous devez disposer d’un processus pour effectuer le suivi de vos prospects, comme décrit dans le tableau ci-dessous :

| Vitrine | Prospects |
|:--- |:--- |
| AppSource | <table> <tr><td>Votre CRM</td></tr> <tr><td>CRM vendeur sur le terrain Microsoft</td></tr> </table> |
| Place de marché Azure | <table> <tr><td>Place de marché Azure</td></tr> <tr><td>CRM vendeur sur le terrain Microsoft</td></tr> </table> |

Vos prospects dans les vitrines sur la Place de marché diffèrent des prospects qualifiés par le commercial Microsoft, qui font partie du programme de covente.

Pour plus d’informations sur la façon d’accéder au programme de co-vente, consultez la page [Promouvoir votre entreprise avec Microsoft](./promote-your-business-with-microsoft.md).
