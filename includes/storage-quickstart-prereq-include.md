Pour accéder à Stockage Azure, vous avez besoin d’un abonnement Azure. Si vous n’avez pas d’abonnement, vous pouvez [créer un compte gratuit](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) avant de commencer.

Tous les accès à Stockage Azure se font via un compte de stockage. Si vous ne disposez pas déjà d’un compte de stockage, vous pouvez facilement en créer un en utilisant le [portail Azure](https://portal.azure.com/#create/Microsoft.StorageAccount-ARM). Vous pouvez également créer un compte de stockage avec Azure PowerShell ou Azure CLI. Si vous avez besoin d’aide pour créer un compte de stockage, consultez [Créez un compte de stockage](../articles/storage/common/storage-quickstart-create-account.md).
