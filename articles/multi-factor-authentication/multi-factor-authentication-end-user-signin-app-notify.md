# Connexion à l'application mobile par notification avec Azure Multi-Factor Authentication

Les informations suivantes décrivent l'utilisation de l'authentification multifacteur avec votre application mobile, lorsqu'une notification vous est envoyée.

## Connexion avec une notification envoyée à votre application mobile

<ol>

<li>Connectez-vous à une application ou à un service comme Office&#160;365 à l'aide de votre nom d'utilisateur et de votre mot de passe.</li>
<li>Microsoft vous envoie une notification.</li>


<center>![Setup](./media/multi-factor-authentication-end-user-signin-app-notify/notify.png)</center>

<li>Répondez au téléphone et appuyez sur la touche de vérification.</li>

<center>![Setup](./media/multi-factor-authentication-end-user-signin-app-notify/phone.png)</center>


<li>Vous devez maintenant être connecté.</li>
