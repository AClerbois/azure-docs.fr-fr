# Intégration des alertes du Centre de sécurité avec les journaux Azure (version préliminaire)

De nombreuses équipes de sécurité et d’intervention utilisent une solution SIEM (Security Information and Event Management) comme point de départ pour le triage et l’examen des alertes de sécurité. Grâce à l’intégration des journaux Azure, les clients peuvent synchroniser quasiment en temps réel les alertes du Centre de sécurité Azure, ainsi que les événements de sécurité des machines virtuelles collectés par les journaux Azure Diagnostics et Azure Audit, avec leurs journaux analytiques ou une solution SIEM.

L’intégration des journaux Azure fonctionne avec HP ArcSight, Splunk, IBM Qradar, entre autres. Pour commencer, téléchargez [Intégration des journaux Azure](https://www.microsoft.com/download/details.aspx?id=53324).
