# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Aide pour les clients Azure Data Lake Storage Gen1

Azure Data Lake Storage Gen2 Preview offre, en plus du système de fichiers compatible Hadoop disponible dans Data Lake Storage Gen1, des fonctionnalités de stockage d’objets blob natives comme la hiérarchisation des données, la gestion du cycle de vie et une sécurité réseau améliorée.

Si vous utilisez actuellement Data Lake Storage Gen1, Microsoft vous encourage à évaluer l’ensemble des nouvelles fonctionnalités de Data Lake Storage Gen2 et des améliorations apportées, puis de [participer à la préversion](https://aka.ms/adlsgen2signup) pour donner votre avis. Notez que Data Lake Storage Gen1 est encore pris en charge. Étant donné que Data Lake Storage Gen1 et Gen2 peuvent coexister, vos ressources et applications existantes qui exploitent Data Lake Storage Gen1 continuent à fonctionner sans mises à jour. Vous pouvez donc développer un plan de migration vers Data Lake Storage Gen2 à votre convenance.

Si vous évaluez actuellement Data Lake Storage Gen1, Microsoft vous encourage à participer à la préversion Gen2 et à tester les nouvelles fonctionnalités de Data Lake Storage Gen2 et les améliorations apportées. De même, si vous êtes actuellement un client du stockage d’objet blob, Microsoft vous encourage à déterminer si un système de fichiers compatible Hadoop est adapté à vos besoins analytiques.
