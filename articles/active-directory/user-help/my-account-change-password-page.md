# <a name="change-your-work-or-school-account-password-from-the-my-account-portal"></a>Changer le mot de passe de votre compte professionnel ou scolaire à partir du portail Mon compte

À partir de la page **Modifier le mot de passe** du portail **Mon compte**, vous pouvez mettre à jour le mot de passe de votre compte professionnel ou scolaire. Cela ne fonctionnera que si vous connaissez votre mot de passe existant et si vous n’êtes pas verrouillé sur votre compte.

Si vous ne vous souvenez pas de votre mot de passe, si votre compte a été verrouillé ou si vous n’avez jamais reçu de mot de passe de l’équipe de support technique de votre entreprise, vous pouvez utiliser vos informations de sécurité et votre appareil mobile pour réinitialiser votre mot de passe. Pour plus d’informations, consultez [Réinitialiser votre mot de passe professionnel ou scolaire](https://docs.microsoft.com/azure/active-directory/user-help/user-help-reset-password).

>[!Note]
>Si vous tentez de récupérer un compte personnel comme pour Xbox, hotmail.com ou outlook.com, essayez les suggestions fournies dans l’article [Quand vous ne pouvez pas vous connecter à votre compte Microsoft](https://support.microsoft.com/help/12429/microsoft-account-sign-in-cant).

## <a name="to-change-your-password-from-the-my-account-portal"></a>Pour modifier votre mot de passe à partir du portail Mon compte

1. Connectez-vous à votre compte professionnel ou scolaire, en utilisant votre mot de passe existant puis accédez à votre page https://myprofile.microsoft.com/.

2. Sélectionnez **Mot de passe** dans le volet de navigation de gauche ou sélectionnez le lien **Modifier le mot de passe** dans le bloc **Mot de passe**.

    ![Page Mon compte, avec le lien Modifier le mot de passe affiché en surbrillance](media/my-account-portal/my-account-portal-change-password.png)

3. Saisissez votre ancien mot de passe, puis créez un nouveau mot de passe et confirmez-le.

    ![Page Modifier le mot de passe, affichant des champs de mot de passe](media/my-account-portal/my-account-portal-change-password-page.png)

4. Sélectionnez **Envoyer**.

    Votre mot de passe sera modifié et vous serez invité à vous reconnecter à votre compte professionnel ou scolaire.

## <a name="next-steps"></a>Étapes suivantes

Après avoir modifié votre mot de passe, vous pouvez :

- Afficher ou gérer vos [informations de sécurité](user-help-security-info-overview.md).

- Afficher ou gérer vos [appareils](my-account-portal-devices-page.md) connectés.

- Afficher ou gérer vos [organisations](my-account-portal-organizations-page.md).

- Afficher votre [activité de connexion](my-account-portal-sign-ins-page.md).

- Afficher la façon dont votre organisation [utilise vos données relatives à la confidentialité](my-account-portal-privacy-page.md).
