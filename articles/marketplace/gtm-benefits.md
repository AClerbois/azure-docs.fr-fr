# <a name="gtm-benefits-on-the-marketplace"></a>Avantages de GTM sur la Place de marché   
Dans les vitrines de la Place de marché, les nouvelles annonces bénéficient gratuitement d’avantages de commercialisation (GTM). Une fois l’annonce effectuée, les spécialistes en marketing chez Microsoft s’adressent à vous ou votre contact marketing pour démarrer rapidement les activités. Vous n’avez rien à faire, si ce n’est collaborer avec Microsoft une fois que vous avez été contacté.  

Les activités fournies par Microsoft diffèrent selon l’état de votre solution dans la Place de marché. Les avantages augmentent considérablement pour les annonces.  

| vitrine | Autres avantages disponibles  |  
|:--- |:--- |  
| AppSource | <table> <tr><td>Partenaires Microsoft Gold</td></tr> <tr><td>Applications d’évaluation</td></tr> </table> |  
| Place de marché Azure | <table> <tr><td>Offres de transaction</td></tr> <tr><td>Offres d’évaluation</td></tr> </table> |  

Pour optimiser l’impact de ces activités, il est préférable selon Microsoft que votre plan de lancement soit prêt à être mis en œuvre. Vous pouvez utiliser votre page d’accueil pour un grand nombre de vos tactiques de lancement.  

>[!NOTE]
>Le catalogue One Commercial Partner Catalog (catalogue OCP) est l’un des avantages dont bénéficient les membres de Microsoft Partner Network.  

![Avantages de GTM sur la Place de marché](./media/marketplace-publishers-guide/gtmmay2018.png)  

Votre support concernant la commercialisation inclut des modèles, du contenu web, des formations et des outils pour promouvoir votre entreprise.  
*   Pour plus d’informations sur le support concernant la commercialisation, visitez la page sur les Services Microsoft Go-To-Market à l’adresse [partner.microsoft.com/reach-customers/gtm](https://partner.microsoft.com/reach-customers/gtm).
