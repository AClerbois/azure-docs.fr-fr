Vous pouvez vous abonner à une base de données entièrement gérée et hébergée par Azure dans l’Azure Store. Pour ce faire, procédez comme suit :

1. Connectez-vous au portail de gestion Azure.
1. Cliquez sur **Nouveau**. ![Nouveau][button-new]
1. Sélectionnez **Store**. ![Store][button-store]
1. Sélectionnez MongoLab. Vous pouvez nous trouver sous la catégorie **App Services** ainsi que sous **Tout**. ![MongoLab][entry-mongolab]
1. Cliquez sur **Next**. ![Suivant][button-next] L'entrée du MongoLab Store s'affiche. ![NewMongoLab][screen-newmongolab]
1. Sélectionnez l’option **Abonnement** souhaitée.
1. Entrez un **Nom** pour la base de données. Le nom peut uniquement être composé de lettres, de chiffres, de tirets, de points et de traits de soulignement. MongoLab requiert également que ce nom soit unique. Il est possible que vous soyez invité à envoyer à nouveau votre demande si le nom est déjà pris.
1. Sélectionnez la **Région** souhaitée.
1. Cliquez sur **Suivant**. ![Suivant][button-next]
1. Vérifiez les informations relatives à votre achat sur le store, puis cliquez sur **Acheter** pour confirmer. ![Suivant][button-purchase]  
1. Le bouton de progression de la barre d’outils vous indique l’état de la configuration. ![ProgressButton][button-progress] Un message de réussite s'affiche lorsque la configuration est terminée. ![SuccessMessage][message-success]

Félicitations ! MongoLab a configuré une base de données MongoDB dans la région Azure que vous avez sélectionnée. Vous avez maintenant accès à notre interface utilisateur de gestion et à notre support 24 h/24 et 7 jours sur 7.
