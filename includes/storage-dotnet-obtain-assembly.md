### Obtention du package de la bibliothèque cliente de stockage
Vous pouvez utiliser NuGet pour obtenir le package de la bibliothèque cliente de stockage et ses dépendances. Cliquez avec le bouton droit sur votre projet dans l’**Explorateur de solutions**, puis sélectionnez **Manage NuGet Packages**. Effectuez une recherche en ligne sur « WindowsAzure.Storage », puis cliquez sur **Install** pour installer le package de stockage Azure et ses dépendances.

Le package de bibliothèque cliente Azure Storage est également disponible dans le [Kit de développement logiciel (SDK) Azure pour .NET](https://azure.microsoft.com/downloads/). Toutefois, nous vous recommandons d’installer la bibliothèque cliente de stockage à partir de NuGet pour vous assurer que vous avez toujours la dernière version de la bibliothèque cliente.
