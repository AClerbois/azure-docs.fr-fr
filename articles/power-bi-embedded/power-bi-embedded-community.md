# Communauté Power BI Embedded
Dans la version préliminaire publique, accédez à [Microsoft Power BI Community/developer](http://community.powerbi.com/t5/Developer/bd-p/Developer) pour obtenir l'aide de la communauté. Dans [Microsoft Power BI Community](http://community.powerbi.com/), vous pouvez également interagir avec les membres de la communauté afin d’obtenir de l’aide concernant l’utilisation de **Power BI Desktop** et **Power BI Service**.
