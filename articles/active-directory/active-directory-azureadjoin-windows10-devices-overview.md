# Windows 10 pour l’entreprise : plusieurs manières d’utiliser des appareils professionnels

Windows 10 vous offre la possibilité d’exploiter Azure Active Directory. Las appareils Windows 10 peuvent être connectés à Azure AD et les utilisateurs peuvent se connecter à Windows avec des comptes Azure AD ou ajouter leur code Azure pour accéder aux ressources et aux applications métier.

![](./media/active-directory-azureadjoin/windows10-overview.png)
