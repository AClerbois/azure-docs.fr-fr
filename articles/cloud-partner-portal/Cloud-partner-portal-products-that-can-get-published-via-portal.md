# <a name="what-products-can-be-published-via-the-cloud-partner-portal"></a>Produits pouvant être publiés via le portail Cloud Partner

Le portail Cloud Partner est votre hub pour publier votre image de machine virtuelle certifiée Azure sur la Place de marché Microsoft Azure, ou votre application Cortana Intelligence/Microsoft R Solutions sur Microsoft AppSource.  

Les images de machine virtuelle sont des images préconfigurées avec un système d’exploitation et une ou plusieurs applications entièrement installés. Pour publier votre image de machine virtuelle dans la Place de marché Azure, vous devez respecter les [prérequis techniques](https://docs.microsoft.com/azure/marketplace-publishing/marketplace-publishing-vm-image-creation-prerequisites) pour certifier votre image.
