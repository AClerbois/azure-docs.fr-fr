# <a name="approvals-page"></a>Page Approbations

La page [**Approbations**](https://cloudpartner.azure.com/#approvals) liste les offres déjà approuvées, avec le nom de l’offre, l’éditeur, le type de l’offre et l’action en cours.  Les offres approuvées sont soit déjà publiées, soit en cours de publication.  Vous pouvez classer les résultats par colonne ou les filtrer sur une chaîne que vous spécifiez.  La zone d’état en bas de la page affiche la valeur agrégée et le nombre de pages, et vous permet de naviguer entre les pages de la liste.  Quand vous sélectionnez une offre dans cette liste, cette offre s’affiche dans l’éditeur d’offre associé.
