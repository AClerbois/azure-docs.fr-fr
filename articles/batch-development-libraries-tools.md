# Outils et bibliothèques de développement Azure Batch 
<p> Récupérez ces bibliothèques et outils pour développer des applications Azure Batch et Batch Apps.

+ [Batch]
+ [Batch Apps]
+ [Ressources supplémentaires]

## Batch
 
+ [Bibliothèque Batch Client pour .NET](http://www.nuget.org/packages/Azure.Batch/) (NuGet) - Développez des applications clientes pour exécuter des travaux à l'aide du service Batch
+ [Bibliothèque Batch Management](http://www.nuget.org/packages/Microsoft.Azure.Management.Batch/) (NuGet) - Gérez les comptes de service Batch
+ [Batch Explorer](https://code.msdn.microsoft.com/windowsazure/Azure-Batch-Explorer-c1d37768) (MSDN) - Créez une application d'interface graphique utilisateur pour afficher et gérer les travaux Batch

## Batch Apps

+ [Kit de développement logiciel (SDK) Batch Apps Cloud](http://www.nuget.org/packages/Microsoft.Azure.Batch.Apps.Cloud/) (NuGet) - Permettez aux applications de se décharger des travaux auprès du service Batch
+ [Extension Visual Studio Batch Apps](https://visualstudiogallery.msdn.microsoft.com/8b294850-a0a5-43b0-acde-57a07f17826a) (galerie Visual Studio) - Créez des applications cloud dans Visual Studio à l'aide du Kit de développement logiciel (SDK) Batch Apps Cloud
+ [Kit de développement logiciel (SDK) Batch Apps Client](http://www.nuget.org/packages/Microsoft.Azure.Batch.Apps/) (NuGet) - Interagissez avec les applications qui peuvent se décharger des travaux auprès du service Batch
