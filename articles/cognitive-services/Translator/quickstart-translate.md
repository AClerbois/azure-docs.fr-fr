# <a name="quickstart-use-the-translator-to-translate-text"></a>Démarrage rapide : Utiliser Translator pour traduire du texte

Dans ce guide de démarrage rapide, vous allez apprendre à traduire une chaîne de texte de l’anglais vers l’allemand, l’italien, le japonais et le thaï à l’aide de l’API REST Translator.

Pour suivre ce démarrage rapide, vous devrez disposer d’un [compte Azure Cognitive Services](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account) avec une ressource Translator.
