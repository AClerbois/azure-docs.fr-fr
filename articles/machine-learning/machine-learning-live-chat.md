# Obtenir de l'aide de la part du support par conversation en direct sur Machine Learning
[Azure Machine Learning Studio](machine-learning-what-is-ml-studio.md) fournit une interface intuitive pour générer des modèles Machine Learning. Une [galerie](machine-learning-gallery-how-to-use-contribute-publish.md) et des [forums](https://social.msdn.microsoft.com/forums/azure/home?forum=MachineLearning) sont à votre disposition pour vous apporter de l'aide.

Cependant, pour vous débloquer, vous avez parfois besoin de poser une question rapide. Dans [Studio](machine-learning-what-is-ml-studio.md), recherchez l'icône de discussion dans le volet de navigation supérieur. Si cette icône est visible, cela signifie qu'un membre de l'équipe produit est en ligne pour vous apporter une aide en temps réel.

![Discussion Azure ML](./media/machine-learning-live-chat/AzureMLChatNavBar.png)

Tapez votre question pour obtenir une réponse !

![Boîte de dialogue de discussion Azure ML](./media/machine-learning-live-chat/AzureMLChat.png)

## Vous ne voyez pas l’icône de conversation en direct ?
Les membres de l’équipe sont en fait en train de répondre à votre question. Si vous ne voyez pas l’icône de conversation en direct, c’est qu’elle n’est pas visible en dehors des heures ouvrables.
