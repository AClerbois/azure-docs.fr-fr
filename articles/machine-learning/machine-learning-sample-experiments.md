# Utilisez des exemples d'expériences pour créer de nouvelles expériences
Pour créer une expérience, recherchez des exemples dans la [galerie Cortana Intelligence](http://gallery.cortanaintelligence.com/). Vous trouverez des exemples d’expériences fournis par l’équipe Machine Learning, ainsi que des exemples partagés par la communauté Machine Learning dans son ensemble. Vous pouvez poser des questions ou publier des commentaires sur les expériences.

[AZURE.INCLUDE [machine-learning-free-trial](../../includes/machine-learning-free-trial.md)]

## Parcourir la galerie Cortana Intelligence
Pour afficher les expériences disponibles dans la galerie, rendez-vous dans la [galerie](http://gallery.cortanaintelligence.com/), puis cliquez sur **Expériences** en haut de la page. Sur cette page, vous pouvez afficher les expériences **récemment ajoutées**, consulter les expériences **les plus populaires** ou découvrir les toutes dernières **expériences Microsoft les plus populaires**.

Pour parcourir toutes les expériences :

1. Cliquez sur **Parcourir tout** en haut de la page.
2. Sous **Affiner par**, sélectionnez **Expérience** pour afficher toutes les expériences de la galerie.
3. Vous pouvez affiner la liste en sélectionnant des filtres sur la gauche. Par exemple, pour parcourir les expériences reposant sur un algorithme de détection des anomalies PCA, sélectionnez **Expérience** sous **Catégories**, puis **Détection des anomalies PCA** sous **Algorithmes utilisés** (si vous ne voyez pas cet algorithme, cliquez sur **Afficher tout** en bas de la liste).<br></br> ![](./media/machine-learning-sample-experiments/refine-the-view.png) 
4. Vous pouvez utiliser la zone de recherche pour affiner vos choix. Par exemple, pour rechercher des expériences partagées par Microsoft à propos de la reconnaissance de chiffres utilisant un algorithme de machine à vecteurs de support à deux classes, entrez « digit recognition » dans la zone de recherche, puis sélectionnez **Expérience**, **Contenu Microsoft uniquement** et **Machine à vecteurs de support à deux classes** : ![](./media/machine-learning-sample-experiments/search-for-experiments.png) 
5. Cliquez sur une expérience pour en savoir plus à propos de celle-ci. Pour exécuter et/ou modifier l’expérience, cliquez sur **Ouvrir dans Studio** sur la page de l’expérience.

> [AZURE.NOTE] Pour ouvrir une expérience dans Machine Learning Studio, vous devez vous connecter avec les informations d’identification de votre compte Microsoft. Si vous n'avez pas encore d'espace de travail Machine Learning, un espace de travail d'essai gratuit est créé. [Que comprend l’essai gratuit de Machine Learning ?](https://azure.microsoft.com/pricing/details/machine-learning/)

![](./media/machine-learning-sample-experiments/example-experiment.png)


## Utiliser un modèle dans Machine Learning Studio

Vous pouvez également créer une expérience dans Machine Learning Studio en prenant un exemple de la galerie comme modèle.

1. Connectez-vous avec les informations d’identification de votre compte Microsoft à [Studio](https://studio.azureml.net), puis cliquez sur **Nouveau*** pour créer une expérience.
2. Parcourez les exemples de contenu, puis cliquez sur l’un d’entre eux.

Une expérience est créée dans votre espace de travail, et l’exemple d’expérience sert de modèle.
