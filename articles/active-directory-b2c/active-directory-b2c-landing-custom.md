# <a name="sign-up-and-sign-in-consumers-in-your-applications-using-custom-policies-in-azure-active-directory-b2c"></a>Inscription et connexion de consommateurs à vos applications à l’aide de stratégies personnalisées dans Azure Active Directory B2C

Les stratégies personnalisées sont des fichiers de configuration qui définissent le comportement de votre client Azure AD B2C. Elles peuvent être entièrement modifiées par un développeur d’identité pour effectuer un nombre presque illimité de tâches.

Découvrez comment utiliser des fonctionnalités spécifiques d’Azure Active Directory B2C :

* Configurer les fournisseurs OIDC comme [Azure AD](active-directory-b2c-setup-aad-custom.md).
* Configurer les fournisseurs SAML comme [Salesforce](active-directory-b2c-setup-sf-app-custom.md).
* Intégration des API RESTful
    * [Obtenir des revendications supplémentaires](active-directory-b2c-rest-api-step-custom.md)
    * [Valider les entrées utilisateur](active-directory-b2c-rest-api-validation-custom.md)
* Personnaliser la connexion :
    * [Configurer des entrées d’utilisateur](active-directory-b2c-configure-signup-self-asserted-custom.md)
    * [Personnaliser l’interface utilisateur](active-directory-b2c-ui-customization-custom.md)
    * [Personnaliser le jeton](active-directory-b2c-reference-manage-sso-and-token-configuration.md)
* Résoudre les problèmes en [collectant les journaux d’activité à l’aide d’Application Insights](active-directory-b2c-troubleshoot-custom.md).
