Cet article utilise des cmdlets PowerShell. Pour exécuter les applets de commande, vous pouvez utiliser Azure Cloud Shell, un environnement d’interpréteur de commandes interactif hébergé dans Azure et utilisée par le biais du navigateur. Azure Cloud Shell est fourni avec les applets de commande PowerShell de Azure préalablement installée.

Pour exécuter n’importe quel code contenue dans cet article sur Azure Cloud Shell, ouvrez une session Cloud Shell, utilisez le **copie** bouton sur un bloc de code pour copier le code et collez-le dans la session Cloud Shell avec __Ctrl + Maj + V__ sur Windows et Linux, ou __Cmd + Maj + V__ sur macOS. Texte collé n’est pas exécutée automatiquement, appuyez sur **entrée** pour exécuter du code.

Vous pouvez lancer Azure Cloud Shell avec :

|  |   |
|-----------------------------------------------|---|
| Sélectionnez **Essayer** dans le coin supérieur droit d’un bloc de code. Cela __ne__ copier automatiquement du texte dans Cloud Shell. | ![Exemple Essayer pour Azure Cloud Shell](./media/cloud-shell-try-it/cli-try-it.png) |
| Ouvrez [shell.azure.com](https://shell.azure.com) dans votre navigateur. | [![Bouton Lancer Azure Cloud Shell](./media/cloud-shell-try-it/launchcloudshell.png)](https://shell.azure.com) |
| Sélectionnez le bouton **Cloud Shell** du menu situé en haut à droite du [portail Azure](https://portal.azure.com). | ![Bouton Cloud Shell du portail Azure](./media/cloud-shell-try-it/cloud-shell-menu.png) |

Si vous ne souhaitez pas utiliser Azure Cloud Shell, installez Azure PowerShell localement à la place. Veillez à installer la dernière version du module Azure PowerShell Az pour obtenir des fonctionnalités à jour. Pour plus d’informations, consultez l’article [Installation et configuration d’Azure PowerShell](/powershell/azure/overview).
