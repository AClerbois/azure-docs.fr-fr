# <a name="azure-government-marketplace"></a>Place de marché Azure Government
Pour les partenaires souhaitant publier des offres sur la Place de marché Azure Government, lisez les détails ci-dessous.

## <a name="publishing"></a>Publication
> [!NOTE]
> Si vous n’êtes pas déjà un partenaire de Place de marché Azure certifié, suivez les étapes mentionnées [ici](../marketplace-publishing/marketplace-publishing-getting-started.md) avant de continuer.
> 
> 

### <a name="step-1"></a>Étape 1
Connectez-vous à [https://publish.windowsazure.com](https://publish.windowsazure.com)

### <a name="step-2"></a>Étape 2
Cliquez sur l’offre que vous voulez publier

### <a name="step-3"></a>Étape 3 :
Cliquez sur **SKU**, puis sur la zone Azure Government Cloud

> [!NOTE]
> Seules les SKU BYOL (apportez votre propre licence) sont prises en charge.  Cette option n’est pas disponible pour les SKU à paiement à l’utilisation (retenues à la source).
> 
> 

![alt text](./media/government-manage-marketplace-partner-1.png)

### <a name="step-4"></a>Étape 4
Cliquez sur le lien + Add Certification (Ajouter une certification) pour ajouter des liens vers d’éventuelles certifications de votre offre.

![alt text](./media/government-manage-marketplace-partner-2.png)

### <a name="step-5"></a>Étape 5
Demandez un compte d’évaluation dans le cloud Microsoft Azure Government pour vous tester votre image dans le portail de publication : [https://azuregov.microsoft.com/trial/azuregovtrial](https://azuregov.microsoft.com/trial/azuregovtrial)

Après une vérification de votre éligibilité en tant que partenaire servant des entités fédérales, d’État ou locales aux États-Unis, une confirmation est envoyée par e-mail.  Votre compte d’évaluation sera disponible dans les 3 à 5 jours ouvrés.

### <a name="step-6"></a>Étape 6
Cliquez sur Publish (Publier), puis sur Push to Staging (Déployer dans un environnement intermédiaire). 

![alt text](./media/government-manage-marketplace-partner-3.png)

Vous devrez entrer un abonnement figurant dans la liste approuvée qui a accès à l’offre intermédiaire. Entrez l’ID d’abonnement à partir de votre nouveau compte d’évaluation Azure Government.

![alt text](./media/government-manage-marketplace-partner-4.png)

### <a name="step-7"></a>Étape 7
Une fois l’offre transférée avec succès, vous pouvez tester votre image en vous connectant à [https://portal.azure.us](https://portal.azure.us) à l’aide de votre compte d’évaluation Azure Government.

### <a name="step-8"></a>Étape 8
Une fois que vous avez validé votre image à l’aide de l’abonnement d’évaluation, vous pouvez activer l’offre en cliquant sur Publish (Publier) et en demandant une approbation pour passer en mode de production. 

![alt text](./media/government-manage-marketplace-partner-5.png)

## <a name="next-steps"></a>Étapes suivantes
Pour obtenir des informations supplémentaires et des mises à jour, inscrivez-vous au [blog Microsoft Azure Government](https://blogs.msdn.microsoft.com/azuregov/).
