# Que faire en cas de modification du bureau d’enregistrement DNS pour votre nom de domaine personnalisé ?

Si vous utilisez uniquement le nom de domaine personnalisé avec Azure AD, vous pouvez continuer à utiliser votre nom de domaine personnalisé sans interruption et sans tâches de configuration supplémentaires dans Azure AD.

## Si vous utilisez d'autres services

Si vous utilisez votre nom de domaine personnalisé avec Office 365, Intune ou autres services s’appuyant sur des noms de domaine personnalisés, reportez-vous à la documentation de ces services.

-   [Utilisation de domaines personnalisés avec Office 365](https://support.office.com/article/Add-your-users-and-domain-to-Office-365-6383f56d-3d09-4dcb-9b41-b5f5a5efd611?ui=fr-FR&rs=fr-FR&ad=US)

-   [Utilisation de domaines personnalisés avec Intune](https://technet.microsoft.com/library/dn646966.aspx#BKMK_DomainNames)
