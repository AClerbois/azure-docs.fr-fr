# <a name="sign-up-for-content-moderator"></a>Inscription à Content Moderator #
Accédez à la page d’[inscription de Content Moderator](https://contentmoderator.cognitive.microsoft.com/Account/Signup). Vous avez la possibilité de vous connecter avec votre compte Microsoft ou de créer un compte sur le site web de Content Moderator.

![Inscription](images/0-signup-1.PNG)

## <a name="name-your-team"></a>Attribuer un nom à votre équipe ##
Saisissez un nom pour votre équipe de révision. Si vous le souhaitez, vous pouvez inviter vos collègues à rejoindre l’équipe.

![Créer une équipe](images/1-Team-Name.PNG)
