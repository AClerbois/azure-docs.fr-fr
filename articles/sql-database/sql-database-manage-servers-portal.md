# <a name="create-and-manage-azure-sql-database-servers-with-the-azure-portal"></a>Créer et gérer des serveurs Azure SQL Database avec le Portail Azure

Vous pouvez créer et gérer un serveur Azure SQL Database avec le [Portail Azure](https://portal.azure.com/), PowerShell, l’API REST ou C#. Cette rubrique porte sur l’utilisation du Portail Azure. Pour PowerShell, consultez la page [Créer et gérer des serveurs avec PowerShell](sql-database-manage-servers-powershell.md).

## <a name="create-an-azure-sql-database-server-using-the-azure-portal"></a>Création d’un serveur Azure SQL Database à l’aide du portail Azure

1. Ouvrez le panneau **Serveurs SQL** dans le [portail Azure](https://portal.azure.com/). 

    ![Serveurs SQL](./media/sql-database-get-started/new-sql-server.png)

2. Cliquez sur **Ajouter** pour créer un serveur SQL

    ![Ajouter un serveur SQL](./media/sql-database-get-started/new-sql-server-add.png)

> [!TIP]
> Pour un didacticiel de prise en main avec le portail Azure et SQL Server Management Studio, consultez [Prise en main des serveurs Azure SQL Database, des bases de données et des règles de pare-feu à l’aide du portail Azure et de SQL Server Management Studio](sql-database-get-started.md).
>

## <a name="view-and-update-azure-sql-server-settings-using-the-azure-portal"></a>Consulter et mettre à jour les paramètres d’un serveur Azure SQL à l’aide du Portail Azure
1. Ouvrez le panneau **Serveurs SQL** dans le [portail Azure](https://portal.azure.com/). 

    ![new sql server](./media/sql-database-get-started/new-sql-server.png)

2. Cliquez sur le serveur que vous souhaitez utiliser, puis cliquez sur le paramètre souhaité dans le panneau Serveurs SQL. 

    ![sql server blade](./media/sql-database-get-started/sql-server-blade.png)

> [!TIP]
> Vous trouverez un didacticiel de démarrage qui crée un serveur à l’aide du Portail Azure à la page [Créer une base de données - Portail Azure](sql-database-get-started.md).
>
