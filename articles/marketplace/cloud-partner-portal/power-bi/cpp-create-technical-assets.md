# <a name="create-power-bi-app-offer-technical-assets"></a>Créer des ressources techniques pour une offre d'application Power BI

La principale ressource technique requise pour ce type d'offre est une application Power BI. Une application Power BI est un ensemble de jeux de données principaux, de rapports ou de tableaux de bord, ainsi que de services connectés et de jeux de données incorporés facultatifs. Il s'agit d'une évolution d'un type de ressources précédemment connu sous le nom de [pack de contenu](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction). Pour plus d'informations sur le développement de ce type d'application, consultez [Que sont les applications de modèles Power BI ?](https://go.microsoft.com/fwlink/?linkid=2028636).


## <a name="obtain-an-installation-url"></a>Obtenir une URL d'installation

Vous ne pouvez créer une application Power BI que dans le cadre du [service Power BI](https://powerbi.microsoft.com/). Pour commencer, vous devez vous connecter avec une [licence Power BI Pro](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro). Après avoir créé et testé votre application de service dans Power BI, enregistrez l'URL d'installation de l'application générée par Power BI. Enfin, fournissez cette URL sous l'onglet [Infos techniques](./cpp-technical-info-tab.md) du formulaire de soumission du Portail Cloud Partner.
 

## <a name="next-steps"></a>Étapes suivantes

Après avoir créé et testé votre application Power BI, et créé l'offre associée, vous pouvez [publier l'offre d'application Power BI](./cpp-publish-offer.md).
