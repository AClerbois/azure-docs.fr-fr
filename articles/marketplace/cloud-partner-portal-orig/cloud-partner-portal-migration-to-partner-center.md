# <a name="cloud-partner-portal-migration-to-partner-center"></a>Migration du Portail des partenaires cloud vers l’Espace partenaires

Pour simplifier votre compte et offrir une expérience de gestion, Microsoft migre les offres de Portail Cloud Partner vers l’espace partenaires. Si vous gérez actuellement la documentation de co-vente dans le catalogue OCP GTM (One Commercial Partner Go-to-Market), vous pourrez bientôt mettre à jour vos offres de co-vente dans l’espace partenaires.

| **Type d’offre** | **Gestion des offres migrées de Portail Cloud Partner à l’espace partenaires** | **Possibilité de mettre à jour la documentation de la co-vente dans l’espace partenaires** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| SaaS (software as a service) |     Déjà disponible       |   Disponibilité générale autour de février 2020      |
| Dynamics 365 for Customer Engagement et PowerApps      |     Déjà disponible        |   Déjà disponible      |
| Dynamics 365 for Operations     |     Déjà disponible       |   Déjà disponible     |
| Dynamics 365 Business Central      |     Déjà disponible        |   Disponibilité générale autour de février 2020     |
| Azure Application     |     Disponibilité générale en décembre 2019       |   Disponibilité générale autour de février 2020     |
| Machine virtuelle Azure      |     Annoncé prochainement       |   Annoncé prochainement    |
| Conteneur Azure     |     Annoncé prochainement       |   Annoncé prochainement     |
| Service de conseil      |     Annoncé prochainement       |   Annoncé prochainement     |
| Module IoT Edge      |     Annoncé prochainement       |   Annoncé prochainement     |
| Application de service Power BI     |     Annoncé prochainement       |   Annoncé prochainement     |
|  |  |
