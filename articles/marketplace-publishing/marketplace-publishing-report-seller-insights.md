# Présentation des rapports Informations du vendeur

**En quoi consiste les informations du vendeur ?**

Tous les éditeurs de services de développement facturés à l’utilisation et de machines virtuelles reçoivent des rapports mensuels de Microsoft, si leurs offres génèrent de l’utilisation.

**Que vais-je recevoir ?**

- **E-mail de bienvenue :** en tant que nouvel éditeur, vous recevez un e-mail de bienvenue pour vous indiquer que vous allez commencer à recevoir les rapports mensuels contenant les informations du vendeur.

- **Rapports de ventes mensuels :** si vos offres génèrent de l’utilisation, vous recevez un deuxième e-mail avec en pièces jointes vos rapports et des instructions sur la façon d’accéder à votre mot de passe

    - Si vous proposez une référence de machine virtuelle ou de services de développement facturés à l’utilisation qui génèrent de l’utilisation, votre rapport mensuel contient les détails relatifs aux commandes, à l’utilisation, aux marchés et des informations sur les clients pour vos offres non-BYOL.
    - Ces fichiers sont verrouillés par un mot de passe que seuls vous et Microsoft connaissez afin de protéger les données de vos clients.
    - Nous n’envoyons PAS de rapport si aucune de vos offres ne génère d’utilisation au cours du mois.

## Comment lire votre rapport Informations du vendeur


**Commandes par référence et type de licence : onglet Commandes Marketplace**

![readingreportbyorders][2]

- Des segments vous aident à filtrer votre rapport par élément.
- Un graphique affiche les commandes mensuelles par type de licence Azure : chaque barre indique le nombre total de commandes pour le mois, en fonction du type de licence Azure.
- Un graphique affiche les commandes mensuelles par référence : chaque barre indique le nombre total de commandes par mois pour chaque référence.
- Un graphique affiche la tendance mensuelle des commandes par type de licence Azure et type de licence Marketplace.
- Des graphiques à secteurs présentent les commandes par type de licence Azure et type de licence Marketplace.
- Un tableau indique le nombre total de commandes mensuelles par type de licence Marketplace, total par mois et total cumulé pour tous les mois.


**Commandes par référence et type de licence : onglet Utilisation Marketplace**

![readingreportbyusage][3]

- Des segments vous aident à filtrer votre rapport par élément.
- Vous devez sélectionner l’utilisation de machines virtuelles normalisée ou l’utilisation brute.
- Un graphique affiche l’utilisation mensuelle par type de licence Azure : chaque barre indique l’utilisation totale pour le mois, en fonction du type de licence Azure.
- Un graphique affiche l’utilisation mensuelle par référence : chaque barre indique l’utilisation totale par mois pour chaque référence.
- Un graphique affiche la tendance mensuelle d’utilisation par type de licence Azure et type de licence Marketplace.
- Des graphiques à secteurs présentent l’utilisation par type de licence Azure et type de licence Marketplace.
- Un tableau indique l’utilisation totale mensuelle par type de licence Marketplace, total par mois et total cumulé pour tous les mois.

## Comment demander un rappel de mot de passe ?

**Accédez à https://publish.windowsazure.com/ et connectez-vous avec les informations d’identification de votre compte Microsoft** ![passwordreminder][7]

**Sélectionnez l’onglet Éditeurs** ![selectpublisherstab][8]


**Recherchez l’ID d’éditeur dans l’URL** : utilisez cet ID comme mot de passe pour ouvrir votre fichier Excel Informations du vendeur. Nous utiliserons ce mot de passe jusqu’à nouvel ordre. En outre, nous vous recommandons d’utiliser MS Office 2013 avec Windows pour lire les classeurs. Certains utilisateurs ont signalé des problèmes liés à l’utilisation de Microsoft Office pour Mac.

![publisherid][9]

Recherchez l’ID d’éditeur dans l’URL + utilisez cet ID comme mot de passe pour ouvrir votre fichier Excel Informations du vendeur. Nous utiliserons ce mot de passe jusqu’à nouvel ordre. En outre, nous vous recommandons d’utiliser MS Office 2013 avec Windows pour lire les classeurs. Certains utilisateurs ont signalé des problèmes liés à l’utilisation de Microsoft Office pour Mac.

## Étapes suivantes  
Si vous avez des questions sur les rapports et les informations, contactez notre équipe de support.

1. Accédez aux pages de support du site https://publish.windowsazure.com/
2. Sélectionnez « Rapports et informations »
3. Sélectionnez « Questions relatives aux rapports »
4. Cliquez sur « Envoyer une demande » ![sellerinsightsquestions][10]
