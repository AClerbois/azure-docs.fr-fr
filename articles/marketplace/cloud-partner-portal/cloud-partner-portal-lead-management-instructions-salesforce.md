# <a name="lead-management-instructions-for-salesforce"></a>Instructions de gestion des prospects pour Salesforce

Ce document fournit des instructions sur la configuration de votre système Salesforce afin que Microsoft puisse vous fournir des prospects. 

1. Accédez à Setup. <br/>
![Image d’installation de Salesforce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce1.png)

2. Dans la barre navigation, accédez à Build->Customize->Leads->Web-to-Lead. <br/>
![Image web to lead Salesforce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce2.png)

3. Cliquez sur le bouton « Create Web-to-Lead Form ». <br/>
![Image de création de formulaire Salesforce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce3.png)

4. Dans la page suivante, ne vous souciez pas du champ du formulaire et cliquez sur « Generate ». <br/>
![Image de génération Salesforce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce4.png)

5. Copiez l’OID et collez-le dans le champ <b>Identificateur d’objet</b> sur le portail Microsoft Cloud Partner. <br/>
![Image d’obtention d’OID Salesforce](./media/cloud-partner-portal-lead-management-instructions-salesforce/salesforce5.png)
