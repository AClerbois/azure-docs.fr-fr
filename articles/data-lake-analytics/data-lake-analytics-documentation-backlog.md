# Backlog de la documentation Azure Data Lake Analytics

Nous continuerons d’alimenter la nouvelle documentation pour accompagner Azure Data Lake Analytics au fur et à mesure de son évolution et pour attirer l'attention sur les options et les cas d’utilisation courants. Cette page répertorie la documentation que nous avons planifiée. Vous pouvez nous faire savoir sur quel sujet vous aimeriez que nous portions notre attention en ajoutant un commentaire à cet article, en envoyant une requête de tirage dans GitHub ou en nous envoyant un e-mail.
