# <a name="bing-search-sdk"></a>Kit de développement logiciel (SDK) pour Recherche Bing
Les exemples d’API Recherche Web Bing incluent les scénarios suivants :
1. Utilisez une requête d’un seul mot pour obtenir des résultats et afficher les nom et URL du premier résultat pour web, image, actualités et vidéo.
2. Lancez une requête sur une phrase, vérifiez le nombre de résultats, puis affichez le nom et l’URL du premier résultat.
3. Lancez une requête sur un terme de recherche avec des filtres de réponse définis sur `news`, puis affichez les détails des résultats d’actualités.
4. Lancez une requête sur un terme de recherche avec les paramètres `answerCount` et `promote`, puis afficher les détails des résultats.

Les Kits de développement logiciel (SDK) pour Recherche Bing rendent les fonctionnalités de recherche sur le web directement accessibles dans les langages de programmation suivants :
* Bien commencer avec les [exemples .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7) 
    * [Package NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.WebSearch/1.2.0)
    * Voir aussi les [bibliothèques .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingWebSearch) pour les définitions et les dépendances.
* Bien commencer avec les [exemples Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Voir aussi les [bibliothèques Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/webSearch) pour les définitions et les dépendances.
* Bien commencer avec les [exemples Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Voir aussi [bibliothèques Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples/tree/master/Search/BingWebSearch) pour les définitions et les dépendances.
* Bien commencer avec les [exemples Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Voir aussi les [bibliothèques Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-websearch) pour les définitions et les dépendances.

Les exemples du SDK pour chaque langage incluent un fichier Lisez-moi avec des détails sur les prérequis et l’installation/exécution des exemples.
