## <a name="prerequisites"></a>Prérequis

Pour suivre ce tutoriel, veillez à disposer des éléments suivants :

* Un ordinateur Windows sur lequel <a href="https://www.visualstudio.com/downloads/" target="_blank">Visual Studio 2017+</a> est installé avec la charge de travail **Développement web et ASP.NET**.
* [SDK .NET Core 2.2](https://dotnet.microsoft.com/download).
* Un ou plusieurs appareils (iOS ou Android) à déployer et exécuter.
  * Si vous utilisez Android :
    * <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3+</a> et <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3+</a> installés sur votre ordinateur Windows.
    * Un appareil Android <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">prêt pour le développement</a> et <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">compatible ARCore</a>.
  * Si vous utilisez iOS :
    * Un ordinateur macOS avec <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Xcode 10+</a>, <a href="https://cocoapods.org" target="_blank">CocoaPods</a> et <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3+</a> installés.
    * Un appareil iOS <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">compatible ARKit</a> prêt pour le développement.
