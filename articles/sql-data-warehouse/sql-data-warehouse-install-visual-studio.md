# Installez Visual Studio 2015

Pour le développement, l’équipe Azure SQL Data Warehouse recommande l’utilisation de Visual Studio 2013 ou plus tard, combiné aux outils de données SL Server (SSDT). Cette section explique comment télécharger et mettre à jour Visual Studio 2015 lorsqu’aucune version utilisable de ce logiciel n’est installée. Les étapes suivantes permettent de télécharger et installer les composants requis pour utilisation avec l’entrepôt Azure SQL Data Warehouse.

+ [Télécharger et installer Visual Studio 2015](#Download-and-Install-Visual-Studio-2015)
+ [Télécharger et installer la version préliminaire des outils SQL Server Data Tools (SSDT)] (#Download-and-Install-SQL-Server-Data-Tools-(SSDT)-Preview)
 
> [AZURE.NOTE]Version d’évaluation Visual Studio 2013 et SQL Server Data Tools (SSDT) peut également servir à l’entrepôt Azure SQL Data Warehouse.
 
##Télécharger et installer Visual Studio 2015

Pour installer Visual Studio 2015, suivez les étapes ci-dessous

1. [Télécharger Visual Studio 2015](https://www.visualstudio.com/downloads) à partir de Visual Studio en ligne.
2. Suivez le guide d’[installation de Visual Studio](https://msdn.microsoft.com/library/e2h7fzkw.aspx) depuis MSDN en choisissant les configurations par défaut 

##Téléchargez et installez la version préliminaire de SQL Server Data Tools (SSDT)

L’entrepôt de données SQL Azure utilise la dernière version d’évaluation de SQL Server Data Tools (SSDT) disponible dans le

1. [Télécharger la version préliminaire de SQL Server Data Tools](https://msdn.microsoft.com/mt429383.aspx) pour Visual Studio 2013 ou 2015.
2. Suivez les instructions d’installation pour installer les outils de données.
