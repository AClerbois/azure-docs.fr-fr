# <a name="create-power-bi-app-technical-assets"></a>Créer des ressources techniques pour une application Power BI

La principale ressource technique requise pour ce type d'offre est une application Power BI, qui regroupe un jeu de données principal, un rapport et/ou un tableau de bord, ainsi que des services connectés et des jeux de données incorporés facultatifs. (Les applications Power BI sont une évolution du type de ressource précédemment connu sous le nom de [packs de contenu](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction).)  Pour plus d'informations sur le développement de ces applications, consultez [Applications Power BI](https://go.microsoft.com/fwlink/?linkid=2028636).


## <a name="obtaining-an-installation-url"></a>Obtention d'une URL d'installation

Pour créer une application Power BI, vous devez utiliser le [service Power BI](https://powerbi.microsoft.com/) et vous connecter à l'aide d'une licence [Power BI Pro](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro). Une fois votre application de service créée et testée dans Power BI, enregistrez l'URL d'installation de l'application générée par Power BI.  Vous devez fournir cette URL sous l'[onglet Infos techniques](./cpp-technical-info-tab.md) du formulaire de soumission du Portail Cloud Partner.
 

## <a name="next-steps"></a>Étapes suivantes

Après avoir créé et testé votre application Power BI, et créé l'offre associée, vous pouvez [publier l'offre Power BI](./cpp-publish-offer.md).
