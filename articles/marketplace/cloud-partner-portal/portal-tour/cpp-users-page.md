# <a name="users-page"></a>Page Utilisateurs

La page [**Utilisateurs**](https://cloudpartner.azure.com/#users) liste tous les utilisateurs inscrits du portail Microsoft Cloud Partner.  Les colonnes affichées incluent l’adresse e-mail de chaque utilisateur, le rôle de portail, le type d’offre et l’organisation de publication. Vous pouvez classer les résultats par colonne ou les filtrer sur une chaîne que vous spécifiez.  

Cette page contient également un bouton **Ajouter un utilisateur** qui permet d’ajouter des utilisateurs à l’organisation de publication.  Pour plus d’informations, consultez [Gestion des utilisateurs sur le portail Microsoft Cloud Partner](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-users.md).
