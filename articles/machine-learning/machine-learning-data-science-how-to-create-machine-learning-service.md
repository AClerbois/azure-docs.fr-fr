# Processus de science des données Cloud pour l'apprentissage automatique

Ce guide du processus de science des données Cloud explique comment utiliser la plateforme Azure pour la science des données et créer des modèles opérationnels à l'aide d'Azure Machine Learning. Il décrit des tâches typiques de science des données, comme l'intégration de données issues de différentes sources, leur préparation en vue de leur exploitation dans Azure Machine Learning et l'utilisation d'Azure Machine Learning pour créer des modèles opérationnels exploitables par les applications des utilisateurs (par programmation ou autre). Si le diagramme détaille les principales étapes mises en œuvre dans un exercice type de science des données, certaines ne sont pas obligatoires et leur ordre peut varier selon l'emplacement, la taille et la complexité des données.

L'étape intitulée **Traitement avancé des données dans Azure** vous permet d'accéder à un second diagramme. Si vous revenez en arrière à partir de celui-ci, vous pouvez poursuivre la procédure en reprenant à cet endroit pour ajouter les données traitées dans Azure Machine Learning, puis pour créer et déployer des modèles.
