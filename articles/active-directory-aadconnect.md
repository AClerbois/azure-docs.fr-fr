<h1 id="vnettut1">Azure Active Directory Connect</h1>

L'assistant Azure Active Directory Connect constitue un outil et une expérience guidée uniques permettant de connecter vos annuaires locaux à Azure Active Directory.  L'assistant déploie et configure tous les composants requis pour démarrer l'intégration et le bon fonctionnement de votre annuaire. Cela inclut les services de synchronisation, la synchronisation des mots de passe ou les services AD FS (Active Directory Federation Services), ainsi que les conditions préalables tel que le module Azure AD PowerShell.

>[AZURE.NOTE] **Azure Active Directory Connect regroupe les fonctionnalités précédemment publiées comme DirSync et AAD Sync. Ces outils ne seront plus disponibles individuellement.** 

> **Azure Active Directory Connect est désormais votre " guichet unique " pour la synchronisation, l'authentification et tous les autres aspects de vos annuaires pour l'intégration vers Azure AD.**


Si vous évaluez ou utilisez l'outil de synchronisation Azure Active Directory (DirSync), les services de synchronisation Azure Active Directory (AAD Sync) ou Forefront Identity Manager 2010 R2, consultez la page [Outils d'intégration Directory](http://msdn.microsoft.com/library/azure/dn757582.aspx "Directory Integration Tools") pour davantage d'informations.


##  Version préliminaire publique d'Azure Active Directory Connect 

La version préliminaire publique actuelle d'Azure Active Directory Connect fournit une expérience guidée pour l'intégration d'une ou plusieurs forêts Windows Server Active Directory avec Azure AD. 

[Téléchargez la version préliminaire publique d'Azure AD Connect](http://connect.microsoft.com/site1164/program8612 "Azure Active Directory Connect") 

Avec cette version préliminaire d'Azure Active Directory Connect, les possibilités suivantes s'offrent à vous : 

- Choisir la configuration rapide pour configurer facilement et rapidement une forêt unique en 4 clics
- Choisir les paramètres personnalisés pour configurer plusieurs forêts ou sélectionner AD FS pour l'authentification unique (SSO)
- Configurer les options de synchronisation supplémentaires tel que le mode Exchange Hybrid, l'écriture différée du mot de passe et l'attribut d'ID alternatif.
