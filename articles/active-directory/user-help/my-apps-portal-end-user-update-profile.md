# <a name="update-your-profile-and-account-info-from-the-my-apps-portal"></a>Mettre à jour vos informations de profil et le compte à partir du portail mes applications
Vous pouvez utiliser votre compte professionnel ou scolaire avec le portail web **Mes applications** pour voir et démarrer la plupart des applications cloud de votre organisation, mettre à jour votre profil et vos informations de compte, consulter les informations de vos **groupes** et effectuer des **révisions d’accès** pour vos applications et vos groupes. Si vous n’avez pas accès au portail **Mes applications**, contactez le support technique pour obtenir l’autorisation.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-portal.md)]

>[!Important]
>Ce contenu est destiné aux utilisateurs. Si vous êtes administrateur, vous trouverez des informations sur la configuration et la gestion de vos applications cloud dans la [documentation sur la gestion des applications](https://docs.microsoft.com/azure/active-directory/manage-apps).

## <a name="view-your-organization-related-profile-information"></a>Afficher vos informations de profil liés à l’organisation
Selon ce que votre organisation a choisi d’inclure dans votre **profil** page de la **mes applications** portail, vous pouvez voir vos informations professionnelles spécifiques, ainsi que vos appareils et activité et les organisations supplémentaires auxquels vous appartenez.

### <a name="to-view-your-profile-information"></a>Pour afficher vos informations de profil
1.  Connectez-vous à votre compte professionnel ou scolaire, [accédez à la **mes applications** portal](my-apps-portal-end-user-access.md).

2. À partir de la **applications** page, sélectionnez votre photo de profil dans l’angle supérieur droit, où il indique que votre nom et votre organisation, puis **profil**.

    Le **profil** page s’affiche, présentant les informations de votre profil.

    ![Page de profil, affichant toutes les informations disponibles](media/my-apps-portal/my-apps-portal-profile-page.png)

3. À partir de la **profil** page, vous pouvez :

    - **Passez en revue vos détails liés à l’organisation.** Afficher votre photo nom, titre, les adresses de messagerie associée et les informations office. Ces informations sont gérées par votre organisation et vous ne pourrez pas le modifier. Si vous voyez une erreur, contactez votre support technique.

    - **Passez en revue vos appareils et une activité**. Assurez-vous que chaque appareil est familier et correctement connecté à votre organisation. Si vous ne reconnaissez pas un appareil, sélectionnez **désactivez le périphérique** pour supprimer l’association avec votre compte. Une fois que vous désactivez un appareil, il est supprimé de cette page.

    - **Passez en revue vos organisations.** Assurez-vous que vous travaillez toujours avec chacun des organisations spécifiées. Si vous ne fonctionnera plus avec une organisation, nous vous recommandons de sélectionner **connectez-vous quitter l’organisation**. Après avoir quitté l’organisation, il est supprimé de cette page.

## <a name="manage-your-work-or-school-account-information"></a>Gérer votre travail ou scolaire les informations de compte
Mettre à jour et gérer votre travail ou scolaire à partir des informations de compte le **profil** page de la **mes applications** portail. Cette page vous aide à :

- Modifier votre travail ou scolaire de mot de passe de compte.

- Allumez le mot de passe réinitialisé (si votre administrateur a activé cette fonctionnalité).

- Fournir des informations de vérification de sécurité supplémentaires.

- Passez en revue les conditions d’utilisation de votre organisation.

- Déconnectez-vous de partout.

## <a name="change-your-password"></a>Modifier votre mot de passe
Si vous souhaitez modifier le mot de passe pour votre compte professionnel ou scolaire, vous pouvez sélectionner **Change password** à partir de la **gérer compte** zone de la **profil** page.

### <a name="to-change-your-password"></a>Pour modifier votre mot de passe
1. À partir de la **gérer compte** zone de la **profil** page, sélectionnez **Change password**.

2. Sur le **Change password** page, vérifiez que votre ID d’utilisateur est correct, puis tapez votre ancien mot de passe et votre nouveau mot de passe dans les zones.

    ![Page de modification de mot de passe](media/my-apps-portal/my-apps-portal-change-password-page.png)

3. Sélectionnez **Envoyer**.

    Votre mot de passe est modifié. Vous serez requis pour la connexion à toutes les applications que vous avez précédemment connecté à l’aide de votre travail ou scolaire.

## <a name="set-up-and-use-password-reset"></a>Configurer et utiliser la réinitialisation du mot de passe
Si vous avez oublié votre mot de passe, jamais a reçu un support de votre entreprise ou avez été verrouillé pour votre compte, vous pouvez réinitialiser votre mot de passe.

>[!Important]
>Votre administrateur doit activer cette fonctionnalité et vous devez vous inscrire pour participer. Pour plus d’informations sur la procédure d’inscription et comment réinitialiser votre mot de passe, consultez [inscrire pour la réinitialisation du mot de passe libre-service](active-directory-passwords-reset-register.md) et [réinitialiser votre mot de passe Professionnel ou scolaire](user-help-reset-password.md).

## <a name="change-your-security-verification-information"></a>Modifier vos informations de vérification de sécurité
Si votre organisation vous impose d’utiliser vérification à deux facteurs, vous pouvez ajouter, mettre à jour et supprimer vos informations de sécurité associée à partir de la **vérification de sécurité supplémentaire** page.

Vérification à deux facteurs vous impose d’utiliser deux éléments d’informations sur la vérification, comme un mot de passe et un code confidentiel, avant de pouvoir obtenir pour votre compte ou pour les informations de votre organisation. Pour plus d’informations sur la vérification à deux facteurs, consultez [configurer mon compte pour la vérification en deux étapes](multi-factor-authentication-end-user-first-time.md).

### <a name="to-change-your-security-information"></a>Pour modifier vos informations de sécurité
1. À partir de la **gérer compte** zone de la **profil** page, sélectionnez **vérification de sécurité supplémentaire**.

    ![Page Vérification de sécurité supplémentaires](media/my-apps-portal/my-apps-portal-additional-verification-page.png)

2. Sur le **vérification de sécurité supplémentaire** page, vous pouvez ajouter, modifier ou supprimer les informations suivantes :

    - **Option de vérification par défaut.** Sélectionnez la méthode secondaire par défaut à utiliser pour la vérification à deux facteurs. Cette méthode est utilisée automatiquement chaque fois que vérification à deux facteurs est requise après avoir entré votre nom d’utilisateur et le mot de passe.

    - **Ajouter, mettre à jour ou supprimer des méthodes de vérification.** Vous pouvez ajouter de nouvelles, mettre à jour existant ou supprimer les anciennes informations qui ne s’applique plus.

    - **Configurer l’application Microsoft Authenticator.** Vous pouvez choisir de configurer l’application Microsoft Authenticator fonctionne comme méthode de vérification. Pour plus d’informations sur l’application Microsoft Authenticator, consultez [quel est l’application Microsoft Authenticator ?](user-help-auth-app-overview.md)

3. Cliquez sur **Enregistrer** pour enregistrer vos modifications.

## <a name="review-your-organizations-terms-of-use-statement"></a>Passez en revue les conditions de votre organisation d’instruction use
Vous pouvez choisir passer en revue les termes du contrat de votre organisation de l’instruction use, s’il est disponible.

1. À partir de la **gérer compte** zone de la **profil** page, sélectionnez **passez en revue les conditions d’utilisation**.

2. Passez en revue les termes du contrat de votre organisation d’utilisation et sélectionnez **Accept** pour confirmer que vous avez lu et compris les conditions d’utilisation pour votre organisation.

    ![Page conditions d’utilisation à partir du portail mes applications](media/my-apps-portal/my-apps-portal-tou-page.png)

    Si votre organisation ne dispose des conditions d’utilisation, vous pouvez sélectionner **fait** pour revenir à la **profil** page.

## <a name="sign-out-of-everywhere"></a>Déconnectez-vous de partout
Vous pouvez choisir pour vous déconnecter partout où que vous êtes actuellement connecté à l’aide de votre travail ou scolaire. Cela inclut toutes les applications et appareils.

### <a name="to-sign-out-of-everywhere"></a>Se déconnecter partout
1. À partir de la **gérer compte** zone de la **profil** page, sélectionnez **se déconnecter partout**.

2. Dans le **se déconnecter partout** boîte de confirmation, sélectionnez **Oui** pour confirmer que vous souhaitez vous déconnecter de tous vos sessions et les appareils. Sélectionnez **non** si vous changez d’avis.
