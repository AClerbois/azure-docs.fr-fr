## <a name="get-account-credentials"></a>Obtenir les informations d’identification du compte

Dans le cadre de cet exemple, vous devez fournir les informations d’identification de vos comptes Azure Batch et de stockage. Pour obtenir rapidement les informations d’identification nécessaires, dirigez-vous vers le portail Azure. (Vous pouvez aussi les obtenir avec les API Azure ou des outils en ligne de commande.)

1. Cliquez sur **Tous les services** > **Comptes Batch**, puis sur le nom de votre compte Batch.

2. Pour afficher les informations d’identification Batch, cliquez sur **Clés**. Copiez les valeurs des champs **Compte Batch**, **URL** et **Clé d’accès principale** dans un éditeur de texte.

3. Pour afficher le nom et les clés du compte de stockage, cliquez sur **Compte de stockage**. Copiez les valeurs des champs **Nom du compte de stockage** et **Clé1** dans un éditeur de texte.
