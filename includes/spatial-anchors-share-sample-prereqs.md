## <a name="prerequisites"></a>Prérequis

Pour suivre ce tutoriel, veillez à disposer des éléments suivants :

* Lisez la page [Vue d’ensemble d’Azure Spatial Anchors](../articles/spatial-anchors/overview.md).
* Effectuez l’un des [guides de démarrage rapide de 5 minutes](../articles/spatial-anchors/index.yml).
* Vous connaissez les bases de C# et Unity.
* Vous connaissez les bases d’<a href="https://developers.google.com/ar/discover/" target="_blank">ARCore</a> si vous souhaitez utiliser Android ou d’<a href="https://developer.apple.com/arkit/" target="_blank">ARKit</a> si vous souhaitez utiliser iOS.
* Un ordinateur Windows sur lequel <a href="https://www.visualstudio.com/downloads/" target="_blank">Visual Studio 2017</a> ou ultérieur est installé avec la charge de travail **Développement web et ASP.NET**.
* [SDK .NET Core 2.2](https://dotnet.microsoft.com/download).
* Un ou plusieurs appareils (iOS ou Android) sur lesquels déployer et exécuter une application.
  * Si vous utilisez Android, vous devez avoir :
    * <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3</a> ou ultérieur, <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3</a> ou ultérieur et <a href="https://git-scm.com/download/win" target="_blank">Git pour Windows</a> doivent être installés sur votre ordinateur Windows.
    * Un appareil Android <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">prêt pour le développement</a> et <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">compatible ARCore</a>.
  * Si vous utilisez iOS, vous devez avoir :
    * Un ordinateur macOS sur lequel <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Xcode 10</a> ou ultérieur, <a href="https://cocoapods.org" target="_blank">CocoaPods</a> et <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3</a> ou ultérieur doivent être installés.
    * Un appareil iOS <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">compatible ARKit</a> prêt pour le développement.
    * Git installé via Homebrew. Entrez la commande suivante sur une seule ligne dans le Terminal : `/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"` Exécutez ensuite `brew install git`.
