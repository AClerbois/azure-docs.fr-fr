1. [Créez un espace de travail Azure Machine Learning](../articles/machine-learning/service/setup-create-workspace.md#portal) si vous n’en avez pas déjà un.

1. Ouvrez votre espace de travail dans le [portail Azure](https://portal.azure.com/).  Si vous ne savez pas comment localiser votre espace de travail dans le portail, consultez [comment trouver votre espace de travail](../articles/machine-learning/service/how-to-manage-workspace.md#view).

1. Dans votre espace de travail, sélectionnez **interface visuelle**.  Puis sélectionnez **interface visuelle de lancement**.  
 
    ![Lancer l’interface visuelle](./media/aml-ui-prereq/launch-ui.png)

    La page Web d’interface s’ouvre dans une nouvelle page de navigateur.
