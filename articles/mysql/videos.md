[Vue d’ensemble : Azure Database pour PostgreSQL et MySQL](https://channel9.msdn.com/Events/Connect/2017/T147)

<iframe src="https://channel9.msdn.com/Events/Connect/2017/T147/player" width="800" height="450" allowFullScreen frameBorder="0"></iframe>

Azure Database pour PostgreSQL et Azure Database pour MySQL rassemblent les moteurs de base de données Community Edition et les fonctionnalités d’un service entièrement administré, afin que vous puissiez vous concentrer sur vos applications au lieu de gérer une base de données. Connectez-vous pour avoir une rapide vue d’ensemble des avantages de l’utilisation du service et pour voir certaines fonctionnalités en action.

[Présentation approfondie des fonctionnalités de service administré pour MySQL et PostgreSQL](https://channel9.msdn.com/Events/Connect/2017/T148)

<iframe src="https://channel9.msdn.com/Events/Connect/2017/T148/player" width="800" height="450" allowFullScreen frameBorder="0"></iframe>

Azure Database pour PostgreSQL et Azure Database pour MySQL rassemblent les moteurs de base de données Community Edition et les fonctionnalités d’un service entièrement géré. Connectez-vous pour obtenir une présentation approfondie du fonctionnement de ces services et sur la façon dont nous assurons une haute disponibilité et une mise à l’échelle rapide (en quelques secondes) afin que vous puissiez répondre aux besoins de vos clients. Vous en apprendrez également sur certains investissements sous-jacents relatifs à la sécurité et la disponibilité dans le monde entier.

[How to get started with the new Azure Database for MySQL service](https://channel9.msdn.com/events/Build/2017/B8045) (Prise en main du nouveau service Azure Database pour MySQL)

<iframe src="https://channel9.msdn.com/Events/Build/2017/B8045/player" width="800" height="450" allowFullScreen frameBorder="0"></iframe>

Dans cette vidéo tirée de la conférence Microsoft //Build de mai 2017, découvrez l’offre MySQL gérée de Microsoft dans Azure. La vidéo présente la stratégie de Microsoft prenant en charge les systèmes de base de données open source dans Azure. Elle explique ce que cela signifie pour vous, en tant que développeur, de développer ou de déployer des applications utilisant MySQL dans Azure. Cette vidéo offre une vue d’ensemble de l’architecture du service et montre l’intégration d’Azure Database pour MySQL dans d’autres services Azure, tels que Web Apps.
