Azure Data Share permet aux organisations de partager leurs données avec leurs clients et partenaires, de façon simple et sécurisée.

| **Ressource** | **Limite** |
| --- | --- |
| Nombre maximal de ressources Data Share par abonnement Azure |50 |
| Nombre maximal de partages envoyés par ressource Data Share |100 |
| Nombre maximal de partages reçus par ressource Data Share |100 |
| Nombre maximal d’invitations par partage envoyé |100 |
| Nombre maximal d’abonnements de partage par partage envoyé |100 |
| Nombre maximal de jeux de données par partage |100 |
| Nombre maximal de planification d’instantanés par partage |1 |
