# <a name="commands"></a>Commandes
|||
|---|---|
|az mysql server|Gérez les serveurs. Les serveurs contiennent des bases de données, des entrepôts de données et des pools élastiques.|
|az mysql server create|Crée un serveur MySQL.|
|az mysql server delete|Supprime un serveur MySQL.|
|az mysql server list|Obtient une liste de tous les serveurs MySQL dans un groupe de ressources.|
|az mysql server show|Affiche les détails d’un serveur MySQL.|
|az mysql server update|Met à jour un serveur.|
|az mysql server restore|Restaurez une version antérieure d’un serveur.|
|az mysql server firewall-rule|Gérez les règles de pare-feu d’un serveur.|
|az mysql server firewall-rule create|Crée une règle de pare-feu.|
|az mysql server firewall-rule delete|Supprime une règle de pare-feu.|
|az mysql server firewall-rule list|Répertorie les règles de pare-feu.|
|az mysql server firewall-rule show|Affiche les détails d’une règle de pare-feu.|
|az mysql server firewall-rule update|Met à jour une règle de pare-feu.|
|az mysql db|Gérez les bases de données.|
|az mysql db list|Répertoriez toutes les bases de données MySQL sur un serveur.|
|az mysql server-logs list|Répertoriez les fichiers journaux pour un serveur.|
|az mysql server-logs download|Téléchargez les journaux en local à partir du serveur.|
|az mysql server configuration list|Répertoriez les paramètres de configuration du serveur.|
|az mysql server configuration show|Affichez les détails des paramètres de configuration du serveur.|
|az mysql server configuration update|Modifiez la valeur de paramètre de configuration du serveur.|
|||
