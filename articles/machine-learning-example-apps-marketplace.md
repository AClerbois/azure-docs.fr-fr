# Exemples d'applications d'apprentissage automatique sur Azure Marketplace

Azure Machine Learning fournit aux scientifiques de données un nouveau canal pour la monétisation de leur savoir-faire et de leur créativité, comme pour les développeurs d'applications pour Smartphone. En publiant vos services d'apprentissage automatique sur Azure Marketplace, vous permettez à quiconque dans le monde entier d'acheter vos nouvelles API publiées sur le cloud et de s'y abonner afin de les utiliser dans leurs applications.

Mais Azure Marketplace peut offrir beaucoup plus que des API d'analyse de base. Par exemple, nous avons créé un ensemble d'applications de ML de bout en bout terminées, toutes à l'aide d'Azure ML, afin de répondre à des besoins professionnels spécifiques. Ces applications ML ne nécessitent pas de scientifique des données ou de savoir-faire en matière de ML : toute la science nécessaire est déjà intégrée dans notre solution. Les utilisateurs peuvent simplement importer leurs propres données et commencer à les utiliser. 

Citons par exemple :

* [Recommandations](http://datamarket.azure.com/dataset/amla/recommendations) : il s'agit d'un moteur de recommandations en libre-service mais très puissant qui vous permet d'ajouter facilement des recommandations de produits personnalisées sur votre site web ou dans vos campagnes par courrier électronique. 
  
* [Fréquemment achetés ensemble](https://datamarket.azure.com/dataset/amla/mba) : API qui permet à vos clients de découvrir des éléments du catalogue qui sont fréquemment achetés ensemble. Utilisez votre historique des achats client pour ajouter des recommandations " Fréquemment achetés ensemble " sur votre site web et pour améliorer les conversions dans votre magasin numérique.

* [Détection d'anomalie](https://datamarket.azure.com/dataset/aml_labs/anomalydetection) : API qui permet de détecter les événements anormaux dans les données de série chronologique (par exemple, télémétrie de serveur ou données de capteur).

Ces API vous permettent de consommer des services puissants d'apprentissage automatique basés sur Azure ML sans avoir à être un scientifique de données ou à passer beaucoup de temps à développer ces API vous-même ; il vous suffit d'importer vos propres données et de commencer à les utiliser.
