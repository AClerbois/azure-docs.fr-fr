### <a name="azure-active-directory-authentication"></a>Authentification Azure Active Directory

Ancres Spatial Azure permet également aux applications pour s’authentifier auprès des jetons d’utilisateur Azure AD (Active Directory). Par exemple, vous pouvez utiliser des jetons Azure AD à intégrer à Azure les ancres spatiale. Si une entreprise gère les utilisateurs dans Azure AD, vous pouvez fournir un jeton d’utilisateur Azure AD dans le SDK ancres spatiale de Azure. Cela vous permet de s’authentifier directement auprès du service ancres Spatial Azure d’un compte qui fait partie du même locataire Azure AD.
