# <a name="azure-synapse-analytics-data-management-partners"></a>Partenaires de gestion de données d’Azure Synapse Analytics
Pour créer votre solution d’entrepôt de données à l’aide de la fonctionnalité SQL Analytics d’Azure Synapse, vous disposez d’un vaste éventail d’outils de pointe. Cet article met en évidence les sociétés partenaires de Microsoft qui proposent des outils et solutions de gestion de données prenant en charge Azure Synapse Analytics.
