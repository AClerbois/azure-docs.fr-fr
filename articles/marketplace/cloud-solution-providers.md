# <a name="cloud-solution-providers"></a>Fournisseurs de solutions cloud

Offres de logiciels peuvent maintenant atteindre des millions de clients Microsoft qualifiés pris en charge par les partenaires de fournisseur de solutions Cloud (CSP) avec un minimum investissement de vente et marketing.  Les éditeurs peuvent désormais tirer parti de canaux de partenaire CSP, en plus de la disponibilité publique des offres via [vitrines web de Microsoft](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace).

Les éditeurs peuvent ne souhaitent participer une offre pour la disponibilité dans le canal de partenaire CSP, pour une offre de nouveau ou existant. Cette option permet de vendre vos produits et de créer des solutions regroupées pour les clients, les partenaires CSP.

Les éditeurs sont chargés de fournir le support de garantie de réparation pour les clients finaux et pour fournir un mécanisme pour les partenaires CSP et/ou les clients de vous contacter pour la prise en charge. Il s’agit des meilleures pratiques pour fournir les partenaires CSP avec la documentation de l’utilisateur, formation et les notifications d’intégrité/panne (le cas échéant) de service afin que les partenaires CSP sont équipées pour gérer les demandes de prise en charge 1 niveau de clients.

Le tableau suivant présente les types d’offre et les modèles de tarification qui peuvent participer au canal partenaire CSP :

| **Type d’offre**    | **BYOL**  |  **Gratuit** | **PAYG**   |
| :---------------- | :---------|:----------|:-----------|
| Machines virtuelles  | Oui | OUI | Oui |
| Containers | N/A | Oui | N/A |
| Modèles de solution | N/A | Oui | N/A |
| Applications gérées | N/A | Oui | Oui |
| Solution informatique SaaS (par site de tarification) | N/A | Oui | Oui |
|   |   |   |

Le canal de partenaire CSP participer paramètre se trouve dans l’onglet place de marché ou Storefront au sein de l’expérience de création de portail Microsoft Cloud Partner offre.
