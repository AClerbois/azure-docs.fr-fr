# Comment acheter ou souscrire un abonnement Azure

Cette rubrique va vous aider avec les tâches suivantes :

* S’inscrire à un abonnement d’essai gratuit Azure
* Mettre à niveau votre abonnement d’essai gratuit Azure vers un abonnement de paiement à l’utilisation
* Activer votre abonnement Azure à l’aide d’une clé d’activation des services en ligne (une clé 5 x 5)

## S’inscrire à un abonnement d’essai gratuit Azure
Pour vous inscrire à un abonnement d’essai gratuit Azure, vous devez fournir un numéro de téléphone, une carte de crédit et un [compte Microsoft](https://www.microsoft.com/fr-FR/account/faq.aspx) (anciennement, Windows Live ID) ou un compte scolaire/professionnel (également appelé ID d’organisation). Votre numéro de téléphone et votre carte de crédit sont uniquement utilisés pour vérifier votre identité ; votre carte de crédit ne sera pas débitée.

>[AZURE.NOTE]1 dollar USD (ou une somme équivalente dans la devise de votre région) fera l’objet d’une retenue d’autorisation sur votre carte de paiement afin de vérifier qu’il s’agit d’une carte valide. La somme sera débloquée dans un délai de 3 à 5 jours, en fonction des politiques de votre établissement financier. Votre carte ne sera pas débitée.

Pour vous inscrire à un abonnement d’essai gratuit Azure :

1. Ouvrez la [page d’essai gratuit Azure](https://azure.microsoft.com/fr-FR/pricing/free-trial/), puis sélectionnez **Essayez-le dès maintenant**.

2. Connectez-vous à votre compte Microsoft ou à un compte scolaire/professionnel.

3. Dans la section **À propos de vous**, entrez vos informations personnelles. Notez que le champ Pays/région ne peut pas être modifié une fois qu’il est spécifié. ![Capture d’écran des informations personnelles](./media/billing-buy-sign-up-azure-subscription/AboutYou.png)

4. La section Vérification par téléphone implique l’envoi d’un code sur votre téléphone à l’aide de votre méthode préférée : appel téléphonique ou SMS. Le fait d’entrer le code reçu nous permet de vérifier votre identité. </br></br> Dans la section **Vérification par téléphone**, sélectionnez l’indicatif téléphonique international de votre pays, puis entrez votre numéro de téléphone (n’entrez pas d’indicatif régional). Ensuite, sélectionnez **Envoyer un message texte** ou **M’appeler**. Après avoir reçu le code de vérification, entrez-le dans la zone **Vérifier le code**, puis sélectionnez le bouton **Vérifier le code** pour vérifier votre numéro de téléphone. Vous ne pouvez pas utiliser un numéro de téléphone VoIP pour la vérification par téléphone.</br> ![Capture d’écran de la vérification par téléphone](./media/billing-buy-sign-up-azure-subscription/PhoneVerify.png)</br>Si vous ne recevez pas de message texte ou d’appel, essayez d’utiliser un autre numéro de téléphone ou contactez le [support Microsoft](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade) pour obtenir de l’aide. </br>
5. Une fois la vérification par téléphone terminée, passez à la section Vérification de carte.

 ![cardverify](./media/billing-buy-sign-up-azure-subscription/VardVerify.png)</br>

6. Une fois la vérification de la carte de crédit terminée, sélectionnez l’option relative à l’**accord**, puis sélectionnez **S’inscrire**. ![Inscription](./media/billing-buy-sign-up-azure-subscription/Signup.png)</br> La configuration de votre abonnement peut prendre quelques minutes. Si la configuration de l’abonnement ne se termine pas dans un délai de 15 minutes contactez le [support Microsoft](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade) pour bénéficier d’une assistance. </br> Si vous obtenez un message d’erreur, suivez les étapes décrites dans le message pour résoudre le problème. Message d’erreur typique :

**Nous sommes désolés, mais nous n’avons pas pu effectuer l’opération. Veuillez réessayer ultérieurement ou recharger la page web.** </br>

Dans ce cas, essayez d’utiliser un autre navigateur pour terminer le processus d’inscription. Si le problème persiste, contactez le [support Microsoft](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade) pour obtenir de l’aide.

![signuperror1](./media/billing-buy-sign-up-azure-subscription/signuperror1.png)</br>7. Lorsque l’abonnement est prêt, sélectionnez **Commencer la gestion de mon service**. ![startservice](./media/billing-buy-sign-up-azure-subscription/startservice.png)

## Mettre à niveau votre abonnement d’essai gratuit Azure vers un abonnement de paiement à l’utilisation

1. Ouvrez une session sur le [portail des comptes Azure](https://account.windowsazure.com/subscriptions) à l’aide du compte Microsoft ou du compte scolaire/professionnel que vous avez utilisé pour vous inscrire pour l’essai gratuit.
2. Sélectionnez l’abonnement, puis **Mettre à niveau**.</br>![billpage](./media/billing-buy-sign-up-azure-subscription/billpage.png)
3. Sélectionnez **Oui**, puis **Mettre à niveau maintenant**. </br>![mettre à niveau](./media/billing-buy-sign-up-azure-subscription/Upgrade.png)

Après la mise à niveau vers l’abonnement de paiement à l’utilisation, vous pouvez continuer à utiliser les avantages de votre version d’essai (30 jours/200 $) jusqu’à épuisement de l’un des deux.

## Activer votre abonnement Azure à l’aide d’une clé d’activation des services en ligne (une clé 5 x 5)

Si vous avez acheté des crédits Azure dans Open auprès d’un revendeur, procédez comme suit pour activer votre abonnement :

1. Ouvrez la page [Clé de licence Azure dans Open](http://azure.microsoft.com/offers/ms-azr-0111p/), puis sélectionnez Activer un nouvel abonnement ou l’abonnement existant.
2. Entrez la clé de produit, puis cochez la première option, relative à l’accord, (ainsi que la seconde option si vous souhaitez recevoir des offres spéciales) pour activer l’abonnement.</br>![CLÉ OSA](./media/billing-buy-sign-up-azure-subscription/OSAkey.png)
3. Sélectionnez **Activer**.
