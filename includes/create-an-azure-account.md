# Créer un compte Azure et activer les fonctionnalités préliminaires

Pour déployer des applications sur Azure, vous avez besoin d'un compte. Cet article explique comment créer un compte d'évaluation gratuit et comment activer les fonctionnalités préliminaires d'Azure, notamment les services BizTalk, HDInsight et Azure Back pour le compte.

## Création d'un compte Azure

1.  Ouvrez un navigateur web et accédez à <a href="http://www.windowsazure.com">http://www.windowsazure.com</a>
2.  Si vous êtes titulaire d'un abonnement MSDN, accédez à la page <a href="/pricing/member-offers/msdn-benefits-details/" target="_blank">Avantage Azure pour les abonnés MSDN</a> et suivez la procédure de création d'un compte en activant vos avantages d'abonné MSDN.

   Sinon, accédez à la page <a href="/pricing/free-trial/" target="_blank">créer un compte d'évaluation gratuit</a> et suivez la procédure de création d'un compte d'évaluation. Vous devrez fournir un numéro de carte de crédit et un numéro de téléphone mobile pour prouver votre identité, mais rien ne vous sera facturé.

<h2><a id="enable"></a>Activer des fonctionnalités préliminaires d'Azure</h2>

1.  Accédez à [https://account.windowsazure.com/](https://account.windowsazure.com/) et connectez-vous avec votre compte Azure.
2.  Cliquez sur **fonctionnalités préliminaires** pour voir les fonctionnalités préliminaires disponibles.<br />
    ![open preview features tab][1]
3.  Repérez la fonctionnalité qui vous intéresse et cliquez sur **essayer maintenant**.<br />
    ![select a preview feature][2]
4.  Sélectionnez votre abonnement et cliquez sur la coche.<br />
    ![select subscription][3]
