# <a name="all-publishers-page"></a>Page Tous les éditeurs

La page [**Tous les éditeurs**](https://cloudpartner.azure.com/#publishers) liste tous les éditeurs cloud Microsoft inscrits, avec leur nom unique d’éditeur et leur nom d’affichage. Vous pouvez classer les résultats par colonne ou les filtrer sur une chaîne que vous spécifiez.  Cette page contient également un bouton **Ajouter un éditeur**, qui est activé uniquement pour les administrateurs du portail.  Vous pouvez sélectionner n’importe quel éditeur dans cette liste pour afficher le profil correspondant.  Pour plus d’informations, consultez [Gestion du profil d’éditeur](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-publisher-profile.md).


## <a name="example-page"></a>Exemple de page

L’illustration suivante montre une liste d’éditeurs filtrée sur la chaîne « contoso ».  

![Page Tous les éditeurs](./media/all-publishers-page1.png)

Si vous cliquez sur la première entrée, la page du profil de Contoso s’affiche.  Vous pouvez modifier uniquement votre propre profil d’éditeur.

![Page du profil de l’éditeur Contoso](./media/all-publishers-page2.png)
