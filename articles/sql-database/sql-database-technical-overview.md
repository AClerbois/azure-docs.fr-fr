# Vue d’ensemble de SQL Database

SQL Database est un service de base de données relationnelle dans le cloud, qui repose sur le moteur Microsoft SQL Server et propose des fonctionnalités stratégiques intégrées. SQL Database offre des fonctions prévisibles de gestion des performances, de l’évolutivité, de la continuité d’activité et de la protection des données, ainsi qu’une administration proche de zéro aux développeurs du cloud et aux architectes de solutions. Mieux encore, SQL Database prend en charge les outils, bibliothèques et API SQL Server existants, ce qui facilite votre transition vers le cloud. Votre base de données SQL peut être [opérationnelle en quelques secondes](sql-database-get-started.md), sans que vous soyez contraint de gérer une infrastructure ou des machines virtuelles. Les fonctionnalités de programmation de type administrateur de bases de données permettent des opérations de développement efficaces, alors que la fonction d’autogestion assure une maintenance proche de zéro pour les bases de données. Ainsi, vous pouvez vous concentrer sur le développement et la gestion de votre application.

SQL Database vous permet d’[augmenter la taille](sql-database-service-tiers.md) des données sans temps mort et avec des performances prévisibles qui peuvent être accrues ou réduites en fonction des besoins, tandis que vous pouvez choisir des [bases de données élastiques](sql-database-elastic-pool.md) pour passer à des milliers de bases de données dans le cadre d’un budget que vous contrôlez, en exploitant les puissantes fonctionnalités de gestion des [tâches de base de données élastique](sql-database-elastic-jobs-overview.md) et les fonctionnalités de programmation des [outils de base de données élastique](sql-database-elastic-scale-get-started.md).

Grâce à un contrat [SLA](http://azure.microsoft.com/support/legal/sla/) proposant une disponibilité de 99,99 %, leader sur le marché, vos applications bénéficient d’une probabilité de coupure de service proche de zéro. La fonction de protection des données intégrée, qui repose sur l’audit, la restauration et la géo-réplication, garantit que vos données sont protégées. SQL Database vous permet de restaurer vos données à n’importe quel moment, à partir de n’importe quelle transaction, sur une période de 35 jours maximum. Vous pouvez également répliquer vos données dans la région Microsoft Azure de votre choix et implémenter aisément une stratégie de récupération d’urgence géographique, conçue pour répondre aux besoins de votre entreprise. La [dernière version de SQL Database](sql-database-preview-whats-new.md), la version 12, offre une compatibilité presque totale avec le moteur Microsoft SQL Server.

Il existe différents [niveaux de service](sql-database-service-tiers.md) pour SQL Database : le niveau de base, le niveau Standard et le niveau Premium. Ces niveaux prennent en charge des charges de travail de base de données légères ou lourdes, selon le cas. Vous pouvez ainsi passer d’un niveau à l’autre ou les associer les uns aux autres, afin de bénéficier d’une conception d’applications innovante. Grâce à la puissance et à la portée de Microsoft Azure, vous pouvez mélanger les différents services Microsoft Azure avec SQL Database, afin de pouvoir répondre aux besoins professionnels uniques de vos nouvelles applications, réduire les coûts, optimiser l’efficacité et mettre en lumière de nouvelles opportunités professionnelles.

Démarrez avec une [version d’évaluation gratuite](http://azure.microsoft.com/pricing/free-trial/), puis [créez votre première base de données SQL](sql-database-get-started.md) en quelques minutes.
