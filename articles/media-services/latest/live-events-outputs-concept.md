# <a name="live-events-and-live-outputs"></a>Événements en direct et Sorties en direct

Azure Media Services vous permet de transmettre des événements en direct auprès de vos clients dans le cloud Azure. Pour configurer vos événements de streaming en direct dans Media Services v3, vous devez en savoir plus sur les [Événements en direct](https://docs.microsoft.com/rest/api/media/liveevents) et les [Sorties en direct](https://docs.microsoft.com/rest/api/media/liveoutputs).
