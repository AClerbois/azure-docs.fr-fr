# <a name="reset-the-password-for-a-user-in-azure-active-directory"></a>Réinitialiser le mot de passe d’un utilisateur dans Azure Active Directory

## <a name="how-to-reset-the-password-for-a-user"></a>Comment réinitialiser le du mot de passe pour un utilisateur
1. Connectez-vous au [centre d’administration d’Azure AD](https://aad.portal.azure.com) avec un compte d’administrateur général pour l’annuaire.
2. Sélectionnez **Utilisateurs et groupes**.

   ![Ouvrir la gestion des utilisateurs](./media/active-directory-users-reset-password-azure-portal/create-users-user-management.png)
3. Sélectionnez **Tous les utilisateurs**.

   ![Ouverture du groupe Tous les utilisateurs](./media/active-directory-users-reset-password-azure-portal/create-users-open-users-blade.png)
1. Sélectionnez un utilisateur dans la liste.
2. Pour l’utilisateur sélectionné, sélectionnez **Vue d’ensemble**, puis, dans la barre de commandes, sélectionnez **Réinitialiser le mot de passe**.

    ![Sélection de la commande Réinitialiser le mot de passe](./media/active-directory-users-reset-password-azure-portal/create-users-reset-password-command.png)
6. Dans **Réinitialiser le mot de passe**, sélectionnez **Réinitialiser le mot de passe**.
