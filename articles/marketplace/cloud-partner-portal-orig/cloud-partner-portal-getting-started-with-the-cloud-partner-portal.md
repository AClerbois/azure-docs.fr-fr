# <a name="getting-started-with-the-cloud-partner-portal"></a>Prise en main du portail Microsoft Cloud Partner

Cet article propose une procédure détaillée d’utilisation du portail Microsoft Cloud Partner. Découvrez ce que vous pouvez faire à l’aide du portail Cloud Partner pour gérer les offres sur la [Place de marché Azure](https://azuremarketplace.microsoft.com/) et sur [AppSource](https://appsource.microsoft.com/).

<a name="portal-tour"></a>Visite guidée de portail
-----------

Le portail Microsoft Cloud Partner contient tout ce dont vous avez besoin pour proposer et exploiter une entreprise prospère dans le cloud. Voici un aperçu rapide des éléments de ce site que vous utiliserez probablement le plus souvent.

### <a name="left-navigation-bar"></a>Barre de navigation gauche

Lorsque vous accédez pour la première fois au portail, vous devez voir la barre de navigation amovible. Il s’agit de l’emplacement à partir lequel vous pouvez naviguer parmi les options de menu.

![Portail des partenaires cloud](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/cloud-partner-portal-page.png)

Utilisez la barre de menus pour obtenir plus d’informations sur chacun des éléments suivants :

- **Nouvelle offre** : lancez une nouvelle offre ici.
- **Toutes les offres** : vérifiez l’état et le statut de toutes vos offres en cours.
- **Utilisateurs** - [gérez et contrôlez l’accès pour votre société](./cloud-partner-portal-manage-users.md).
- **Apprendre** : démarrez votre apprentissage et tenez-vous informé des nouveautés sur le portail Microsoft Cloud Partner.
- **Informations** : recueillez des informations sur l’utilisation de vos produits et sur vos clients en visitant le portail Insights.

### <a name="offer-page"></a>Page de l’offre

Dans la suite de la visite guidée, lorsque vous démarrez une nouvelle offre ou revenez en arrière pour la modifier, vous effectuez la plupart des tâches dans la page de l’offre. C’est là que vous configurez tous les paramètres de l’offre et vérifiez son état.

![Page de l’offre](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/offer-page.png)

Sur la page Offre, vous pouvez :
- Ajouter du contenu ou vérifiez le statut de votre offre en basculant entre les deux modes de niveau supérieur : **Éditeur** et **état**.
- Utiliser la **Barre d’action** pour gérer explicitement les modifications de votre offre en enregistrant et en ignorant votre entrée.
- Naviguer entre les **Formulaires** obligatoires pour chaque offre tandis que vous modifiez les paramètres de votre offre.

### <a name="menu-bar"></a>Barre de menus

Enfin, en haut de votre écran, vous pouvez toujours accéder à la barre de menus supérieure. Cette barre contient des actions interactives pour le support et les commentaires. Vous pouvez également l’utiliser pour accéder à toutes les informations de votre compte d’éditeur.

![Barre de menus](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/menu-bar.png)

-   **Vous avez besoin d’aide ?** - Si vous vous sentez désorienté et avez besoin d’aide, créez une demande de support et recherchez des liens vers la documentation.
-   **Commentaires** : vous avez quelque chose à nous dire ? Faites-nous en part ici. Microsoft souhaite rendre la Place de marché Microsoft Azure, le processus de publication sur AppSource et l’expérience globale du portail Microsoft Cloud Partner aussi faciles et intuitifs que possible. Nous trions et passons en revue tous les commentaires dans le respect constant de votre vie privée. Nous vous contactons uniquement lorsque vous avez besoin d’informations supplémentaires.
- **Compte d’éditeur** : lorsque vous visitez le portail pour la première fois, vous voyez s’afficher le contrat d’éditeur par lequel vous pouvez accepter de devenir un éditeur. À partir de là, vous pouvez apporter des modifications à votre profil d’éditeur ou revoir le contrat. Il s’agit également de l’emplacement à partir duquel vous vous connectez à votre compte du Centre de développement afin de pouvoir vendre vos offres.
