# <a name="audit-history-page"></a>Page Historique d’audit

La page [**Historique d’audit**](https://cloudpartner.azure.com/#history) liste les événements de publication pour toutes les offres de la Place de marché.  Les colonnes affichées incluent le type d’événement, la date et l’heure de l’occurrence, le nom de l’offre, l’éditeur et l’utilisateur à l’origine de l’action.  Vous pouvez classer les résultats par colonne ou les filtrer sur une chaîne que vous spécifiez.  À partir de cette page, l’utilisateur peut également télécharger la liste des audits en cours, dans un fichier de valeurs séparées par des virgules (.csv).
