> [!IMPORTANT]
>
> Pour utiliser cette fonctionnalité Azure à partir de PowerShell, vous devez installer le module `AzureRM`. Il s’agit d’un module antérieur disponible uniquement pour Windows PowerShell 5.x qui reçoit des mises à jour critiques, mais n’obtient plus de nouvelles fonctionnalités. Si le module `Az` est installé, vous devez également installer `AzureRM`. Pour installer `AzureRM` :
> 
> 1. Désactivez les alias AzureRM dans `Az` (si activés) avec `Disable-AzAlias`. L’exécution de cette commande sans que `Az` soit installé ou avec des alias désactivés ne présente aucun danger.
> 2. Suivez les [dernières instructions d’installation pour AzureRM](/powershell/azure/install-az-ps). Même si vous avez installé `AzureRM`, effectuez un mise à jour vers la dernière version.
