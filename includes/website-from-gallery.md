Azure Marketplace met à votre disposition une large gamme d'applications web populaires, développées par Microsoft, par des sociétés tierces ou par des initiatives de logiciel open source. Les applications web créées à partir d'Azure Marketplace ne nécessitent pas l'installation de logiciel autre que le navigateur utilisé pour la connexion au [portail Azure](http://go.microsoft.com/fwlink/?LinkId=529715). 

Ce didacticiel vous apprendra à effectuer les opérations suivantes :

- création d'une application web via Azure Marketplace ;

- déploiement de l'application web via le portail Azure.
 
Vous allez configurer un blog WordPress avec un modèle par défaut. L'illustration suivante présente l'application terminée :


![Blog Wordpress][13]

>[AZURE.NOTE] Si vous souhaitez prendre en main Azure App Service avant de créer un compte Azure, accédez à [Essayer App Service](http://go.microsoft.com/fwlink/?LinkId=523751), où vous pouvez immédiatement créer une application web de démarrage de courte durée dans App Service. Aucune carte de crédit n'est requise ; vous ne prenez aucun engagement.

## Créer une application web dans le portail

1. Connectez-vous au portail Azure.

2. Pour ouvrir Azure Marketplace, cliquez sur l'icône **Marketplace** ou sur l'icône **Nouveau** en bas à gauche du tableau de bord, sélectionnez **Web + mobile**, puis **Azure Marketplace** en bas.
	
	<!--todo:![Create New][5]-->
	
3. Dans Marketplace, sélectionnez **Web Apps**.

4. Cliquez sur l'icône **Scalable WordPress**.

	<!--todo:![WordPress from list][7]-->
	
5. Après avoir lu la description de l'application WordPress, sélectionnez **Créer**.

6. Cliquez sur chaque partie (**APPLICATION WEB**, **BASE DE DONNÉES** et **ABONNEMENT**), puis entrez ou sélectionnez des valeurs pour les champs obligatoires :
	
- Entrez le nom d'URL de votre choix.	
- Sélectionnez la région la plus proche de vous.

	<!--todo:![configure your app][8]-->

7. Lorsque vous avez fini de définir l'application web, cliquez sur **Créer**.

## Lancer et gérer l'application web WordPress
	
1. Dans le panneau **Web Apps**, cliquez sur la nouvelle application web pour ouvrir le tableau de bord.

	<!--todo:![launch dashboard][10]-->

2. Dans la page **Essentials**, cliquez sur le lien situé sous **URL** pour ouvrir la page d'accueil de l'application web.

	<!--todo:![site URL][11]--> 

3. Si vous n'avez pas installé WordPress, entrez les informations de configuration appropriées requises par WordPress, puis cliquez sur **Installer WordPress** pour finaliser la configuration et ouvrir la page de connexion de l'application web.

	<!--todo:![login to WordPress][12]-->

4. Cliquez sur **Connexion** et entrez vos informations d'identification.  
	<!--todo:screenshot-->

5. Vous disposez d'une nouvelle application web WordPress, qui ressemble à l'application web indiquée ci-dessous.    

	<!--todo:![your WordPress site][13]-->
