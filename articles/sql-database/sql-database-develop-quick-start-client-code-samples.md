# Développement de clients et échantillons de code de démarrage rapide pour une base de données SQL


Cette rubrique fournit des liens vers des échantillons de code de démarrage rapide que vous pouvez utiliser pour vous connecter à la base de données SQL Azure. Les exemples regroupent :


- Une variété de langages de programmation.
- Windows et Linux comme système d'exploitation de votre programme client.
- Échantillons courts de code de démarrage rapide, ainsi que des échantillons plus longs qui gèrent les erreurs temporaires avec logique de nouvelle tentative automatique.
- Échantillons de code qui convertissent des jeux de résultats relationnels en format orienté objet.


> [AZURE.NOTE]Depuis le 19 mai 2015, des échantillons de code pour davantage de langages sont en cours de préparation et des liens vers ceux-ci seront ajoutés à cette rubrique.
