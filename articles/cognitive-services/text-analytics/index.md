    <h1>Documentation sur les API Analyse de texte</h1>
    <div class="intro" style="min-width: 200px">
        <p>L’API Analyse de texte est un service cloud qui fournit un traitement en langage naturel avancé sur le texte brut et inclut trois fonctions principales : analyse des sentiments, extraction d’expressions clés et détection de la langue. Découvrez comment analyser le contenu de différentes façons avec nos démarrages rapides, didacticiels et exemples.</p>
    </div>
