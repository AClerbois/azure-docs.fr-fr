> [!IMPORTANT]
> Cet article comporte un contenu qui nécessite Azure Classic CLI. Il s’agit d’une version antérieure de l’interface Azure CLI à utiliser uniquement avec le modèle de déploiement Azure Classic.
>
> Vous pouvez installer l’interface Azure Classic CLI côte à côte avec l’interface Azure CLI de nouvelle génération. Toutefois, nous vous recommandons d’utiliser l’interface Azure CLI pour tous les nouveaux scripts et déploiements.
> Pour installer l’interface Azure Classic CLI, consultez [Installer Azure Classic CLI](/cli/azure/install-classic-cli). Pour installer l’interface CLI actuelle, consultez [Installer Azure CLI](/cli/azure/install-azure-cli).
