# Module AzureRM.FrontDoor
Les rubriques de cette section documentent les applets de commande Azure PowerShell pour Azure Front Door Service qui utilisent Azure Resource Manager. Les applets de commande existent dans l’espace de noms Microsoft.Azure.Commands.FrontDoor.

## Applets de commande AzureRM.FrontDoor
### [Get-AzureRmFrontDoor](Get-AzureRmFrontDoor.md)
Obtenir l’équilibreur de charge Front Door

### [Get-AzureRmFrontDoorFireWallPolicy](Get-AzureRmFrontDoorFireWallPolicy.md)
Obtenir la stratégie de pare-feu d’applications web (WAF)

### [New-AzureRmFrontDoor](New-AzureRmFrontDoor.md)
Créer un équilibreur de charge Azure Front Door

### [New-AzureRmFrontDoorBackendObject](New-AzureRmFrontDoorBackendObject.md)
Créer un objet PSBackend

### [New-AzureRmFrontDoorBackendPoolObject](New-AzureRmFrontDoorBackendPoolObject.md)
Créer un objet PSBackendPool pour la création d’une porte d’entrée

### [New-AzureRmFrontDoorCustomRuleObject](New-AzureRmFrontDoorCustomRuleObject.md)
Créer un objet CustomRule pour la création d’une stratégie de pare-feu d’applications web (WAF)

### [New-AzureRmFrontDoorFireWallPolicy](New-AzureRmFrontDoorFireWallPolicy.md)
Créer une stratégie de pare-feu d’applications web (WAF)

### [New-AzureRmFrontDoorFrontendEndpointObject](New-AzureRmFrontDoorFrontendEndpointObject.md)
Créer un objet PSFrontendEndpoint pour la création d’une porte d’entrée

### [New-AzureRmFrontDoorHealthProbeSettingObject](New-AzureRmFrontDoorHealthProbeSettingObject.md)
Créer un objet PSHealthProbeSetting pour la création d’une porte d’entrée

### [New-AzureRmFrontDoorLoadBalancingSettingObject](New-AzureRmFrontDoorLoadBalancingSettingObject.md)
Créer un objet PSLoadBalancingSetting pour la création d’une porte d’entrée

### [New-AzureRmFrontDoorManagedRuleObject](New-AzureRmFrontDoorManagedRuleObject.md)
Créer un objet ManagedRule pour la création d’une stratégie de pare-feu d’applications web (WAF)

### [New-AzureRmFrontDoorMatchConditionObject](New-AzureRmFrontDoorMatchConditionObject.md)
Créer un objet MatchCondition pour la création d’une stratégie de pare-feu d’applications web (WAF)

### [New-AzureRmFrontDoorRoutingRuleObject](New-AzureRmFrontDoorRoutingRuleObject.md)
Créer un objet PSRoutingRuleObject pour la création d’une porte d’entrée

### [New-AzureRmFrontDoorRuleGroupOverrideObject](New-AzureRmFrontDoorRuleGroupOverrideObject.md)
Création d’un objet RuleGroupOverride pour la création d’une stratégie de pare-feu d’applications web (WAF)

### [Remove-AzureRmFrontDoor](Remove-AzureRmFrontDoor.md)
Supprimer un équilibreur de charge Front Door

### [Remove-AzureRmFrontDoorFireWallPolicy](Remove-AzureRmFrontDoorFireWallPolicy.md)
Supprimer la stratégie de pare-feu d’applications web (WAF)

### [Set-AzureRmFrontDoor](Set-AzureRmFrontDoor.md)
Mettre à jour un équilibreur de charge Front Door

### [Set-AzureRmFrontDoorFireWallPolicy](Set-AzureRmFrontDoorFireWallPolicy.md)
Mettre à jour une stratégie de pare-feu d’applications web (WAF)
