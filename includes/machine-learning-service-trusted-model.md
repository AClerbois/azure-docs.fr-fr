> [!IMPORTANT]
> Vous devez uniquement utiliser les modèles que vous créez ou que vous obtenez à partir d’une source approuvée. Modèles sérialisés doivent être traités en tant que code, comme les failles de sécurité ont été détectés dans plusieurs formats courants. En outre, les modèles peuvent être formés intentionnellement ayant des intentions malveillantes pour fournir la sortie biaisée ou inexacte.
