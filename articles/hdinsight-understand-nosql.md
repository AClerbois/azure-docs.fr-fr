# Gestion des données : compréhension des technologies NoSQL sur Azure

**Résumé :** Azure propose un vaste choix d'options pour vos charges de travail NoSQL. Cependant, il peut être difficile de distinguer les différences entre toutes les options disponibles. Ce document décrit les technologies NoSQL sur Azure, ainsi que les différences au niveau de la gestion par rapport aux modèles relationnels de gestion des données.

La technologie relationnelle a été pendant longtemps l'approche prédominante pour l'exploitation des données. Cependant, lorsque vous utilisez de très grandes quantités de données de types différents, comme c'est le cas aujourd'hui, vous êtes confronté à de nouveaux défis au niveau de leur gestion. La technologie relationnelle est, dans certains cas, la solution idéale pour gérer ces données. Ce guide passe en revue les différentes options, en décrivant le rôle de chacune et en expliquant dans quelles circonstances vous pouvez les utiliser.
