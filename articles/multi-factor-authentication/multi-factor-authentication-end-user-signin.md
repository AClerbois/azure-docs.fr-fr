# Expérience de connexion avec Azure Multi-Factor Authentication
Les informations suivantes sont fournies pour vous guider dans le processus de connexion.


## Quelle sera votre expérience de connexion ?
Selon la méthode de connexion et l'authentification multifacteur utilisées, le processus est différent. Dans cette section, nous fournissons des informations sur le processus de connexion. Choisissez ce qui décrit le mieux votre méthode de connexion :


Que faites-vous ?|Description
:------------- | :------------- | 
[Connexion avec un téléphone mobile ou de bureau](multi-factor-authentication-end-user-signin-phone.md) | Voici le processus de connexion habituel lors d’une connexion à l'aide de votre téléphone mobile ou de bureau.
[Connexion avec l'application mobile à l'aide de la notification](multi-factor-authentication-end-user-signin-app-notify.md) | Voici le processus de connexion habituel lors d’une connexion à l'aide de l’application mobile avec des notifications.
[Connexion avec l'application mobile à l'aide du code de vérification](multi-factor-authentication-end-user-signin-app-verify.md)|Voici le processus de connexion habituel lors d’une connexion à l'aide de l’application mobile avec un code de vérification.
[Connexion avec une autre méthode](multi-factor-authentication-end-user-signin-alt.md)|Ceci vous présente le processus de connexion habituel si vous souhaitez utiliser une autre méthode.
