# <a name="azure-spatial-anchors-support-options"></a>Options de support Azure ancres Spatial

## <a name="open-a-tech-support-ticket"></a>Ouvrez un ticket de support technique

Pour ouvrir un ticket de support technique au sein du portail Azure pour les ancres Spatial Azure :

1. Avec le [Azure portal](https://azure.microsoft.com/account/) ouvrir, sélectionnez l’icône d’aide dans la barre de menus supérieure, puis sélectionnez le **aide + support** bouton. 

![Prise en charge et l’aide du portail azure](./media/spatial-anchor-support.png)

1. À l’aide + support page ouverte, sélectionnez **+ nouveau demande de support**.

![Azure portail nouvelle demande de support](./media/spatial-anchor-support2.png)

1. Lorsque vous terminez les champs de ticket de support : 

- Type de problème : Techniques
- Service : Ancres spatiales

![Champs de ticket de support concernant le portail Azure](./media/spatial-anchor-support3.png)

## <a name="community-support"></a>Support de la communauté pour les objets blob

Pour la prise en charge à partir de la Communauté des utilisateurs, consultez [ancres spatiale d’Azure sur Stack Overflow](https://stackoverflow.com/questions/tagged/azure-spatial-anchors). 

## <a name="provide-content-feedback"></a>Fournir des commentaires sur les ressources

Au bas de chaque article de contenu, il existe une opportunité pour ouvrir un problème GitHub et de fournir des commentaires sur le contenu de la documentation d’ancrage Spatial Azure. 

## <a name="provide-product-feedback"></a>Indiquer des commentaires sur le produit

Pour fournir des commentaires, voter pour des idées que d’autres personnes ont soumis ou partager une idée ou une suggestion pour le service ancres Spatial Azure, visitez le [Forum de commentaires Azure Spatial ancres](https://feedback.azure.com/forums/919252-azure-spatial-anchors).
