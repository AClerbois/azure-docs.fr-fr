# <a name="azure-saas-application-offer"></a>Offre d’application Azure SaaS

<table> <tr> <td>Cette section explique comment publier une application software-as-a-service (SaaS) sur la <a href="https://azuremarketplace.microsoft.com/marketplace/">Place de marché Azure</a> Microsoft ou <a href="https://appsource.microsoft.com">Microsoft AppSource</a>. Utilisez les applications SaaS quand votre solution est déployée dans votre propre abonnement Azure et que les clients se connectent via une interface que vous concevez et gérez pour tester l’application. <a href="https://azure.microsoft.com/services/active-directory">Azure Active Directory</a> (AAD) permet de tirer parti de votre environnement d’essai existant. </td> <td><img src="./media/cloud-partner-portal-publish-web-app/saasapp-icon1.png"  alt="Azure SaaS application icon" /></td> </tr> </table>


## <a name="publishing-benefits"></a>Avantages de la publication

Voici quelques-uns des avantages qu’offre la publication de vos applications sur une place de marché Microsoft :

- Atteindre 100 millions d’utilisateurs Azure Active Directory via Office 365 et Dynamics 365.
- Étendre votre équipe de vente : atteindre des utilisateurs professionnels dans le monde entier et obtenir un canal de vente qui implique les utilisateurs finaux, ce qui vous aide à générer des prospects et initie des conversations avec de nouveaux clients dans de multiples secteurs.

- Obtenir des informations détaillées exploitables : nous partagerons des informations détaillées sur les performances de votre application sur AppSource, ce qui fonctionne bien, et comment améliorer vos procédures de ventes.


## <a name="publishing-process"></a>Processus de publication

Pour créer et publier vos applications sur la Place de marché Microsoft Azure ou Microsoft AppSource, suivez les étapes de cette section :

1. Vérifiez que vous remplissez les conditions préalables, y compris les enregistrements de partenaire : [inscription dans le centre de développement](https://dev.windows.com/en-us/registration?accountProgram=Azure) *ou* [inscription dans Microsoft Partner Network](https://partner.microsoft.com/).
2.  Définissez votre offre de place de marché et vos plans.
3.  Fournissez le contenu marketing et vos tarifs.
4.  Testez et certifiez votre offre dans le [portail Microsoft Cloud Partner](https://cloudpartner.azure.com).
5.  Publiez votre offre.
