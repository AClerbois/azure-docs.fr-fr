# Gestion des paramètres Azure Multi-Factor Authentication dans le portail Office 365


Si vous utilisez l’authentification multifacteur avec Office 365, vous souhaitez peut-être gérer vos paramètres de vérification de sécurité supplémentaires par le biais du portail Office 365.

## Pour accéder aux paramètres de vérification de sécurité supplémentaires dans le portail Office 365

<ol>

<li>Connectez-vous au [portail Office&#160;365] (https://login.microsoftonline.com/).</li>
<li>Dans la partie supérieure, cliquez sur l'icône qui ressemble à une petite roue dentée. Ceci affichera une liste déroulante indiquant les paramètres Office&#160;365.</li>


<center>![Setup](./media/multi-factor-authentication-end-user-manage-o365/o365a.png)</center>

<li>Cliquez sur les paramètres Office&#160;365. La page des paramètres s’ouvre.</li>
<li>Sur la gauche, cliquez sur Vérification de sécurité supplémentaire.</li>

<center>![Setup](./media/multi-factor-authentication-end-user-manage-o365/o365.png)</center>

<li>Sur la droite, cliquez sur le lien indiquant **Mettre à jour mes numéros de téléphone utilisés pour la sécurité du compte**. Cette opération vous redirige vers une nouvelle page. Vous devez vérifier à nouveau si vous utilisez votre téléphone ou l’application mobile.</li>
<li>Une fois cette opération terminée, vous devez être sur la page de vérification où vous pouvez modifier vos paramètres.</li>

<center>![Cloud](./media/multi-factor-authentication-end-user-manage-o365/addsec.png)</center>
