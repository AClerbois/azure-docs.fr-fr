# <a name="azure-database-for-postgresql-videos"></a>Base de données Azure pour PostgreSQL
[How to get started with the new Azure Database for PostgreSQL service](https://channel9.msdn.com/events/Build/2017/B8046) (Prise en main du nouveau service de base de données Azure pour PostgreSQL)

<iframe src="https://channel9.msdn.com/Events/Build/2017/B8046/player" width="800" height="450" allowFullScreen frameBorder="0"></iframe>

Dans cette vidéo de la conférence //Build 2017 de Microsoft, écoutez deux clients ayant adopté très tôt le service de base de données Azure pour PostgreSQL discuter de la façon dont ils ont utilisé ce service pour innover plus rapidement. Nous apprenons comment ils ont migré vers le service, et ils abordent les étapes qui ont suivi pour le développement de leur application. Nous parcourons certaines des fonctionnalités clés du service et discutons de la façon dont vous pouvez, en tant que développeur, migrer vos applications existantes ou développer de nouvelles applications qui utilisent ce service PostgreSQL géré dans Azure.
