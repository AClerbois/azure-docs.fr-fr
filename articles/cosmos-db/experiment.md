# <a name="azure-cosmos-db-documentation-experiment"></a>Expérience de documentation Azure Cosmos DB

Azure Cosmos DB est une base de données multimodèle distribuée à l’échelle mondiale. Azure Cosmos DB permettant d’effectuer de nombreuses actions, notamment créer un graphique ou une base de données de documents ou encore migrer des données provenant de MongoDB, de Cassandra ou du Stockage Table Azure, l’organisation de la documentation peut se révéler complexe. Nous avons donc besoin de votre aide.

Nous sommes en train de tester l’organisation de la table des matières et de la [page de destination](https://docs.microsoft.com/en-us/azure/cosmos-db/) Azure Cosmos DB, illustrées ci-dessous, et nous aimerions connaître votre avis sur les modifications proposées.

## <a name="to-load-both-views"></a>Charger les deux affichages

1. Pour ouvrir les deux affichages côte à côte, accédez à la [page de destination](https://docs.microsoft.com/en-us/azure/cosmos-db/) avec le navigateur web de votre choix. 
2. Utilisez les images ci-dessus pour identifier chaque version (la version d’origine ou la version expérimentale). 
3. Ouvrez une fenêtre de navigation privée ou InPrivate et chargez-y la [page de destination](https://docs.microsoft.com/en-us/azure/cosmos-db/), puis utilisez une nouvelle fois les images pour identifier chaque version. 

   Si les deux affichages n’apparaissent pas, fermez et rouvrez une nouvelle fenêtre de navigation privée ou InPrivate, puis chargez à nouveau la [page de destination](https://docs.microsoft.com/en-us/azure/cosmos-db/). Plusieurs tentatives seront peut-être nécessaires pour que les deux versions se chargent dans des fenêtres distinctes. L’actualisation de la fenêtre privée ne charge pas l’autre version ; il est indispensable d’ouvrir une nouvelle fenêtre. 

4. Vous pouvez maintenant redimensionner les fenêtres afin de les placer côte à côte, ou basculer entre les affichages pour les comparer. 

5. Une fois que vous avez bien observé l’organisation et la disposition de l’expérience expérimentale, cliquez sur le bouton **Enquête sur l’expérience** ci-dessous pour nous faire part de vos commentaires. Envoyez le formulaire avant le 4 avril 2018 à minuit, pour que les avis puissent être analysés et intégrés.
