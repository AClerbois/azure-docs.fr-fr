# Création d'un service mobile

Cette rubrique montre comment utiliser le [portail de gestion Azure][portail de gestion Azure] pour créer dans Azure Mobile Services un serveur principal JavaScript qui utilise JavaScript pour la logique métier côté serveur. Pour créer à la fois le serveur principal du service mobile et une application cliente mobile qui consomme le nouveau service mobile, consultez plutôt la page [Prise en main de Mobile Services][Prise en main de Mobile Services].

> [WACOM.NOTE] Pour suivre ce didacticiel, vous devez disposer d'un compte Azure sur lequel la fonctionnalité Azure Mobile Services est activée. Si vous ne possédez pas de compte, vous pouvez créer un compte d'évaluation gratuit en quelques minutes. Pour plus d'informations, consultez la page [Version d'évaluation gratuite d'Azure][Version d'évaluation gratuite d'Azure].
