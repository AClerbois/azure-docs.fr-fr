# Consultation et modification de votre code source

Dans la version préliminaire du portail Microsoft Azure, il est facile de consulter et de modifier le code source de votre projet d'équipe Visual Studio Online.


Une fois le projet d'équipe créé sur le portail et que vous êtes enregistré (contrôle de version Team Foundation) ou que vous avez validé le code (git), vous pouvez facilement accéder au code et le consulter via le portail.


1. Ouvrez d'abord le volet du projet d'équipe, puis localisez le filtre Code.  Ensuite, dans la partie réservée à la source, cliquez sur le nom du référentiel que vous souhaitez parcourir. Dans ce cas, il s'agit de " BrowseCode ".  Remarque : si vous avez plus de deux référentiels, vous pouvez cliquer sur le numéro du référentiel (1 dans ce cas) pour afficher la liste complète.
![Code Lens](./media/visual-studio-online-browse-edit-source-code/Code-Lens.png)
2. Maintenant que le volet du référentiel est ouvert, vous pouvez cliquer sur la partie réservée au code pour parcourir le code source.Repository Blade.
![Repository Blade](./media/visual-studio-online-browse-edit-source-code/Repo-Blade.png)
3. Un volet avec navigation sous forme d'arborescence standard s'affiche. L'intégralité de votre projet est disponible.  Vous pouvez cliquer sur les nœuds pour ouvrir le dossier ou sur un fichier spécifique (Index.cshtml dans ce cas) pour afficher ou modifier le fichier.
![Tree Navigation](./media/visual-studio-online-browse-edit-source-code/Tree-Nav.png)
4. Pour modifier un fichier, cliquez simplement sur le bouton de modification de la barre de commandes.  Le contenu du fichier passera en mode de modification et vous pourrez procéder aux changements nécessaires. Conseil : cliquez sur le bouton d'agrandissement en haut à droite afin de voir l'écran de modification complet.
![Edit Mode](./media/visual-studio-online-browse-edit-source-code/Edit-Mode.png)
5. Lorsque vous avez fini de modifier vos fichiers, cliquez sur le bouton de validation (ou d'archivage) de la barre de commandes.  Cela ouvre un nouveau volet dans lequel vous pouvez saisir votre message de validation.  Cliquez sur OK lorsque vous avez terminé et le fichier sera validé vers le référentiel.
![Commit Code](./media/visual-studio-online-browse-edit-source-code/Commit-Code.png)
6. La partie supérieure du volet affiche un bref message indiquant que la validation vers le référentiel a réussi.
![Commit Success](./media/visual-studio-online-browse-edit-source-code/Commit-Success.png)
