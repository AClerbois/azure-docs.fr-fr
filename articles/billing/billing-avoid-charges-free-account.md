# <a name="avoid-getting-charged-for-your-azure-free-account"></a>Éviter les frais pour votre compte gratuit Azure

Un compte gratuit Azure vous donne 200 $ de crédit Azure à dépenser durant les 30 premiers jours et 12 mois de services gratuits. Pour plus d’informations, consultez [Compte Azure gratuit](https://azure.microsoft.com/free/) et [Créer des services gratuits avec un compte gratuit Azure](billing-create-free-services-included-free-account.md). En fonction du statut de votre crédit, vous pouvez utiliser votre crédit ou être facturé pour l’utilisation non couverte par les avantages du compte gratuit Azure.

## <a name="you-still-have-active-azure-free-account-credit"></a>Vous avez encore du crédit actif pour votre compte gratuit Azure 
Si avez encore du crédit Azure qui n’a pas expiré, vous utilisez ce crédit pour payer les frais encourus sur votre compte gratuit Azure.

## <a name="your-credit-ran-out-or-has-expired"></a>Votre crédit est épuisé ou a expiré
Si votre crédit est épuisé ou a expiré à la fin des 30 jours, Azure désactive votre abonnement. Vous n’êtes pas facturé, sauf si vous activez votre abonnement en le mettant à niveau vers un abonnement de type « paiement à l’utilisation ». Pour plus d’informations, consultez [Mise à niveau de votre abonnement d’essai gratuit ou Microsoft Imagine pour le paiement à l’utilisation](billing-upgrade-azure-subscription.md). Une fois que vous avez mis à niveau votre abonnement, des frais de paiement à l’utilisation vous sont facturés pour tous les frais encourus sur votre compte gratuit Azure. 


Examinons quelques-unes des raisons qui peuvent occasionner des frais sur votre compte gratuit Azure.


### <a name="your-usage-exceeds-the-limits-of-free-services-included-with-azure-free-account"></a>Votre utilisation dépasse les limites de services gratuits inclues avec le compte gratuit Azure

Vous obtenez une quantité limitée de services gratuits chaque mois avec votre compte gratuit Azure. La quantité gratuite expire à la fin du mois et n’est pas reportée au mois suivant. Par exemple, vous obtenez 5 Go de stockage de fichiers chaque mois. Si durant un mois vous utilisez seulement 2 Go, les 3 Go restants ne sont pas reportés au mois suivant. Pour éviter toute facturation, maintenez votre utilisation dans les limites. Pour en savoir plus sur les limites de services gratuits, consultez [What is Azure free account (Qu’est-ce qu’un compte gratuit Azure ?)](https://azure.microsoft.com/free/free-account-faq/). Pour vérifier votre utilisation des services gratuits, consultez [Vérifier l’utilisation des services gratuits inclus avec le compte gratuit Azure](billing-check-free-service-usage.md). 

### <a name="you-are-using-services-not-included-with-azure-free-account"></a>Vous utilisez des services non inclus avec le compte gratuit Azure

Des frais de paiement à l’utilisation vous sont facturés pour l’utilisation des services qui ne sont pas inclus gratuitement avec votre compte gratuit Azure. Pour en savoir plus sur les services inclus avec un compte gratuit, consultez [What is Azure free account (Qu’est-ce qu’un compte gratuit Azure ?)](https://azure.microsoft.com/free/free-account-faq/). Vous pouvez vérifier votre utilisation dans le portail Azure ou par le biais du fichier d’utilisation Azure. Pour plus d’informations, consultez [Vérifiez régulièrement la répartition des coûts et le taux d’avancement dans le portail](billing-getting-started.md#costs) et [Télécharger ou afficher votre facture Azure et vos données d’utilisation quotidienne](billing-download-azure-invoice-daily-usage-date.md). 

### <a name="you-have-reached-the-end-of-your-azure-free-account-benefits"></a>Vous avez atteint la fin des avantages de votre compte gratuit Azure

Les avantages de votre compte gratuit Azure expirent après 12 mois. Passé ce délai, Azure vous facture des frais de paiement à l’utilisation pour les services que vous utilisez. Vous pouvez utiliser le portail Azure pour supprimer les ressources pour les services que vous n’utilisez pas. Si vous prévoyez de n’utiliser aucun service Azure, vous pouvez [annuler votre abonnement](billing-how-to-cancel-azure-subscription.md).
