# <a name="how-to-create-a-commercial-marketplace-account-in-partner-center"></a>Comment créer un compte de la place de marché Commercial dans l’espace partenaires

Pour publier des offres pour [place de marché Azure](https://azuremarketplace.microsoft.com/) ou [AppSource](https://appsource.microsoft.com/) via la [portail place de marché Commercial](https://partner.microsoft.com/dashboard/commercial-marketplace/offers), vous devez créer un compte espace partenaires.  

Dans cet article, nous allons aborder le créer un compte espace partenaires, notamment comment : 

- [S’inscrire à l’aide de la page d’inscription de partenaires](#register-for-an-account-using-the-enrollment-page)
- [Connectez-vous avec un compte professionnel](#sign-in-with-a-work-account)
- [Acceptez les termes et conditions](#agree-to-terms-and-conditions) 
- [Fournissez votre profil d’éditeur](#provide-your-publisher-profile)

Si vous aviez un compte le [portail de partenaire de Cloud (CPP)](https://cloudpartner.azure.com) qui a été migré vers Partner Center, vous n’avez pas besoin créer un nouveau compte. Consultez [les éditeurs de migration à partir de CPP](#publishers-migrated-from-cpp). 

## <a name="prerequisites"></a>Conditions préalables

Pour créer un compte sur les partenaires, vérifiez que vous disposez :

- Autorisé à signer les accords juridiques en nom de votre société
- Nom d’entreprise juridique de votre entreprise, adresse et contact principal (il peut s’agir vous)

Nous allons vérifier ces informations pendant le processus de création de compte.

## <a name="register-for-an-account-using-the-enrollment-page"></a>Inscrivez-vous à un compte à l’aide de la page d’inscription 

Visitez le [ **Bienvenue dans Microsoft Partner Center** ](https://partner.microsoft.com/dashboard/account/v3/enrollment/introduction/azureisv) page d’inscription et examinez les informations d’inscription y trouvent.

### <a name="sign-in-with-a-work-account"></a>Connectez-vous avec un compte professionnel

Dans le cadre de votre inscription Partner Center, nous demandons que vous liez un domaine de compte de messagerie de travail de votre société à votre nouveau compte de partenaires. En associant ces comptes, vos employés de la société peuvent se connecter aux partenaires avec leurs noms d’utilisateur de compte de travail et les mots de passe.

Pour vous aider à la vérification selon que votre entreprise possède déjà un compte professionnel, comment créer un nouveau compte de travail ou comment configurer plusieurs comptes de travail à utiliser avec les partenaires, visitez [votre entreprise compte professionnel et partenaires](./company-work-accounts.md). 

### <a name="agree-to-terms-and-conditions"></a>Acceptez les conditions générales

Vous devez accepter les deux ensembles de conditions générales, y compris le [Microsoft Online Subscription Agreement](https://go.microsoft.com/fwlink/?LinkId=870457) et [contrat d’éditeur place de marché Microsoft](https://go.microsoft.com/fwlink/?linkid=843476).


### <a name="provide-your-publisher-profile"></a>Fournissez votre profil d’éditeur

Votre profil d’éditeur inclut votre nom de la société et l’ID MPN. Si vous ne le n'avez pas encore fait, vous devez joindre le [Microsoft Partner Network](https://partner.microsoft.com/commercial). En rejoignant Microsoft Partner Network, vous recevrez un ID. MPN 

Vous devez également déclarer votre ID d’éditeur pour l’instant. Votre ID d’éditeur identifiera de votre entreprise dans la place de marché Azure et AppSource. 

Une fois que vous avez confirmé que vos informations de profil de serveur de publication, vous pouvez accepter les termes et conditions et créez votre compte espace partenaires en sélectionnant **accepter et continuer**. *Vous devez être autorisé à agir pour le compte de votre entreprise afin d’accepter les termes du contrat.*

Vous venez de terminer l’inscription de votre centre de partenaires et que vous serez dirigé vers le [vue d’ensemble de la place de marché Commercial](./commercial-marketplace-overview.md) page.


## <a name="publishers-migrated-from-cpp"></a>Les éditeurs de migration à partir de CPP

Si votre compte a été migré à partir de la [portail de partenaire de Cloud (CPP)](https://cloudpartner.azure.com), vous n’avez pas besoin créer un nouveau centre de partenaire de compte, mais recevez un lien personnalisé vers votre nouveau compte espace partenaires par courrier électronique et une notification de bannière affiche une fois connecté à votre compte CPP existant.

Une fois que vous avez activé votre nouveau compte de partenaires en vous rendant sur ce lien personnalisé, vous pouvez revenir à votre compte en vous rendant sur le [tableau de bord de la place de marché Commercial](https://partner.microsoft.com/dashboard/commercial-marketplace/) dans Partner Center.

Les informations de profil de contrat et d’entreprise publication seront migrées vers votre nouveau compte de partenaires, ainsi que les précédemment configuré informations de profil de paiement de compte, les comptes d’utilisateur et les autorisations, et active offre associé à votre compte CPP. 

Une fois que votre compte est migré à partir de CPP pour partenaires, le compte de partenaires devient le compte principal à utiliser pour les mises à jour du compte gestion des utilisateurs, autorisations et gestion du paiement. Ces mises à jour du compte sont automatiquement synchronisées sur votre compte CPP en lecture seule jusqu'à ce que le portail CPP est finalement déconseillé.
