# <a name="how-to-code-and-test-azure-functions-locally"></a>Procédure locale de codage et de test d’Azure Functions
> [!NOTE]
> L’expérience des outils et de développement Azure Functions est disponible en préversion et sera considérablement améliorée avant la version Release finale. Pour exécuter l’hôte Azure Functions localement, le plus simple est d’utiliser [l’interface de ligne de commandes Azure Functions](https://go.microsoft.com/fwlink/?linkid=832752) et de l’installer à partir de npm. Actuellement, seul Windows est pris en charge. Une documentation complète sera bientôt disponible. Pour envoyer vos commentaires ou signaler des bogues concernant l’interface de ligne de commandes Azure Functions, signalez un problème dans le référentiel GitHub [azure-webjobs-sdk-script](https://github.com/Azure/azure-webjobs-sdk-script).
