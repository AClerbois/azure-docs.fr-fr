# <a name="customize-your-azure-security-center-for-iot-solution"></a>Personnaliser votre solution Microsoft Azure Security Center pour IoT

Dans ce guide, découvrez comment personnaliser différents paramètres dans Azure Security Center pour IoT.

> [!div class="checklist"]
> * Configurer les recommandations de solution
> * Modifier les paramètres

## <a name="change-settings"></a>Modifier les paramètres

Gérez votre paramètre Azure Security Center pour IoT :

Sur votre IoT Hub, accédez au panneau de vue d’ensemble de la sécurité dans l’angle supérieur gauche, consultez les « paramètres ». Pour configurer vos paramètres Azure Security Center pour IoT, procédez comme suit :

1. Ouvrez votre **hub IoT** dans le portail Azure.
1. Dans le menu de gauche, sous **Sécurité**, sélectionnez et ouvrez **Vue d’ensemble**.
1. Sous Paramètres, sélectionnez le paramètre de solution que vous souhaitez modifier.
1. N’oubliez pas de cliquer sur **Enregistrer** en haut de l’écran des paramètres pour enregistrer les modifications apportées aux paramètres.

## <a name="configure-solution-recommendations"></a>Configurer les recommandations de solution

Pour configurer vos recommandations de solution Azure Security Center pour IoT, procédez comme suit :

1. Ouvrez votre **hub IoT** dans le portail Azure.
1. Sélectionnez et ouvrez **Vue d’ensemble** sous **Sécurité** dans le menu de gauche.
1. Sous paramètres, sélectionnez **Configuration recommandée** dans le menu de gauche.
1. Désactivez/activez les recommandations de solution pertinentes pour votre organisation et vos flux de travail.
1. Sélectionnez **Enregistrer** en haut de l’écran pour enregistrer vos sélections.
