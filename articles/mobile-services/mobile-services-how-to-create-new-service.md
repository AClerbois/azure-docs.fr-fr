# Création d'un service mobile

Cette rubrique explique comment utiliser le [portail de gestion Azure][portail de gestion] pour créer dans Azure Mobile Services un nouveau back-end JavaScript qui utilise JavaScript pour la logique métier côté serveur. Pour créer à la fois le serveur principal du service mobile et une application cliente mobile qui consomme le nouveau service mobile, consultez plutôt la page [Prise en main de Mobile Services].

>[AZURE.NOTE]Pour suivre ce didacticiel, vous devez disposer d'un compte Azure sur lequel la fonctionnalité Azure Mobile Services est activée. Si vous ne possédez pas de compte, vous pouvez créer un compte d'évaluation gratuit en quelques minutes. Pour plus d'informations, consultez la <a href="http://www.windowsazure.com/pricing/free-trial/?WT.mc_id=A30A4DDE2&amp;returnurl=http%3A%2F%2FFfr-fr%2Fdocumentation%2Farticles%2Fmobile-services-how-to-create-new-service%2F" target="_blank">version d'évaluation gratuite d'Azure</a>.
