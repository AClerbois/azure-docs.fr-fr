# <a name="managing-users-on-the-cloud-partner-portal"></a>La gestion des utilisateurs sur le portail Cloud Partner

Le [portail Microsoft Cloud Partner](https://cloudpartner.azure.com) afin de pouvoir ajouter des utilisateurs avec accès en fonction du rôle pour les offres de machine virtuelle. Cela aide à gérer les accès et les autorisations lorsque plusieurs personnes travaillent sur les offres.

## <a name="add-users-and-assign-roles"></a>Ajouter des utilisateurs et attribuer des rôles 

Utilisez les étapes suivantes pour ajouter des utilisateurs et attribuer des rôles pour la publication de l’offre.

1. Cliquez sur le **utilisateurs** onglet dans le volet de navigation gauche.

    ![Onglet Utilisateurs](./media/userstab.png)


2. Cliquez sur **Add User**.

    ![Sélectionnez Ajouter un utilisateur](./media/adduser.png)


3. Tapez l’adresse de messagerie de l’utilisateur et sélectionnez une attribution de rôle.  Vous pouvez ajouter le nouvel utilisateur en tant que « propriétaire » ou « contributeur ».

    **E-mail** : ajoutez les adresses e-mail des collègues qui contribueront à la publication de l'offre. Les comptes Microsoft (Outlook, Hotmail et Live) et ID d’organisation sont pris en charge.

    - Ajouter un « groupe d’équipe messagerie alias/sécurité groupe » pour vous prémunir contre l’individu quitter l’organisation.
    - Vérifiez que cet e-mail ID fournis sur le portail Cloud Partner sont surveillés pour les communications de Microsoft.
    
    ![Affecter le rôle](./media/assignrole.png)

    **Rôle** : reportez-vous au tableau ci-dessous pour identifier le type de rôle utilisateur approprié.

    ![Niveau de rôles](./media/roleaccesslevel.png)

    Seuls les propriétaires peuvent accéder les onglets de paiements de Azure et Azure client dans [Insights de vendeur](../../cloud-partner-portal-orig/si-getting-started.md).


4. Sélectionnez le nom de serveur de publication que vous souhaitez ajouter au nouvel utilisateur, puis cliquez sur Ajouter pour terminer l’ajout de l’utilisateur.

    > [!NOTE]
    > Les « noms de serveur de publication » pour lequel votre adresse de messagerie est ajouté en tant que propriétaire ou collaborateur seront afficheront dans la liste déroulante des **Publisher** pour la sélection.


    ![Attribuer un rôle : sélectionner un éditeur](./media/assignselectpublisher.png)


## <a name="delete-an-existing-user"></a>Supprimer un utilisateur existant

Pour supprimer un utilisateur existant, recherchez leur courrier électronique dans le **utilisateurs** onglet, puis appuyez sur la **supprimer** bouton sur le côté droit.


## <a name="next-steps"></a>Étapes suivantes

Vous pouvez également utiliser le portail pour [gérer les profils de serveur de publication](./cpp-manage-publisher-profile.md).
