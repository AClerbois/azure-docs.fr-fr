# <a name="build-your-landing-page"></a>Créer votre page d’accueil  
La page d’accueil du site web de votre entreprise constitue la passerelle entre vos activités de génération de la demande et votre annonce dans la Place de marché.  

Commencez par identifier vos objectifs. Pour vos solutions, choisissez votre client cible et l’action que votre client doit effectuer.  Votre solution peut inclure les actions suivantes.  
*   Tester la solution  
*   Obtenir une solution maintenant  
*   et bien plus...  

Vous pouvez référencer votre page d’accueil avec plusieurs stratégies marketing. Vous pouvez utiliser les stratégies marketing.  
*   Événements  
*   Webdiffusions  
*   Médias sociaux  
*   Livres blancs  
*   Sessions de formation technique  
*   Communiqués de presse  

Plus votre message et votre appel à l’action sont cohérents, plus votre solution sera facile à trouver pour vos clients.  

Lorsque vous disposez d’un plan de campagne, créez votre page d’accueil. Passez en revue le tableau suivant pour optimiser l’efficacité de votre page d’accueil.  

| Bonne pratique | À éviter |  
|:--- |:--- |  
| ***Expliquer les problèmes que votre solution peut résoudre, et comment vous utilisez Azure à cette fin*** |  Ne pas expliquer la manière dont votre solution fonctionne avec Azure. |  
| ***Créer une URL courte et facile à mémoriser*** | Utiliser des URL longues. Les URL longues sont difficiles à retenir et à localiser. |  
| ***Ajouter du contenu visuel approprié comme une vidéo de témoignage client ou l’illustration de l’architecture d’une solution*** | Si vous utilisez trop de texte, vos clients ignorent votre contenu et arrêtent leur exploration. |  
| ***Créer un appel d’action clair pour diriger les visiteurs vers la page de votre catalogue sur la Place de marché*** | Avoir trop de liens ou d’actions dans la page. |  
| ***Placer votre appel à l’action dans un en-tête ou dans une section clairement délimitée*** |  Incorporez les liens dirigeant vers votre annonce dans un paragraphe. |  
| ***Rechercher les mots clés les plus efficaces et optimiser la page pour la recherche*** | Supposer que le nom de votre produit suffit à être mieux référencé. |  
| ***Utiliser les mots clés de vos campagnes publicitaires*** |  Si vous utilisez trop de mots-clés différents dans vos propriétés web, vos campagnes publicitaires perdent en efficacité. |  
| ***Faire apparaître les noms de produits et les mots clés importants `above the fold`*** | Obligez les utilisateurs à faire défiler la page pour voir le produit ou la solution que vous présentez. |  
| ***Utiliser des logos pour valider votre solution, en respectant les règles de Microsoft relatives à l’utilisation des marques et des logos, <br />comme Azure Certified*** | L’utilisation de l’image de marque Microsoft n’est pas autorisée. |  

*   Pour plus d’informations sur le badge Azure Certified, consultez la page Instructions relatives à la certification Microsoft Azure Certified à l’adresse [azure.microsoft.com/support/legal/marketplace/certified-guidelines](https://azure.microsoft.com/support/legal/marketplace/certified-guidelines).  
*   Pour obtenir des ressources supplémentaires sur la personnalisation, consultez la page Brand Central à l’adresse [microsoft.sharepoint.com/teams/brandcentral](https://microsoft.sharepoint.com/teams/brandcentral).  
*   Pour plus d’informations sur l’outil Générateur de logo, consultez la page Générateur de logo à l’adresse [logobuilder.partner.microsoft.com](https://logobuilder.partner.microsoft.com).  
*   Pour plus d’informations, consultez la page Fonctionnement à l’adresse [partner.microsoft.com/membership/how-it-works](https://partner.microsoft.com/membership/how-it-works).
