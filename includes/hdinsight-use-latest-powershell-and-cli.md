> [!IMPORTANT]
> La prise en charge de la gestion des ressources HDInsight à l’aide d’Azure Service Manager (ASM) par le biais d’Azure PowerShell et de la CLI Azure est **déconseillée**. Elle a pris fin le 1er janvier 2017. Dans ce document, la procédure repose sur les nouvelles applets de commande Azure PowerShell et les commandes de la CLI Azure qui fonctionnent avec Azure Resource Manager.
> 
> Pour installer les dernières versions, suivez les étapes des articles [Installation et configuration de l’interface Azure CLI](../articles/cli-install-nodejs.md) et [Installation et configuration d’Azure PowerShell](/powershell/azureps-cmdlets-docs). Si vous devez modifier certains scripts existants pour utiliser les nouvelles applets de commande ou les nouvelles commandes de l’interface de ligne de commande, consultez [Migration vers les outils de développement Azure Resource Manager pour les clusters HDInsight](../articles/hdinsight/hdinsight-hadoop-development-using-azure-resource-manager.md).
