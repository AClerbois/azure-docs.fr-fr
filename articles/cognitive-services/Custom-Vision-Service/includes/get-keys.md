## <a name="get-the-training-and-prediction-keys"></a>Obtenir les clés d’entraînement et de prédiction

Le projet doit avoir un jeu de clés d’abonnement valide afin d’interagir avec le service. Pour obtenir un jeu de clés gratuit, rendez-vous sur le [site web Custom Vision](https://customvision.ai) et connectez-vous avec un compte Microsoft. Sélectionnez l’__icône d’engrenage__ dans le coin supérieur droit. Dans la section __Comptes__, affichez les valeurs des champs __Training Key__ (Clé de formation) et __Prediction Key__ (Clé de prédiction). Vous en aurez besoin ultérieurement.
