# <a name="activity-management-overview"></a>Vue d’ensemble de la gestion des activités

CloudSimple effectue le suivi de toutes les activités qui peuvent affecter le fonctionnement de vos clouds privés. Ces activités, notamment les alertes, événements, tâches et activité d’audit. Le [pages des activités](https://docs.azure.cloudsimple.com/activity/) synthétiser toutes les activités en cours et vous permettent d’accéder à des détails supplémentaires.

## <a name="events"></a>Événements

Événements de suivi d’utilisateur et l’activité du système sur le portail CloudSimple.  L’activité associée à une ressource spécifique et la gravité de l’impact sur les événements s’affichent.  Vous pouvez afficher les événements à partir du portail CloudSimple.

## <a name="alerts"></a>Alertes

Les alertes sont des notifications de toutes les activités significatives sur votre environnement CloudSimple.  Événements, ce qui a un impact sur les accès utilisateur ou de facturation, sont affichés sous forme d’alertes.  Vous pouvez reconnaissez les alertes déclenchées à partir du portail CloudSimple.

## <a name="tasks"></a>Tâches

Tâches suivre toute opération de l’utilisateur, qui prend plus de 30 secondes.  Vous pouvez surveiller la progression d’une tâche à partir du portail CloudSimple.  Terminé show tâches le temps total nécessaire pour la tâche se termine.

## <a name="audit"></a>Audit

L’utilisateur a effectué des opérations de suivi de journaux d’audit.  Les journaux contiennent les paramètres fournis par l’utilisateur pour l’opération.  Vous pouvez utiliser les journaux d’audit pour surveiller l’activité des utilisateurs pour tous les utilisateurs.
