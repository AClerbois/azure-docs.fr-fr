<a name="cloud-partner-portal-gdpr-compliance"></a>Conformité RGPD du portail Microsoft Cloud Partner
====================================

Le [règlement général sur la protection des données (RGPD)](https://ec.europa.eu/info/law/law-topic/data-protection/reform_en/) est une loi de l’Union européenne sur la protection et la confidentialité des données. Le règlement RGPD établit des règles que doivent respecter les sociétés, les administrations, les ONG et d’autres organisations qui proposent des biens et services aux personnes de l’Union européenne ou qui collectent et analysent des données liées à des résidents de l’Union européenne.

Le portail Microsoft Cloud Partner fournit des informations détaillées sur le traitement des données des clients et les mesures de sécurité utilisées pour protéger ces données. Ces informations sont accessibles par le biais de ses expériences à l’intérieur du produit.

<a name="discover"></a>Découvrez
--------

Le Portail Cloud Partner donne accès à toutes vos données dans l’onglet Utilisateurs. Vous pouvez y identifier vos données personnelles. Pour savoir comment ajouter/supprimer des utilisateurs, consultez notre [documentation](./cloud-partner-portal-manage-users.md).

<a name="manage"></a>gérer
------

Vous pouvez gérer vos données personnelles en cliquant sur Supprimer de votre compte utilisateur dans l’onglet Utilisateurs. Il est possible d’attribuer le rôle **propriétaire** et le rôle **contributeur** à chaque utilisateur, mais aussi de les copier/mettre à jour/supprimer.

<a name="consent"></a>Consentement
-------

Pour la gestion du consentement dans le Portail Microsoft Cloud Partner, un accord éditeur vous est toujours présenté dans un premier temps ; vous devez accepter ses conditions pour être éditeur auprès de Microsoft, faute de quoi l’éditeur ne peut pas effectuer de ventes avec Microsoft.
