# <a name="overview-about-changing-your-security-info-preview-methods-and-information"></a>Vue d’ensemble du changement des méthodes et données relatives aux informations de sécurité (préversion)

Vous pouvez ajouter, modifier, supprimer et changer la méthode de vérification des informations de sécurité par défaut dans les pages **Informations de sécurité**. Pour plus de détails sur chaque méthode et sur la façon de changer vos informations, consultez l’article de procédure associé.

## <a name="two-factor-verification-versus-password-reset-authentication"></a>Vérification à deux facteurs et réinitialisation de mot de passe

Les méthodes relatives aux informations de sécurité sont utilisées pour la vérification à deux facteurs et la réinitialisation de mot de passe. Toutefois, ces techniques d’authentification ne prennent pas en charge toutes les méthodes.

| Méthode | Utilisé pour |
| ------ | -------- |
| Application Authenticator | Vérification à deux facteurs et réinitialisation de mot de passe. |
| SMS | Vérification à deux facteurs et réinitialisation de mot de passe. |
| Appels téléphoniques | Vérification à deux facteurs et réinitialisation de mot de passe. |
| Compte e-mail | Authentification par réinitialisation de mot de passe uniquement. Vous devez choisir une autre méthode pour la vérification à deux facteurs. |
| Questions de sécurité | Authentification par réinitialisation de mot de passe uniquement. Vous devez choisir une autre méthode pour la vérification à deux facteurs. |
