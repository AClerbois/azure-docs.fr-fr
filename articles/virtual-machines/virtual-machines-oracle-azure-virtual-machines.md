# Logiciel Oracle hébergé sur des machines virtuelles Azure

Configurez votre première ou prochaine base de données Oracle de développement/test ou de production en exécutant une machine virtuelle dans les services d’infrastructure Microsoft Azure. Tirez parti de la facilité de configuration et de la possibilité d’étendre rapidement la charge de travail de la base de données Oracle pour inclure de nouvelles capacités.

## Images de machines virtuelles Oracle

Consultez la page [Liste d’images de machines virtuelles Oracle](virtual-machines-oracle-list-oracle-virtual-machine-images.md) pour le dernier jeu d’images de machines virtuelles Windows et Linux.

Consultez la page [Considérations diverses relatives aux images de machines virtuelles Oracle](virtual-machines-miscellaneous-considerations-oracle-virtual-machine-images.md) pour comprendre les limites du logiciel Oracle sur les machines virtuelles Azure Windows Server.
