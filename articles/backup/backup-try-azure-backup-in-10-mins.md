# Essayez de sauvegarde Azure dans 10mins
Il est très facile du programme d'installation sauvegarde Azure et les modalités-à-vidéos suivantes devraient vous aider à démarrer.

## Étape 1: Obtenir un abonnement Azure
+ Si vous n'avez pas d'abonnement Azure, vous pouvez commencer avec un essai gratuit qui vous permet d'accéder à n'importe quel service Azure. Consultez le lien suivant [http://azure.microsoft.com/en-in/pricing/free-trial/](http://azure.microsoft.com/en-in/pricing/free-trial) pour en savoir plus.
+ Vous pouvez ignorer cette étape, vous avez un abonnement Azure.

## Étape 2: Configurer le service de sauvegarde Azure
Passez en revue les étapes présentées dans la vidéo à [http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-1-of-3-set-up-a-backup-vault-on-azure/](http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-1-of-3-set-up-a-backup-vault-on-azure/) à :

+	Créer un coffre de sauvegarde

+	Téléchargez « identification de coffre »

+	Télécharger l'agent de sauvegarde Azure

> [AZURE.VIDEO getting-started-with-azure-backup-1-of-3-set-up-a-backup-vault-on-azure]

## Étape 3: Préparer votre ordinateur local
Passez en revue les étapes présentées dans la vidéo à [http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-2-of-3-prepare-your-production-server-for-azure-backup/](http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-2-of-3-prepare-your-production-server-for-azure-backup/) à :

+ Installez l'agent de sauvegarde Azure sur votre client Windows Server ou le serveur DPM.

+ Inscrire l'ordinateur sur le coffre de sauvegarde


> [AZURE.VIDEO getting-started-with-azure-backup-2-of-3-prepare-your-production-server-for-azure-backup]

## Étape 4: Sauvegarde et restauration de données
Passez en revue les étapes présentées dans la vidéo à [http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-3-of-3-start-backing-up-your-production-server/](http://azure.microsoft.com/en-in/documentation/videos/getting-started-with-azure-backup-3-of-3-start-backing-up-your-production-server/) à :

+ Données de sauvegarde à partir d'un serveur

+ Restauration des données sauvegardées à partir de Azure

> [AZURE.VIDEO getting-started-with-azure-backup-3-of-3-start-backing-up-your-production-server]
