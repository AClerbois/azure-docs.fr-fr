# <a name="view-dashboard"></a>Affichage du tableau de bord #

Cliquez sur l’onglet **Tableau de bord** pour consulter les métriques clés de la révision de contenu effectuée dans l’outil. Vous pouvez afficher le total des révisions, les révisions terminées et celles qui sont en cours, qu’il s’agisse de texte ou d’images. Vous pouvez afficher les métriques pour votre équipe par défaut et pour vos sous-équipes.
