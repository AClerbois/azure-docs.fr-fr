# <a name="bing-visual-search-sdk"></a>SDK Recherche visuelle Bing

Recherche visuelle Bing fournit les SDK client suivants, que vous pouvez utiliser pour tester l’API Recherche visuelle.

* [Exemples relatifs à .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [Package NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.VisualSearch/1.2.0)
    * [Bibliothèques .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingVisualSearch) pour les définitions et les dépendances
* [Exemples relatifs à Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * [Bibliothèques Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) pour les définitions et les dépendances
* [Exemples Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * [Bibliothèques Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples/tree/master/Search) pour les définitions et les dépendances
* [Exemples Python Batch](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * [Bibliothèques Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) pour les définitions et les dépendances

Les exemples de SDK pour chaque langage incluent un fichier Lisez-moi comportant les prérequis pour l’utilisation du SDK, ainsi que les étapes nécessaire à l’installation et l’exécution des exemples.
