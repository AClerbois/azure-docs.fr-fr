# <a name="discover-experiments-in-cortana-intelligence-gallery"></a>Découvrir des expérimentations dans la galerie Cortana Intelligence
[!INCLUDE [machine-learning-gallery-item-selector](../../../includes/machine-learning-gallery-item-selector.md)]

## <a name="experiments-for-machine-learning-studio"></a>Expérimentations pour Machine Learning Studio
La galerie contient une grande variété d’[expérimentations](https://gallery.cortanaintelligence.com/experiments) qui ont été développées dans [Azure Machine Learning Studio](https://studio.azureml.net). Vous trouverez aussi bien des expérimentations de démonstration rapides qui illustrent une technique spécifique que des solutions entièrement développées pour les problèmes d’apprentissage automatique complexes.

> [!NOTE]
> Une ***expérimentation*** est un canevas dans Machine Learning Studio que vous pouvez utiliser pour créer un modèle d’analytique prédictive. Vous créez le modèle en connectant des données avec divers modules analytiques. Vous pouvez essayer différentes idées, effectuer des essais, puis publier votre modèle sous la forme d’un service web dans Azure. Pour obtenir un exemple de création d’une expérimentation simple, consultez [Didacticiel d’apprentissage automatique : Créer votre première expérimentation dans Azure Machine Learning Studio](create-experiment.md). Pour une description plus complète de la création d’une solution d’analytique prédictive, consultez [Procédure pas à pas : Développement d’une solution d’analytique prédictive pour l’évaluation des risques de crédit dans Azure Machine Learning](walkthrough-develop-predictive-solution.md).
>
>

## <a name="discover"></a>Découvrir
Pour parcourir les expérimentations [dans la galerie](http://gallery.cortanaintelligence.com), sélectionnez **Expérimentations** en haut de la page d’accueil de la galerie.

La page **[Expérimentations](https://gallery.cortanaintelligence.com/experiments)** affiche la liste des expérimentations récemment ajoutées et populaires. Pour voir toutes les expérimentations, sélectionnez le bouton **Afficher tout**. Pour rechercher une expérimentation spécifique, sélectionnez **Afficher tout**, puis sélectionnez les critères de filtre. Vous pouvez également entrer des termes de recherche dans la zone **Recherche** située en haut de la page de la galerie.

Vous pouvez obtenir plus d’informations sur une expérimentation dans la page de détails de l’expérimentation. Pour ouvrir une page de détails de l’expérimentation, sélectionnez l’expérimentation. Dans la section **Commentaires** d’une page de détails de l’expérimentation, vous pouvez commenter, donner un avis ou poser des questions sur l’expérimentation. Vous pouvez même partager l’expérimentation avec vos amis ou collègues sur Twitter ou LinkedIn. Vous pouvez également envoyer un lien vers la page de détails de l’expérimentation afin d’inviter d’autres utilisateurs à consulter cette page.

![Partager cet élément avec des amis](./media/gallery-how-to-use-contribute-publish/share-links.png)

![Ajouter vos propres commentaires](./media/gallery-how-to-use-contribute-publish/comments.png)

## <a name="download"></a>Télécharger
Vous pouvez télécharger une copie d’une expérimentation à partir de la galerie dans votre espace de travail Machine Learning Studio. Vous pouvez ensuite modifier votre copie pour créer vos propres solutions.

La galerie Cortana Intelligence propose deux méthodes pour importer une copie d’une expérimentation :

* **À partir de la galerie**. Si vous trouvez une expérimentation qui vous plaît dans la galerie, vous pouvez facilement en télécharger une copie et l’ouvrir dans votre espace de travail Machine Learning Studio.
* **À partir de Machine Learning Studio**. Dans Machine Learning Studio, vous pouvez utiliser n’importe quelle expérimentation de la galerie comme modèle pour créer une nouvelle expérimentation.

### <a name="from-the-gallery"></a>À partir de la galerie

1. Dans la galerie, ouvrez la page de détails de l’expérimentation.
2. Sélectionnez **Ouvrir dans Studio**.

    ![Ouvrez l’expérimentation à partir de la galerie](./media/gallery-experiments/open-experiment-from-gallery.png)

Lorsque vous sélectionnez **Ouvrir dans Studio**, l’expérimentation s’ouvre dans votre espace de travail Machine Learning Studio. (Si vous n’êtes pas déjà connecté à Machine Learning Studio, vous êtes invité à vous connecter une première fois avec votre compte Microsoft.)

### <a name="from-within-machine-learning-studio"></a>À partir de Machine Learning Studio

1. Dans Machine Learning Studio, sélectionnez **NOUVEAU**.
2. Sélectionnez **Expérimentation**. Vous pouvez faire votre choix dans la liste des expérimentations de la galerie ou rechercher une expérimentation précise à l’aide de la zone **Recherche**.
3. Pointez votre souris sur une expérimentation et sélectionnez **Ouvrir dans Studio**. (Pour en savoir plus sur l’expérimentation, sélectionnez **Afficher dans la galerie**. Vous accéderez à la page de détails de l’expérimentation dans la galerie.)

    ![Ouvrez une expérimentation de la galerie dans Machine Learning Studio](./media/gallery-experiments/open-experiment-from-studio.png)

Vous pouvez personnaliser, itérer et déployer une expérimentation téléchargée comme n’importe quelle autre expérimentation que vous créez dans Machine Learning Studio.

![Expérimentation ouverte dans Studio](./media/gallery-experiments/experiment-open-in-studio.png)

## <a name="contribute"></a>Participez
Lorsque vous vous connectez à la galerie, vous devenez membre de la communauté de la galerie. En tant que membre de la communauté, vous pouvez proposer vos propres expérimentations afin que d’autres utilisateurs puissent profiter des solutions que vous avez découvertes.

### <a name="publish-your-experiment-to-the-gallery"></a>Publier votre expérimentation dans la galerie

1. Connectez-vous à Machine Learning Studio à l’aide de votre compte Microsoft.
2. Créez votre expérimentation et exécutez-la.
3. Lorsque vous êtes prêt à publier votre expérimentation dans la galerie, sélectionnez **Publier dans la galerie** dans la liste des actions sous le canevas de l’expérimentation.

    ![Sélectionnez « Publier dans la galerie »](./media/gallery-experiments/publish-experiment-to-gallery.png)
4. Sur la page **Description de l’expérimentation**, entrez un titre et des mots clés. Faites en sorte que le titre et les mots clés soient descriptifs. Mettez en avant les techniques que vous avez employées ou le problème réel que vous résolvez. Voici un exemple de titre d’expérimentation descriptif : « Classification binaire : analyse des sentiments sur Twitter ».

    ![Entrez un titre et des mots clés pour la publication](./media/gallery-experiments/experiment-description.png)
5. Dans la zone **RÉSUMÉ** , entrez un résumé de votre expérimentation. Décrivez brièvement le problème que résout l’expérimentation et comment vous l’avez abordé.
6. Dans la zone **DESCRIPTION DÉTAILLÉE**, décrivez les étapes que vous avez suivies dans chaque partie de votre expérimentation. Certaines rubriques utiles à inclure ici sont les suivantes :
   * Capture d’écran du graphe de l’expérimentation
   * Sources de données et explication
   * Traitement des données
   * Ingénierie des caractéristiques
   * Description du modèle
   * Résultats et évaluation des performances du modèle

   Vous pouvez utiliser Markdown pour mettre en forme votre description. Pour voir à quoi ressemblent vos entrées sur la page de description de l’expérimentation lors de la publication de l’expérimentation, sélectionnez **Aperçu**.

   ![Sélectionnez « Aperçu » pour voir à quoi ressemblera votre texte](./media/gallery-experiments/preview-markdown-text.png)

   > [!TIP]
   > Les zones de texte fournies pour l’édition avec Markdown et l’aperçu sont petites. Nous vous recommandons d’écrire la documentation de votre expérimentation dans un éditeur Markdown, puis de la copier-coller dans la zone de texte dans la galerie. Après avoir publié votre expérimentation, vous pouvez apporter des corrections à l’aide d’outils web standard qui utilisent Markdown pour la modification et l’aperçu.

7. Sur la page **Sélection de l’image**, choisissez une image miniature pour votre expérimentation. L’image miniature apparaît en haut de la page de détails de l’expérimentation et dans le titre de l’expérimentation. Les autres utilisateurs voient alors l’image miniature lorsqu’ils parcourent la galerie. Vous pouvez télécharger une image à partir de votre ordinateur ou sélectionner une image stockée dans la galerie.
    </br>
    ![Charger ou sélectionner une image pour la galerie](./media/gallery-experiments/select-gallery-image.png)
8. Sur la page **Paramètres**, sous **Visibilité**, choisissez si vous souhaitez publier votre contenu publiquement (**Visible au public**) ou si vous souhaitez qu’il ne soit accessible qu’aux personnes qui disposent d’un lien vers la page (**Non visible au public**).

    ![Choisissez si vous souhaitez publier publiquement ou en tant que contenu non répertorié](./media/gallery-experiments/choose-public-or-unlisted.png)

    <!-- -->

   > [!TIP]
   > Si vous souhaitez vérifier que la documentation est correctement présentée avant de la diffuser publiquement, vous pouvez d’abord publier l’expérimentation en tant que **Non visible au public**. Vous pourrez opter plus tard pour le réglage de visibilité **Visible au public** sur la page de détails de l’expérimentation.
   >
   >
9. Cochez la case **OK** pour publier l’expérimentation dans la galerie.

    ![Cochez la case OK pour publier l’expérimentation](./media/gallery-experiments/ok-checkmark.png)

Pour obtenir des conseils sur la publication d’une expérimentation de galerie de qualité, consultez [Conseils pour documenter et publier votre expérimentation](#tips-for-documenting-and-publishing-your-experiment).

Et voilà, vous avez terminé.

Vous pouvez désormais voir votre expérimentation dans la galerie et partager le lien avec les autres. Si vous avez publié votre expérimentation avec le réglage de visibilité **Visible au public**, l’expérimentation s’affiche dans les résultats de navigation et de recherche dans la galerie. Vous pouvez à tout moment modifier la documentation de votre expérimentation sur la page de détails de l’expérimentation lorsque vous êtes connecté à la galerie.

Pour afficher la liste de vos contributions, sélectionnez votre image dans le coin supérieur droit de n’importe quelle page de la galerie. Sélectionnez ensuite votre nom pour ouvrir la page de votre compte.

![Sélectionnez le nom de votre compte](./media/gallery-experiments/click-account-name.png)

### <a name="update-your-experiment"></a>Mettre à jour votre expérimentation
Si besoin, vous pouvez apporter des modifications au workflow (modules, paramètres, etc.) dans une expérimentation que vous avez publiée dans la galerie. Dans Machine Learning Studio, apportez les modifications souhaitées sur l’expérimentation, puis publiez-la à nouveau. Votre expérimentation publiée sera mise à jour.

Vous pouvez modifier les informations suivantes pour votre expérimentation directement dans la galerie :

* Nom de l'expérimentation
* Résumé ou description
* Mots clés
* Image
* Réglage de visibilité (**Visible au public** ou **Non visible au public**)

Vous pouvez également supprimer l’expérimentation de la galerie.

Vous pouvez apporter ces modifications, ou supprimer l’expérimentation, à partir de la page de détails de l’expérimentation ou à partir de votre page de profil dans la galerie.


#### <a name="from-the-experiment-details-page"></a>À partir de la page de détails de l’expérimentation
Sur la page de détails de l’expérimentation, cliquez sur **Modifier** pour modifier les détails de votre expérimentation.

![Sélectionnez Modifier pour modifier votre expérimentation](./media/gallery-experiments/edit-button.png)

La page Détails entre en mode édition. Pour apporter des modifications, sélectionnez l’option **Modifier** située à côté du nom, du résumé ou des balises de l’expérimentation. Lorsque vous avez terminé d’apporter des modifications, sélectionnez **Terminé**.

![Sélectionnez « Modifier » pour modifier les détails, puis sélectionnez « Terminé » une fois terminé](./media/gallery-experiments/edit-details-page.png)

Pour modifier le réglage de visibilité de l’expérimentation (**Visible au public** ou **Non visible au public**) ou supprimer l’expérimentation de la galerie, sélectionnez l’icône **Paramètres**.

![Sélectionnez « Paramètres » pour modifier la visibilité ou supprimer l’expérimentation](./media/gallery-experiments/settings-button.png)

#### <a name="from-your-profile-page"></a>À partir de votre page de profil
Sur la page de votre profil, sélectionnez la flèche bas de l’expérimentation, puis sélectionnez **Modifier**. Vous accédez alors à la page de détails de votre expérimentation en mode édition. Une fois que vous avez terminé d’apporter des modifications, sélectionnez **Terminé**.

Pour supprimer l’expérimentation de la galerie, sélectionnez **Supprimer**.

![Sélectionnez « Modifier » ou « Supprimer »](./media/gallery-experiments/edit-delete-buttons.png)

### <a name="tips-for-documenting-and-publishing-your-experiment"></a>Conseils pour la documentation et la publication de votre expérimentation
* Vous pouvez partir du principe que le lecteur a de l’expérimentation en matière de science des données, mais il peut être utile d’utiliser un langage simple. Expliquez les choses en détail dès que possible.
* La suite Cortana Intelligence Suite est relativement nouvelle. Tous les lecteurs ne savent pas comment l’utiliser. Vous devez donc fournir suffisamment d’informations et d’explications détaillées pour aider les lecteurs à parcourir votre expérimentation.
* Des éléments visuels peuvent permettre aux lecteurs d’interpréter et d’utiliser correctement la documentation de votre expérimentation. Parmi les visuels, il y a les graphes de l’expérimentation et les captures d’écran de données. Pour plus d’informations sur l’ajout d’images dans la documentation de votre expérimentation, consultez la collection [Publishing Guidelines and Examples](https://gallery.cortanaintelligence.com/Collection/Publishing-Guidelines-and-Examples-1) (Guide et exemples de publications) .
* Si vous incluez un jeu de données dans votre expérimentation (autrement dit, si vous n’importez pas le jeu de données via le module Importer des données), le jeu de données fait partie de votre expérimentation et est publié dans la galerie. Assurez-vous que le jeu de données que vous publiez dispose de conditions de licence qui autorisent le partage et le téléchargement par toute personne. Les partages dans la galerie sont abordés dans les [Conditions d’utilisation](https://azure.microsoft.com/support/legal/website-terms-of-use/)d’Azure.

## <a name="frequently-asked-questions"></a>Forum Aux Questions
**Quelles sont les conditions requises pour envoyer ou modifier une image pour mon expérimentation ?**

Les images que vous envoyez avec votre expérimentation seront utilisées pour créer une vignette d'expérimentation pour votre participation. Nous recommandons des images de moins de 500 Ko, avec des proportions 3:2 et une résolution de 960 x 640.

**Qu'advient-il du jeu de données que j'ai utilisé dans l'expérimentation ? Le jeu de données est-il également publié dans la galerie ?**

Si votre jeu de données fait partie de votre expérimentation et n’est pas importé via le module Importer des données, il est publié dans la galerie dans le cadre de votre expérimentation. Assurez-vous que le jeu de données que vous publiez avec votre expérimentation dispose des conditions de licence appropriées. Les conditions de licence doivent autoriser quiconque à partager et télécharger les données.

**J’ai une expérimentation qui utilise un module Importer des données pour extraire les données à partir d’Azure HDInsight ou de SQL Server. Elle utilise mes informations d'identification pour récupérer les données. Puis-je publier ce genre d’expérimentation ? Comment être sûr que mes informations d’identification ne seront pas partagées ?**

Pour l’instant, vous ne pouvez pas publier une expérimentation qui utilise des informations d’identification dans la galerie.

**Comment entrer plusieurs mots clés ?**

Après avoir entré un mot clé, appuyez sur la touche Tab pour en entrer un autre.
