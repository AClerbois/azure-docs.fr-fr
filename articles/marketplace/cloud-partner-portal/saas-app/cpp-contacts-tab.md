# <a name="saas-application-contacts-tab"></a>Onglet Contacts de l’application SaaS

Cet article décrit comment utiliser l’onglet Contacts afin d’identifier les contacts pour les clients qui utilisent votre offre.

## <a name="identify-contacts"></a>Identifier les contacts

Utilisez la capture d’écran suivante pour vous guider et renseigner les informations requises. Tous les champs marqués d’un astérisque (*) doivent être renseignés.

![Formulaire des contacts ingénierie et support](./media/saas-offer-contacts.png)

1. Sous Contact ingénierie, renseignez les champs suivants :

   - Nom : entrez le nom associé au contact ingénierie pour votre application. Cette personne recevra des informations techniques de Microsoft.
   - Adresse e-mail : entrez l’adresse e-mail du contact ingénierie.
   - Téléphone : entrez le numéro de téléphone du contact ingénierie. Les notations de numéro de téléphone ISO sont prises en charge. Pour plus d’informations, voir [E.123](https://en.wikipedia.org/wiki/E.123)
  
2. Sous Contact du support, renseignez les champs suivants :

   - Nom : entrez le nom du contact du support pour votre application. Cette personne recevra des informations de support de Microsoft.
   - Adresse e-mail : entrez l’adresse e-mail du contact du support pour votre application.
   - Téléphone : entrez le numéro de téléphone du contact du support. Les notations de numéro de téléphone ISO sont prises en charge. Pour plus d’informations, voir [E.123](https://en.wikipedia.org/wiki/E.123)
   - URL du support : entrez l’URL de votre page de support.

3. Sélectionnez **Enregistrer** lorsque vous avez fini de remplir les formulaires.
