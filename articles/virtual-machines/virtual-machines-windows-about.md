# <a name="about-windows-virtual-machines-in-azure"></a>À propos des machines virtuelles Windows dans Azure
[!INCLUDE [learn-about-deployment-models](../../includes/learn-about-deployment-models-both-include.md)]

Azure Virtual Machines est l’un des nombreux types de ressources informatiques évolutives, à la demande proposées par Azure. En règle générale, une machine virtuelle sera mieux adaptée à vos besoins si vous devez surtout améliorer le contrôle de votre environnement informatique.

Une machine virtuelle Azure vous donne la flexibilité de la virtualisation sans devoir acheter le matériel physique qui exécute la machine virtuelle ni en assurer la maintenance. Toutefois, vous devez toujours assurer la maintenance de la machine virtuelle : configuration, mise à jour corrective et maintenance du système du logiciel qui s’exécute sur la machine virtuelle. Vous pouvez également en savoir plus [sur les machines virtuelles Linux dans Azure](virtual-machines-linux-about.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json).
