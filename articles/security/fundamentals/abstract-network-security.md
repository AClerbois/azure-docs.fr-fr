# <a name="azure-network-security"></a>Azure Network Security
## <a name="abstract"></a>Résumé
Par leur conception, les services réseau d’Azure maximisent la flexibilité, la disponibilité, la résilience, la sécurité et l’intégrité. Ce livre blanc fournit des détails sur les fonctions de mise en réseau d’Azure. Il décrit également comment les clients peuvent utiliser les fonctionnalités de sécurité natives dans Azure pour aider à protéger leurs ressources d’informations.
Ce livre blanc s’adresse essentiellement aux publics suivants :
- responsables techniques, administrateurs réseau et développeurs à la recherche de solutions de sécurité prises en charge et disponibles dans Azure ;
- responsables de PME ou de processus d’entreprise désireux d’obtenir une vue de haut niveau des technologies et services Azure en rapport avec la sécurité réseau dans le cloud public Azure.
