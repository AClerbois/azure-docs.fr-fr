# <a name="azure-data-factory-whitepapers"></a>Azure Data Factory livres blancs 

Livres blancs vous permettent d’Explorer Azure Data Factory à un niveau plus profond. Cet article vous fournit une liste de livres blancs disponibles pour Azure Data Factory.

| **Livre blanc** | **Description** |
| --- | --- |
|[Azure Data Factory, intégration de données dans le Cloud](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | Cet article décrit comment Azure Data Factory peut vous permettre de créer un entrepôt de données modernes, activer l’analytique avancée offrir des applications SaaS intelligentes et portez vos packages SQL Server Integration Services vers Azure.|
|[Migration des données à partir de l’entrepôt de données relationnelles locales vers Azure à l’aide d’Azure Data Factory](https://azure.microsoft.com/mediahandler/files/resourcefiles/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/Data_migration_from_on-prem_RDW_to_ADLS_using_ADF.pdf) | Cet article traite de la complexité de la migration des dizaines de To de données partir d’un entrepôt de données relationnelles sur site (par exemple, Netezza, Oracle, Teradata, SQL server) vers Azure (par exemple, stockage d’objets Blob ou Azure Data Lake Storage) à l’aide d’Azure Data Factory. Les défis et les meilleures pratiques sont illustrées autour de résilience, de performances, de l’évolutivité, de gestion et de sécurité pour le parcours d’ingestion de données volumineuses à Azure par Azure Data Factory. |
|[Azure Data Factory : SSIS dans le Cloud](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| Ce livre dépasse de la raison pour laquelle vous souhaitez migrer vos charges de travail SSIS existantes pour Azure Data Factory et les considérations courantes d’adresse et les problèmes. Nous allons ensuite vous guident dans les détails techniques de création d’un runtime d’intégration Azure-SSIS et vous montrer comment télécharger, exécuter et surveiller vos packages via Azure Data Factory à l’aide des outils que vous êtes probablement familiarisés avec telles que SQL Server Management Studio (SSMS). |
