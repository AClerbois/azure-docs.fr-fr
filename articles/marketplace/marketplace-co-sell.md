# <a name="co-sell-partner-engagement"></a>Engagement en tant que partenaire de co-vente

Si vous être un partenaire, le programme de co-vente de Microsoft vous permet de travailler directement avec Microsoft sur des opportunités de vente conjointe. Le programme aide nos partenaires à atteindre une vaste communauté de clients et à profiter de nouvelles opportunités de vente qui accéléreront la croissance de leur entreprise.

## <a name="partner-selling-motions"></a>Méthodes de vente partenaires

Microsoft propose trois méthodes de vente partenaires :

* *Vente par le biais du partenaire* : vous pouvez revendre la technologie Microsoft par l’intermédiaire de vos propres vendeurs.
* *Co-vente de services* : les vendeurs et les partenaires collaborent pour vendre votre projet ou vos services gérés en y intégrant la technologie Microsoft.
* *Co-vente IP* : les vendeurs et les partenaires collaborent pour vendre votre solution IP renouvelable et empaquetée.*

\* Certaines de ces solutions de co-vente IP peuvent bénéficier d’avantages supplémentaires lorsqu’elles sont vendues et déployées avec succès par les équipes commerciales de terrain de Microsoft.
