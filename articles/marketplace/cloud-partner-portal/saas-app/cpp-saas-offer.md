# <a name="azure-saas-application-offer"></a>Offre d’application Azure SaaS

Le [portail Microsoft Cloud Partner](https://cloudpartner.azure.com/) n’est plus prend en charge la création et la gestion des offres d’application SaaS.  Au lieu de cela, cette fonctionnalité a été déplacée vers Microsoft [portail espace partenaires](https://partner.microsoft.com/pcv/).  Pour plus d’informations, consultez [créer une offre SaaS](../../partner-center-portal/create-new-saas-offer.md). 


## <a name="next-steps"></a>Étapes suivantes

Avant de créer une nouvelle offre d’application SaaS, vous devez lire le [Guide de publication proposent des applications SaaS](../../marketplace-saas-applications-technical-publishing-guide.md) et examinez les étapes de portail partenaires répertoriés dans le [liste de vérification de la création offre](../../partner-center-portal/offer-creation-checklist.md).
