Analytique de LAC de données Azure rend la tâche complexe de la gestion d’infrastructure distribuée et code complexe. Il approvisionne dynamiquement des ressources, et vous pouvez l’utiliser pour effectuer l’analytique sur des exaoctets de données. Lorsque le travail est terminé, il ralentit ressources automatiquement. Vous payez uniquement pour la puissance de traitement qui a été utilisée. Il n'est pas nécessaire de réécrire le code à mesure que vous augmentez ou diminuez la taille des données stockées ou le volume de calcul utilisé. Pour augmenter les limites par défaut pour votre abonnement, contactez le support technique.
