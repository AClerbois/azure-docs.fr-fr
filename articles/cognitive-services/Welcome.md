# <a name="what-is-cognitive-services"></a>Qu’est-ce que Cognitive Services ?

Microsoft Cognitive Services (anciennement Projet Oxford) est un ensemble d’API, de kits SDK et de services qui permettent aux développeurs de créer des applications plus intelligentes, attrayantes et exposées. Microsoft Cognitive Services développe le portefeuille évolutif d’API d’apprentissage automatique de Microsoft et permet aux développeurs d’ajouter facilement des fonctionnalités intelligentes telles que la détection d’émotion et vidéo, la reconnaissance faciale, vocale et de la vision ainsi que la compréhension vocale et du langage, dans leurs applications. Notre objectif est de proposer des expériences informatiques plus personnelles ainsi qu’une productivité assistée par des systèmes qui améliorent leurs capacités à voir, entendre, parler, comprendre et qui commencent même à raisonner.

Vous trouverez une documentation sur chaque produit et les références API correspondantes dans le tableau de navigation gauche.

## <a name="getting-started-with-free-trials"></a>Bien démarrer avec les essais gratuits
Pour s’inscrire aux essais gratuits, un email et quelques [étapes simples](https://azure.microsoft.com/try/cognitive-services/ "Aide à l’inscription") suffisent. Vous avez besoin d’un compte Microsoft si vous n’en avez pas encore un. Vous recevrez une paire unique de clés pour chaque API demandée. La seconde est une simple clé de secours. Ne partagez les clés secrètes avec personne. Les essais ont à la fois un débit limité en termes de transactions par seconde ou minute et un plafond d’utilisation mensuel. Une transaction est un simple appel d’API. Vous pouvez effectuer une mise à niveau vers des niveaux payants pour débloquer ces restrictions.

## <a name="subscription-management"></a>Gestion des abonnements
Une fois connecté avec votre compte Microsoft, vous pouvez accéder à [Mes abonnements] (https://www.microsoft.com/cognitive-services/en-us/subscriptions « Mes abonnements ») pour afficher les produits que vous utilisez, les quotas restants et la possibilité d’ajouter des produits supplémentaires à votre abonnement.

## <a name="upgrade-to-unlock-limits"></a>Mettre à niveau pour débloquer les limites
Toutes les API disposeront d’un plan d’essai gratuit.  Lorsque des offres payantes vous sont proposées pour les API, vous êtes dirigé vers le portail Azure pour y effectuer l’achat.  Vous pouvez trouver des liens *Acheter* dans la page Abonnements si vous les utilisez déjà ou vous pouvez ignorer complètement la version d’essai et acheter via les liens fournis dans [Tarifs](https://www.microsoft.com/cognitive-services/en-us/pricing "tarifs").  Vous devez configurer un compte d’abonné Azure avec une carte de crédit et un numéro de téléphone. Si vous avez des exigences spéciales ou souhaitez simplement communiquer avec le service commercial, cliquez sur le bouton « Nous contacter » en haut de la page des tarifs.

## <a name="regional-availability"></a>Disponibilité régionale
Les API figurant dans Cognitive Services sont hébergées sur un réseau croissant de centres de données Azure gérés par Microsoft. Vous trouverez la disponibilité régionale de chaque API dans la [liste des régions Azure](https://azure.microsoft.com/regions).
    
À la recherche d’une région encore non prise en charge ? Faites-le nous savoir en créant une demande de fonctionnalité sur notre [forum UserVoice](https://cognitive.uservoice.com/).

## <a name="support"></a>Support
Nous sommes là pour vous tout au long de votre processus de développement. Contactez-nous à tout moment en utilisant les liens ci-dessous. 
* Pour une demande d’assistance ou des questions techniques, publiez votre demande dans [Stack Overflow](https://stackoverflow.com/questions/tagged/microsoft-cognitive).
* Pour une demande de commentaires ou de fonctionnalité, accédez à [UserVoice](https://cognitive.uservoice.com/).
