Ce didacticiel est basé sur le code téléchargé dans le démarrage rapide de Mobile Services. Avant de démarrer ce didacticiel, vous devez suivre [Prise en main de Mobile Services] ou [Ajout de Mobile Services à une application existante].

> [AZURE.IMPORTANT]Si vous avez effectué le didacticiel de démarrage rapide avant la sortie d’Azure Mobile Services Android SDK 2.0, vous devez le suivre de nouveau, car le Kit de développement logiciel (SDK) n’est pas rétrocompatible. Pour vérifier la version, étudiez la section **dépendances** du fichier **build.gradle** de votre projet.
