# <a name="manage-azure-and-appsource-marketplace-offers"></a>Gérer les offres sur les places de marché Azure et AppSource

Cette section présente les opérations générales utilisées pour gérer les offres sur les places de marché Azure et AppSource.  Par exemple, elle explique comment utiliser le [Portail Microsoft Cloud Partner](https://cloudpartner.azure.com/) pour créer et publier une offre de place de marché.  Cette section explique également des points techniques communs à la plupart des types d’offre, tels que les ressources de texte et d’image.


## <a name="standard-offer-operations"></a>Opérations standard sur les offres

Le Portail Cloud Partner permet à un éditeur d’effectuer les opérations de base suivantes sur une offre.

|     Opération      |  Description                                           |
|     ---------      |  -----------                                           |
| [Créer l’offre](./cpp-create-offer.md)   | Crée une entrée d’offre en attente dans le dépôt d’intégration, mais ne la publie pas automatiquement sur une place de marché Microsoft | 
| [Afficher l’état de l’offre](./cpp-view-status-offer.md)   | Indique l’état de publication d’une offre.  Il existe plusieurs emplacements dans le portail où apparaissent les informations d’état. |
| [Publier l’offre](./cpp-publish-offer.md) | Valide, puis publie une entrée d’offre terminée sur la place de marché Microsoft appropriée.  L’offre est ensuite dite *en ligne*. |
| [Mettre à jour l’offre](./cpp-update-offer.md)   | Permet de modifier certaines caractéristiques d’une offre existante.  S’applique aux offres en attente et publiées. |
| [Supprimer l’offre](./cpp-delete-offer.md)   | Supprime une offre en attente ou en ligne.  | 
|  |  |
