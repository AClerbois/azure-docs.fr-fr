# <a name="container-publishing-prerequisites"></a>Conditions préalables à la publication des conteneurs

Cet article décrit les conditions préalables à la publication d’une offre de conteneur sur la Place de marché Azure.

## <a name="publishing-prerequisites"></a>Publication des conditions préalables

Pour publier une nouvelle image de conteneur, vous devez respecter les conditions préalables suivantes :

- Accéder au portail Cloud Partner. Pour plus d’informations, consultez le Guide de publication sur AppSource et la Place de marché Azure.
- Accepter les conditions de la Place de marché Azure
- Héberger la ressource technique de votre conteneur dans un registre Azure Container Registry.
- Avoir vos métadonnées de conteneur prêtes à être utilisées. Par exemple, la liste non exhaustive suivante :
  - Un titre
  - Une description (au format HTML)
  - Une image de logo (au format PNG) avec les tailles d’images fixes suivantes : 40 x 40 px, 90 x 90 px, 115 x 115 px et 255 x 115 px.
- Des conditions d’utilisation et une politique de confidentialité
- Documentation
- Contacts du support
