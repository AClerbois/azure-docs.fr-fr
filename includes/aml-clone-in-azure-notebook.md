* **Azure Notebooks** -Jupyter gratuit en fonction des blocs-notes dans le cloud Azure
    1. [![Azure Notebooks](https://notebooks.azure.com/launch.png)](https://aka.ms/aml-clone-azure-notebooks)
    [Importer des exemples de blocs-notes dans Azure Notebooks](https://aka.ms/aml-clone-azure-notebooks).
    1.  Créez un espace de travail et son fichier de configuration (**config.json**) en suivant les instructions de [Guide pratique pour configurer un environnement de développement pour le service Azure Machine Learning](https://aka.ms/aml-how-to-configure-environment).
    1. Sélectionnez `+New` dans la barre d’outils Azure Notebooks pour ajouter votre fichier **config.json** dans le même dossier que le bloc-notes.
    1. Ouvrez le bloc-notes.

* **Votre propre serveur de blocs-notes Jupyter**

    Clonez le [dépôt GitHub](https://aka.ms/aml-notebooks) et suivez les instructions du fichier **Lisez-moi** pour installer le logiciel sur votre serveur.
