# <a name="view-your-recent-sign-in-activity-on-the-my-profile-preview-portal"></a>Afficher votre activité de connexion récente sur le portail Mon profil (préversion)
Vous pouvez afficher toutes vos activités de connexion récentes à l’aide de votre compte professionnel ou scolaire, à partir de la page **Activité récente** du portail **Mon profil**.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

## <a name="view-your-recent-sign-in-activity"></a>Afficher votre activité de connexion récente
1. Connectez-vous à votre compte professionnel ou scolaire, puis accédez à votre page https://myprofile.microsoft.com/.

2. Sélectionnez **Activité récente** dans le volet de navigation gauche ou sélectionnez le lien **Vérifier l’activité** à partir du bloc **Activité récente**.

    ![Page Mon profil, avec les liens d’activité récente en surbrillance](media/myprofile-portal/myprofile-portal-recent-activity-links.png)

3. Développez et passez en revue chacun des éléments de connexion, en veillant à reconnaître chacun d’eux. Si vous trouvez un élément de connexion qui ne vous semble pas familier, nous vous recommandons fortement de changer votre mot de passe pour protéger votre compte s’il est compromis.

    ![Page d’activité récente avec les détails de connexion étendus](media/myprofile-portal/myprofile-portal-recent-activity-page.png)

    >[!Note]
    >Si vous pensez que votre compte a été compromis, consultez [Procédure d’accès à un compte Microsoft piraté](https://support.microsoft.com/help/10494/microsoft-account-how-to-access-a-compromised-account) pour une aide sur la réinitialisation de votre mot de passe ou le déblocage de votre compte.

## <a name="next-steps"></a>Étapes suivantes

- Après avoir consulté votre activité de connexion récente, vous pouvez choisir d’afficher ou de gérer vos [informations de sécurité](user-help-security-info-overview.md), vos [appareils](myprofile-portal-devices-page.md) connectés ou [organisations](myprofile-portal-organizations-page.md) connectées, ou d’afficher la façon dont votre organisation [utilise vos données](myprofile-portal-privacy-page.md).
