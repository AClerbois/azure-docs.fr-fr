# <a name="azure-cosmos-db-documentdb-api-documentation"></a>Azure Cosmos DB : Documentation de l’API DocumentDB

Cet article fournit des liens vers tous les contenus Azure Cosmos DB propres à l’API DocumentDB.

Ces articles ne s’appliquent pas à l’API Graph, à l’API Table ou à l’API MongoDB.

À votre disposition ici, des rubriques et des ressources dont la lecture se révèle indispensable pour découvrir l’API DocumentDB pour Azure Cosmos DB.

- Site Web : [Query playground](https://www.documentdb.com/sql/demo) (interface de requête)
