# <a name="micro-focus-devtest-environments-on-azure"></a>Environnements de développement/test micro-charges Focus sur Azure

Mise en route sur Azure avec un environnement de développement et de test est un excellent moyen pour optimiser vos efforts de modernisation. Les applications mainframe qui dépendent de systèmes hérités tels que IBM CICS (Customer Information contrôle System) et IMS (Information Management System) peuvent être réhébergées sur Azure à l’aide de solutions de partenaires.

Micro Focus a conclu un partenariat avec Microsoft pour vous offrir des outils de migration et le réhébergement du macroordinateur attrayante. Vous pouvez exécuter des applications mainframe pour le développement, test, les scénarios de production sur Azure et la plateforme Microsoft Windows.

Micro Focus offre réhébergement du développement et test du réhébergement de solutions qui étendent COBOL existant et PL / j’applications d’entreprise vers Azure.
