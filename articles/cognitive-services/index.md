#<a name="cognitive-services-documentation"></a>Documentation Cognitive Services

Cognitive Services (version préliminaire) est une série d’API vous permettant d’autoriser les interactions naturelles et contextuelles au sein de vos applications.  Cognitive Services offre les API Language, Speech, Vision, Search et Knowledge.
