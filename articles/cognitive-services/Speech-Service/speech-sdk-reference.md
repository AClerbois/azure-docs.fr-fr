# <a name="speech-sdk-reference"></a>Référence du Kit de développement logiciel (SDK) Speech

Le Kit de développement logiciel (SDK) Speech fournit à vos applications un accès aux fonctions du service Speech, facilitant le développement de logiciels de reconnaissance vocale. Les Kits de développement logiciel (SDK) permettent actuellement d’accéder à **Reconnaissance vocale** et **Traduction vocale**. La synthèse vocale utilise les appels [REST] (https://docs.microsoft.com/azure/cognitive-services/speech-service/rest-apis) POST sur HTTP.

Le tableau ci-dessous répertorie les systèmes d’exploitation et langages de programmation actuellement pris en charge.

|Langage de programmation|Plateforme|Référence API|
|-|-|-|
|C#|Windows|[Browse](https://aka.ms/csspeech/csharpref)
|C/C++|Windows, Linux|[Browse](https://aka.ms/csspeech/cppref)
|Java\*|Appareils|[Browse](https://aka.ms/csspeech/javaref)

\* *Le Kit de développement logiciel (SDK) Java fait partie du [Kit de développement logiciel (SDK) Speech Devices](speech-devices-sdk.md).*
