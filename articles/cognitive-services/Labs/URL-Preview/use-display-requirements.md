# <a name="project-url-preview-use-and-display-requirements"></a>Conditions d’utilisation et d’affichage de l’aperçu de l’URL du projet

Utilisez uniquement les données de l’aperçu de l’URL du projet pour afficher les extraits de code de l’aperçu et les vignettes sous forme de liens hypertexte vers les sites sources dans une URL créée par l’utilisateur final et partagée sur les réseaux sociaux, un chatbot ou toute offre similaire. Vous ne devez en aucun cas copier, stocker ou mettre en cache les données que vous recevez d’un projet de l’aperçu d’URL d’un projet. Vous devez répondre à toutes les requêtes visant à désactiver les aperçus éventuellement reçus des propriétaires de site web ou de contenu.
