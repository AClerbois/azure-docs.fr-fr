# Gestion d'un espace de travail Azure Machine Learning 
Vous pouvez utiliser votre espace de travail Microsoft Azure Machine Learning (Azure ML) pour gérer les accès, déployer des services web de production et gérer vos services web Azure ML. De plus, l'onglet du tableau de bord affiche l'utilisation de l'espace de travail, ainsi que des informations sur ce dernier en un clin d'œil.  

Dans l'onglet **SERVICES WEB**, vous pouvez ajouter, mettre à jour ou supprimer des services web Azure ML.  

## Accès à la page de gestion de l'espace de travail ##
1.	Connectez-vous à votre compte Microsoft Azure.
2.	Dans le volet des services Microsoft Azure, cliquez sur **MACHINE LEARNING**.
3.	Dans la liste **d'apprentissage automatique**, cliquez sur un espace de travail.
