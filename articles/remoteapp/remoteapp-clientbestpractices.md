# Meilleures pratiques pour les clients RemoteApp

Les informations suivantes peuvent vous aider à utiliser des clients RemoteApp :

- Utilisez toujours la dernière version du client. Vous avez ainsi la garantie que la version du client que vous exécutez dispose des dernières résolutions de bogues, améliorations et fonctionnalités. Vous devrez peut-être vous inscrire pour recevoir automatiquement les mises à jour du client dans le magasin approprié.

- RemoteApp vous déconnecte automatiquement si vous êtes inactif pendant un certain temps. Afin d'éviter de perdre des données, nous vous conseillons de fermer vos applications lorsque vous avez terminé d'utiliser le service.
