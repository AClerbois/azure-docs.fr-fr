# <a name="cloudsimple-node-quota-limits"></a>Limites de quota de nœud CloudSimple

Quatre nœuds est la quantité par défaut disponible à l’achat, lorsque votre abonnement est activé pour le service de CloudSimple.  Vous pouvez acheter un [type de nœud](cloudsimple-node.md) à partir du portail Azure.  Un minimum de trois nœuds de la même référence SKU sont requis pour créer un Cloud privé.  Si vous avez acheté les nœuds, vous pouvez voir une erreur lorsque vous tentez d’acheter des nœuds supplémentaires.

## <a name="quota-increase"></a>Augmentation du quota

Vous pouvez augmenter le quota de nœuds en envoyant une demande de prise en charge. L’équipe d’opérations de service sera évaluer la requête et travailler avec vous permettent d’augmenter le quota de nœuds.  Sélectionnez les options suivantes lorsque vous ouvrez un nouveau ticket :

* Type de problème : **Technique**
* Abonnement : **Votre ID d’abonnement**
* Type de service : **Solution de VMware à CloudSimple**
* Type de problème : **Quota de nœuds dédié**
* Sous-type du problème : **Augmenter le quota de nœuds dédiés**
* Objet : **Augmentation du quota**

Les détails du ticket de support, indiquez le nombre de nœuds et de nœud de référence (SKU) requis.

Vous pouvez également contacter votre responsable de compte Microsoft à [ azurevmwaresales@microsoft.com ](mailto:azurevmwaresales@microsoft.com) pour augmenter le quota de nœuds de votre abonnement.  Vous devrez fournir le :

* Identifiant d’abonnement
* Nœud de référence (SKU)
* Nombre de nœuds supplémentaires pour lequel vous demandez l’augmentation du quota
