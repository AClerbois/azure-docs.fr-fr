# En savoir plus sur Azure AD

###Autres articles sur le sujet
[Présentation d’Azure AD](active-directory-whatis.md)<br> [Comment cela fonctionne-t-il ?](active-directory-works.md)<br> [Prise en main](active-directory-get-started.md)<br> [Étapes suivantes](active-directory-next-steps.md)<br> [En savoir plus](active-directory-learn-map.md)

J’ai déjà un abonnement Azure. Comment Azure AD est-il lié à cela ? [Association des abonnements Azure avec un répertoire](active-directory-how-subscriptions-associated-directory.md)

Pour des tâches administratives Azure AD telles que la gestion des utilisateurs, la gestion de domaine ou la configuration de l'authentification unique, utilisez le module Azure Active Directory pour les cmdlets Windows PowerShell. [Gestion d’Azure AD à l’aide de Windows PowerShell](https://msdn.microsoft.com/library/azure/jj151815.aspx)

Utilisez l’API Graph Azure Active Directory pour accéder par programme à Azure AD via les points de terminaison de l’API REST. [API Graph d'Azure AD](https://msdn.microsoft.com/library/azure/hh974476.aspx)

Vous pouvez ajouter votre propre nom de domaine personnalisé à Azure Active Directory pour l'utiliser avec les services en ligne de Microsoft. [Gestion de domaine internet](active-directory-add-domain.md)

Pour comprendre les contraintes d'utilisation et autres limites de service pour le service Azure Active Directory, consultez les [limites et restrictions de service Azure AD](active-directory-service-limits-restrictions.md)
