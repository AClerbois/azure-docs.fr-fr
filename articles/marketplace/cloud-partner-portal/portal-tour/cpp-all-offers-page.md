# <a name="all-offers-page"></a>Page Toutes les offres

La page [**Toutes les offres**](https://cloudpartner.azure.com/#alloffers) permet aux partenaires de voir la liste des offres existantes, avec le nom, le type, l’éditeur et l’état actuel de chaque offre.  Vous pouvez filtrer les résultats sur une chaîne que vous spécifiez.   La zone d’état en bas de la page affiche la valeur agrégée et le nombre de pages, et vous permet de naviguer entre les pages de la liste. Quand vous sélectionnez une offre dans cette liste, cette offre s’affiche dans l’éditeur d’offre associé.
