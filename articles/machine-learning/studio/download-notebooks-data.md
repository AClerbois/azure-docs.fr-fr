# <a name="download-notebookspreview-data"></a>Télécharger des données de notebooks (préversion)

La fonctionnalité Notebooks (préversion) a été supprimée le *13 avril 2020*. La suppression de l’onglet Notebooks (préversion) et des fichiers de notebook utilisateur est en cours dans les régions Azure du monde entier.

Cet article fournit des instructions pas à pas sur la façon de télécharger des données de notebooks (préversion).

1. Accédez à l’onglet **Notebooks** dans Azure Machine Learning Studio (classique).
    ![Accédez à Azure Notebooks !](./media/download-notebooks/notebooks-list.png)

1. Sélectionnez le notebook que vous voulez télécharger. Il s’ouvre alors dans Jupyter.
    ![Sélectionner un notebook](./media/download-notebooks/select-notebook.png) Le téléchargement simultané de plusieurs notebooks n’est pas pris en charge.

1. Accédez à **Fichiers** -> **Télécharger au format**, puis sélectionnez une option de format.
    ![Options de téléchargement de notebook](./media/download-notebooks/download-options.PNG)

1. Une fois que vous avez sélectionné une option, le téléchargement du fichier de notebook commence. Ignorez les erreurs 500 ou 502 qui s’affichent. Elles n’ont aucun impact sur votre téléchargement.
