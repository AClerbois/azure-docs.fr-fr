Une machine virtuelle (VM) est automatiquement attribuée une adresse IP privée à partir d’une plage que vous spécifiez, en fonction du sous-réseau, dans qu'il est déployé. L’adresse est conservée par une machine virtuelle jusqu'à ce que la machine virtuelle est supprimée. Azure attribue dynamiquement l’adresse IP privée disponible suivante à partir du sous-réseau que vous créez une machine virtuelle dans. Affecter une adresse IP statique si vous souhaitez une adresse IP spécifique à partir du sous-réseau attribué à la machine virtuelle.
