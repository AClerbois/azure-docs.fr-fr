# Exemples

[Télécharger l'exemple de version préliminaire publique Power BI Embedded](http://go.microsoft.com/fwlink/?LinkID=761493)

Pour faciliter vos premiers pas, nous avons créé un exemple d'application web . Vous pouvez télécharger et lire l'article qui l'accompagne [Prise en main de l'exemple](power-bi-embedded-get-started-sample.md). Cet exemple vous montre comment :

  -	Approvisionner du contenu
      - Créer un espace de travail
      - Importer un fichier PBIX
      - Mettre à jour les chaînes de connexion et définir les informations d'identification de vos jeux de données

  - Intégrer un rapport en toute sécurité
