# <a name="azure-active-directory-documentation"></a>Documentation Azure Active Directory

Azure Active Directory (Azure AD) est le service Microsoft de gestion des annuaires et des identités basé sur le cloud mutualisé.  Azure AD propose des fonctionnalités d’identité et d’accès pour les applications qui s’exécutent dans Azure et localement.
