# <a name="api-reference"></a>Référence API

L’API Visage Azure est une API basée sur le cloud qui fournit des algorithmes de détection et de reconnaissance de visage. Les API Visage comprennent les catégories suivantes :

- [API d’algorithme Visage](https://docs.microsoft.com/rest/api/cognitiveservices/face/face) : couvrent les fonctions principales, telles que [Détection](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/detectwithstream), [Rechercher des éléments similaires](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/findsimilar), [Vérification](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/verifyfacetoface), [Identification](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify) et [Grouper](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/group).
- [API de gestion FaceList](https://docs.microsoft.com/rest/api/cognitiveservices/face/facelist) : permet de gérer un FaceList pour [Rechercher des éléments similaires](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/findsimilar).
- [API de gestion LargePersonGroup Person](https://docs.microsoft.com/rest/api/cognitiveservices/face/largepersongroupperson) : permettent de gérer les visages LargePersonGroup Person pour [Identification](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify).
- [API de gestion LargePersonGroup](https://docs.microsoft.com/rest/api/cognitiveservices/face/largepersongroup) : permettent de gérer un jeu de données LargePersonGroup pour [Identification](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify).
- [API de gestion LargeFaceList](https://docs.microsoft.com/rest/api/cognitiveservices/face/largefacelist) : permettent de gérer un LargeFaceList pour [Rechercher des éléments similaires](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/findsimilar).
- [API de gestion PersonGroup Person](https://docs.microsoft.com/rest/api/cognitiveservices/face/persongroupperson) : permettent de gérer les visages PersonGroup Person pour [Identification](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify).
- [API de gestion PersonGroup](https://docs.microsoft.com/rest/api/cognitiveservices/face/persongroup) : permettent de gérer un jeu de données PersonGroup pour [Identification](https://docs.microsoft.com/rest/api/cognitiveservices/face/face/identify).
