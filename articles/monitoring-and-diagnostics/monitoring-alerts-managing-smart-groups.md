# <a name="manage-smart-groups"></a>Gérer des groupes intelligents
Les [groupes intelligents](https://aka.ms/smart-groups) utilisent des algorithmes de Machine Learning afin de regrouper des alertes en fonction de leur similarité ou de leur concomitance. L’utilisateur peut désormais gérer des groupes intelligents au lieu de gérer chaque alerte individuellement. Grâce à cet article, découvrez comment accéder aux groupes intelligents et les utiliser dans Azure Monitor.
1.  Pour consulter les groupes intelligents créés pour vos instances d’alerte, deux possibilités s’offrent à vous :
     1. Cliquez sur **Groupes intelligents** sur la page **Résumé des alertes**.    
    ![Surveillance](./media/monitoring-alerts-managing-smart-groups/sg-alerts-summary.jpg)
     2. Cliquez sur Alertes par groupes intelligents sur la page Toutes les alertes.   
     ![Surveillance](./media/monitoring-alerts-managing-smart-groups/sg-all-alerts.jpg)
2.  Vous accédez alors à la liste de tous les groupes intelligents créés sur vos instances d’alerte. Au lieu d’examiner différentes alertes individuellement, vous pouvez désormais traiter les groupes intelligents.   
![Surveillance](./media/monitoring-alerts-managing-smart-groups/sg-list.jpg)
3.  Cliquez sur un groupe intelligent pour ouvrir la page des détails comportant le motif du regroupement ainsi que les alertes de membre. Cette agrégation vous permet de traiter un seul groupe intelligent, au lieu d’examiner individuellement différentes alertes.   
![Surveillance](./media/monitoring-alerts-managing-smart-groups/sg-details.jpg)
