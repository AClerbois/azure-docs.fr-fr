# <a name="azure-cosmos-db-whitepapers"></a>Livres blancs sur Azure Cosmos DB 

Les livres blancs vous permettent d’explorer plus en profondeur les concepts Azure Cosmos DB. Cet article contient une liste de livres blancs disponibles pour Azure Cosmos DB.

| **Livre blanc** | **Description** |
| --- | --- |
|[Schema-Agnostic Indexing with Azure Cosmos DB](http://www.vldb.org/pvldb/vol8/p1668-shukla.pdf) | Ce document décrit le sous-système d’indexation d’Azure Cosmos DB. Il décrit des fonctionnalités Azure Cosmos DB telles que la représentation sous forme de document, le langage de requête, l’approche de l’indexation de document, la prise en charge des index de base et les premières expériences de production.|
| [Azure Cosmos DB et données personnelles](https://servicetrust.microsoft.com/ViewPage/TrustDocuments?command=Download&downloadType=Document&downloadId=87cc6456-4b23-473c-94d3-6c713b8b8956&docTab=6d000410-c9e9-11e7-9a91-892aae8839ad_FAQ_and_White_Papers)| Ce document fournit des conseils pour les clients Azure Cosmos DB qui gèrent une base de données cloud, une base de données locale, ou les deux, et qui doivent s’assurer que les données personnelles dans leurs systèmes de base de données sont gérées et protégées conformément aux réglementations en vigueur. |
