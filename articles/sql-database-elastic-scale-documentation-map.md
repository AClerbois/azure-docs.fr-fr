# Rubriques sur l'infrastructure élastique de base de données SQL Azure
L'infrastructure élastique de base de données SQL Azure (version préliminaire) permet à la couche Données d'une application de faire l'objet d'une mise à l'échelle via les pratiques de partitionnement standard du secteur, tout en rationalisant de manière significative le développement et la gestion de vos applications cloud partitionnées. L'infrastructure élastique propose des fonctionnalités de développement et de gestion via un ensemble de bibliothèques .net et de modèles de service Azure que vous pouvez héberger dans votre propre abonnement Azure pour gérer vos applications hautement évolutives. L'infrastructure élastique de Base de données SQL Azure implémente les aspects du partitionnement ayant trait à l'infrastructure, ce qui vous permet de vous concentrer sur la logique métier de votre application.
