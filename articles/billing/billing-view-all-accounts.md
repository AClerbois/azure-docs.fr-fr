# <a name="view-your-billing-accounts-in-azure-portal"></a>Afficher vos comptes de facturation dans le portail Azure  

Un compte de facturation est créé pour chaque contrat que vous signez avec Microsoft pour utiliser Azure. Vous utilisez votre compte de facturation pour gérer vos factures et méthodes de paiements et effectuer le suivi des coûts des services Azure. Vous pouvez accéder à plusieurs comptes de facturation. Par exemple, vous pouvez vous être inscrit à Azure pour vos projets personnels. Vous pouvez aussi avoir accès à Azure via le Contrat Entreprise de votre organisation ou le Contrat client Microsoft. Pour chacun de ces scénarios, vous disposez d’un compte de facturation distinct.

Le portail Azure prend actuellement en charge le type suivant de comptes de facturation :

- **Programme Microsoft Online Services** : Un compte de facturation pour le Programme Microsoft Online Services est créé lorsque vous souscrivez Azure via le site web Azure. Par exemple, quand vous souscrivez un [compte gratuit Azure](https://azure.microsoft.com/offers/ms-azr-0044p/), une [offre Azure assortie d’un paiement à l’utilisation](https://azure.microsoft.com/offers/ms-azr-0003p/) ou en tant qu’[abonné Visual studio](https://azure.microsoft.com/pricing/member-offers/credit-for-visual-studio-subscribers/).

- **Contrat Entreprise** : Un compte de facturation pour un Contrat Entreprise est créé quand votre organisation signe un Contrat Entreprise pour utiliser Azure. Les administrateurs du compte de facturation pour un Contrat Entreprise utilisent des services et des étendues de facturation de compte pour gérer et organiser la facturation pour leur organisation.

- **Contrat client Microsoft** : Un compte de facturation pour compte de facturation est créé quand votre organisation signe un Contrat client Microsoft pour utiliser Azure. Les administrateurs du compte de facturation pour un Contrat client Microsoft utilisent des profils de facturation et des étendues de facturation de section de facture pour gérer et organiser la facturation pour leur organisation. Pour plus d’informations, voir [Prise en main de votre compte de facturation dans le cadre d’un Contrat client Microsoft](billing-mca-overview.md).

## <a name="change-billing-account-view-in-the-azure-portal"></a>Modifier l’affichage d’un compte de facturation dans le portail Azure

Une étendue est un nœud à l’intérieur d’un compte de facturation que les utilisateurs utilisent pour afficher et gérer la facturation. Par exemple, les comptes de facturation pour le Programme Microsoft Online Services ont pour unique étendue le compte de facturation, tandis que les comptes de facturation pour un Contrat Entreprise ont pour étendue un compte de facturation, un service et un compte. Si vous avez accès à plusieurs étendues de facturation, par défaut, l’une d’elles s’affiche dans la page Azure Cost Management + Facturation. Vous pouvez modifier l’étendue en suivant les instructions ci-dessous :

1. Connectez-vous au [Portail Azure](https://portal.azure.com).

2. Effectuez une recherche sur **Gestion des coûts + facturation**.

   ![Capture d’écran montrant la recherche dans le portail Azure](./media/billing-view-all-accounts/billing-search-cost-management-billing.png)

3. Sélectionnez **Toutes les étendues de facturation** du côté gauche.

   ![Capture d’écran montrant toutes les étendues de facturation](./media/billing-view-all-accounts/billing-list-of-accounts.png)

4. Sélectionnez une étendue pour afficher les détails.
