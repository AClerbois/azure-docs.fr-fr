# Suivi de l'utilisation avec des événements et des mesures personnalisés

### Bientôt disponible

Le Kit de développement logiciel (SDK) Application Insights permet d'insérer des lignes de code dans votre application pour surveiller les activités des utilisateurs de façon à adapter votre application à leurs besoins.

Nous portons actuellement cette fonction dans le nouveau portail Application Insights Microsoft Azure. (Si vous le souhaitez, vous pouvez [voir comment elle fonctionne dans l'ancien portail](http://msdn.microsoft.com/library/dn481100.aspx)).

Cependant, vous pouvez maintenant [capturer et rechercher des données dans les journaux d'événements de diagnostics générés avec Trace, NLog ou Log4Ne][diagnostic].
