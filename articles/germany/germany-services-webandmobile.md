# <a name="azure-germany-web-and-mobile-services"></a>Services web et mobiles Azure Germany
## <a name="app-service"></a>App Service
Azure App Service est mis à la disposition générale dans Azure Germany.
### <a name="variations"></a>Variantes
L’adresse des applications Azure App Service créées dans Azure Germany est différente de celle des applications créées pour les services Azure globaux :

| Type de service | Azure global | Azure Germany |
| --- | --- | --- |
| App Service |*.azurewebsites.net |*.azurewebsites.de|
