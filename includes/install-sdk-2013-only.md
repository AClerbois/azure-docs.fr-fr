## <a name="a-namesetupdevenvaset-up-the-development-environment"></a><a name="setupdevenv"></a>Configuration de l’environnement de développement
Pour commencer, configurez votre environnement de développement en installant le [Kit de développement logiciel (SDK) Azure pour Visual Studio 2013][Azure SDK for Visual Studio 2013].

Si vous n’avez pas installé Visual Studio, Visual Studio Express pour le web est également installé.

> [!NOTE]
> En fonction du nombre de dépendances du Kit de développement logiciel (SDK) qui se trouvent déjà sur votre ordinateur, l'installation du SDK peut prendre un certain temps (de plusieurs minutes à une demi-heure, voire plus).
> 
> 

[Azure SDK for Visual Studio 2013]: http://go.microsoft.com/fwlink/?LinkID=324322
