# <a name="azure-synapse-analytics-data-integration-partners"></a>Partenaires d’intégration de données d’Azure Synapse Analytics
Pour créer votre solution d’entrepôt de données à l’aide de la fonctionnalité SQL Analytics d’Azure Synapse, vous pouvez choisir dans un vaste éventail d’outils de pointe. Cet article met en évidence les sociétés partenaires de Microsoft qui proposent des solutions d’intégration de données officielles prenant en charge Azure Synapse.
