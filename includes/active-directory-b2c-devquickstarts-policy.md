> [!NOTE]
> Dans Azure AD B2C, le nom de votre flux d’utilisateurs est précédé de `b2c_1_`, comme `b2c_1_sign_up`.  Vous êtes libre d’utiliser vos flux d’utilisateurs sur l’ensemble de vos applications (clientes et serveur).  Si vous avez déjà créé des flux d’utilisateurs dans une autre procédure B2C, il est inutile de recommencer l’opération. Vous pouvez réutiliser les flux d’utilisateurs que vous avez déjà créées sur le portail si elles répondent aux exigences de l’application.
