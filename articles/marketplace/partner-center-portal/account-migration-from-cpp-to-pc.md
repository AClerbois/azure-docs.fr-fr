# <a name="account-migration-from-cloud-partner-portal-to-partner-center"></a>Migration du compte à partir du portail Microsoft Cloud Partner pour partenaires

Lorsque vos offres sont migrés à partir du portail partenaire Cloud (CPP) pour Partner Center (PC), ils obtient verrouillés pour modification dans le CPP. À ce stade, votre paramètres compte doivent être migrées vers les partenaires.  Paramètres de votre compte, ainsi que vos offres, peuvent être gérés dans Partner Center.

## <a name="account-migration-process"></a>Processus de migration de compte

Lorsque les offres sont migrés sur le CPP, votre compte est configuré pour la migration. 
 
Si vous êtes un utilisateur avec le rôle de propriétaire dans le CPP pour un compte donné, une bannière jaune s’affiche sur votre page de profil d’éditeur.  Vous êtes invité à déplacer vos paramètres de compte vers l’espace partenaires. 

Cliquez sur la bannière pour lancer votre processus de migration de compte. Vous êtes tenu d’entrer les éléments suivants :

### <a name="work-e-mail-address"></a>**Adresse de messagerie professionnelle**

Dans la plupart des cas, connectez-vous avec l’adresse de messagerie que vous utilisez pour vous connecter à CPP. Dans certains cas, une autre adresse de messagerie doit être utilisée :

* Compte Microsoft : Si le compte de RPC est un Account de Microsoft, vous devez entrer une adresse de messagerie de travail valide associée au locataire, dont l’ID MPN est inscrit. 

* Incompatibilité de locataire : Si votre adresse de messagerie professionnelle n’appartient pas au locataire qui est associé à l’ID Microsoft Partner Network présents sur votre compte CPP, vous verrez une erreur. Pour déplacer cette erreur survient, entrez une adresse de messagerie associée au locataire. Un message d’erreur fournira le nom du locataire. 

### <a name="sign-up-for-microsoft-partner-network-program"></a>Inscrivez-vous au programme Microsoft Partner Network

Dans le cas où votre compte RPC n’a pas un ID Microsoft Partner Network ou possède un qui n’est pas valide, vous devrez vous inscrire au programme Microsoft Partner Network, dans le cadre du processus d’activation.

## <a name="account-activation-is-complete"></a>Activation du compte terminée

La migration de compte doit se ne produire qu’une seule fois pour un compte donné. Une fois un partenaire donné a terminé la migration du compte, tous les propriétaires voyez ce comportement sur la page de profil d’éditeur :

1. Vous vous apprêtez à voir la page des paramètres de partenaire dans le Microsoft Partner Network, où vous pouvez gérer les paramètres du compte de la Microsoft Partner. 
1. Une fois la migration de compte est terminée, une bannière jaune sur votre page de profil d’éditeur s’affiche demandent aux utilisateurs appartenant au rôle de propriétaire dans CPP pour un compte donné, pour gérer leurs paramètres de compte Centre de partenaires. 
1. La page de paramètres de compte dans le CPP est ensuite convertie en mode lecture seule.
