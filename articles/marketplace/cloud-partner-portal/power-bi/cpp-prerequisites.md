# <a name="power-bi-apps-prerequisites"></a>Conditions préalables des applications Power BI

Cet article répertorie les exigences techniques et de l’entreprise qui doivent être atteintes avant que vous puissiez publier une offre d’application Power BI App sur la Place de marché Microsoft AppSource.

## <a name="technical-requirements"></a>Exigences techniques

Le principal atout technique dont vous aurez besoin pour ce type d’offre est une [Application Power BI](https://go.microsoft.com/fwlink/?linkid=2028636). Pour plus d’informations, consultez [Créer des ressources techniques d’application Power BI](./cpp-create-technical-assets.md).

Une fois votre application de service créée et testée dans Power BI, vous devez enregistrer l’URL d’installation de l’application générée par Power BI. Cela est nécessaire pour [créer une offre d’application Power BI](./cpp-create-offer.md).

## <a name="business-requirements"></a>Exigences commerciales

Les exigences de l’entreprise incluent des obligations procédurales, contractuelles et légales :

-   Vous devez être inscrit en tant qu’éditeur de la place de marché Cloud. Si vous n’êtes pas encore inscrit, suivez les étapes décrites dans l’article [Become a Cloud Marketplace Publisher](../../become-publisher.md) (Devenir un éditeur de place de marché Cloud).

    > [!NOTE] 
    > Vous devez utiliser le même compte d’inscription à Microsoft Developer Center pour se connecter au [portail Cloud Partner](https://cloudpartner.azure.com). Vous ne devez utiliser qu’un seul compte Microsoft pour vos offres AppSource. Celui-ci ne doit pas être propre à des services ou offres individuels.
 
-   Il vous incombe de consentir tous les efforts raisonnables pour mettre à la disposition de vos clients un support technique.
-   Vous devez fournir un contenu conforme aux critères définis pour que votre offre soit répertoriée sur AppSource. Pour plus d’informations, voir [Vous avez une application à répertorier sur AppSource ? Voici comment procéder](https://appsource.microsoft.com/blogs/have-an-app-to-list-on-appsource-here-s-how).
-   Vous devez vous conformer à la [Déclaration de confidentialité Microsoft](https://privacy.microsoft.com/privacystatement).


## <a name="next-steps"></a>Étapes suivantes

Une fois que vous avez rempli toutes les conditions requises, vous êtes prêt à [créer une offre Power BI](./cpp-create-offer.md) dans le Portail Cloud Partner.
