# Remove-AzureRmFrontDoorFireWallPolicy
Supprimer la stratégie de pare-feu d’applications web (WAF)

## SYNTAXE

### ByFieldsParameterSet (valeur par défaut)
```
Remove-AzureRmFrontDoorFireWallPolicy -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByObjectParameterSet
```
Remove-AzureRmFrontDoorFireWallPolicy -InputObject <PSPolicy> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzureRmFrontDoorFireWallPolicy -ResourceId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Description
L’applet de commande **Remove-AzureRmFrontDoor** supprime une stratégie de pare-feu d’applications web sous l’abonnement actuel

## EXEMPLES

### Exemple 1
```powershell
PS C:\> Remove-AzureRmFrontDoorFireWallPolicy -Name $name -ResourceGroupName $resourceGroup
```

Supprimer une stratégie de pare-feu d’applications web (WAF) appelée $name dans $resourceGroup.

### Exemple 2
```powershell
PS C:\> Get--AzureRmFrontDoorFireWallPolicy -ResourceGroupName $resourceGroup | Remove-AzureRmFrontDoorFireWallPolicy
```

Supprimer toutes les stratégies de pare-feu d’applications web (WAF) de $resourceGroup.
