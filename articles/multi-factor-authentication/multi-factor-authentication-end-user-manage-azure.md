# Gestion des paramètres Azure Multi-Factor Authentication dans le portail Azure

Si vous utilisez l’authentification multifacteur avec Azure, vous souhaitez peut-être gérer vos paramètres de vérification de sécurité supplémentaires par le biais du portail Azure.

## Pour accéder aux paramètres de vérification de sécurité supplémentaires dans le portail Azure

<ol>
<li>Connectez-vous au portail Azure. <li>En haut du portail Azure, cliquez sur votre nom d’utilisateur. Une zone de liste déroulante s’affiche. <li>Dans la liste déroulante, sélectionnez Vérification de sécurité supplémentaire. La section Vérification de sécurité supplémentaire s’ouvre.

<center>![Setup](./media/multi-factor-authentication-end-user-manage-azure/changesettings.png)</center>

<li>Vous devez maintenant être sur la page de vérification où vous pouvez modifier vos paramètres.</li>

<center>![Setup](./media/multi-factor-authentication-end-user-manage-azure/proofup.png)</center>
