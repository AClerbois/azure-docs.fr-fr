# Azure AD Connect Sync : personnalisation des options de synchronisation

Les services Azure Active Directory Connect Synchronization (Azure AD Connect Sync) est un composant principal d’Azure AD Connect qui prend en charge toutes les opérations liées à la synchronisation des données d’identité entre votre environnement local et Azure AD dans le cloud. Du point de vue de la synchronisation, Azure AD Sync connecter est le successeur de DirSync, Azure AD Sync et Forefront Identity Manager avec le connecteur Active Directory configuré.
