## <a name="clean-up-resources"></a>Supprimer des ressources

Une fois que vous avez fini de travailler avec les ressources créées par cet article, utilisez la commande suivante pour les supprimer de votre abonnement :

```azurecli-interactive
az group delete --name myResourceGroup
```

Quand vous y êtes invité, tapez `y`. Cette commande supprime le groupe de ressources `myResourceGroup` et toutes les ressources qui lui appartiennent.
