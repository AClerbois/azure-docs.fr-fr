# Azure Privileged Identity Management : modification ou affichage des paramètres d'activation par défaut d’un rôle

## Modification et affichage de l’activation de rôle par défaut
1. Dans le tableau de bord, cliquez sur le rôle à configurer dans la table de rôles.
2. Cliquez sur **Paramètres**.
3. Définissez le nombre d’heure d’activation par défaut en déplaçant le curseur ou en entrant ce nombre dans le champ de texte.
4. Cliquez sur **Activer** ou **Désactiver** selon que vous souhaitez ou non que des notifications relatives à l’activation soient envoyées aux administrateurs.
5. Cliquez sur **Activer** ou **Désactiver** selon que vous souhaitez ou non autoriser les administrateurs à entrer des informations de billetterie dans leur demande d’activation.
6. Cliquez sur **Activer** ou **Désactiver** selon que vous souhaitez ou non exiger l’application de la solution Multi-Factor Authentication (MFA) pour une demande d’activation. Pour plus d’informations sur l’utilisation de la solution MFA avec Privileged Identity Management, voir [Exigence de l’application de la solution MFA](active-directory-privileged-identity-management-how-to-require-mfa.md).
7. Cliquez sur **Activer** ou **Désactiver** selon que vous voulez autoriser ou non que les administrateurs généraux soient temporaires. Pour plus d’informations sur ce paramètre, voir ci-dessous.
8. Cliquez sur **Save**.
