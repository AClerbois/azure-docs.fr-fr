# <a name="azure-policy-built-in-definitions-for-azure-storage"></a>Définitions intégrées d’Azure Policy pour le stockage Azure

Cette page est un index des définitions de stratégie intégrées d’[Azure Policy](../../governance/policy/overview.md) pour Stockage Azure. Pour obtenir des éléments intégrés supplémentaires d’Azure Policy pour d’autres services, consultez [Définitions intégrées d’Azure Policy](../../governance/policy/samples/built-in-policies.md).
