# Aide-mémoire pour l'applet de commande Microsoft Azure HDInsight PowerShell pour les tâches courantes de Hadoop

Microsoft Azure HDInsight fournit des applets de commande PowerShell pour effectuer les tâches Hadoop sur un cluster. L'**aide-mémoire pour l'applet de commande Microsoft Azure HDInsight PowerShell** est conçu pour vous fournir des applets de commande PowerShell pour les tâches Hadoop les plus courantes, telles que créer un cluster, exécuter des travaux, supprimer un cluster, etc. L'aide-mémoire fournit également une liste de toutes les applets de commande PowerShell disponibles pour Microsoft Azure HDInsight.

## Aide-mémoire HDInsight : modèle de gestion des services
