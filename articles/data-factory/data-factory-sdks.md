# Kits de développement logiciel (SDK) Azure Data Factory

Vous pouvez créer, surveiller et gérer les fabriques à l'aide du portail Azure, d’Azure PowerShell, de la bibliothèque de classes .NET ou de l’API REST.

Méthode | Emplacement de la ressource | Références destinées aux développeurs
---------------------------------------------------- | ------------------------------ | -----------
Portail Azure | [https://portal.azure.com/](https://portal.azure.com) | 
Azure PowerShell | Téléchargez la dernière version d’[Azure PowerShell](http://go.microsoft.com/?linkid=9811175&clcid=0x409) | [Référence des applets de commande](https://msdn.microsoft.com/library/dn820234.aspx)<p>[Référence sur la création de scripts JSON](https://msdn.microsoft.com/library/azure/dn835050.aspx) </p> 
Bibliothèque de classes .NET | Le Kit de développement logiciel (SDK) Azure Data Factory .NET vous permet de créer, surveiller et gérer des fabriques de données Azure et d’étendre Data Factory à l'aide d'une activité .NET. Consultez les articles [Utilisation des activités personnalisées dans un pipeline Azure DataFactory](data-factory-use-custom-activities.md) et [Créer, surveiller et gérer des fabriques de données Azure à l'aide du Kit de développement logiciel (SDK) Data Factory .NET](data-factory-create-data-factories-programmatically.md) pour vous aider à démarrer.<p><b>Téléchargement de la dernière version de NuGet</b><br/>Vous pouvez télécharger le dernier package NuGet de la bibliothèque de gestion Azure Data Factory à l’adresse : [https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/](https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/)</p><p>**Utilisation de la Console du Gestionnaire de package à l'aide de Visual Studio**<br/>Vous pouvez exécuter la commande suivante dans la Console du Gestionnaire de package de Visual Studio pour obtenir la dernière version de la bibliothèque de gestion Azure Data Factory</p>Install-Package Microsoft.Azure.Management.DataFactories</p> | [Référence du Kit de développement logiciel (SDK) .NET](https://msdn.microsoft.com/library/dn883654.aspx)
API REST | Vous pouvez utiliser l’API REST Data Factory pour créer, surveiller et gérer des fabriques de données Azure. | [Référence d’API REST](https://msdn.microsoft.com/library/dn906738.aspx)
