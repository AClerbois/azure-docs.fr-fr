# <a name="how-to-manage-pricing-tier-of-your-azure-maps-account"></a>Comment gérer le niveau tarifaire de votre compte Azure Maps

Vous pouvez gérer le niveau tarifaire de votre compte Azure Maps dans le portail Azure. Une fois que vous avez créé un [compte](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), vous pouvez afficher ou modifier le niveau tarifaire de votre compte.

Pour plus d’informations sur le choix d’un niveau tarifaire adapté, consultez [Choisir le bon niveau tarifaire](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier).

## <a name="view-pricing-tier"></a>Afficher le niveau tarifaire

Pour afficher le niveau tarifaire choisi, accédez à l’option **Niveau tarifaire** dans le menu Paramètres.

![Afficher le niveau tarifaire choisi](./media/how-to-manage-pricing-tier/view-pricing-tier.png)

## <a name="change-pricing-tier"></a>Changer le niveau tarifaire

Une fois que vous avez créé votre compte Azure Maps, vous pouvez passer à un niveau tarifaire inférieur ou supérieur. Pour passer à un niveau inférieur ou supérieur, accédez à l’option **Niveau tarifaire** dans le menu Paramètres, puis sélectionnez le niveau tarifaire qui n’est pas mis en surbrillance. Cliquez sur le bouton Enregistrer pour enregistrer l’option de niveau tarifaire choisie.

![Changer le niveau tarifaire](./media/how-to-manage-pricing-tier/change-pricing-tier.png)

> [!Note]
> Vous n’avez pas à générer de nouvelles clés d’abonnement pour modifier le niveau tarifaire de votre compte Azure Maps.
