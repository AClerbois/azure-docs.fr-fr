# Requêtes dans Azure Search
> [AZURE.SELECTOR]
- [Overview](search-query-overview.md)
- [Search Explorer](search-explorer.md)
- [Fiddler](search-fiddler.md)
- [.NET](search-query-dotnet.md)
- [REST](search-query-rest-api.md)

Les solutions personnalisées qui intègrent Azure Search pour prendre en charge une expérience de recherche doivent inclure du code permettant de construire une requête de recherche et de l’envoyer pour traitement vers le moteur de recherche.

Vous pouvez utiliser le SDK .NET ou l’API REST pour écrire des méthodes d’exécution des requêtes. Pour les tests préliminaires ou l’exploration, vous pouvez utiliser des outils comme l’Explorateur de recherche intégré dans le portail Azure pour interroger un index, ou utiliser Fiddler pour émettre une requête valide vers votre service de recherche.

Visitez [Vidéos, didacticiels, démonstrations et exemples de code](search-video-demo-tutorial-list.md) pour obtenir la liste des exemples et des procédures pas à pas qui montrent comment construire des requêtes dans le code. Tous les exemples ou presque que nous fournissons incluent du code qui interroge un service de recherche.
