# <a name="azure-data-factory-developer-reference"></a>Référence du développeur Azure Data Factory
Vous pouvez créer, surveiller et gérer les fabriques à l'aide du portail Azure, d’Azure PowerShell, de la bibliothèque de classes .NET ou de l’API REST.

| Méthode | Emplacement de la ressource | Références destinées aux développeurs |
| --- | --- | --- |
| Portail Azure |[https://portal.azure.com/](https://portal.azure.com) |[Prise en main d’Azure Data Factory (Portail Azure)](data-factory-build-your-first-pipeline-using-editor.md) |
| Azure PowerShell |Téléchargez la dernière version d’ [Azure PowerShell](http://go.microsoft.com/?linkid=9811175&clcid=0x409) |[Référence sur les cmdlets](https://msdn.microsoft.com/library/dn820234.aspx) |
| Bibliothèque de classes .NET |Le Kit de développement logiciel (SDK) Azure Data Factory .NET vous permet de créer, surveiller et gérer des fabriques de données Azure et d’étendre Data Factory à l'aide d'une activité .NET. Consultez les articles [Utilisation des activités personnalisées dans un pipeline Azure Data Factory](data-factory-use-custom-activities.md) et [Créer, surveiller et gérer des fabriques de données Azure à l’aide du Kit de développement logiciel (SDK) Data Factory .NET](data-factory-create-data-factories-programmatically.md) pour vous aider à démarrer.<br/><br/><b>Téléchargement du dernier package Nuget</b><br/>Vous pouvez télécharger le dernier package Nuget de la bibliothèque de gestion Azure Data Factory sur : [https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/](https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/)<br/><br/>**Utilisation de la console du gestionnaire de package dans Visual Studio**<br/>Vous pouvez exécuter la commande suivante dans la Console du Gestionnaire de packages de Visual Studio pour obtenir la dernière version de la bibliothèque de gestion Azure Data Factory<br/><br/>Install-Package Microsoft.Azure.Management.DataFactories |[Référence du Kit de développement logiciel (SDK) .NET](https://msdn.microsoft.com/library/mt415893.aspx) |
| API REST |Vous pouvez utiliser l’API REST Data Factory pour créer, surveiller et gérer des fabriques de données Azure. |[Référence d’API REST](https://msdn.microsoft.com/library/dn906738.aspx) |
