
Ce didacticiel requiert les éléments suivants :

* Visual Studio 2015
* Package NuGet [MicrosoftAzure.MobileEngagement](http://go.microsoft.com/?linkid=9864592)

> [!IMPORTANT]
> Ce didacticiel suppose l’utilisation de Visual Studio 15. Vous pouvez également utiliser Visual Studio 13, mais cette interface utilisateur est légèrement différente de ce qui est décrit dans ce didacticiel.
> 
>

Pour plus d'informations, consultez la page [Version d'évaluation gratuite d'Azure](https://azure.microsoft.com/pricing/free-trial/?WT.mc_id=A0E0E5C02&amp;returnurl=http%3A%2F%2Fazure.microsoft.com%2Ffr-FR%2Fdocumentation%2Farticles%2Fmobile-engagement-windows-store-dotnet-get-started).
