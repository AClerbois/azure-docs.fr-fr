# <a name="storefront-details-tab"></a>Onglet des informations sur les vitrines

Vous devez ensuite entrer les détails de votre vitrine. Les **détails de la vitrine** se composent des sections suivantes :

-   Détails de l’offre
-   Informations de l’éditeur
-   Détails de l’annonce
-   Artefacts marketing

![Création d’une offre de service de conseil : onglet Informations sur les vitrines](media/consultingoffer-storefront-details.png)

## <a name="offer-details"></a>Détails de l’offre

La section des détails de l’offre contient les champs suivants :

-   Résumé de l’offre
-   Description de l’offre

### <a name="offer-summary"></a>Résumé de l’offre

Le résumé de l’offre est une description courte de votre offre, qui figure juste en dessous du nom de l’offre. Entrez le résumé de l’offre en texte brut, sans sauts de ligne. Voici quelques exemples de résumés d’offre, ainsi que les noms correspondants :

*Exemple 1*

-   **Nom de l’offre :** Cloud Analytics : atelier de 3 jours
-   **Résumé de l'offre :** vue d’ensemble de Microsoft Azure et Power BI, évaluation de l'environnement actuel et mini preuve de concept.

*Exemple 2*

-   **Nom de l’offre :** IoT industriel Azure : preuve de concept de 30 jours
-   **Résumé de l'offre :** créer un pilote industriel de produit connecté pour connecter en toute sécurité des appareils de terrain à une solution Azure IoT Hub, avec tableaux de bord, rapports et notifications.

*Exemple 3*

-   **Nom de l’offre :** Services professionnels : réunion d'information d'une heure
-   **Résumé de l'offre :** vue d’ensemble et démonstration de la solution étendue Dynamics 365 for Operations préconfigurée, avec améliorations de la gestion des projets, de la facturation et des ressources pour les services professionnels.

*Exemple 4*

-   **Nom de l’offre :** Power BI dans votre monde : atelier de 4 heures
-   **Résumé de l'offre :** être rapidement opérationnel avec votre premier tableau de bord et découvrir les meilleures pratiques. 12 participants au maximum, sur site.

*Exemple 5*

-   **Nom de l’offre :** Dynamics et projets : évaluation de 3 jours
-   **Résumé de l'offre :** collecte des exigences et évaluation de la solution ERP conçue pour les sociétés de services professionnels et les entreprises orientées projet.

### <a name="offer-description"></a>Description de l’offre

Saisissez ici la description de l’offre de services de conseil. Une bonne description comprend les détails de l’engagement et les livrables proposés au client. Elle doit être claire pour aider le client à comprendre ce qu’il peut en obtenir. La description de votre offre doit mentionner son lien avec le produit Microsoft pour lequel vous proposez des services de conseil.

Ne mentionnez pas d’adresses e-mail ni de numéros de téléphone dans la description de votre offre. Votre offre comportera un bouton de **contact** qui chargera les prospects dans l’environnement de gestion des prospects identifié dans votre offre.

Entrez la description de l’offre au format Markdown. Si vous ne connaissez pas le format Markdown ou la mise en forme HTML, consultez les ressources sur l’[utilisation de la syntaxe Markdown pour la rédaction de documents](https://docs.microsoft.com/contribute/how-to-write-use-markdown).

Ces formats garantissent une lisibilité maximale pour les clients.

Restez bref dans la description de votre offre, et respectez la limite de nombre de caractères, car les utilisateurs n’aiment pas lire un texte abondant. Vous aurez encore la possibilité de charger des brochures marketing, des fiches techniques et d’autres documents décrivant votre offre en détail.

L’exemple suivant montre une description d’offre bien composée, avec son nom et son résumé :

**Nom de l’offre :** Cloud Analytics : atelier de 3 jours

**Résumé de l'offre :** vue d’ensemble de Microsoft Azure et Power BI, évaluation de l'environnement actuel et mini preuve de concept.

**Description de l’offre :** cet atelier de 3 jours est destiné aux responsables techniques et commerciaux. Il a lieu sur site chez le client.

***Agenda***

Jour 1

-   Sécuriser, mettre à l’échelle et organiser les données dans le cloud Microsoft avec Azure Data Lake, HDInsight ou Azure SQL Data Warehouse.

Jour 2

-   Configurer et déployer des solutions d’analytique avancées avec Microsoft R et Azure Machine Learning.

Jour 3

-   Extraire des informations exploitables et faire fonctionner l’analytique avec Power BI, avec une session de collaboration pour co-créer un tableau de bord Power BI.

À la fin de l’atelier, le client sera en mesure de définir un plan de haut niveau et un plan d’implémentation des solutions de données et d’analytique dans le cloud Microsoft.

*Exemple de fichier Markdown d’une offre dans ce format :*

    This 3-day workshop is for technical and business leaders and is held on-site at the client’s facility.

      ### Agenda

      **Day 1**

      * Focuses on how to secure, scale, and organize data within the Microsoft cloud using Azure Data Lake, HDInsight, or Azure SQL Data Warehouse

      **Day 2**

      * Covers how to configure and deploy advanced analytics solutions with Microsoft R and Azure Machine Learning

      **Day 3**

      * Covers how to draw actionable insights and operationalize analytics with Power BI, including a collaborative session to co-build a Power BI dashboard.


      ### Deliverables
      By the end of the workshop, the client will be able to define a high-level plan and an implementation roadmap for data and analytics solutions in the Microsoft cloud.


## <a name="publisher-information"></a>Informations sur l’éditeur

**ID MPN**

Votre ID Microsoft Partner Network (MPN) à 9 chiffres. Si vous n’avez pas d’ID MPN, vous pouvez accéder au centre des partenaires Microsoft pour obtenir un.

**ID de l’espace partenaires**

ID du nouvel Espace partenaires, si vous en avez un.

**ID MPN**

Entrez une clé secrète pour afficher un aperçu de votre offre sur AppSource avant de commencer.
Cet identificateur n’est pas un mot de passe.

## <a name="listing-details"></a>Détails de l’annonce

**Type de service de conseil**

Microsoft se concentre exclusivement les services de conseil de portée définie, de durée fixe, de prix estimé ou fixe (ou gratuits), principalement orientés vers la préparation de la vente, pour un client unique et pour l’évaluation, l’implémentation, la preuve de concept, la réunion d’information ou les offres en atelier, soit sur site ou soit par un canal virtuel. Le marketplace des services de conseil AppSource ne prend pas en charge les annonces pour des services managés ou par abonnement.

>[!Note]
>Les services de conseil AppSource ne sont pas l’endroit approprié pour les formations à la demande ou par abonnement.

Les cinq types d’offres suivants sont inclus :

-   **Évaluation :** évaluation de l’environnement d’un client pour déterminer la mise en application d’une solution et fournir une estimation du coût et de la durée.
-   **Réunion d’information :** présentation d’une solution ou d’un service de conseil pour susciter l’intérêt du client à l’aide de frameworks, de démonstrations et d’exemples de clients. Les réunions d’information doivent se tenir sur site.
-   **Implémentation :** installation complète qui résulte en une solution entièrement fonctionnelle. Pour ce programme pilote, Microsoft vous recommande de vous limiter à des solutions qui peuvent être implémentées en une semaine au maximum.
-   **Preuve de concept :** implémentation limitée permettant de déterminer si la solution répond aux besoins du client.
-   **Atelier :** engagement interactif effectué sur le site d’un client et qui peut comprendre des sessions de formation, des réunions d’information, des évaluations ou des démonstrations basées sur les données ou l’environnement du client.

**Disponibilité par pays/région**

Sélectionnez le pays et région où cette offre de services de conseil sera disponible. Il est impossible de publier une offre unique dans plusieurs pays ou régions. Une nouvelle offre doit être créée pour chaque pays ou région.

>[!Note]
>Les services de conseil AppSource sont actuellement disponibles aux États-Unis, au Royaume-Uni et au Canada. Vous pouvez soumettre une offre pour un pays qui n’est pas encore desservi. Elle sera passée en revue et préparée pour la mise en service. Un nombre minimal d’offres prêtes à l’emploi est nécessaire pour desservir un nouveau pays. Nous vous encourageons donc à soumettre des offres pour les pays qui ne sont pas encore desservis.

**Secteurs d’activité**

Sélectionnez les secteurs auxquels s’applique votre offre de service de conseil.

**Durée**

Sélectionnez un nombre (par exemple, 3, 4, etc.) sous Durée, puis sélectionnez heure, jour ou semaine.

**Produits principaux**

Pour publier sur la Place de marché Microsoft Azure, sélectionnez **Azure** comme produit principal, puis sélectionnez les zones applicables à la solution.

Pour publier sur AppSource, sélectionnez **Dynamics 365**, **Power BI** ou **PowerApps** comme produit principal. Vous pouvez également sélectionner les autres produits applicables, et votre offre de services de conseil s’affiche dans les listes associées à chacun de ces produits sur AppSource.

**Compétences appropriées**

Sélectionnez les compétences adaptées à cette offre pour qu’elles soient affichées avec les détails de l’offre.

## <a name="marketing-artifacts"></a>Artefacts marketing

**Logo de l’entreprise (au format .png, 48 x 48 pixels)**

Chargez l’image qui s’affichera sur la vignette de votre offre dans la page de la galerie des offres. Cette image doit être une image .png avec une résolution de 48 x 48 pixels.

**Logo de l’entreprise (au format .png, 216 x 216 pixels)**

Chargez l’image qui s’affichera sur la page des détails de votre offre. Cette image doit être une image .png avec une résolution de 216 x 216 pixels.

**Vidéos (Max 4)**

Chargez jusqu’à quatre vidéos d’étude de cas client ou vidéos de référence de vos clients. Si vous n’en avez pas, chargez une vidéo expliquant l’expertise de votre entreprise dans le cadre de cette offre. Si vous avez une présentation de la solution Power BI ou PowerApps, chargez la vidéo de présentation ici. Les liens vidéo doivent être YouTube ou Vimeo.

**Documents (Max 3)**

Chargez la brochure marketing qui décrit votre offre de services de conseil en détail. Vous pouvez aussi charger une présentation de l’entreprise, des fiches techniques ou des études de cas. Veillez à ce que vos documents utilisent les noms actuels des produits proposés et pas ceux de produits concurrents de Microsoft.

**Captures d’écran (Max 5)**

Chargez jusqu’à cinq images avec plus d’informations sur l’offre, sur les livrables ou sur votre société. Par exemple, un extrait de votre brochure marketing, une diapositive extraite d’une présentation ou une image illustrant l’orientation ou l’expertise de l’entreprise.
