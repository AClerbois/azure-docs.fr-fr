# Présentation des sites web

[WACOM.INCLUDE [disclaimer](../includes/disclaimer.md)]

Les sites web sont des hôtes d'applications web qui prennent en charge les technologies d'applications web populaires telles que .NET, Node.js et PHP sans que vous ayez à changer le code de vos applications existantes. L'ensemble des modèles de programmation répandus et des ressources utilisées par les développeurs .NET, Node.js et PHP pour accéder aux ressources telles que des bases de données et des fichiers continuent à fonctionner comme prévu sur les sites web déployés dans Azure. Les sites web permettent d'accéder à des ressources de fichiers et de bases de données, ainsi qu'à tous les fichiers de configuration standard (comme web.config, php.ini et config.js) utilisés par les applications web actuelles. Un site web peut héberger une application web exécutée dans le cloud que vous pouvez contrôler et gérer.
