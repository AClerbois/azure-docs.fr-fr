Cet article contient des informations sur le service Azure Machine Learning et sur le support que vous pouvez obtenir pour résoudre vos problèmes.

Consultez nos ressources d’apprentissage :
+ [Didacticiels, notes de publication et articles sur les procédures](../service/index.yml)

+ [Présentation de l'architecture](../service/concept-azure-machine-learning-architecture.md)

+ [Vidéos](https://azure.microsoft.com/resources/videos/index/?services=machine-learning)

## <a name="submit-doc-feedback"></a>Envoyer des commentaires sur la documentation 

Vous pouvez **soumettre des requêtes** pour demander des documents de formation supplémentaires à l’aide du bouton **Commentaires sur le contenu** situé au bas de chaque article.

## <a name="get-service-support"></a>Bénéficier d'un support de service 

Pour le support, consultez les ressources suivantes :

+ **Support technique pour les clients Azure** : [Envoyer et gérer les demandes de support](https://docs.microsoft.com/azure/azure-supportability/how-to-create-azure-support-request) via le portail Azure.

+ **Forum utilisateurs** : posez des questions, répondez à des questions et soyez en contact avec d’autres utilisateurs dans le [forum de support du Azure Machine Learning service sur le site MSDN](https://aka.ms/aml-forum-service).

+ **Stack Overflow** : consultez la page de la communauté Azure Machine Learning sur [StackOverflow](https://stackoverflow.com/questions/tagged/azure-machine-learning), balisée avec « Azure-Machine-Learning ».

+ **Partagez des suggestions sur le produit** et envoyez des requêtes sur les fonctionnalités sur notre [canal Azure Machine Learning Feedback Channel](https://feedback.azure.com/forums/257792-machine-learning). Sélectionnez le bouton **Commentaires sur le produit** situé en bas de chaque article.
