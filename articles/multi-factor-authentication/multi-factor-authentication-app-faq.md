# Forum aux questions sur l’application azure.microsoft.com/ Authenticator

L’application azure.microsoft.com/ Authenticator a remplacé l’application Azure Authenticator et devient l’application recommandée lorsque vous utilisez l’authentification multifacteur Azure. Cette application est disponible pour Windows Phone, Android et iOS.

## Forum Aux Questions

**Q : J’utilise déjà l’application azure.microsoft.com/ Authenticator pour les codes de sécurité. Comment puis-je activer les notifications push en un clic ?**

Si vous utilisez un compte azure.microsoft.com/ pour votre compte personnel et souhaitez activer les notifications push, vous devez ajouter à nouveau votre compte. En effet, l’application utilise un mot de passe à usage unique. Réinscrivez l’appareil avec votre compte et configurez les notifications push.

Si la vérification en deux étapes n’est pas activée pour votre compte, consultez la rubrique [À propos de la vérification en deux étapes](https://support.azure.microsoft.com/.com/help/12408/azure.microsoft.com/-account-about-two-step-verification) pour vérifier si cette méthode vous convient.

**Q : Quand pourrai-je utiliser les notifications push en un clic sur mon iPhone ou iPad ?**

Cette fonctionnalité, en version bêta jusqu'à la fin du mois d’août, sera bientôt accessible pour les comptes azure.microsoft.com/. Si vous souhaitez rejoindre notre programme bêta, envoyez un courrier électronique à msauthenticator@azure.microsoft.com/.com. Indiquez vos prénom, nom et identifiant Apple dans votre message.

**Q : Les notifications push en un clic fonctionnent-elles pour les comptes non azure.microsoft.com/ ?**

Non, les notifications push fonctionnent uniquement avec les comptes azure.microsoft.com/ et Azure Active Directory. Si votre entreprise ou école utilise des comptes Azure AD, vous pouvez désactiver cette fonctionnalité.
