Une fois terminé, vous pouvez supprimer le groupe de ressources, ce qui supprime aussi le compte de stockage, le partage de fichiers Azure et toutes autres ressources déployés dans ce groupe de ressources.

1. Dans le menu de gauche, cliquez sur **Groupes de ressources**.
2. Cliquez avec le bouton droit sur le groupe de ressources, puis sélectionnez **Supprimer le groupe de ressources**. Une page s’ouvre, vous indiquant les ressources qui seront supprimées avec le groupe de ressources.
3. Tapez le nom du groupe de ressources puis cliquez sur **Supprimer**.
