Ce didacticiel utilise Android Studio et Android 4.2 ou une version ultérieure. 

Ce didacticiel est basé sur le démarrage rapide de Mobile Services. Avant de démarrer ce didacticiel, vous devez suivre [Prise en main de Mobile Services] ou [Ajout de Mobile Services à une application existante].

> [AZURE.NOTE] Le Kit de développement logiciel (SDK) Mobile Services pour Android a été mis à jour vers la version 2.0. Si vous avez effectué les deux didacticiels ci-dessus il y a longtemps, vous devez les refaire s'ils n'utilisent pas le SDK mis à jour. Pour vérifier la version du SDK que votre application utilise, ouvrez-la dans Android Studio et contrôlez le contenu du dossier**libs**. Les numéros de version indiqués dans les fichiers jar Mobile Services doivent être 2.0 ou plus.
