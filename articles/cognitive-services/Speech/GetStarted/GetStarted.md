# <a name="get-started-with-speech-service"></a>Prise en main du service Speech

Pour utiliser le service de reconnaissance vocale de Microsoft afin de convertir des données audio en texte, consultez les [API REST](GetStartedREST.md) de reconnaissance vocale ou les [bibliothèques clientes](GetStartedClientLibraries.md).

Pour utiliser le service Microsoft de synthèse vocale, consultez la section [Référence de l’API Synthèse vocale](../api-reference-rest/bingvoiceoutput.md).
