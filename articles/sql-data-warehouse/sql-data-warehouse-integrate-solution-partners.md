# Partenaires de solutions SQL Data Warehouse

Pour créer votre solution d’entrepôt de données de bout en bout, vous pouvez choisir parmi un large éventail d’outils performants. Cet article répertorie les entreprises partenaires qui disposent de solutions pour SQL Data Warehouse.


| Partenaire | Description | En savoir plus |
| ------- | ----------- |------|
| ![Attunity][1] |**Attunity**<br>Attunity CloudBeam offre une solution automatisée et optimisée de chargement de données dans SQL Data Warehouse. Dotée d’une conception de type « cliquer pour charger », elle simplifie et accélère à la fois le chargement par lot et la réplication incrémentielle à partir de nombreuses bases de données hétérogènes, notamment SQL Server, Oracle, DB2, Sybase et mySQL. | [ En savoir plus ][attunity_more]|
| ![Birst][2] |**Birst**<br>Birst est un leader mondial en matière d’outils décisionnels cloud et d’outils d’analyse. Les entreprises peuvent créer une source de données approuvée dans le contexte de chaque utilisateur professionnel clé. Les utilisateurs professionnels de l’organisation peuvent alors générer des rapports et analyser les informations à l’aide d’outils décisionnels de premier ordre.| [En savoir plus][birst_more] |
| ![Bryte Systems][3] |**Bryte Systems**<br>Bryte Systems automatise le transfert de données vers SQL Data Warehouse à partir d’une multitude de bases de données et de sources. Bryte Systems est spécialisé dans les technologies d’intégration et de déplacement de données. Cette entreprise crée des entrepôts de données et des lacs de données cloud destinés à découvrir les perspectives. | [En savoir plus][bryte_systems_more] |
| ![ClearStory Data][4] |**ClearStory Data**<br>ClearStory Data assure une analyse rapide d’un ensemble de données disparates stockées dans SQL Data Warehouse. La plateforme Spark intégrée et l’application d’analyse de ClearStory accélèrent l’accès aux données et l’harmonisation de jeux de données hétéroclites. Elles permettent de procéder à une exploration collaborative rapide qui offre aux utilisateurs professionnels l’assurance d’acquérir de précieuses informations.| [En savoir plus][clearstory_data_more] |
| ![Coffing Data Warehousing][5] |**Coffing Data Warehousing**<br>Coffing Data Warehousing propose Nexus Chameleon, un outil résultant de 10 années de conception dédié à l’interrogation de systèmes. Nexus est disponible en tant qu’outil de requête pour SQL Data Warehouse. Utilisez Nexus pour interroger des ordinateurs en entreprise et cloud et pour associer les données de différentes plateformes. Il vous suffit de pointer et cliquer pour générer un rapport ! | [En savoir plus][coffing_data_warehousing_more] |
| ![Dell Statistica][6] |**Dell Statistica**<br>Dell Statistica est une plateforme d’analyse avancée permettant aux organisations d’effectuer des prévisions, d’optimiser les processus , d’alléger les coûts et d’augmenter les revenus. Dell Statistica simplifie l’analyse des données volumineuses (« Big Data »), rationalise l’implémentation et le déploiement, extrait des informations exploitables à partir d’un texte et répond aux besoins de votre entreprise. | [En savoir plus][dell_statistica_more] |
| ![Dundas][7] |**Logiciels Dundas**<br>Dundas Data Visualization est un fournisseur mondial majeur de logiciels décisionnels et de solutions de visualisation de données. Les tableaux de bord, les fonctions de création de rapports et les outils d’analyse visuelle des données conçus par Dundas s’intègrent de façon transparente aux applications d’entreprise, contribuant ainsi à améliorer la prise de décision et à accélérer l’obtention de vues d’ensemble. | [En savoir plus][dundas_software_more] |
| ![Inbrein][8] |**Inbrein**<br>Inbrein MicroERD vous offre les outils dont vous avez besoin pour créer un modèle de données précis, réduire la redondance des données, améliorer la productivité et respecter les normes. Grâce à son interface utilisateur, dont le développement repose sur des expériences utilisateur intensives, un modélisateur peut travailler sur des modèles de base de données en toute facilité. En outre, vous pouvez bénéficier en permanence d’ajouts ou d’optimisations de fonctions MicroERD par le biais de mises à jour et d’améliorations fonctionnelles rapides. | [En savoir plus][inbrein_more] |
| ![Infolibrarian][9] |**Infolibrarian**<br>InfoLibrarian catalogue, stocke et gère les métadonnées pour vous aider à résoudre les principaux points faibles de la gestion des données. En outre, InfoLibrarian fournit des solutions logicielles de gestion des métadonnées, de gouvernance des données et de gestion des actifs à des fins de gestion et de publication des métadonnées à partir d’un large éventail d’outils et de technologies. | [En savoir plus][infolibrarian_more] |
| ![Informatica][10] |**Informatica**<br>Informatica est l’un des fournisseurs indépendants de logiciels d’intégration de données les plus connus au monde. Informatica Vibe, l’une des premières machines de données virtuelles intégrables du secteur, assure les fonctionnalités de « mappage unique pour un déploiement universel » de la plateforme Informatica. Plus de 5 500 entreprises à travers le monde dépendent d’Informatica pour tirer pleinement parti de leurs actifs informationnels.| [En savoir plus][informatica_more] |
| ![Jinfonet][11] |**Jinfonet**<br>JReport est une solution décisionnelle intégrable pour l’entreprise. Cette solution permet aux utilisateurs de créer des rapports, des tableaux de bord et des analyses de données à partir de sources de données de type cloud, Big Data et transactionnelles. En visualisant les données, les utilisateurs peuvent procéder à leurs propres tâches de création de rapports et de découverte de données, bénéficiant ainsi d’une possibilité de prise de décision agile à la volée.| [En savoir plus][jinfonet_more] |
| ![LogiAnalytics][12] |**Logi Analytics**<br>Utilisés conjointement, Logi Analytics et Microsoft Azure SQL Data Warehouse permettent désormais à votre organisation de collecter, d’analyser et d’exploiter immédiatement les jeux de données les plus volumineux et les plus diversifiés du monde afin d’optimiser le rendement de l’entreprise. | [En savoir plus][logianalytics_more] |
| ![Looker][13] |**Looker**<br>Looker permet à tous les utilisateurs de votre entreprise d’explorer et de comprendre les données sur lesquelles repose votre activité, tout en offrant à l’analyste de données une couche de modélisation flexible et réutilisable pour le contrôle et le traitement de ces données. Looker a fait office de catalyseur en contribuant à révolutionner la culture des entreprises. | [En savoir plus][looker_more] |
| ![Qlik][14] |**Qlik**<br>Qlik propose une approche de l’analyse visuelle orientée plateforme qui s’appuie sur Microsoft Azure SQL Data Warehouse. En tirant parti des fonctionnalités d’entreprise disponibles dans l’entrepôt de données élastique de Microsoft, Qlik permet à l’ensemble de l’organisation de prendre des décisions rapides en toute confiance. Qlik offre des informations utiles et précises au moment stratégique, à savoir au point de prise de décision. | [En savoir plus][qlik_more] |
| ![Redpont Global][15] |**Redpoint Global**<br>RedPoint Interaction et RedPoint Data Management permettent aux mercaticiens d’exploiter l’ensemble de leurs données pour favoriser l’engagement des clients sur tous les canaux tout en assurant une gestion des données structurées et non structurées. En tirant parti de Microsoft Azure SQL Data Warehouse, les utilisateurs RedPoint peuvent optimiser la valeur de l’intégralité de leurs données structurées ou non structurées afin de fournir les interactions hyper-personnalisées et contextuelles requises pour encourager l’implication des clients omnicanaux d’aujourd’hui. L’interface de type glisser-déplacer facilite la conception et l’exécution des processus de gestion des données. | [En savoir plus][redpoint_global_more] |
| ![SiSense][16] |**SiSense**<br>SiSense est un logiciel décisionnel à pile complète intégrant les outils dont une entreprise a besoin pour analyser et visualiser les données : une base de données analytique hautement performante, la possibilité d’associer plusieurs sources, une extraction de données simple (ETL, extraction, transformation et chargement), ainsi qu’une visualisation des données web. Commencez à analyser et à visualiser des jeux de données volumineux avec le logiciel décisionnel et d’analyse SiSense dès aujourd’hui. | [En savoir plus][sisense_more] |
| ![SnapLogic][17] |**SnapLogic**<br>La solution iPaaS (Integration Platform as a Service) de SnapLogic permet aux clients de transférer rapidement des données à destination ou en provenance de Microsoft Azure SQL Data Warehouse. SnapLogic offre la possibilité d’intégrer des centaines d’applications, des services et l’Internet des objets dans le cloud, sur site ou en mode hybride pour présenter des vues d’ensemble de façon rapide et efficace. | [En savoir plus][snaplogic_more] |
| ![SQL Sentry][18] |**SQL Sentry**<br>SQL Sentry Performance Advisor offre des vues en temps réel et historiques des métriques de performances Azure SQL Data Warehouse les plus pertinentes, ce qui permet aux clients de visualiser les tendances des performances et d’accéder aisément à des informations plus détaillées à des fins d’analyse approfondie et de résolution des problèmes. Performance Advisor autorise la visualisation de données de performances portant sur un point dans le temps ou une période quelconques. | [En savoir plus][sql_sentry_more] |
| ![Tableau][19] |**Tableau**<br>Les outils d’analyse en libre service Tableau permettent à tous les utilisateurs de visualiser et de comprendre une multitude de types de données, depuis les fichiers plats jusqu’aux bases de données. Tableau comporte un connecteur optimisé natif pour Microsoft Azure SQL Data Warehouse qui prend en charge à la fois les données actives et les analyses en mémoire. | [En savoir plus][tableau_more] |
| ![Targit][20] |**Targit**<br>Targit Decision Suite offre une plateforme décisionnelle et d’analyse regroupant dans une même solution cohésive des tableaux de bord en temps réel, des analyses en libre service, des fonctions de création de rapports conviviales, de remarquables fonctionnalités mobiles et une technologie de découverte de données simple. Targit encourage les entreprises à passer à l’action. | [En savoir plus][targit_more] |
| ![Yellowfin][21] |**Yellowfin**<br>Yellowfin, figurant en tête du classement de l’étude sur le décisionnel menée par BARC (Business Application Research Center), est un fournisseur de solutions décisionnelles cloud pour la création de rapports et de tableaux de bord ad hoc. Connectez-vous à Azure SQL Data Warehouse, puis créez et partagez de remarquables rapports et tableaux de bord par le biais de fonctionnalités décisionnelles et géodécisionnelles collaboratives primées. | [En savoir plus][yellowfin_more] |
