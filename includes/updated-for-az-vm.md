> [!NOTE]
>
> Cet article a été mis à jour pour tenir compte de l'utilisation du nouveau [module Az d'Azure PowerShell](/powershell/azure/new-azureps-module-az). AzureRM continuera à obtenir les mises à jour de correction critiques, mais les nouvelles fonctionnalités seront uniquement exécutées via le module Az.
> * Le module Az est déjà disponible sur Cloud Shell. La compatibilité avec AzureRM est activée.
> * Pour installer le module Az localement, consultez [Installer Azure PowerShell](/powershell/azure/install-az-ps).
> * Si vous installez le module Az localement, vous pouvez activer la compatibilité AzureRM en exécutant `Enable-AzureRmAlias`.
