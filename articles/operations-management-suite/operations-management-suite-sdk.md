# <a name="operations-management-suite-(oms)-sdk"></a>Kit de développement logiciel (SDK) Operations Management Suite (OMS)
[Operations Management Suite (OMS)](../operations-management-suite/operations-management-suite-overview.md) est une solution de gestion informatique basée sur le cloud de Microsoft qui permet de gérer et de protéger votre infrastructure locale et dans le cloud.  Cet article répertorie la documentation et d’autres ressources disponibles pour l’accès aux services OMS en dehors des portails OMS et Azure.  Celles-ci incluent l’API REST pour l’accès à partir de différentes interfaces de programmation et les moteurs de script tels que PowerShell.
