# <a name="scheduler-documentation"></a>Documentation Scheduler

Azure Scheduler vous permet de façon déclarative de décrire des actions à exécuter dans le cloud.  Ensuite, il planifie et exécute ces actions automatiquement.  Scheduler conserve un historique consultable des résultats de l’exécution des travaux.
