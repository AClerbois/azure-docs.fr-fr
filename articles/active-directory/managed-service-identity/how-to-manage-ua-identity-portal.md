# <a name="create-list-or-delete-a-user-assigned-managed-service-identity-msi-using-the-azure-portal"></a>Créer, répertorier ou supprimer une identité du service administré (MSI) affectée par l’utilisateur à l’aide du portail Azure

[!INCLUDE[preview-notice](~/includes/active-directory-msi-preview-notice-ua.md)]

Managed Service Identity fournit aux services Azure une identité administrée dans Azure Active Directory. Vous pouvez utiliser cette identité pour vous authentifier auprès de services prenant en charge l’authentification Azure AD, sans avoir recours à des informations d’identification dans votre code. 

À ce stade, il n’est pas possible de créer, répertorier ou supprimer des identités affectées par l’utilisateur à l’aide du portail Azure.  Revenez ultérieurement pour des mises à jour.
