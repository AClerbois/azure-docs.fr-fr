# <a name="activity-management-overview"></a>Vue d’ensemble de la gestion des activités

AVS effectue le suivi de toutes les activités qui peuvent affecter le fonctionnement de votre environnement cloud privé AVS. Les activités incluent les alertes, les événements, les tâches et l’activité d’audit. Les [pages Activité](monitor-activity.md) synthétisent toutes les activités en cours et vous permettent d’accéder à des détails supplémentaires.

## <a name="events"></a>Événements

Les événements suivent l’activité de l’utilisateur et du système sur le portail AVS. Les événements affichent l’activité associée à une ressource spécifique et la gravité de l’impact.  Vous pouvez afficher les événements à partir du portail AVS.

## <a name="alerts"></a>Alertes

Les alertes sont des notifications de toutes les activités significatives dans votre environnement AVS. Les événements qui ont un impact sur la facturation ou l’accès utilisateur sont affichés sous forme d’alertes. Vous pouvez confirmer les alertes à partir du portail AVS.

## <a name="tasks"></a>Tâches

Les tâches effectuent le suivi de toute opération de l’utilisateur qui prend plus de 30 secondes. Vous pouvez surveiller la progression d’une tâche à partir du portail AVS. Pour les tâches terminées, les informations incluent la durée totale d’exécution.

## <a name="audit"></a>Audit

Les journaux d’audit assurent le suivi des opérations de l’utilisateur. Les journaux d’audit contiennent les paramètres fournis par l’utilisateur pour l’opération. Vous pouvez utiliser les journaux d’audit pour surveiller l’activité de tous les utilisateurs.
