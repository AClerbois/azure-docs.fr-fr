# Applications d'apprentissage automatique dans Azure Marketplace

Azure Machine Learning fournit aux scientifiques de données un canal pour la monétisation de leur savoir-faire et de leur créativité, comme le font les développeurs d'applications pour smartphone. En publiant vos services d'apprentissage automatique dans Azure Marketplace, vous permettez à tous les utilisateurs du monde entier d'acheter vos API publiées sur le cloud et de s'y abonner afin de les utiliser dans leurs applications.

[AZURE.INCLUDE [machine-learning-free-trial](../includes/machine-learning-free-trial.md)] 

Mais Azure Marketplace peut offrir beaucoup plus que des API d'analyse de base. Par exemple, nous avons créé un ensemble d'applications d'apprentissage automatique de bout en bout afin de répondre à des besoins métiers spécifiques. L'utilisation de ces applications d'apprentissage automatique ne requiert aucun scientifique de données ni expertise concernant Machine Learning. La science des données est déjà intégrée à nos solutions. Les utilisateurs n'ont plus qu'à alimenter ces dernières avec leurs propres données avant de commencer à utiliser les applications. 

Ces applications comprennent notamment :

##Recommandations##
 Moteur de recommandations libre-service mais très puissant qui vous permet d'ajouter facilement des recommandations de produits personnalisées à votre site web ou à vos campagnes par courrier électronique.

[Lien vers DataMarket](http://datamarket.azure.com/dataset/amla/recommendations)

[Guide de démarrage rapide](machine-learning-recommendation-api-quick-start-guide.md) 
  
##Fréquemment achetés ensemble##
API aidant vos clients à identifier les articles de votre catalogue qui sont fréquemment achetés ensemble. Utilisez votre historique des achats client pour ajouter des recommandations " Fréquemment achetés ensemble " à votre site web et pour améliorer les conversions dans votre magasin numérique.

[Lien vers DataMarket](https://datamarket.azure.com/dataset/amla/mba)

[Documentation](machine-learning-apps-frequently-bought-together-api.md)

##Détection des anomalies##
API détectant les événements anormaux dans les données de série chronologique (telles que les données de télémétrie de serveur ou les données transmises par des capteurs).

[Lien vers DataMarket](https://datamarket.azure.com/dataset/aml_labs/anomalydetection)

[Documentation](machine-learning-apps-anomaly-detection.md)

Ces API vous permettent de consommer de puissants services d'apprentissage automatique reposant sur Azure Machine Learning sans qu'il vous soit nécessaire d'être un scientifique de données ni de passer beaucoup de temps à développer vous-même des applications. Il vous suffit d'importer vos propres données et de commencer à les exploiter.

##Analyse de texte##
Cette API permet d'analyser un texte non structuré pour des tâches comme l'analyse des opinions et l'extraction des expressions clés. L'utilisation de cette API ne requiert aucune formation. Il vous suffit d'importer vos données de texte. Cette API utilise des techniques avancées sous-jacentes de traitement du langage naturel. 

[Lien vers DataMarket](https://datamarket.azure.com/dataset/aml_labs/anomalydetection)

[Documentation](machine-learning-apps-text-analytics.md)
