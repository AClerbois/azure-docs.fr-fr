# <a name="updates-to-the-sign-in-experience-for-online-services"></a>Mises à jour de l’expérience de connexion pour les services en ligne

Nous mettons à jour notre expérience de connexion pour les services en ligne de Microsoft, tels qu’Office 365 et Azure. Bien qu’il ne s’agisse que d’une modification visuelle, vous remarquerez que les écrans semblent moins encombrés et plus simples, vous permettant ainsi de mieux vous concentrer sur ce que vous devez faire : vous connecter.

## <a name="current-ui-versus-new-ui"></a>Interface utilisateur en cours et nouvelle interface utilisateur

Comparez l’interface utilisateur existante à la nouvelle. La nouvelle interface utilisateur est plus rationalisée et ciblée, avec une apparence plus moderne.

![ancienne expérience de connexion Microsoft](./media/sign-in-experience-updates/microsoft-sign-in-experience-old.jpg)

![nouvelle expérience de connexion Microsoft](./media/sign-in-experience-updates/microsoft-sign-in-experience-new.jpg)

## <a name="when-will-the-new-ui-appear"></a>Quand la nouvelle interface utilisateur apparaîtra-t-elle ?

Voici une chronologie de l’application de cette modification :

| Délai d’exécution | Modifier |
|-----------|--------|
| Début mai | Vous commencerez à voir une bannière de notification sur votre page de connexion existante. Vous en saurez alors plus et pourrez vous préparer aux modifications. |
| Mi-mai   | La bannière de notification change pour inclure un lien vers un aperçu de l’interface utilisateur mise à jour. Dans la préversion, vous pouvez toujours utiliser un lien pour revenir à l’expérience existante. |
| Mi-juin  | L’interface utilisateur de connexion mise à jour est disponible, et tous les utilisateurs sont orientés par défaut vers la nouvelle expérience d’interface utilisateur. |
