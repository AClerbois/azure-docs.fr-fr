## <a name="speech-recognition-with-a-customized-model"></a>Reconnaissance vocale avec un modèle personnalisé

[Custom Speech Service (CRIS)](https://www.cris.ai/) vous permet de personnaliser le moteur de reconnaissance vocale Microsoft pour votre application.
L’extrait de code suivant vous montre comment utiliser la reconnaissance vocale à partir d’un microphone en utilisant votre modèle CRIS. Renseignez votre clé d’abonnement CRIS et l’identification de votre propre déploiement avant d’exécuter le code.
