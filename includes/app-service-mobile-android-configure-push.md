1. Dans le [portail Azure](https://portal.azure.com/), sélectionnez **Parcourir tout** > **App Services**. Puis sélectionnez votre serveur principal Mobile Apps. 
2. Sous **Paramètres**, sélectionnez **Notifications push**. Ensuite, sélectionnez **Configurer les services de notification Push**.
2. Accédez à **Google (GCM)**. Entrez la clé serveur héritée FCM obtenue à partir de la console Firebase, puis sélectionnez **Enregistrer**.

Votre service est désormais configuré pour fonctionner avec Firebase Cloud Messaging.
