## <a name="enable-event-grid-resource-provider"></a>Activer le fournisseur de ressources Event Grid

Si vous n’avez jamais utilisé Event Grid dans votre abonnement Azure, vous devrez peut-être inscrire le fournisseur de ressources Event Grid.

Dans le portail Azure :

1. Sélectionnez **Abonnements**.
1. Sélectionnez l’abonnement que vous utilisez pour Event Grid.
1. Sous **Paramètres**, sélectionnez **Fournisseurs de ressources**.
1. Recherchez **Microsoft.EventGrid**.
1. Si l’inscription n’est pas encore faite, sélectionnez **S’inscrire**. 

L’inscription peut prendre un certain temps. Sélectionnez **Actualiser** pour mettre à jour l’état. Lorsque l’**état** est **Inscrit**, vous êtes prêt à continuer.
