#<a name="virtual-machine-scale-sets-documentation"></a>Documentation sur les groupes de machines virtuelles identiques

Virtual Machine Scale Sets est une ressource de calcul Azure qui vous permet de déployer et gérer un ensemble de machines virtuelles identiques.  Toutes les machines virtuelles étant configurées de la même façon, les groupes de machines virtuelles identiques sont conçus pour prendre en charge une véritable mise à l’échelle automatique (aucun pré-approvisionnement de machine virtuelle n’est nécessaire). Ceci facilite la création de services à grande échelle pour le Big Compute, le Big Data et les charges de travail en conteneurs.
