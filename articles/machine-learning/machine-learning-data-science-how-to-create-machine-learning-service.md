# Création de solutions d’analyse avancées dans Azure

Le guide ADAPT (Advanced Analytics Process and Technology) présente les tâches de science de données généralement impliquées dans la création et le déploiement de modèles prédictifs avec Azure Machine Learning. Il décrit comment la plateforme Azure permet des tâches classiques comme la réception de données issues de différentes sources, leur préparation en vue de leur exploitation dans Azure Machine Learning et l’utilisation d’Azure Machine Learning pour créer des modèles opérationnels exploitables par les applications des utilisateurs (par programmation ou autre). Si le diagramme détaille les principales étapes mises en œuvre dans un exercice type de science des données, certaines ne sont pas obligatoires et leur ordre peut varier selon l’emplacement, la taille et la complexité des données.

[AZURE.INCLUDE [machine-learning-free-trial](../../includes/machine-learning-free-trial.md)]

L'étape intitulée **Traitement des données avancé dans Microsoft Azure** permet d’accéder à un deuxième guide qui décrit le processus de préparation des données à utiliser dans une expérimentation d’Azure Machine Learning. Si, depuis ce guide, vous revenez en arrière, continuez à partir de l’étape suivante dans ce guide d’intégration des données traitées dans le studio Azure Machine Learning jusqu’à l’étape finale de la consommation du modèle.
