# <a name="visualize-your-graph-data-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>Visualiser vos données de graphe dans l’API Gremlin d’Azure Cosmos DB avec des solutions de visualisations de données

Les solutions suivantes sont recommandées par la [Communauté Apache Tinkerpop](http://tinkerpop.apache.org/#poweredby) pour la visualisation de graph Gremlin.

## <a name="linkurious-enterprise"></a>Linkurious Enterprise

![Logo de Linkurious](./media/graph-visualization/linkurious-logo.jpg)

Linkurious Enterprise utilise une technologie de graph et la visualisation de données pour transformer des jeux de données complexes en réseaux visuels interactifs. La plateforme se connecte à vos sources de données et permet à des enquêteurs de naviguer en toute transparence parmi des milliards d’entités et de relations. Elle vous offre la possibilité de détecter les relations suspectes sans jongler avec des tables ou des requêtes.

L’interface de visualisation interactive de Linkurious Enterprise offre un moyen facile d’examiner les données connectées complexes. Recherchez des entités spécifiques, développez les connexions pour découvrir les relations masquées et appliquez les dispositions de votre choix pour démêler les réseaux complexes. Linkurious est désormais compatible avec l’API Gremlin d’Azure Cosmos DB. Elle est adaptée aux scénarios de visualisation de graphique de bout en bout et prend en charge la lecture et l’écriture de fonctionnalités à partir de l’interface utilisateur. 

![Démonstration de Linkurious](./media/graph-visualization/linkurious-demo.gif)

**Étapes suivantes :** Vous pouvez demander une [démonstration de Linkurious avec Azure Cosmos DB](https://linkurio.us/contact/).

## <a name="cambridge-intelligence"></a>Cambridge Intelligence

![Logo de Cambridge Intelligence](./media/graph-visualization/ci-logo.png)


Les kits de ressources pour la visualisation de graphique de Cambridge Intelligence prennent désormais en charge Azure Cosmos DB :
- [KeyLines pour développeurs JavaScript](https://cambridge-intelligence.com/keylines/) 
- [Re-Graph pour développeurs React](https://cambridge-intelligence.com/regraph/)

![Démonstration de Cambridge Intelligence](./media/graph-visualization/ci-demo-2.gif)

Les kits de ressources vous permettent de concevoir la visualisation de graphique à performances élevées et des applications d’analyse pour votre cas d’usage. Ils exploitent un rendu WebGL puissant et un code élaboré avec soin pour offrir aux utilisateurs une expérience de visualisation rapide et pertinente.

Ils sont compatibles avec n’importe quel navigateur, périphérique, serveur ou base de données et sont fournis avec des didacticiels pas à pas, des API entièrement documentées et des démonstrations interactives.

![Démonstration de Cambridge Intelligence](./media/graph-visualization/ci-demo-1.gif)
