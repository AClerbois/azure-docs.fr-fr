# <a name="cloud-solution-providers"></a>Fournisseurs de solutions cloud

Les offres de logiciels peuvent maintenant être diffusées auprès de millions de clients Microsoft qualifiés grâce aux partenaires CSP (Cloud Solution Provider) avec un minimum d’investissement dans la vente et le marketing.  Les éditeurs peuvent désormais bénéficier des canaux des partenaires CSP, en plus de la disponibilité publique des offres via les [vitrines web de Microsoft](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace).

Les éditeurs peuvent proposer une offre via le canal partenaire CSP, qu’il s’agisse d’une nouvelle offre ou d’une offre existante. Cette option permet aux partenaires CSP de vendre vos produits et de proposer des solutions groupées pour les clients.

Les éditeurs sont chargés de fournir le support de garantie de réparation pour les clients finaux et pour fournir un mécanisme pour les partenaires CSP et/ou les clients de vous contacter pour la prise en charge. Il est recommandé de fournir aux partenaires CSP la documentation utilisateur, des formations et les notifications d’état ou de défaillance du service (le cas échéant), afin que les partenaires CSP soient en mesure de traiter les demandes de support de niveau 1 des clients.

Le tableau suivant présente les types d’offres et les modèles de tarification qui peuvent participer au canal partenaire CSP :

| **Type d’offre**    | **BYOL**  |  **Gratuit** | **PAYG**   |
| :---------------- | :---------|:----------|:-----------|
| Machines virtuelles  | OUI | OUI | OUI |
| Containers | N/A | OUI | N/A |
| Modèles de solution | N/A | OUI | N/A |
| Applications gérées | N/A | OUI | OUI |
| Solution informatique SaaS (tarification par site) | N/A | OUI | OUI |
|   |   |   |

## <a name="how-to-opt-in"></a>Comment participer

Le paramètre d’acceptation du canal de partenaire CSP se trouve dans l’onglet Place de marché ou Storefront au sein de l’expérience de création du Portail Cloud Partner.
