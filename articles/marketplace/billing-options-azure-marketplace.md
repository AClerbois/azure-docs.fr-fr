# <a name="billing-options-in-the-azure-marketplace"></a>Options de facturation sur la Place de marché Azure

Cet article décrit les options de facturation disponibles sur la [Place de marché Azure](https://azuremarketplace.microsoft.com).

## <a name="commercial-considerations-in-the-marketplace"></a>Considérations d’ordre commercial concernant la Place de marché
La Place de marché ne partage pas les revenus pour les types de liste suivants : 
*   Liste
*   Essai
*   Transact suivant le modèle de facturation BYOL (apportez votre propre licence)

La participation aux vitrines de la Place de marché n’implique pas de frais supplémentaires.

Pour plus d’informations, consultez les [Politiques concernant la participation à la Place de marché Microsoft Azure](https://azure.microsoft.com/support/legal/marketplace/participation-policies).  

## <a name="pay-as-you-go-and-byol-billing-options"></a>Options de facturation pour le paiement à l’utilisation et BYOL
Quand vous utilisez le modèle de facturation Paiement à l’utilisation comme option de publication, les revenus générés par les licences logicielles basées sur l’utilisation sont répartis selon un rapport 80 %/20 % entre vous et Microsoft. Vous pouvez facturer une même offre selon les deux modèles de facturation (Paiement à l’utilisation et BYOL). Ils coexistent au niveau de l’offre en tant que références SKU distinctes. Pour configurer les modèles de facturation de votre offre, rendez-vous sur le Portail Cloud Partner. 

Voici quelques exemples :
*   Si vous activez l’option Paiement à l’utilisation, vous obtiendrez le résultat suivant :
    | Coût de votre licence | 1 $/heure |
    |:--- |:--- |
    | Coût d’utilisation Azure (D1/1 cœur) | 0,14 $/heure |
    | *Microsoft facture au client le montant suivant :* | *1,14 $/heure* |

    Dans ce scénario, Microsoft facture 1,14 $ l’heure d’utilisation de votre image de machine virtuelle publiée :
    | Microsoft facture | 1,14 $/heure |
    |:--- |:--- |
    | Microsoft vous verse 80 % des revenus générés par les licences | 0,80 $/heure |
    | Microsoft conserve 20 % des revenus générés par les licences | 0,20 $/heure |
    | Microsoft conserve les revenus générés par l’utilisation Azure | 0,14 $/heure |

*   Si vous activez l’option BYOL, vous obtiendrez le résultat suivant :
    | Coût de votre licence | Frais de licence négociés et facturés par vos soins |
    |:--- |:--- |
    | Coût d’utilisation Azure (D1/1 cœur) | 0,14 $/heure |
    | *Microsoft facture au client le montant suivant :* | *0,14 $/heure* |

    Dans ce scénario, Microsoft facture 0,14 $ l’heure d’utilisation de votre image de machine virtuelle publiée : 
    | Microsoft facture | 0,14 $/heure |
    |:--- |:--- |
    | Microsoft conserve les revenus générés par l’utilisation Azure | 0,14 $/heure |
    | Microsoft conserve 0 % des revenus générés par les licences | 0 $/heure |

## <a name="single-billing-and-payment-methods"></a>Facturation unique et modes de paiement
L’option de publication Transact de la Place de marché Azure présente un avantage important : vos coûts de licence et l’utilisation d’Azure sont directement facturés à votre client en une seule fois.

Dans ce scénario, Microsoft gère la facturation et les paiements à votre place. La facturation Microsoft vous évite d’avoir à créer votre propre relation d’approvisionnement avec votre client. Elle peut vous faire gagner du temps et économiser des ressources. Elle peut également vous aider à vous concentrer sur la vente plutôt que sur la collecte des paiements. 

## <a name="enterprise-agreement"></a>Contrat Entreprise  
Si vous êtes un client Contrat Entreprise Microsoft, vous pouvez utiliser votre Contrat Entreprise pour payer les produits Microsoft. Certains produits, notamment votre utilisation d’Azure, sont susceptibles de vous être facturés. L’option de paiement Contrat Entreprise est destinée aux entreprises désireuses d’acheter des licences de logiciels et de services cloud pour au moins trois ans. Vous échelonnez les paiements au lieu de n’effectuer qu’un seul paiement initial. Si vous utilisez l’option de publication Paiement à l’utilisation, la facturation de vos frais de licence de logiciels suit le cycle trimestriel de facturation de dépassement du Contrat Entreprise.  

### <a name="monetary-commitment"></a>Engagement financier
Si vous êtes un client Contrat Entreprise, vous pouvez ajouter Azure à votre contrat. Pour cela, vous pouvez régler d’avance un engagement financier portant sur Azure. Il courra tout au long de l’année. Votre engagement inclut toute combinaison d’utilisation des services Azure.
