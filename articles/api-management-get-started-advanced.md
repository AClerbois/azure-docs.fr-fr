# Prise en main de la configuration avancée du service Azure API Management

Les didacticiels de cette section fournissent les consignes pour la configuration et les tâches propres au service Gestion des API.

-	[Personnalisation du portail des développeurs dans Gestion des API Azure][] - Ce didacticiel vous montre comment changer l'apparence générale du portail des développeurs pour la cohérence de votre marque.
-	[Utilisation de l'inspecteur d'API pour le suivi des appels dans Gestion des API Azure][] - Ce didacticiel vous montre comment utiliser l'inspecteur d'API pour effectuer le suivi et la collecte des informations concernant les appels d'opération.
-	[Création et configuration de paramètres de produit avancés dans Gestion des API Azure][] - Ce didacticiel présente d'une part comment configurer certains paramètres de produit avancés, tels que les stratégies de quota et de limite de débit, et d'autre part une démonstration de l'appel d'opérations jusqu'au dépassement de la limite de débit.
-	[Délégation de l'inscription des utilisateurs et des abonnements aux produits][] - Ce didacticiel montre comment utiliser votre site web existant pour gérer les connexions/inscriptions des développeurs et l'abonnement aux produits au lieu de faire appel aux fonctionnalités intégrées du portail des développeurs.
