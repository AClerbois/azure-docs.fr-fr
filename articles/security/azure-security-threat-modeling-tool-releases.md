# <a name="threat-modeling-tool-releases"></a>Versions de Threat Modeling Tool

Threat Modeling Tool est actuellement publié comme application Windows gratuite [à télécharger](https://aka.ms/threatmodelingtool). Ce mécanisme de livraison nous permet de transmettre les dernières améliorations et correctifs de bogues pour les clients chaque fois qu’ils ouvrent l’outil, rendant ainsi la gestion et l’utilisation plus facile.

## <a name="system-requirements"></a>Configuration requise

- Systèmes d'exploitation pris en charge
  - Microsoft Windows 10
- Conditions supplémentaires
  - Une connexion Internet est nécessaire pour recevoir les mises à jour de l’outil ainsi que les modèles.
