> [!NOTE]
>
> Cet article a été récemment mis à jour pour tenir compte de l’utilisation des commandes du nouveau module Az d’Azure PowerShell. Ce module est le nouveau module PowerShell pour Azure, compatible avec les versions 5.x et 6 de PowerShell. AzureRM continuera à obtenir les mises à jour de correction critiques, mais les nouvelles fonctionnalités seront uniquement exécutées via le module Az.
>
> * Pour en savoir plus sur le module Az et l’extension de prise en charge du module AzureRM, consultez [Présentation du nouveau module Azure PowerShell Az](/powershell/azure/new-azureps-module-az).
> * Le module Az est déjà disponible sur Azure Cloud Shell. La compatibilité avec AzureRM est automatiquement activée.
>   Pour obtenir des instructions sur l’installation du module Az, consultez [Installation d’Azure PowerShell](/powershell/azure/install-az-ps).
> * Pour activer les commandes de compatibilité AzureRM, exécutez `Enable-AzureRmAlias` après l’installation.
>   Pour en savoir plus sur la compatibilité, consultez [Migrate from Azure PowerShell AzureRM to Az](/powershell/azure/migrate-from-azurerm-to-az) (Migration d’Azure PowerShell AzureRM à Az).
