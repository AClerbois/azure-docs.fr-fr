Un nouveau hub contient la passerelle, dont la création prend 30 minutes. Une fois le hub créé, vous serez facturé, même si vous n’y joignez aucun site. 
1. Sur la page WAN, sous **Virtual WAN Architecture** (Architecture WAN virtuel), cliquez sur **Hubs** pour ouvrir la page Hubs.
2. Dans la page Hubs, cliquez sur **+New Hub** (+Nouveau hub) pour ouvrir la page **Hubs**.
3. Dans la page Hubs, renseignez les champs suivants :

  * Lieu
  * NOM
  * Espace d’adressage privé du hub
4. Cliquez sur **Confirmer** pour créer le hub. Cliquez sur **Actualiser** pour afficher le hub sur la page Hubs.
