<a name="getting-started-with-seller-insights"></a>Prise en main de Seller Insights
====================================

Cet article décrit la fonctionnalité Seller Insights disponible dans le [portail Cloud Partner](https://cloudpartner.azure.com/#insights).


<a name="insights-tour"></a>Visite guidée d’Insights
-------------

Le portail Cloud Partner vous propose des informations relatives à vos clients Azure et Visual Studio et à leur utilisation. Voici une vue d’ensemble rapide des données et des informations que vous rencontrerez probablement.

<a name="top-navigation-bar"></a>Barre de navigation supérieure
------------------

Après avoir sélectionné **Insights** dans la barre de menus de gauche, vous voyez une barre de navigation dans la partie supérieure, qui vous indique les modules insights auxquels vous avez accès.

1.  **Résumé** : cet onglet présente des graphiques, des tendances et des valeurs des données les plus recherchées par les éditeurs.

2.  **Paiement** : cet onglet contient des informations sur les paiements et les transactions associées, sous forme de graphiques et de formats téléchargeables.

3.  **Commandes et utilisation** : cet onglet contient les informations sur les commandes et l’utilisation, sous forme de graphiques et de formats téléchargeables.

4.  **Client** : cet onglet contient des informations sur vos clients et leurs achats.

5.  **Déploiement** : cet onglet contient les informations de réussite et d’échec du déploiement sous forme de graphiques et des formats au niveau événement.

6.  **Téléchargements** : le téléchargement de jeux de données volumineux est plus facile et moins perturbateur avec la nouvelle expérience de téléchargement.

Vous pouvez constater que vous voyez uniquement un jeu limité parmi les modules répertoriés ci-dessus.
Seuls les utilisateurs avec les autorisations *propriétaire* peuvent voir les modules **paiement** et **client** en raison d’informations sensibles sur le client et l’entreprise. Utilisez un rôle propriétaire dans votre organisation pour modifier les autorisations si vous devez accéder à ces modules.


<a name="tips"></a>Conseils :
-----

-   ajustez les dates pour afficher les informations qui vous intéressent.
-   Téléchargez les données au niveau de la transaction pour effectuer d’autres analyses sur les informations fournies par les informations Selle Insights.
-   Si vous recherchez des informations sur le paiement ou sur le client, assurez-vous que vous êtes connecté avec un rôle de propriétaire et non un rôle de contributeur. Pour en savoir plus sur les autorisations des utilisateurs, consultez [Gérer les utilisateurs](./cloud-partner-portal-manage-users.md).
