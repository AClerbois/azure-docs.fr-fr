# <a name="azure-cosmos-db-documentation-experiment"></a>Expérience de documentation Azure Cosmos DB

Azure Cosmos DB est une base de données multimodèle distribuée à l’échelle mondiale. Azure Cosmos DB vous permettant d’effectuer de nombreuses actions, telles que créer un graphique ou une base de données de documents ou encore migrer vos données MongoDB, Cassandra ou de stockage de table Azure, l’organisation de la documentation peut être complexe. Nous avons donc besoin de votre aide.

Nous testons l’organisation de la table des matières et de la [page d’accueil](https://docs.microsoft.com/en-us/azure/cosmos-db/) Azure Cosmos DB, comme indiqué dans les images suivantes, et nous aimerions avoir votre avis sur les modifications proposées.

## <a name="to-load-both-views"></a>Pour charger les deux affichages

1. Pour ouvrir les deux affichages côte à côte, accédez à la [page d’accueil](https://docs.microsoft.com/en-us/azure/cosmos-db/) depuis votre navigateur web favori. 
2. Utilisez les images ci-dessus pour déterminer quelle version vous voyez (celle d’origine ou la version expérimentale). 
3. Ouvrez une fenêtre de navigation privée et chargez la [page d’accueil](https://docs.microsoft.com/en-us/azure/cosmos-db/) dans la nouvelle fenêtre de navigateur, puis utilisez une nouvelle fois les images pour déterminer la version que vous consultez. 

   Si les deux affichages n’apparaissent pas, fermez et ouvrez une nouvelle fenêtre de navigation privée, puis chargez à nouveau la [page d’accueil](https://docs.microsoft.com/en-us/azure/cosmos-db/). Plusieurs tentatives peuvent être nécessaires pour que les deux versions soient chargées dans des fenêtres distinctes. L’actualisation de la fenêtre privée ne chargera pas l’autre version, vous devez ouvrir une nouvelle fenêtre. 

4. Vous pouvez ensuite redimensionner les fenêtres pour les placer côte à côte, ou basculer entre les affichages pour les comparer. 

5. Après vous être familiarisé avec l’organisation et la disposition de l’expérience expérimentale, cliquez sur le bouton **Experiment survey** (Enquête sur l’expérience) pour donner votre avis. Envoyez le formulaire avant le 4 avril 2018 à minuit, pour que les avis puissent être consultés et intégrés.
