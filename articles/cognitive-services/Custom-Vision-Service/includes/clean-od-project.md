## <a name="clean-up-resources"></a>Supprimer des ressources
Si vous voulez implémenter votre projet de détection d’objets (ou essayer un projet de [classification d’images](../csharp-tutorial.md)), supprimez le projet de détection de fourchettes/ciseaux de cet exemple. Avec l’essai gratuit, vous pouvez essayer deux projets Vision personnalisée.

Sur le [site web Custom Vision](https://customvision.ai), accédez à **Projects** (Projets) puis sélectionnez la corbeille sous My New Project (Nouveau projet).
