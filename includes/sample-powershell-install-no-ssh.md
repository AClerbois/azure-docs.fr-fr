Cet exemple nécessite Azure PowerShell. Exécutez `Get-Module -ListAvailable Az` pour voir s’il est installé. Si vous devez l’installer, consultez [Installer le module Azure PowerShell](/powershell/azure/install-az-ps). 

Exécutez `Connect-AzAccount` pour créer une connexion avec Azure.
