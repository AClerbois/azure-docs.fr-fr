# <a name="cloud-partner-portal-tour"></a>Visite guidée du portail Microsoft Cloud Partner

Cet article est une introduction au [portail Microsoft Cloud Partner](https://cloudpartner.azure.com) : il présente son contenu, ses mécanismes de navigation et ses fonctionnalités.   À partir de ce portail, les partenaires inscrits peuvent créer des offres et gérer les offres existantes pour la [Place de marché Azure](https://azuremarketplace.microsoft.com) et la [Place de marché AppSource](https://azuremarketplace.microsoft.com).  Le portail Microsoft Cloud Partner est uniquement accessible aux partenaires inscrits. Pour plus d’informations, consultez [Devenir un éditeur sur la Place de marché dans le cloud](https://docs.microsoft.com/azure/marketplace/become-publisher).

## <a name="section-contents"></a>Contenu de la section

Après l’explication des mécanismes de [navigation](./cpp-portal-navigation.md) dans le portail, nous présenterons les entrées principales du volet de navigation de gauche, dans l’article correspondant à chaque entrée : 
- Le menu [Nouvelle offre](./cpp-new-offer-menu.md) permet aux utilisateurs de créer des offres du type spécifié.
- La page [Toutes les offres](./cpp-all-offers-page.md) permet aux partenaires de voir toutes les offres publiées ainsi que certaines de leurs caractéristiques principales. 
- La page [Approbations](./cpp-approvals-page.md) permet aux partenaires de voir toutes leurs offres actuellement approuvées.
- La page [Tous les éditeurs](./cpp-all-publishers-page.md) liste tous les éditeurs actuellement inscrits dans Cloud Partner.
- La page [Utilisateurs](./cpp-users-page.md) liste tous les comptes d’utilisateur actuellement associés à des partenaires inscrits dans Cloud Partner.
- La page [Historique](./cpp-history-page.md) liste les événements de publication et de modification d’offres pour tous les éditeurs. 
- La page [Insights](./cpp-insights-page.md) contient les différents tableaux de bord qui composent la fonctionnalité *Seller Insights* de ce portail.
