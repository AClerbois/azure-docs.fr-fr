# <a name="how-to-manage-your-azure-location-based-services-account-and-keys"></a>Guide pratique pour gérer votre compte et vos clés Azure Location Based Services

Vous pouvez gérer votre compte et vos clés Azure Location Based Services par le biais du portail Azure. Lorsque vous avez un compte et une clé, vous pouvez mettre en œuvre les API dans votre site web ou votre application mobile.

Si vous n’avez pas d’abonnement Azure, créez un [compte gratuit](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) avant de commencer.

## <a name="log-in-to-azure"></a>Connexion à Azure 

Connectez-vous au portail Azure sur http://portal.azure.com.

## <a name="create-a-new-account"></a>Créer un nouveau compte

1. Cliquez sur **Créer une ressource** en haut à gauche du portail Azure.

2. Recherchez et sélectionnez **Location Based Services**, puis cliquez sur **Créer**.

3. Entrez les informations du compte Location Based Services. 

![Entrer les informations du compte dans le portail](./media/how-to-manage-account-keys/new-account-portal.png)

## <a name="manage-keys-on-the-account-page"></a>Gérer les clés dans la page du compte

Dans le menu de gauche, sous Paramètres, cliquez sur **Clés**.

![Gérer les clés du compte dans le portail](./media/how-to-manage-account-keys/account-keys-portal.png)

Utilisez les clés pour vous authentifier auprès d’Azure Location Based Services et commencer à développer votre application basée sur l’emplacement.

## <a name="delete-an-account"></a>Supprimer un compte

Vous pouvez supprimer un compte en cliquant sur le bouton **Supprimer** dans le volet de navigation supérieur.

![Supprimer votre compte dans le portail](./media/how-to-manage-account-keys/account-delete-portal.png)
