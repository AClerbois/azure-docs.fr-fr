Pour obtenir une prédiction LUIS dans un chatbot ou une autre application, vous devez publier l’application. 

Vous n’êtes pas obligé de créer une clé de point de terminaison LUIS dans le portail Azure avant de publier ou de tester l’URL du point de terminaison. Chaque application LUIS possède une clé de démarrage gratuite pour la création. Vous bénéficiez d’une création illimitée et de [quelques accès de point de terminaison](../articles/cognitive-services/luis/luis-boundaries.md#key-limits). 

1. Dans le volet de navigation supérieur droit, sélectionnez **Publier**.

2. Sélectionnez l’emplacement Production et le bouton **Publier**.

3. La publication est terminée lorsque la barre d’état verte s’affiche en haut du site web, confirmant ainsi sa réussite.
