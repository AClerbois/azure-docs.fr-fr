> [!TIP]
> Dans cet article, le contenu s’applique au stockage de Table Azure d’origine. Cependant, il existe désormais une offre Premium pour le stockage de tables : l’API Table Azure Cosmos DB. Cette API offre des tables optimisées pour le débit, une distribution globale et des index secondaires automatiques. Il existe des [différences de fonctionnalités entre l’API Table d’Azure Cosmos DB et Stockage Table Azure](../articles/cosmos-db/faq.md#where-is-table-api-not-identical-with-azure-table-storage-behavior). Pour en savoir plus et pour essayer l’expérience Premium, consultez [API Table Azure Cosmos DB](https://aka.ms/premiumtables).
