# <a name="threat-modeling-tool-update-release-71510231---1112018"></a>Mise à jour de Threat Modeling Tool version 7.1.51023.1 - 01/11/2018

À l’origine indiqué dans les [Notes de publication de la disponibilité générale](./azure-security-threat-modeling-tool-releases-71509112.md), nous avons publié une mise à jour (7.1.51023.1) de Microsoft Threat Modeling Tool qui oblige les utilisateurs de la préversion (clients de la préversion avec la version < 7.1.50911.2) à mettre à niveau vers la version en disponibilité générale prise en charge. Cette version ne contient aucune nouvelle fonctionnalité ou correctif inédit.

- Les utilisateurs de la préversion téléchargent automatiquement la mise à niveau à l’ouverture du client. Si vous choisissez de ne pas installer la nouvelle mise à jour, la préversion de l’outil se ferme.
- Les utilisateurs de la version en disponibilité générale de l’outil sont invités à indiquer leur choix de mettre à niveau ou non.
- Les utilisateurs de la version 2016 de l’outil ne sont pas concernés.

## <a name="feature-changes"></a>Évolution des fonctionnalités

- Aucun

## <a name="system-requirements"></a>Conditions requises pour le système

- Systèmes d'exploitation pris en charge
  - Microsoft Windows 10
- Version .NET obligatoire
  - .NET 3.5.2
- Conditions supplémentaires
  - Une connexion Internet est nécessaire pour recevoir les mises à jour de l’outil ainsi que les modèles.
