Pour que votre application logique puisse accéder à tout service, vous devez créer une *connexion* entre votre application logique et ce service. Si vous n’avez pas créé cette connexion, vous êtes invité à fournir des informations de connexion lorsque vous ajoutez un déclencheur ou une action pour ce service vers votre application logique. Le Concepteur d’applications logiques vous permet de créer facilement cette connexion, directement à partir de votre application logique.
