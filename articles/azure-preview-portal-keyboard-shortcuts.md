# Raccourcis clavier du portail Azure 

## Ouvrir les concentrateurs

| Raccourci | Action |
|--------|----------|
| N | Ouvrir les concentrateurs de notification |
| S ou / | Ouvrir le concentrateur d'exploration (services) | 
| A | Sélectionner une session active | 
| C | Créer | 

## Navigation

| Raccourci | Action |
|--------|----------|
| F | Accéder au premier panneau ouvert | 
| H | Basculer entre le portail d'accueil Azure et le panneau actuel | 
| J | Accéder au précédent panneau ouvert | 
| K | Accéder au panneau ouvert suivant | 
| L | Accéder au dernier panneau ouvert | 
| - | Réduire le panneau ou annuler l'agrandissement | 
| + | Agrandir le panneau ou annuler la réduction | 
| K | Accéder au panneau ouvert suivant | 


## Commandes
| Raccourci | Action |
|--------|----------|
| Échap | Fermer un menu de concentrateur, une boîte de dialogue, un menu de compte, ou n'importe quel menu ouvert | 
| # | Fermer tous les panneaux ouverts | 
| Accueil | Faites défiler le panneau actuel vers le haut | 
| Fin | Faites défiler le panneau actuel vers le bas | 
| CTRL+M | Pour basculer via tabulation entre les commandes et l'utilisation des onglets dans une commande d'édition |
