# <a name="update-your-own-password"></a>Mettre à jour votre mot de passe
Si vous ne savez pas comment gérer votre mot de passe de compte professionnel ou scolaire, vous êtes au bon endroit. Découvrez comment vous inscrire pour demander la réinitialisation du mot de passe, modifier votre mot de passe, réinitialiser votre mot de passe et déverrouiller votre compte.

## <a name="dont-lose-access-to-your-account"></a>Ne perdez pas l’accès à votre compte.

> [!IMPORTANT]
> **Pourquoi est-ce que je reçois ce message ?** Si vous avez suivi un lien jusqu’ici, vous voyez probablement ce message parce que votre administrateur exige que vous demandiez une réinitialisation du mot de passe pour pouvoir accéder à votre application. Il se peut que vous soyez invité à fournir votre numéro de téléphone ou votre adresse de messagerie, ou encore à définir des questions de sécurité. Ne vous inquiétez pas. Nous n’utilisons ces informations que pour renforcer la sécurité de votre compte. Les étapes présentées ici devraient vous aider à atteindre votre objectif.
>

Le moyen le plus rapide de vous inscrire pour demander une réinitialisation du mot de passe est de vous rendre sur la page [de réinitialisation du mot de passe](http://aka.ms/ssprsetup).  

1. Accédez à la [page d’inscription à la réinitialisation de mot de passe](http://aka.ms/ssprsetup).
2. Entrez votre nom d'utilisateur et votre mot de passe.
3. Cliquez sur **Le configurer maintenant** pour choisir une option d’inscription. L’exemple suivant montre comment effectuer l’inscription à l’aide de l’option **Téléphone d’authentification**. Il est possible que votre administrateur vous demande de vous inscrire à plusieurs options. 

   ![Options d’installation][101]
4. Sélectionnez votre pays dans la liste déroulante et entrez votre numéro de téléphone complet (indicatif inclus), puis choisissez l’option de **SMS** ou d’**appel**. Quelle que soit l’option choisie, vous recevez un code. 

   ![Vérification par téléphone][102]
5. Entrez le code que vous avez reçu, puis cliquez sur **Vérifier**. 

   ![Exemple de code de vérification][103]
6. Cliquez sur **Terminer** pour terminer l’Assistant. Vous pouvez maintenant utiliser l’option à laquelle vous vous êtes inscrit pour réinitialiser votre mot de passe à tout moment en accédant à la page [Who are you ? (Qui êtes-vous ?)](https://passwordreset.microsoftonline.com).

   ![Informations de récupération][104]

> [!IMPORTANT]
> Si votre administrateur vous permet de vous inscrire à plusieurs options, nous vous recommandons vivement de vous inscrire également à une option de sauvegarde, pour le cas où vous perdriez votre téléphone ou n’auriez plus accès à votre adresse de messagerie.
   >
   >

## <a name="change-your-password-in-office-365"></a>Modifier votre mot de passe dans Office 365
Suivez les étapes ci-dessous pour modifier le mot de passe de votre compte professionnel ou scolaire dans Office 365. Si vous avez oublié votre mot de passe et souhaitez le réinitialiser, suivez les étapes indiquées dans [Réinitialiser votre mot de passe](#how-to-reset-your-password).

1. Connectez-vous à Office 365 avec votre compte professionnel ou scolaire.
2. Accédez à **Paramètres** > **Office 365** > **Mot de passe** > **Modifier le mot de passe**.
3. Entrez votre ancien mot de passe, puis tapez un nouveau mot de passe et confirmez-le.
4. Cliquez sur **Save**.

Vous pouvez en apprendre plus sur la modification de votre mot de passe dans Office 365 dans le [Centre de documentation Office 365](https://support.office.com/article/Change-my-password-in-Office-365-for-business-d1efbaee-63a7-4c08-ab1d-71bf932bbb5d).

## <a name="change-your-password-from-the-access-panel"></a>Modifier votre mot de passe à partir du volet d’accès
Suivez les étapes ci-dessous pour modifier le mot de passe de votre compte professionnel ou scolaire à partir du [volet d’accès](https://myapps.microsoft.com). Si vous avez oublié votre mot de passe et souhaitez le réinitialiser, suivez les étapes indiquées dans [Réinitialiser votre mot de passe](#how-to-reset-your-password).

1. Connectez-vous à https://myapps.microsoft.com avec votre compte professionnel ou scolaire.
2. Cliquez sur l’onglet **Profil**. 
3. Cliquez sur **Modifier mon mot de passe**.
4. Entrez votre ancien mot de passe, puis tapez un nouveau mot de passe et confirmez-le.
5. Cliquez sur **Envoyer**.

Vous avez rencontré un problème lors de la modification de votre mot de passe ? Lisez [Problèmes courants et leurs solutions](#common-problems-and-their-solutions).

## <a name="reset-your-password"></a>Réinitialiser votre mot de passe
Suivez les étapes ci-dessous pour réinitialiser le mot de passe de votre compte professionnel ou scolaire depuis n’importe quelle page de connexion à un compte professionnel ou scolaire.

> [!IMPORTANT]
> Cette fonctionnalité est disponible uniquement si votre administrateur l’a activée. Si elle est désactivée, vous voyez un message indiquant que votre compte n’est pas activé pour cette fonctionnalité. Utilisez le lien **Contacter votre administrateur** pour demander à votre administrateur de déverrouiller votre compte.
>
> Si votre administrateur a autorisé votre compte à utiliser cette fonctionnalité, vous devez vous connecter avant de pouvoir l’utiliser. Pour ce faire, accédez à la [page d’inscription à la réinitialisation de mot de passe](http://aka.ms/ssprsetup).
>
>

1. Sur une page de connexion à un compte professionnel ou scolaire, cliquez sur le lien **Vous ne pouvez pas accéder à votre compte ?** ou accédez à la page [Who are you ? (Qui êtes-vous ?)](https://passwordreset.microsoftonline.com).

   ![Connexion à un compte][110]

2. Entrez votre identifiant de compte professionnel ou scolaire et prouvez que vous n’êtes pas un robot en passant le test CAPTCHA. 

   ![Confirmation d’identité][111]

3. Cliquez sur **Suivant**.
4. Choisissez une option pour réinitialiser votre mot de passe. Selon la façon dont votre administrateur a configuré le système, vous pouvez voir une ou plusieurs des options suivantes :

   * **Envoyer un courrier électronique sur mon adresse de messagerie secondaire** : envoie un message électronique avec un code à 6 chiffres à votre **autre adresse e-mail** ou à votre **adresse e-mail d’authentification** (au choix).
   * **Envoyer un SMS à mon téléphone mobile** : envoie un code à 6 chiffres à votre **téléphone mobile** ou à votre **adresse e-mail d’authentification** (au choix).
   * **Appeler mon téléphone mobile** : appelle votre **téléphone mobile** ou votre **téléphone d’authentification** (au choix). Appuyez sur la touche *#* pour vérifier l’appel.
   * **Appeler mon téléphone de bureau** : appelle votre **téléphone de bureau**. Appuyez sur la touche *#* pour vérifier l’appel.
   * **Répondre à mes questions de sécurité** : affiche vos questions de sécurité préenregistrées pour vous permettre de répondre.

   ![Options de réinitialisation de mot de passe][109]

5. Cet exemple montre comment utiliser l’option **Envoyer un SMS à mon téléphone mobile**. Pour l’option avec vérification par téléphone, entrez votre numéro de téléphone complet, puis cliquez sur **SMS** pour vérifier qu’il est correct. Un texte avec un code à 6 chiffres est ensuite envoyé à votre téléphone.

6. Lorsque vous recevez le SMS, veillez à utiliser le code de vérification dans le corps du message, pas le numéro à partir duquel le code a été envoyé. La réception du SMS peut prendre plusieurs minutes.

7. Entrez le code à 6 chiffres que vous avez reçu, puis cliquez sur **Suivant**.

8. Il se peut que votre administrateur demande une étape de vérification supplémentaire. Si tel est le cas, répétez l’étape 4 en sélectionnant une autre option.

9. Dans l’écran **Choisir un nouveau mot de passe**, entrez un nouveau mot de passe et confirmez votre choix, puis cliquez sur **Terminer**.

   ![Confirmation du nouveau mot de passe][107]

10. Une fois que votre mot de passe est accepté, vous pouvez vous connecter avec le nouveau mot de passe.

    ![Confirmation de la réinitialisation du mot de passe][108]

Vous avez rencontré un problème lors de la réinitialisation de votre mot de passe ? Lisez [Problèmes courants et leurs solutions](#common-problems-and-their-solutions).

## <a name="unlock-your-account"></a>Déverrouiller votre compte
Suivez les étapes ci-dessous pour déverrouiller votre compte local depuis n'importe quelle page de connexion à un compte professionnel ou scolaire. 

> [!NOTE]
> Vous seul pouvez déverrouiller votre compte s’il a été verrouillé localement.

> [!IMPORTANT]
> Cette fonctionnalité est disponible uniquement si votre administrateur l’a activée. Si elle est désactivée, vous voyez un message indiquant que votre compte n’est pas activé pour cette fonctionnalité. Utilisez le lien **Contacter votre administrateur** pour demander à votre administrateur de déverrouiller votre compte.
>
> Si votre administrateur a autorisé votre compte à utiliser cette fonctionnalité, vous devez vous connecter avant de pouvoir l’utiliser. Pour ce faire, accédez à la [page d’inscription à la réinitialisation de mot de passe](http://aka.ms/ssprsetup).
>
>

1. Sur une page de connexion à un compte professionnel ou scolaire, cliquez sur le lien **Vous ne pouvez pas accéder à votre compte ?** ou accédez à la page [Who are you ? (Qui êtes-vous ?)](https://passwordreset.microsoftonline.com).

   ![Connexion à un compte][110]
2. Sur la page **Who are you? (Qui êtes-vous ?)**, entrez votre identifiant de compte professionnel ou scolaire et prouvez que vous n'êtes pas un robot en passant le test CAPTCHA. 
3. Cliquez sur **Suivant**.

   ![Confirmation d’identité][111]
4. Choisissez une option pour déverrouiller votre compte. Selon la façon dont votre administrateur a configuré le système, vous pouvez voir une ou plusieurs des options suivantes :

   * **Envoyer un courrier électronique sur mon adresse de messagerie secondaire** : envoie un message électronique avec un code à 6 chiffres à votre **autre adresse e-mail** ou à votre **adresse e-mail d’authentification** (au choix).
   * **Envoyer un SMS à mon téléphone mobile** : envoie un code à 6 chiffres à votre **téléphone mobile** ou à votre **adresse e-mail d’authentification** (au choix).
   * **Appeler mon téléphone mobile** : appelle votre **téléphone mobile** ou votre **téléphone d’authentification** (au choix). Appuyez sur la touche *#* pour vérifier l’appel.
   * **Appeler mon téléphone de bureau** : appelle votre **téléphone de bureau**. Appuyez sur la touche *#* pour vérifier l’appel.
   * **Répondre à mes questions de sécurité** : affiche vos questions de sécurité préenregistrées pour vous permettre de répondre.

   ![Options de vérification d’identité][112]
5. Cet exemple montre l’option **Répondre à mes questions de sécurité** sélectionnée. Fournissez les réponses à vos questions de sécurité, puis cliquez sur **Suivant** pour vérifier votre identité.

6. Il se peut que votre administrateur demande une étape de vérification supplémentaire. Si tel est le cas, répétez l’étape 4 en sélectionnant une autre option.
7. Lorsque la page de réussite s’affiche, votre compte local a été déverrouillé et vous pouvez maintenant vous connecter.

   ![Compte de connexion déverrouillé][113]

> [!IMPORTANT]
> Veillez à mettre à jour le mot de passe sur tous vos appareils. Souvent, une application non autorisée avec un mot de passe obsolète (comme votre client de messagerie de téléphone) peut être la raison pour laquelle votre compte a été verrouillé en premier lieu.
   >
   >
