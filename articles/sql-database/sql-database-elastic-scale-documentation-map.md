# Rubriques sur les outils de bases de données élastiques Azure SQL

Les fonctionnalités de base de données élastique de Base de données SQL Azure sont conçues pour simplifier le développement et la gestion des couches de données, en particulier pour les développeurs de logiciels en tant que services (SaaS) où de grandes quantités de bases de données sont utilisées pour prendre en charge une base de client final dynamique.

Les **outils de bases de données élastiques** vous permettent de développer et de gérer des applications utilisant des modèles de partitionnement pour procéder à une montée en charge dans de nombreuses bases de données. Ces outils incluent une bibliothèque cliente de base de données élastique prenant en charge la gestion du partitionnement, le routage dépendant des données et les fonctions de requête entre partitions pour ADO.Net. Ils incluent également un outil de fusion et de fractionnement qui simplifie la gestion des partitions en vous permettant de déplacer des données (via le fractionnement) d’une partition à l’autre (via la fusion). Ces outils complètent les pools de bases de données élastiques qui permettent une tarification prévisible avec des caractéristiques de performances garanties pour n’importe quel nombre de bases de données. La fonctionnalité de tâche associée vous offre un outil simple et fiable pour la mise à niveau et la gestion de chaque base de données dans le pool à l’aide de scripts T-SQL.
