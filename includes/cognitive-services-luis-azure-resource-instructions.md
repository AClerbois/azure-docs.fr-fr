Si vous découvrez Cognitive Services ou Azure, créez une [clé temporaire](https://azure.microsoft.com/try/cognitive-services/). Si vous avez un compte Azure, créez une ressource sur le portail. Les requêtes de point de terminaison Language Understanding prennent en charge deux types de ressource Azure : **Language Understanding** ou **Cognitive Service**. Vous en avez besoin quand vous [affectez](../articles/cognitive-services/luis/luis-how-to-azure-subscription.md) la clé à l’application.
