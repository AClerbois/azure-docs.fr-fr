# <a name="view-and-update-an-azure-sql-database-server-and-its-settings"></a>Affichage et mise à jour d’un serveur Azure SQL Database et ses paramètres

Vous pouvez afficher et mettre à jour un serveur Azure SQL Database et ses paramètres à l’aide du [portail Azure]((sql-database-manage-portal.md), [PowerShell](sql-database-manage-powershell.md), l’API REST ou C#. 

## <a name="view-and-update-an-azure-sql-server-settings-using-the-azure-portal"></a>Affichage et mise à jour des paramètres d’un serveur Azure SQL à l’aide du portail Azure
1. Ouvrez le panneau **Serveurs SQL** dans le [portail Azure](https://portal.azure.com/). 

    ![new sql server](./media/sql-database-get-started/new-sql-server.png)

2. Cliquez sur le serveur que vous souhaitez utiliser, puis cliquez sur le paramètre souhaité dans le panneau Serveurs SQL. 

    ![sql server blade](./media/sql-database-get-started/sql-server-blade.png)

> [!TIP]
> Pour un didacticiel de prise en main avec le portail Azure, consultez [Prise en main des serveurs Azure SQL Database, des bases de données et des règles de pare-feu à l’aide du portail Azure et de SQL Server Management Studio](sql-database-get-started.md).
>
