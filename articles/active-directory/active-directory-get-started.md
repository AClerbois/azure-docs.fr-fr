# Prise en main d’Azure AD

[Présentation d’Azure AD](active-directory-whatis.md)<br> [Comment cela fonctionne-t-il ?](active-directory-works.md)<br> [Étapes suivantes](active-directory-next-steps.md)<br> [En savoir plus](active-directory-learn-map.md)


Vous êtes prêt à commencer ? Azure AD est disponible pour tous, gratuitement. Inscrivez-vous maintenant pour un abonnement d'évaluation Azure : votre répertoire vous attend. L'annuaire est à vous, même si vous n'achetez rien. [Essayez Azure AD maintenant](http://azure.microsoft.com/trial/get-started-active-directory/)

## Comment dois-je choisir le répertoire à utiliser pour le test, pour l’organisation, pour la production ?

Si vous avez un abonnement existant pour des services cloud de Microsoft tels que Office 365, vous disposez déjà d'un répertoire dans Azure AD que vous déjà pouvez utiliser. Ce répertoire déjà authentifie déjà les utilisateurs qui se connectent à Office 365. Il vous suffit de vous inscrire à Azure à l’aide du même compte professionnel ou scolaire qui vous utilisez pour vous connecter à Office 365. Vous pouvez ensuite gérer le répertoire dans le Portail de gestion Azure et ajouter des répertoires pour les tests et ainsi de suite, en fonction de vos besoins. [En savoir plus sur la gestion des répertoires avec Office 365 dans Azure](active-directory-how-subscriptions-associated-directory.md#manage-the-directory-for-your-office-365-subscription-in-azure).

Si vous n'avez pas encore un abonnement pour un service cloud Microsoft, un nouveau répertoire vous sera fourni pour votre travail ou votre école lorsque vous vous inscrivez à Azure. Vous pouvez donc l'utiliser comme répertoire principal.


## Comment puis-je ajouter mon propre nom de domaine enregistré ?
Vous souhaitez ajouter un nom de domaine personnalisé ou utiliser votre propre nom de domaine enregistré ? [En savoir plus sur l’ajout d’un domaine personnalisé](active-directory-add-domain.md).

## Comment puis-je ajouter ma société ?
Avec Azure AD Premium, vous pouvez ajouter le logo de votre société et d’autres éléments d’image de marque sur la page de connexion d’Azure et les pages du panneau d’accès. [Ajout de votre société à vos pages de connexion et du panneau d’accès](active-directory-add-company-branding.md)

## Comment puis-je intégrer Azure AD à un autre répertoire ?
Vous pouvez configurer la synchronisation de répertoires entre un répertoire local et Azure AD, y compris la synchronisation de mots de passe, et activer l'authentification unique pour l'accès aux applications locales et dans le cloud. [En savoir plus sur l'intégration de répertoire](active-directory-aadconnect.md).

## Comment puis-je créer et modifier des utilisateurs ?
Vous devez créer un compte pour chaque utilisateur qui accédera à un service cloud Microsoft. Par défaut, les utilisateurs ne reçoivent pas d'autorisations d’administrateur, mais vous pouvez leur en affecter. [En savoir plus sur la création d’utilisateurs Azure AD](active-directory-create-users.md).

## Affichage de vos rapports d’accès et d’utilisation
Azure AD fournit plusieurs rapports pour les administrateurs. Des rapports supplémentaires sont disponibles dans les éditions payantes d'Azure AD. Il existe également une API rapports pour la création de vos propres rapports personnalisés. [En savoir plus sur les rapports Azure AD](active-directory-view-access-usage-reports.md).
