# Premiers pas avec Azure AD Privileged Identity Management

## Démarrage de l’extension Privileged Identity Management

1. Créez un répertoire dans Azure Active Directory.
2. Ajoutez des utilisateurs à votre nouveau répertoire, y compris quelques administrateurs généraux.

    >[AZURE.NOTE] Assurez-vous de configurer des utilisateurs avec leurs comptes professionnels ou scolaires. Les comptes Microsoft ne peuvent pas faire l’objet d’une inscription à Azure AD Privileged Identity Management pour l’instant.

3. Connectez-vous au [portail Azure](https://portal.azure.com/) en tant qu’administrateur général de ce répertoire.
4. Cliquez sur votre nom d’utilisateur dans le coin supérieur droit du portail Azure.
5. Sélectionnez l’annuaire que vous venez de créer.
6. Cliquez sur l’icône **Nouveau** dans le volet de navigation à gauche.
7. Dans le menu Créer, sélectionnez **Sécurité + Identité**.
8. Sélectionnez **Azure AD Privileged Identity Management**.
9. Laissez la case à cocher **Épingler au tableau de bord** activée, puis cliquez sur le bouton **Créer**. Le tableau de bord Privileged Identity Management s’ouvre.
