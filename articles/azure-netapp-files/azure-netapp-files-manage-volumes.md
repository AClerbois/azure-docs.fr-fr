# <a name="manage-volumes-by-using-azure-netapp-files"></a>Gérer les volumes avec Azure NetApp Files
Vous pouvez gérer un volume en visualisant, en mettant à jour ou en supprimant le volume. 

Si vous voulez supprimer un volume, vous devez d’abord supprimer tous ses sous-objets. 

1.  Cliquez sur le panneau **Volumes** à partir du panneau Gérer les pools de capacités. 
2.  Utilisez la vue **Parcourir les volumes** pour visualiser, mettre à jour ou supprimer des volumes existants.
