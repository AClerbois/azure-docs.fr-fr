### <a name="add-utterances-from-the-command-line"></a>Ajouter des énoncés à partir de la ligne de commande

Appeler add-utterance sans argument permet d’ajouter un énoncé à l’application, sans apprentissage.

```console
> python add-utterances-3-6.py
```

La réponse suivante indique que les énoncés ont été ajoutés au modèle.
