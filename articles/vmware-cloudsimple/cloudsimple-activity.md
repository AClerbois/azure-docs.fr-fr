# <a name="activity-management-overview"></a>Vue d’ensemble de la gestion des activités

CloudSimple effectue le suivi de toutes les activités qui peuvent affecter le fonctionnement de vos clouds privés. Ces activités incluent les alertes, les événements, les tâches et l’activité d’audit. Les [pages Activité](https://docs.azure.cloudsimple.com/activity/) synthétisent toutes les activités en cours et vous permettent d’accéder à des détails supplémentaires.

## <a name="events"></a>Événements

Les événements effectuent le suivi de l’activité de l’utilisateur et du système sur le portail CloudSimple.  Les événements affichent l’activité associée à une ressource spécifique et la gravité de l’impact.  Vous pouvez afficher les événements à partir du portail CloudSimple.

## <a name="alerts"></a>Alertes

Les alertes sont des notifications de toutes les activités significatives dans votre environnement CloudSimple.  Les événements, qui ont un impact sur la facturation ou l’accès utilisateur, sont affichés sous forme d’alertes.  Vous pouvez confirmer les alertes déclenchées à partir du portail CloudSimple.

## <a name="tasks"></a>Tâches

Les tâches effectuent le suivi de toute opération de l’utilisateur, qui prend plus de 30 secondes.  Vous pouvez surveiller la progression d’une tâche à partir du portail CloudSimple.  Les tâches terminées affichent le temps total nécessaire pour terminer la tâche.

## <a name="audit"></a>Audit

Les journaux d’audit effectuent le suivi des opérations effectuées par l’utilisateur.  Les journaux contiennent les paramètres fournis par l’utilisateur pour l’opération.  Vous pouvez utiliser les journaux d’audit pour surveiller l’activité de tous les utilisateurs.
