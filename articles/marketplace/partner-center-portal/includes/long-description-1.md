Fournissez une description plus longue de votre offre, jusqu’à 3 000 caractères. Les clients verront ceci dans la vue d’ensemble du référencement sur la Place de marché.

Incluez un ou plusieurs des éléments suivants dans votre description :

- La valeur et les avantages principaux de votre offre
- Catégories et/ou associations sectorielles
- Opportunités d’achats dans l’application
- Toutes les informations requises
