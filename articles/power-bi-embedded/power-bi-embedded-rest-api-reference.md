# API REST Power BI Embedded

Avec **Microsoft Power BI Embedded**, vous pouvez utiliser une API REST pour développer par programmation des applications qui intègrent des **rapports Power BI** à vos applications.

Pour en savoir plus sur l'API REST **Power BI Embedded**, consultez [Opérations REST Power BI](https://msdn.microsoft.com/library/mt711493).
