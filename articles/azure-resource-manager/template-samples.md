# <a name="azure-resource-manager-templates-for-management-features"></a>Modèles Azure Resource Manager pour les fonctionnalités de gestion

Le tableau suivant comprend des liens vers les modèles Azure Resource Manager pour les fonctionnalités fournies par Resource Manager.

| | |
|-|-|
|**Affectations de rôle**||
| [Affecter un rôle pour le groupe de ressources](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-resourcegroup)| Affecte un rôle intégré à un utilisateur pour un groupe de ressources existant. |
| [Affecter un rôle pour une machine virtuelle existante](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-virtualmachine)| Affecte un rôle intégré à un utilisateur pour une machine virtuelle existante. |

| [Affecter un rôle pour un abonnement Azure](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/subscription-role-assigment)| Affecte un rôle à un utilisateur pour un abonnement Azure. |
|**Définition de rôle**||
| [Créer une définition de rôle personnalisée](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/create-role-def)| Crée une définition de rôle dans un abonnement Azure. |
|**Verrou de ressource**||
| [Verrouiller un groupe de ressources](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/create-rg-lock-role-assignment)| Crée un groupe de ressources et lui applique un verrou **DoNotDelete**. Affecte le rôle de contributeur à un utilisateur. |
