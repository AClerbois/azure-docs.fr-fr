## <a name="set-expiration"></a>Définir une expiration

Il est également possible de configurer votre point d’ancrage pour faire expirer automatiquement à une date donnée à l’avenir. Lorsqu’un point d’ancrage expire, il sera n’est plus situé ou mis à jour. Expiration peut uniquement être définie lorsque le point d’ancrage est créé. La mise à jour d’expiration par la suite n’est pas possible. Par conséquent, vous pouvez définir sa date d’expiration avant l’enregistrement dans le cloud.
