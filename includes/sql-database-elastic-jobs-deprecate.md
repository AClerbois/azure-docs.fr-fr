> [!IMPORTANT]
> Cet article concerne la version hébergée par le client des *tâches de base de données élastique*. Les tâches de base de données élastique sont en cours de dépréciation et de remplacement par les **tâches de base de données élastique** hébergées sur Azure. Pour les nouvelles tâches, utilisez [les dernières **tâches de base de données élastique**](../articles/sql-database/elastic-jobs-overview.md). Si vous utilisez actuellement les anciennes tâches hébergées par le client, consultez [Migrer vers les nouvelles tâches de base de données élastique](../articles/sql-database/elastic-jobs-migrate.md) pour connaître les instructions et scripts de migration pour rapidement mettre à niveau vers la dernière version.
