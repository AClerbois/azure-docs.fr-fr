# <a name="export-or-delete-settings-in-the-azure-portal"></a>Exporter ou supprimer des paramètres dans le portail Azure
Vous pouvez exporter et supprimer les paramètres suivants dans le menu Paramètres du portail Azure :
* Tableaux de bord privés du portail Azure
* Paramètres utilisateur tels que les abonnements ou répertoires favoris, et le répertoire de la dernière ouverture de session
* Thèmes et personnalisations

Nous vous recommandons vivement d’exporter et de passer en revue vos paramètres avant de les supprimer. La reconstruction des tableaux de bord et le rajout de vos personnalisations peuvent prendre beaucoup de temps.

[!INCLUDE [GDPR-related guidance](../../includes/gdpr-intro-sentence.md)]

**Exporter ou supprimer vos paramètres du portail**

1. Connectez-vous au [Portail Azure](http://portal.azure.com).
2. En haut à droite du portail, sélectionnez **Paramètres**.

    ![Capture d’écran qui affiche que l’icône d’engrenage du portail Azure](media/azure-portal-export-delete-settings/azure-portal-settings-icon.png)
3. Sélectionnez **Exporter tous les paramètres** ou **Supprimer tous les paramètres et les tableaux de bord privés**.

    ![Capture d’écran qui affiche l’exportation et la suppression des paramètres du portail](media/azure-portal-export-delete-settings/azure-portal-export-delete-settings.png)

      Le tableau suivant décrit ces paramètres. 

      | Paramètre | Description |
      | --- | --- |
      | **Exporter tous les paramètres** | Crée un fichier .json qui contient vos paramètres utilisateur tels que le thème de votre tableau de bord.|
      | **Supprimer tous les paramètres et les tableaux de bord privés** | Supprime tous les liens à des tableaux de bord privés et d’autres personnalisations utilisateur que vous avez effectuées dans le tableau de bord du portail. |
