# <a name="monitor-and-manage-wunderlist-by-using-azure-logic-apps"></a>Superviser et gérer Wunderlist à l’aide d’Azure Logic Apps

Depuis le 6 mai 2020, le [service Wunderlist s’arrête](https://www.wunderlist.com/) et a demandé à ses clients de [migrer leurs données et de commencer à utiliser Microsoft To Do](https://www.wunderlist.com/blog/goodbye-from-wunderlist/). Vous pouvez utiliser le [connecteur Microsoft To Do](https://docs.microsoft.com/connectors/todo/) pour remplacer l’ancien [connecteur Wunderlist](https://docs.microsoft.com/connectors/wunderlist/).
