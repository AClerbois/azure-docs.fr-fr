## <a name="intent-recognition-from-a-file-with-events"></a>Reconnaissance de l’intention à partir d’un fichier avec événements

Le code suivant indique comment faire fonctionner la reconnaissance de l’intention de façon continue dans la langue par défaut (en-US). Le code vous permet d’accéder à des informations supplémentaires, telles que des résultats intermédiaires. L’entrée provient d’un fichier audio. Le format pris en charge est WAV à canal unique (mono)/PCM avec un taux d’échantillonnage de 16 KHz.
