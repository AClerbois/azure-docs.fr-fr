- Pour accéder au service de données boîte Edge, votre organisation doit avoir un abonnement contrat entreprise (EA) ou fournisseur de solutions Cloud (CSP). Pour plus d’informations, consultez [souscrire un abonnement Azure](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/).
- Étant donné que ce service de gestion est hébergé dans Azure, il est protégé par les fonctionnalités de sécurité Azure. Pour plus d’informations sur les fonctionnalités de sécurité fournies par Azure, accédez à la [Microsoft Azure Trust Center](https://azure.microsoft.com/support/trust-center/security/).
- Pour les opérations de gestion du Kit de développement logiciel, vous pouvez obtenir la clé de chiffrement pour votre ressource dans **propriétés de l’appareil**. Vous pouvez afficher la clé de chiffrement uniquement si vous disposez des autorisations pour l’API Graph de ressources.
