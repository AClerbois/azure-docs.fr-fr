## <a name="speech-recognition-with-a-customized-model"></a>Reconnaissance vocale avec un modèle personnalisé

[Custom Speech Service (CRIS)](https://www.cris.ai/) vous permet de personnaliser le moteur de reconnaissance vocale Microsoft pour votre application.
L’extrait de code suivant vous montre comment utiliser la reconnaissance vocale à partir d’un microphone en utilisant votre modèle CRIS. Indiquez votre clé d’abonnement et l’identification de votre propre point de terminaison avant d’exécuter le code.
