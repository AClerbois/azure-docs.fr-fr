# <a name="iot-edge-module-offer-publishing-overview"></a>Vue d’ensemble de la publication d’une offre de module Azure IoT Edge

Cet article décrit le processus de publication et les principaux éléments d’une offre de module Azure IoT Edge. Utilisez cet article comme point de départ pour publier votre offre dans la [Place de marché Microsoft Azure](https://azuremarketplace.microsoft.com).

## <a name="publishing-process"></a>Processus de publication

Les étapes principales pour la publication d’une offre de module Azure IoT Edge sont les suivantes :

1. Créer l’offre<br> Fournissez des informations détaillées sur l’offre. Ces informations incluent : la description de l’offre, les documents marketing, les informations de prise en charge et les spécifications des ressources.

2. Créer l’activité et les ressources techniques<br> Créer les ressources de l’activité (documents juridiques et supports marketing) et les ressources techniques de la solution associée (images de module Azure IoT Edge hébergés dans un registre Azure Container Registry.

3. Créer la référence SKU<br> Créez la ou les références SKU associées à l’offre. Chaque image que vous prévoyez de publier doit avoir une référence SKU unique.

4. Certifier et publier l’offre <br>Une fois l’offre et les ressources techniques prêtes, vous pouvez envoyer l’offre. Cette soumission commence le processus de publication. Pendant ce processus, la solution est testée, validée et certifiée, puis publiée sur la Place de marché Microsoft Azure.
