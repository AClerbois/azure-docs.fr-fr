# <a name="azure-vmware-solution-by-cloudsimple"></a>Solution Azure VMware de CloudSimple

Bienvenue dans la solution Azure VMware de CloudSimple. Découvrez comment utiliser CloudSimple et comment activer des solutions sur le service. Recherchez des articles ou parcourez la table des matières.

- Consultez la [vue d’ensemble](cloudsimple-vmware-solutions-overview.md) pour en savoir plus sur la solution Azure VMware de CloudSimple.
- Sélectionnez [Vue d’ensemble du cloud privé CloudSimple](cloudsimple-private-cloud.md) pour en savoir plus sur la solution de cloud privé CloudSimple.
- Sélectionnez [Nœuds](cloudsimple-node.md) pour en savoir plus sur les nœuds CloudSimple.
- Sélectionnez [Initialiser et approvisionner une capacité](quickstart-create-cloudsimple-service.md) pour commencer à utiliser la solution Azure VMware de CloudSimple.

| **Création de service**     | Service CloudSimple         | Configurer le réseau de périmètre du service CloudSimple                                    | Portail Azure       |
|                          | Nœud CloudSimple            | Provisionner la capacité à utiliser dans le service CloudSimple                          | Portail Azure       |
| **Consommation de service**  | Cloud privé               | Créer des environnements VMware                                                    | Portail CloudSimple |
|                          | Réseau local virtuel/sous-réseau                 | Créer des réseaux underlay pour la consommation VMware                               | Portail CloudSimple |

|                          | Connexion Azure            | Établir une connectivité à des réseaux locaux ou virtuels dans Azure         | Portail CloudSimple |
| **Gestion unifiée**   | Machine virtuelle CloudSimple | Gérer des machines virtuelles VMware dans le portail Azure                                | Portail Azure       |
