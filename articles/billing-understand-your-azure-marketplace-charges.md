# Présentation de vos frais Azure Marketplace
Cet article vous permet de comprendre les frais associés à vos commandes Marketplace. Les éléments à mémoriser tout en passant en revue votre facture sont les différents types de tarification (De base uniquement, Usage uniquement et De base + Usage). Notez que chaque commande Marketplace passée possède son propre cycle de facturation mensuel. Ces sujets sont traités en détail ci-dessous.

## Passer en revue votre facture Marketplace
Une fois connecté au [portail de comptes Azure](https://account.windowsazure.com/subscriptions/), vous pouvez passer en revue et télécharger les factures passées et actuelles.

Pour afficher ou télécharger votre facture :

1. Connectez-vous au portail de comptes Azure en utilisant votre ID d’organisation.
2. Cliquez sur **Marketplace**. La liste affichée est la liste complète de toutes les commandes Marketplace qui ont été passées par votre compte Azure. Vous remarquerez que les commandes sont regroupées par nom et ID d’abonnement Azure parent.
![Commandes Marketplace](./media/billing-understand-your-azure-marketplace-charges/marketplace-orders.png)
3. Cliquez sur l’une des commandes affichées pour accéder à la présentation résumée du cycle de facturation actuel de cette commande. Vous trouverez une explication des différents modèles de tarification sur le [FAQ sur Marketplace](https://azure.microsoft.com/marketplace/faq/).
![Résumé de la commande](./media/billing-understand-your-azure-marketplace-charges/order-summary.png)

La vue récapitulative contient les informations suivantes pour chaque commande :
- Prix
- Cycle de facturation
- Date d’achat
- Abonnement parent associé
- Devise de facturation

Notez que nous avons apporté quelques changements à l’affichage de ces informations. Jusqu’à maintenant, nous indiquions le coût TTC de la commande. Désormais, le prix peut être hors taxe ou TTC en fonction du pays dans lequel vous vous trouvez. L’emplacement du prix a également été déplacé du volet de navigation droit vers le centre. Pour suivre l’abonnement Azure auquel ces frais sont associés, nous avons également ajouté le nom et l’ID d’abonnement Azure dans le volet de navigation droit, ainsi que l’option **Modifier le mode de paiement**.

## Mettre à jour le mode de paiement et la gestion des commandes
La page de résumé comprend des actions de l’utilisateur, ce qui vous permet de mettre à jour les fonctions de gestion des commandes et de modèle de paiement :

> [AZURE.NOTE] Si vous utilisez votre ID d’organisation pour modifier vos informations personnelles, vous devrez enregistrer un ticket auprès du support.

Pour mettre à jour votre mode de paiement, cliquez sur le lien **Modifier le mode de paiement** sur le côté droit de la page.
![Résumé de la commande](./media/billing-understand-your-azure-marketplace-charges/order-summary.png)

Ce lien vous dirige vers un autre portail où vous pouvez apporter des modifications à votre mode de paiement favori.

Pour changer votre mode de paiement, procédez comme suit :

1. Cliquez sur **Changer votre mode de paiement**. ![Abonnements](./media/billing-understand-your-azure-marketplace-charges/subscriptions.jpg)
2. Sélectionnez le mode de paiement que vous souhaitez adopter. L’option **Payer avec** vous permet de sélectionner votre carte de crédit. L’option **Ajouter un nouveau moyen de paiement** vous permet d’ajouter une nouvelle carte de crédit. ![Modifier le mode de paiement](./media/billing-understand-your-azure-marketplace-charges/change-payment-method.jpg)

## Gestion des commandes
La page de synthèse contient un lien **Gérer les commandes**. Ce lien vous dirige vers votre portail où vous pouvez consulter l’ensemble de vos commandes Marketplace, afficher l’état du service et demander de l’assistance. ![Portail](./media/billing-understand-your-azure-marketplace-charges/portal.jpg)

Pour demander de l’assistance, cliquez sur la vignette **Aide + support**. La page **Aide + support** suivante s’affiche dans laquelle vous pouvez effectuer les opérations suivantes :
- Ouvrir une nouvelle demande de support.
- Gérer les demandes de support existantes.
- Gérer l’intégrité des ressources.

![Demande d'assistance](./media/billing-understand-your-azure-marketplace-charges/request-support.jpg)

## Historique de facturation
Une nouvelle fonctionnalité dans la page de résumé permet d’afficher les offres basées sur l’utilisation, par exemple les machines virtuelles. Vous pouvez désormais télécharger l’utilisation en fonction du cycle de facturation pour les périodes actuelles et clôturées. Pour voir l’utilisation, cliquez sur le lien **Télécharger l’utilisation** pour la période que vous souhaitez afficher.

Si vous souhaitez afficher vos frais réels, cliquez sur **Afficher les frais**. Vous êtes redirigé vers un autre portail où vous pouvez voir tous les frais, notamment les taxes. Si vous utilisez un ID d’organisation, le bouton **Afficher les frais** n’est pas activé, et vous devez envoyer un ticket de support pour demander un résumé de vos frais. ![Historique de facturation](./media/billing-understand-your-azure-marketplace-charges/billing-history.png)

Pour afficher les frais réels, procédez comme suit :

1. Cliquez sur **Afficher les frais** dans la page **HISTORIQUE DE FACTURATION**. Un nouvel onglet s’ouvre et affiche l’[historique des commandes](https://account.microsoft.com/billing/orders#/). Notez que si vous essayez d’accéder à votre utilisation brute dans le portail de facturation et de gestion de comptes, vous devrez contacter le support en cliquant sur le lien correspondant au bas de la page.) ![Historique des commandes](./media/billing-understand-your-azure-marketplace-charges/order-history.jpg)
2. Identifiez la commande que vous souhaitez passer en revue, puis cliquez sur **Détails** pour afficher la répartition de vos frais : sous-total, taxe et total. ![Détails de la commande](./media/billing-understand-your-azure-marketplace-charges/order-details.jpg)
