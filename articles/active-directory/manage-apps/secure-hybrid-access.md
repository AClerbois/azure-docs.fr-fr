# <a name="secure-hybrid-access-secure-legacy-apps-with-app-delivery-controllers-and-networks"></a>Sécuriser l’accès hybride : sécuriser les applications héritées avec les réseaux et les contrôleurs de livraison d’applications

Vous pouvez désormais protéger vos applications d’authentification locales et dans le Cloud héritées en les connectant à Azure AD avec votre contrôleur ou votre réseau de livraison d’application. De cette façon, vous pouvez mener à bien vos transitions et renforcer la sécurité de l’ensemble de vos applications grâce à des fonctionnalités de Azure AD telles que Azure AD Conditional Access et Azure AD Identity Protection.

Votre contrôleur de livraison et de mise en réseau vous permet de protéger facilement vos applications héritées, qui sont toujours critiques pour vos processus d’entreprise, mais que vous ne pouviez pas protéger avant avec Azure AD. Il est probable que vous disposiez déjà de tout ce dont vous avez besoin pour commencer à protéger ces applications.
