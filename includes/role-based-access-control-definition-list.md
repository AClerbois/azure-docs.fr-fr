Le [contrôle d’accès en fonction du rôle Azure(RBAC)](../articles/role-based-access-control/overview.md) est le système d’autorisation que vous utilisez pour gérer l’accès aux ressources Azure. Pour déterminer les ressources auxquelles les utilisateurs, les groupes, les principaux de service et les identités managées ont accès, vous devez lister les rôles qui leur sont attribués.
