# <a name="create-a-dynamics-365-for-customer-engagement-application-offer"></a>Créer une offre d’application Dynamics 365 for Customer Engagement

Cette section explique comment créer et publier une offre d’application Dynamics 365 for Customer Engagement pour la Place de marché AppSource. Toutes les offres apparaissent sous la forme d’entités distinctes dans la Place de marché. Elles se composent des regroupements de ressources et des services de support suivants : détails de l’offre, informations techniques sur le package, artefacts marketing, informations de support et spécifications facultatives de la version Test Drive. 


## <a name="new-offer-form"></a>Formulaire de nouvelle offre

Connectez-vous au [Portail Cloud Partner](https://cloudpartner.azure.com/), puis sélectionnez **+ Nouvelle offre** dans la barre de menus de gauche. Dans le menu Nouvelle offre, sélectionnez **Dynamics 365 for Customer Engagement** :

![Élément de menu Nouvelle offre de l’application Dynamics 365 for Customer Engagement](./media/new-dyn365ce-menu-item.png)

Le formulaire **Nouvelle offre** est affiché, montrant l’onglet [**Paramètres de l’offre** initial](./cpp-offer-settings-tab.md).
