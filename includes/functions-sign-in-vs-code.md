## <a name="sign-in-to-azure"></a>Connexion à Azure

Avant de pouvoir publier votre application, vous devez vous connecter à Azure.

1. Dans la zone **Azure : Fonctions**, choisissez **Se connecter à Azure...** . Si vous ne l’avez pas encore fait, vous pouvez **Créer un compte Azure gratuit**.

    ![Réponse de la fonction localhost dans le navigateur](./media/functions-sign-in-vs-code/functions-sign-into-azure.png)

1. Lorsque vous y êtes invité, sélectionnez **Copier et ouvrir**, ou copiez le code affiché et ouvrez <https://aka.ms/devicelogin> dans votre navigateur.

1. Collez le code copié dans la page **Connexion à l’appareil**, vérifiez la connexion à Visual Studio Code, puis sélectionnez **Continuer**.  

1. Finalisez la connexion à l’aide des informations d’identification de votre compte Azure. Une fois connecté, vous pouvez fermer le navigateur.
