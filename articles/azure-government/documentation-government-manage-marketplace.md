# <a name="azure-marketplace-for-government"></a>Place de marché Azure pour Azure Government
La Place de marché Azure est disponible pour Azure Government avec une liste actualisée d’images de nos éditeurs. 

## <a name="variations"></a>Variantes
Voici quelques aspects à prendre en considération lors de l’utilisation de la Place de marché Azure pour Azure Government :

* Seules des images BYOL (avec apport de sa propre licence) sont disponibles. Vous ne pouvez accéder à aucune image nécessitant une transaction d’achat sur la Place de marché Azure.
* Seul un sous-ensemble d’images est actuellement disponible en comparaison avec la Place de marché publique. La liste des images disponibles figure [ici](../azure-government-image-gallery.md). 
* Avant le déploiement d’une image, votre administrateur d’entreprise doit activer les achats sur la Place de marché pour votre abonnement Azure.
  * Connectez-vous au portail en tant qu’administrateur d’entreprise.
  * Accédez à *Gérer*.
  * Sous *Détails de l’inscription*, cliquez sur l’icône crayon en regard de la ligne *Place de marché Azure*.
  * *Activez/Désactivez* selon le besoin.
  * Cliquez sur *Enregistrer*.

> [!NOTE]
> Si vous souhaitez rendre vos images disponibles dans Azure Government, pour plus d’informations, voir [instructions d’intégration des partenaires](documentation-government-manage-marketplace-partners.md).
>
