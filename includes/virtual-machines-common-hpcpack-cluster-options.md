Créez et gérez un cluster de calcul haute performance (HPC) basé sur le cloud en tirant parti de [Microsoft HPC Pack](https://technet.microsoft.com/library/jj899572.aspx) et des services d’infrastructure et de calcul d’Azure. HPC Pack est disponible en téléchargement gratuit, est basé sur les technologies Microsoft Azure et Windows Server et prend en charge un vaste éventail de charges de travail HPC.

Pour obtenir plus d’options HPC dans Azure, consultez la page [Ressources techniques pour Batch et HPC (calcul hautes performances)](../articles/batch/big-compute-resources.md).
