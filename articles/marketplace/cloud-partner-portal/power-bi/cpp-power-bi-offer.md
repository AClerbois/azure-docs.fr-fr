# <a name="power-bi-app-offer"></a>Offre d’application Power BI

Le Portail Cloud Partner ne prend plus en charge la création et la gestion des offres d’application Power BI. Cette fonctionnalité a été déplacée vers l’[Espace partenaires](https://partner.microsoft.com/pcv/) de Microsoft. Pour plus d’informations, consultez [Créer une application Power BI pour Microsoft AppSource](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-power-bi-app-offer).
