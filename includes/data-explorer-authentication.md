## <a name="authentication"></a>Authentication

Pour exécuter l’exemple suivant, vous avez besoin d’une application Azure AD et d’un principal de service ayant accès aux ressources. Consultez [Créer une application Azure AD](/azure/active-directory/develop/howto-create-service-principal-portal) pour créer une application Azure AD gratuite et ajouter une attribution de rôle au niveau de l’abonnement. Vous avez également besoin des éléments suivants : `directory (tenant) ID`, `application ID` et `client secret`.
