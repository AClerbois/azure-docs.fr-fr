# <a name="power-bi-app-offer-prerequisites"></a>Application Power BI offre des conditions préalables

Cet article fournit des techniques et les besoins dont vous avez besoin pour répondre aux avant de pouvoir publier une application Power BI offrent à la place de marché AppSource.  Si vous ne le n'avez pas déjà fait, passez en revue la [Office 365, Dynamics 365, PowerApps et Power BI offrent Guide de publication](../../appsource-offer-publishing-guide.md).


## <a name="technical-requirements"></a>Exigences techniques

La ressource technique principale, vous aurez besoin pour ce type d’offre est un [application Power BI](https://go.microsoft.com/fwlink/?linkid=2028636). Pour plus d’informations, consultez [Créer des ressources techniques d’application Power BI](./cpp-create-technical-assets.md).

Une fois que votre application de service est créée et testée dans Power BI, vous devez enregistrer l’URL d’installation d’application Power BI génère. Vous en aurez besoin pour [créer une offre d’application Power BI](./cpp-create-offer.md).

## <a name="business-requirements"></a>Exigences commerciales

Exigences de l’entreprise incluent des obligations procédurales, contractuelles et légales.  Vous devez respecter les consignes suivantes :

- Être un serveur de publication de la place de marché cloud inscrit. Si vous n’êtes pas inscrit, suivez les étapes de [devenir un éditeur de place de marché Cloud](../../become-publisher.md).

    > [!NOTE] 
    > Utiliser le même compte de l’inscription de Microsoft Developer Center pour vous connecter à la [portail Microsoft Cloud Partner](https://cloudpartner.azure.com). Vous ne devez utiliser qu’un seul compte Microsoft pour vos offres AppSource. Les comptes ne doivent pas être spécifiques à des services ou offres.
 
- Consentez tous les efforts raisonnables pour mettre à la disposition de vos clients un support technique.
- Fournir du contenu qui répond aux critères de votre offre soit répertoriée sur AppSource. Pour plus d’informations, voir [Vous avez une application à répertorier sur AppSource ? Voici comment procéder](https://appsource.microsoft.com/blogs/have-an-app-to-list-on-appsource-here-s-how).
- Respecter la [déclaration de confidentialité Microsoft](https://privacy.microsoft.com/privacystatement).


## <a name="next-steps"></a>Étapes suivantes

Une fois que vous avez rempli toutes les conditions requises, vous êtes prêt à [créer une offre d’application Power BI](./cpp-create-offer.md) dans le portail Cloud Partner.
