Cet exemple nécessite Azure PowerShell. Exécutez `Get-Module -ListAvailable Az` pour trouver la version. Si vous devez installer ou mettre à niveau, consultez [Installer le module Azure PowerShell](/powershell/azure/install-az-ps). 

Exécutez `Connect-AzAccount` pour créer une connexion avec Azure.
