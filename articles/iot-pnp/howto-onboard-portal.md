# <a name="onboard-to-the-azure-certified-for-iot-portal"></a>Intégrer au portail Azure Certified pour IoT

Vous pouvez utiliser le [portail Azure Certified pour IoT](https://aka.ms/ACFI) :

- Terminez le processus de certification pour votre appareil IoT.
- Recherchez des modèles de fonctionnalité de l’appareil mobile IoT Plug-and-Play (préversion). Vous pouvez utiliser ces modèles pour créer rapidement des appareils prêts pour l’IoT et les intégrer à des solutions.
- Publiez un modèle de fonctionnalité de l’appareil dans le référentiel public.

## <a name="sign-in"></a>Se connecter

Utilisez votre _compte professionnel_ Microsoft pour vous connecter au [portail Azure Certified pour IOT](https://aka.ms/ACFI). Le portail vérifie que vous êtes membre de l’Espace partenaires Microsoft lorsque vous vous connectez :

- Si votre organisation est déjà membre de l’[Espace partenaires Microsoft](https://docs.microsoft.com/partner-center/), vous avez accès à toutes les fonctions du portail.
- Si votre organisation n’est pas membre de l’[Espace partenaires Microsoft](https://docs.microsoft.com/partner-center/), vous pouvez exécuter des tests de certification et parcourir les référentiels de modèles de capacités des appareils. Vous ne pouvez pas publier votre appareil dans le catalogue d’appareils certifiés IoT.
- Si vous voyez un message de notification indiquant que votre ID Microsoft Partner Network est manquant, vous devez [créer un compte d’Espace partenaires](https://docs.microsoft.com/partner-center/mpn-create-a-partner-center-account). Une fois votre compte approuvé, vous pouvez revenir au portail et publier votre appareil. Pour plus d’informations, consultez le [FAQ de l’Espace partenaires](https://support.microsoft.com/help/4340639/partner-center-account-faqs).

Avant de publier votre appareil sur le catalogue d’appareils certifiés IoT, lisez et acceptez le contrat du programme. Vous trouverez le contrat sur la page Paramètres du portail Azure Certified pour IoT.
