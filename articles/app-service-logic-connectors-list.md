# Liste des connecteurs et applications API dans Microsoft Azure App Service

Le tableau suivant répertorie tous les connecteurs et toutes les applications API disponibles créées par Microsoft, dans l'ordre alphabétique : 

	Nom  | Description
------------- | ------------- 
Connecteur AS2 | Le connecteur AS2 peut recevoir et envoyer des messages via le protocole de transport AS2 dans les communications entreprise-entreprise. Les données sont transportées de manière fiable et sécurisée sur Internet à l'aide de certificats numériques et du chiffrement.<br/><br/>[Connecteurs B2B](app-service-logic-b2b-connectors.md)
Azure HDInsight | Azure HDInsight déploie et approvisionne des clusters Apache Hadoop dans le cloud, et fournit une infrastructure logicielle conçue pour gérer et analyser les données volumineuses, et générer des rapports à leur sujet. Ce connecteur crée un cluster Hadoop sur Azure et soumet différents travaux Hadoop, notamment Hive, Pig, MapReduce et Streaming MapReduce. À l'aide de ce connecteur, vous pouvez faire tourner un cluster, soumettre un travail et attendre que la tâche soit terminée.<br/><br/>[Connecteurs de services de données et d'applications](app-service-logic-data-connectors.md)
Azure Mobile Services | Azure Mobile Services réunit un ensemble de services Azure qui dotent vos applications mobiles de fonctionnalités principales. À l'aide de ce connecteur, vous pouvez vous connecter à Azure Mobile Services et créer, mettre à jour, obtenir et supprimer des entrées, ainsi qu'effectuer des appels d'API personnalisées sur une table. <br/><br/>[Connecteurs de services de données et d'applications](app-service-logic-data-connectors.md)
Azure Service Bus | Azure Service Bus est un système de messagerie générique, basé sur le cloud qui permet de pratiquement tout connecter. Il permet l'échange de messages d'une façon faiblement couplée pour une meilleure évolutivité et une meilleure fiabilité. À l'aide de ce connecteur, vous pouvez envoyer des messages à partir d'entités Service Bus comme les files d'attente et les rubriques et recevoir des messages d'entités Service Bus comme les files d'attente et les abonnements.<br/><br/>[Connecteurs de services de données et d'applications](app-service-logic-data-connectors.md)
Azure Storage Blob | Azure Storage Blob stocke de gros volumes de données non structurées, telles que du texte ou des données binaires, accessibles depuis n'importe où dans le monde. À l'aide de ce connecteur, vous pouvez vous connecter au stockage d'objets blob et télécharger, obtenir, supprimer, copier un objet blob, en répertorier dans un conteneur, en créer un instantané et utiliser un déclencheur pour en récupérer.<br/><br/>[Connecteurs de services de données et d'applications](app-service-logic-data-connectors.md)
Azure Storage Table | Le service Azure Storage Table stocke de gros volumes de données structurées. Le service est un magasin de données NoSQL idéal pour stocker des données non relationnelles structurées. Ce connecteur se connecte à un service Azure Storage Table et peut obtenir une entité, interroger des entités, insérer une entité, mettre à jour une entité, supprimer une entité et utiliser un déclencheur pour récupérer des données.<br/><br/>[Connecteurs de services de données et d'applications](app-service-logic-data-connectors.md)
BizTalk EDIFACT | L'application API EDIFACT reçoit et envoie des messages à l'aide du protocole EDIFACT dans les communications entreprise-entreprise.<br/><br/>[Connecteurs B2B](app-service-logic-b2b-connectors.md)
BizTalk X12 | L'application API X12 reçoit et envoie des messages à l'aide du protocole X12 dans les communications entreprise-entreprise.<br/><br/>[Connecteurs B2B](app-service-logic-b2b-connectors.md)
Encodeur de fichier plat BizTalk | Fournit une interopérabilité entre des données de fichier plat (comme Excel et csv) et des données XML. Cette application API peut convertir une instance de fichier plat au format XML et vice versa.<br/><br/>[Connecteurs d'intégration BizTalk](app-service-logic-integration-connectors.md)
Encodeur JSON BizTalk | Fournit une interopérabilité entre des données JSON et XML. Cette application API peut convertir une instance JSON au format XML et vice versa.<br/><br/>[Connecteurs d'intégration BizTalk](app-service-logic-integration-connectors.md)
Règles BizTalk | Les règles BizTalk définissent et contrôlent la structure, le fonctionnement et la stratégie d'une organisation. Les stratégies d'entreprise peuvent être mises à jour sans recompiler et redéployer les applications associées.<br/><br/>[Connecteurs d'intégration BizTalk](app-service-logic-integration-connectors.md)
Gestion des partenaires commerciaux BizTalk | L'application API TPM (Trading Partner Management) de gestion des partenaires commerciaux définit et rend persistantes les relations entreprise-entreprise à l'aide de partenaires, contrats, schémas et certificats utilisés dans les contrats. Ces relations sont mises en œuvre à l'aide des applications API AS2, EDIFACT et X12.<br/><br/>[Connecteurs B2B](app-service-logic-b2b-connectors.md)
Service BizTalk Transform | L'application API BizTalk Transform convertit des données d'un format à l'autre. Vous pouvez également télécharger un mappage existant (fichier .trfm), afficher la représentation graphique du mappage qui affiche les liens entre les schémas source et cible, et utiliser une fonctionnalité de test avec un exemple de contenu XML d'entrée. Différentes fonctions intégrées sont également disponibles, notamment les manipulations de chaînes, l'affectation conditionnelle, les expressions arithmétiques, les formateurs de date et d'heure, l'itération, etc.<br/><br/>[Connecteurs d'intégration BizTalk](app-service-logic-integration-connectors.md)
Validateur XML BizTalk | L'application API Validateur valide les données XML par rapport aux schémas XML prédéfinis. Vous pouvez utiliser les schémas existants ou en générer en fonction d'une instance de fichier plat, d'une instance JSON ou de connecteurs existants.<br/><br/>[Connecteurs d'intégration BizTalk](app-service-logic-integration-connectors.md)
Extracteur XPath BizTalk | L'application API Extracteur XPath recherche et extrait des données du contenu XML en fonction d'une expression XPath donnée.<br/><br/>[Connecteurs d'intégration BizTalk](app-service-logic-integration-connectors.md)
Box | Box est un service de partage de fichiers. Le connecteur Box se connecte à Box et permet de télécharger, d'obtenir, de supprimer un fichier, de répertorier des fichiers et utilise un déclencheur pour récupérer des fichiers.<br/><br/>[Connecteurs de services de données et d'applications](app-service-logic-data-connectors.md)
Chatter | Chatter est un réseau social d'entreprise. Le connecteur Chatter se connecte à Chatter et permet de publier un message, de rechercher et utiliser un déclencheur pour récupérer les nouveaux messages.<br/><br/>[Connecteurs de réseaux sociaux](app-service-logic-social-connectors.md)
DropBox | DropBox est un service d'hébergement de fichiers. À l'aide de ce connecteur, vous pouvez vous connecter à DropBox et télécharger, obtenir, supprimer un fichier, répertorier des fichiers et utiliser un déclencheur pour récupérer des fichiers.<br/><br/>[Connecteurs de services de données et d'applications](app-service-logic-data-connectors.md)
Facebook | Facebook est un service de réseau social. Le connecteur Facebook se connecte à Facebook et permet de publier un message, une vidéo, des photos, des offres et ainsi de suite. Vous pouvez également récupérer des messages, commentaires, événements, flux d'utilisateur, informations utilisateur sur les différents " J'aime ", comme les livres, jeux, films et ainsi de suite. Ce connecteur inclut également un déclencheur pour récupérer les nouveaux messages publiés sur une page.<br/><br/>[Connecteurs de réseaux sociaux](app-service-logic-social-connectors.md)
File  | Le connecteur File permet aux clients d'envoyer ou de télécharger des fichiers vers un système de fichiers ou un réseau, et de recevoir ou télécharger des fichiers à partir d'un système de fichiers ou un réseau. À l'aide de ce connecteur, vous pouvez vous connecter au serveur de fichiers local et télécharger, obtenir, supprimer un fichier, répertorier des fichiers et utiliser un déclencheur pour récupérer des fichiers.<br/><br/>[Connecteurs de protocoles](app-service-logic-protocol-connectors.md)
FTP<br/>FTPS | Le protocole FTP (File Transfer Protocol) est un protocole réseau couramment utilisé pour transférer des fichiers d'un hôte à l'autre. Le connecteur FTP permet de télécharger, obtenir, supprimer un fichier, répertorier des fichiers et également d'utiliser un déclencheur pour récupérer des fichiers.<br/><br/>[Connecteurs de protocoles](app-service-logic-protocol-connectors.md)
HTTP<br/>HTTPS | L'écouteur HTTP ouvre un point de terminaison qui agit comme serveur HTTP et écoute les requêtes HTTP entrantes.<br/><br/>[Connecteurs de protocoles](app-service-logic-protocol-connectors.md)
HTTP<br/>HTTPS | Le connecteur HTTP envoie des données aux serveurs HTTP à l'aide du protocole HTTP ou HTTPS.<br/><br/>[Connecteurs de protocoles](app-service-logic-protocol-connectors.md)
Marketo | Marketo développe des logiciels d'automatisation marketing qui offrent des services de marketing entrant, marketing social, CRM et d'autres services associés. Le connecteur Marketo se connecte à Marketo et permet de créer/mettre à jour des prospects, d'obtenir les modifications de prospect, de planifier une campagne, de demander une campagne, d'obtenir des prospects, d'obtenir des campagnes/répertorier des informations, d'ajouter des prospects à la liste et d'en supprimer de la liste.<br/><br/>[Connecteurs d'entreprise](app-service-logic-enterprise-connectors.md)
Microsoft Office 365 | Le connecteur Office 365 permet d'envoyer et recevoir des courriers électroniques, de gérer votre calendrier et de gérer vos contacts à l'aide de votre compte Office 365. Vous pouvez également envoyer, recevoir et obtenir des messages électroniques, créer et supprimer des événements dans votre calendrier, ainsi que créer, mettre à jour, obtenir et supprimer vos contacts.<br/><br/>[Connecteurs de services de données et d'applications](app-service-logic-data-connectors.md)
Microsoft OneDrive | Microsoft OneDrive est un service d'hébergement de fichiers. Le connecteur OneDrive  se connecte à votre Microsoft OneDrive personnel et permet de télécharger, obtenir, supprimer un fichier, de répertorier des fichiers et d'utiliser un déclencheur pour récupérer des fichiers.<br/><br/>[Connecteurs de services de données et d'applications](app-service-logic-data-connectors.md)
Microsoft SharePoint | Le connecteur Microsoft SharePoint se connecte au serveur Microsoft SharePoint Server local ou à SharePoint Online et permet de gérer des documents et éléments de liste. Vous pouvez également créer, mettre à jour, obtenir et supprimer des documents et éléments de liste. Différentes méthodes d'authentification comme les informations d'identification par défaut, OAuth 2.0, l'authentification Windows et l'authentification par formulaire sont prises en charge. <br/><br/>[Connecteurs de services de données et d'applications](app-service-logic-data-connectors.md)
Microsoft SQL Server | Le connecteur Microsoft SQL se connecte au serveur SQL Server local ou à la base de données SQL Azure. Vous pouvez créer, mettre à jour, obtenir et supprimer des entrées dans une table de base de données SQL.<br/><br/>[Connecteurs de services de données et d'applications](app-service-logic-data-connectors.md)
Microsoft Yammer | Microsoft Yammer est un réseau social d'entreprise. Le connecteur Yammer se connecte à Yammer, inclut une action de publication de message et un déclencheur pour récupérer les nouveaux messages.<br/><br/>[Connecteurs de réseaux sociaux](app-service-logic-social-connectors.md)
MongoDB | Le connecteur MongoDB se connecte à la base de données du serveur Mongo (cloud ou locale) et permet de créer, mettre à jour, obtenir et supprimer des entrées dans une collection MongoDB.<br/><br/>[Connecteurs d'entreprise](app-service-logic-enterprise-connectors.md)
Oracle Database | Le connecteur Oracle Database se connecte à la base de données Oracle locale et  permet de créer, mettre à jour, obtenir et supprimer des entrées dans une table de base de données. <br/><br/>[Connecteurs de services de données et d'applications](app-service-logic-data-connectors.md)
POP3<br/>IMAP | POP3 (Post Office Protocol) est le protocole utilisé par un client de messagerie pour récupérer du courrier électronique auprès d'un serveur de messagerie. Le connecteur POP3 se connecte à un serveur POP3 et inclut un déclencheur pour récupérer les messages électroniques avec des pièces jointes.<br/><br/>[Connecteurs de protocoles](app-service-logic-protocol-connectors.md)
Quickbooks | À l'aide du connecteur QuickBooks, vous pouvez créer, mettre à jour, lire, supprimer et interroger différentes entités d'Intuit QuickBooks, comme des clients, articles, factures, etc.<br/><br/>[Connecteurs d'entreprise](app-service-logic-enterprise-connectors.md)
Salesforce | Le connecteur Salesforce gère différentes entités, comme des comptes, des prospects, des opportunités, des cas, etc. dans votre compte Salesforce. Vous pouvez également créer, mettre à jour, insérer, interroger et supprimer diverses entités.<br/><br/>[Connecteurs d'entreprise](app-service-logic-enterprise-connectors.md)
SAP | Le connecteur SAP se connecte à un serveur SAP local et appelle des RFC, BAPI et TRFC, et permet d'envoyer des IDOC.<br/><br/>[Connecteurs d'entreprise](app-service-logic-enterprise-connectors.md)
SFTP | SFTP (SSH File Transfer Protocol) est un protocole commun utilisé pour transférer des fichiers de manière sécurisée. Le connecteur SFTP permet de télécharger, obtenir, supprimer un fichier, répertorier des fichiers et également d'utiliser un déclencheur pour récupérer des fichiers.<br/><br/>[Connecteurs de protocoles](app-service-logic-protocol-connectors.md)
SMTP | SMTP (Simple Mail Transfer Protocol) est le protocole utilisé entre un client de messagerie et un serveur de messagerie pour envoyer des messages. Le connecteur SMTP se connecte à un serveur SMTP et permet d'envoyer du courrier électronique avec des pièces jointes. <br/><br/>[Connecteurs de protocoles](app-service-logic-protocol-connectors.md)
SugarCRM | Le connecteur SugarCRM se connecte à SugarCRM Online et permet de créer, mettre à jour, lire et supprimer différents types de modules comme des comptes, des contacts, des produits et ainsi de suite.<br/><br/>[Connecteurs d'entreprise](app-service-logic-enterprise-connectors.md)
Twilio | Twilio est un service SaaS orienté communication. Le connecteur Twilio se connecte à Twilio et permet d'envoyer, obtenir, répertorier des messages, de répertorier l'utilisation, d'obtenir des numéros verts disponibles, d'obtenir des numéros de portables disponibles, d'obtenir des numéros locaux disponibles, de répertorier les numéros de téléphone entrants, d'obtenir le numéro de téléphone entrant et d'ajouter un numéro de téléphone entrant.<br/><br/>[Connecteurs de réseaux sociaux](app-service-logic-social-connectors.md)
Twitter | Twitter est un service de réseau social en ligne qui permet aux utilisateurs d'envoyer et de lire des messages courts de 140 caractères. Le connecteur Twitter se connecte à Twitter et permet d'obtenir le fil d'actualité de l'utilisateur, de rechercher des tweets, d'obtenir des abonnés, d'obtenir des amis, de rechercher un utilisateur, d'obtenir le fil d'actualité de la page d'accueil, celui des mentions, de publier un tweet, de publier un tweet pour un utilisateur et d'envoyer un message direct. Le connecteur Twitter utilise également des déclencheurs pour obtenir des tweets par mot clé, par descripteur d'utilisateur et par hashtag.<br/><br/>[Connecteurs de réseaux sociaux](app-service-logic-social-connectors.md)
