# Azure Mobile Engagement - Guides de dépannage

## Introduction
Les guides de résolution des problèmes suivants couvrent la plupart des problèmes courants avec Azure Mobile Engagement, à l’exception des questions liées à la [facturation][Link 11] et de l’assistance pour se connecter à l’[aperçu][Link 7]. Si vous ne parvenez pas à résoudre votre problème à l’aide de ces guides de résolution des problèmes, vous pouvez ouvrir une demande de service Azure - si vous avez un [contrat de Support Azure](http://azure.microsoft.com/support/options/) - ou demander de l'aide sur le [Forum MSDN][Link 8]. Veillez à inclure toutes les informations de résolution des problèmes des <a href="#SR" title="SR">informations de la demande de service</a> lorsque vous ouvrez une demande de service afin d’aider le support technique à étudier votre problème.
