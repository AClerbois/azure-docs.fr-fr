# Création de votre première solution de recherche à l'aide du service Azure Search

## Bientôt disponible

Nous mettons actuellement la touche finale à un exemple d'application illustrant les fonctionnalités d'Azure Search. En attendant, nous vous remercions de votre patience. Cette application devrait normalement être disponible dans quelques jours. Vous pourrez alors télécharger l'exemple de code et suivre les instructions pour l'exécuter dans votre service Azure Search.

Vous pouvez cependant exécuter quelques étapes dès maintenant. Vous pouvez, d'ores et déjà, télécharger quelques éléments et configurer le service :

-   Visual Studio 2013 Express pour le web
-   Configuration de la version préliminaire d'Azure Search

### Téléchargement de Visual Studio 2013 Express pour le web

L'exemple d'application est créé dans Visual Studio 2013. Si vous possédez déjà une édition existante de Visual Studio 2013, vous devez l'utiliser. Dans le cas contraire, vous pouvez télécharger la version gratuite sur le site de Microsoft.

Accédez au [Centre de téléchargement](http://www.microsoft.com/en-us/download/details.aspx?id=40747) pour télécharger Visual Studio 2013 Express pour le web.

### Configuration du service de recherche gratuit

Il est conseillé d'utiliser le service gratuit pour les besoins de ce didacticiel. Il s'agit d'une version limitée du service Search, mise gratuitement à la disposition de tous les abonnés. Si vous n'avez pas d'abonnement, vous pouvez vous enregistrer pour une version d'évaluation.

Pour plus d'informations sur la configuration du service, rendez-vous sur la page [Configuration d'Azure Search dans la version préliminaire du portail Azure](../search-configure/).

Les instructions comportent plus d'étapes que nécessaire. Vous pouvez vous arrêter dès que le service gratuit a été créé. Pour effectuer ce didacticiel, vous aurez besoin d'une URL de service et d'une clé API.
