# <a name="azure-ad-domain-services---contact-us"></a>Services de domaine Azure AD : nous contacter

## <a name="feedback"></a>Commentaires
Si vous avez des commentaires pour notre service ou des questions non techniques, vous pouvez envoyer un e-mail au groupe dédié aux produits du service du domaine Azure AD, à l’adresse suivante : [Commentaires sur Azure AD Domain Services](mailto:aaddsfb@microsoft.com). Les e-mails envoyés à cette adresse sont toujours consultés. Les membres du groupe dédié aux produits **peuvent** vous répondre qu’ils ont besoin d’informations supplémentaires. Par contre, il n’y aura pas de réponse aux questions d’ordre technique.

## <a name="technical-assistance"></a>Support technique
Si vous rencontrez des problèmes avec votre domaine géré, vérifiez si les étapes décrites dans le [guide de dépannage](troubleshoot.md) apportent une solution. Si vous rencontrez toujours des problèmes, connectez-vous au portail Microsoft Azure. Ouvrez la page d’accueil **Azure AD Domain Services** et sélectionnez **Nouvelle demande de support** dans le volet d’action sous la section relative à la **prise en charge et à la résolution des problèmes**.

![nouvelle demande de support](./media/contact-us/supportRequest.png) 

Remplissez chaque section de la demande de support aussi précisément que possible. Lors du choix d’un service, veillez à sélectionner **Azure Active Directory Domain Services (VM - Domain Controllers)** (Azure Active Directory Domain Services (VM - contrôleurs de domaine)) pour acheminer votre requête au spécialiste du support adéquat.

> [!IMPORTANT]
> Sélectionnez **Azure Active Directory Directories, Domains, and Objects** (Répertoires, domaines et objets Azure Active Directory) dans les listes de services pour obtenir de l’aide lors de la validation des domaines auprès de Microsoft Azure Active Directory.
