Dans cet article, vous allez créer un service backend qui ajoute un appareil dans le registre des identités, puis ajoute un module à cet appareil. Pour effectuer ces opérations, votre service requiert l’autorisation d’**écriture du registre**. Par défaut, chaque IoT Hub est créé avec une stratégie d’accès partagé nommée **registryReadWrite** qui accorde cette autorisation.
