1.  Dans l'Explorateur de solutions de Visual Studio, cliquez avec le bouton droit sur le projet, puis cliquez sur **Gérer les packages NuGet**.

2.  Dans le Gestionnaire de package NuGet, cliquez sur **En ligne** et sur **Inclure la version préliminaire**. Entrez **Microsoft.IdentityModel.Clients.ActiveDirectory** comme terme de recherche. Cliquez ensuite sur **Installer** pour installer le package NuGet de la bibliothèque d'authentification Active Directory.
