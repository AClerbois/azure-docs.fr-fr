1. Copiez l’installation pertinente sur le serveur et lancez le programme d’installation.
2. Dans **Avant de commencer**, sélectionnez **Service Mobilité**.

    ![Service Mobilité](./media/site-recovery-install-mob-svc-gui/mobility3.png)
3. Dans la zone **Détails du serveur de configuration**, spécifiez l’adresse IP du serveur de configuration et la phrase secrète associée.

    ![Service Mobilité](./media/site-recovery-install-mob-svc-gui/mobility6.png)
4. Dans la zone **Emplacement de l’installation**, conservez les paramètres par défaut, puis cliquez sur **Suivant** pour lancer l’installation.
5. Dans la zone **Progression de l’installation**, surveillez l’installation et redémarrez la machine si vous y êtes invité. Après l’installation du service, la mise à jour de l’état sur le portail peut prendre environ 15 minutes.
