### <a name="azure-active-directory-authentication"></a>Authentification Azure Active Directory

Ancres Spatial Azure permet également aux applications pour s’authentifier auprès des jetons d’utilisateur Azure Active Directory. Par exemple, vous pouvez utiliser des jetons AAD à intégrer avec ASA. Si une entreprise gère les utilisateurs dans AAD, vous pouvez fournir un jeton d’utilisateur AAD dans le SDK d’ancres Spatial Azure. Cela vous permet de s’authentifier directement auprès du service ASA pour un compte qui fait partie du même locataire AAD.
