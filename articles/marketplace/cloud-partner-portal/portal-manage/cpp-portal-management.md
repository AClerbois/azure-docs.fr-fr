# <a name="cloud-partner-portal-management"></a>Gestion du portail Cloud Partner

Cette section décrit les types d’opérations disponibles pour créer et gérer les comptes nécessaires à l’utilisation du [portail Cloud Partner](https://cloudpartner.azure.com).  Ces comptes sont nécessaires pour la [Place de marché Azure](https://azuremarketplace.microsoft.com) et [AppSource](https://appsource.microsoft.com).  


## <a name="background-information"></a>Informations contextuelles

Avant d’utiliser le portail Cloud Partner, vous devez vous familiariser avec les articles suivants :

- L’article [Stratégies et conditions de la Place de marché Azure](../../policies-terms.md) liste les différentes exigences et règles légales et techniques qui s’appliquent aux utilisateurs et éditeurs de la [Place de marché Azure](https://azuremarketplace.microsoft.com) et d’[AppSource](https://appsource.microsoft.com).
- L’article [Devenir un éditeur sur la Place de marché dans le cloud](https://docs.microsoft.com/azure/marketplace/become-publisher) fournit une vue d’ensemble de toutes les étapes nécessaires pour devenir un éditeur cloud Microsoft officiel.


## <a name="account-creation-and-management"></a>Création et gestion de comptes

Cette section contient les articles suivants qui expliquent comment gérer des comptes d’éditeur de la Place de marché.

- L’article [Prérequis pour la gestion des comptes](./cpp-manage-prerequisites.md) explique comment devenir un éditeur cloud inscrit, ce qui est impératif pour utiliser le portail Cloud Partner.
- L’article [Gérer un profil d’éditeur de la Place de marché Azure et d’AppSource](./cpp-manage-publisher-profile.md) explique comment utiliser le portail Cloud Partner pour afficher et gérer votre profil d’éditeur de la Place de marché Microsoft. 
- L’article [Gestion des utilisateurs sur le portail Cloud Partner](./cpp-manage-users.md) explique comment ajouter des utilisateurs au compte de publication cloud de votre organisation et comment gérer leur accès, afin de permettre à plusieurs personnes de travailler sur les mêmes offres.


## <a name="next-steps"></a>Étapes suivantes

Vous devez respecter les [prérequis pour la gestion des comptes](./cpp-manage-prerequisites.md) avant de continuer.  Ensuite, vous devez également suivre la [visite guidée du portail Cloud Partner](../portal-tour/cpp-portal-tour.md) pour vous familiariser avec l’interface utilisateur du portail.
