Ce guide de démarrage rapide utilise la clé de démarrage pour des raisons pratiques. La clé sert principalement à créer le modèle, mais elle permet aussi de créer une petite quantité (1000) de requêtes de point de terminaison. Quand vous êtes prêt à créer d’autres requêtes de point de terminaison dans un environnement de test, intermédiaire ou de production, créez une ressource **Language Understanding** dans le portail Azure et affectez-la à l’application LUIS dans le portail LUIS.
