# <a name="getting-started"> </a>Prise en main de Mobile Services

Cette rubrique n'est plus prise en charge. Aucun projet Windows Store autonome ne peut faire l'objet d'un téléchargement à partir du portail. Vous pouvez toutefois télécharger un projet Windows universel. Pour plus d'informations, consultez la page [Prise en main de Mobile Services](mobile-services-javascript-backend-windows-store-dotnet-get-started.md). 

Pour obtenir des instructions sur l'ajout de Mobile Services à une application Windows Store existante, consultez la page [Ajout de Mobile Services à une application existante](mobile-services-windows-store-dotnet-get-started-data.md).
