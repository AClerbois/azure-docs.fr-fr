# <a name="account-management-overview"></a>Vue d’ensemble de gestion de compte

Lorsque vous créez votre service CloudSimple, il crée un compte sur CloudSimple.  Le compte est associé à votre abonnement Azure où se trouve le service.  Tous les utilisateurs avec **propriétaire** et **contributeur** rôles dans l’abonnement ont accès au portail CloudSimple.  L’ID d’abonnement Azure et l’ID de client associé au service CloudSimple sont trouvent sur le [page comptes](https://docs.azure.cloudsimple.com/account/).

## <a name="additional-alert-emails"></a>Messages d’alerte supplémentaires

Vous pouvez configurer les identifiants de courrier électronique dans CloudSimple pour recevoir des alertes :

* Liés à votre service
* Pour le traitement automatique

## <a name="cloudsimple-operator-access"></a>Accès de l’opérateur CloudSimple

Vous pouvez contrôler l’accès au portail CloudSimple pour le personnel des opérations de service.  Personnel des opérations de service vous connecter au portail lorsque vous soumettez un ticket de support.  Opérations de service résoudra les problèmes signalés et les actions effectuées sont disponibles dans les journaux d’audit.

## <a name="users"></a>Utilisateurs

Tous les utilisateurs qui ont **propriétaire** et **contributeur** rôle dans l’abonnement a accès au portail CloudSimple.  Lorsque vous accédez au portail, l’utilisateur est créé sur le compte CloudSimple.  Vous pouvez désactiver l’accès au portail CloudSimple pour des utilisateurs spécifiques à partir de la page comptes.
