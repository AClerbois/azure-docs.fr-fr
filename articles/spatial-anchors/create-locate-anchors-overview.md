# <a name="create-and-locate-anchors-using-azure-spatial-anchors"></a>Créer et localiser des ancres à l’aide d’Azure Spatial Anchors

Azure Spatial Anchors vous permet de partager des ancres dans le monde entre différents appareils.

Ce service a été mis au point pour bien fonctionner avec l’environnement de développement de votre choix.
