> [!Note]
> Azure IoT Central introduit actuellement un nouveau modèle d'interaction entre les appareils et les modèles d'appareils. Cette nouvelle interface utilisateur sépare les appareils et les modèles d'appareils, ce qui vous permet d'interagir indépendamment avec chacun d'eux. Si vous voyez un bouton **Modèles d'appareils** dans le menu de navigation de gauche, cela signifie que vous disposez de la nouvelle interface utilisateur :
>
> ![Bouton Modèle d'appareil](media/iot-central-experimental-note/devicetemplate.png)
>
> * [Consultez la documentation de l'interface utilisateur actuelle](../articles/iot-central/overview-iot-central.md).
> * [Consultez la documentation de la nouvelle interface utilisateur](../articles/iot-central/overview-iot-central.md?toc=/azure/iot-central-experimental/toc.json&bc=/azure/iot-central-experimental/breadcrumb/toc.json).
