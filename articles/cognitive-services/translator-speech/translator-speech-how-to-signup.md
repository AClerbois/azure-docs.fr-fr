# <a name="how-to-sign-up-for-the-translator-speech-api"></a>Comment s’inscrire à l’API de traduction de conversation Translator Speech

Suivez ces étapes pour vous inscrire à l’API de traduction de conversation Translator Speech.

## <a name="1-sign-into-azure"></a>1. Se connecter à Azure

- Vous n’avez pas de compte ? [Inscrivez-vous pour obtenir un compte Microsoft Azure](http://azure.com/).
- Vous disposez déjà d’un compte ? [Connexion](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2. S’abonner à Microsoft Translator

- Une fois connecté à Azure, accédez à la section [Cognitive Services](https://portal.azure.com/#create/Microsoft.CognitiveServices)
- Sous « Type d’API », sélectionnez l’API de traduction de conversation Translator Speech.
- Dans la section Niveau tarifaire, sélectionnez le niveau tarifaire qui répond le mieux à vos besoins.
    - Chaque abonnement a un niveau gratuit. Le niveau gratuit a les mêmes fonctionnalités et fonctions que les plans payants et n’a pas de date d’expiration.
    - Vous ne pouvez avoir qu’un seul abonnement gratuit dans votre compte.
- Remplissez le reste du formulaire et sélectionnez le bouton Créer.
- Vous pouvez à présent utiliser l’abonnement.

## <a name="3-authentication-key"></a>3. Clé d’authentification

- Pour récupérer votre clé d’authentification, accédez au menu Toutes les ressources, puis cliquez sur votre abonnement.
- La valeur de la clé étant utilisée pour l’authentification, ne la divulguez pas. Vous aurez besoin de la valeur de la clé pour développer votre application.
