# <a name="new-offer-menu"></a>Menu Nouvelle offre

Le menu **Nouvelle offre** permet aux partenaires de créer des instances de types d’offre pris en charge dans le [portail Microsoft Cloud Partner](https://cloudpartner.azure.com).  La sélection de l’élément de menu **Nouvelle offre**, dans la barre du menu de navigation de gauche, affiche la liste suivante de types d’offre.  Quand vous sélectionnez un de ces types, le processus de création et de publication de l’offre démarre.

> [!NOTE]
> Comme ce menu est régulièrement mis à jour pour refléter les nouveaux types d’offre et les changements de nom d’offre, les éléments que vous voyez peuvent différer de ceux listés dans le tableau ci-dessus.


### <a name="next-steps"></a>Étapes suivantes

Si vous créez une offre, vous pouvez utiliser le tableau précédent pour accéder à la section de documentation correspondante et obtenir des conseils sur la création et la publication de votre offre.  Sinon, la visite guidée du portail continue par la présentation de la [page Toutes les offres](./cpp-all-offers-page.md).
