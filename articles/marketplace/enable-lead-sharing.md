# <a name="enable-lead-sharing"></a>Activer le partage des prospects  
Assurez-vous que la gestion des prospects est activée pour votre annonce sur la Place de marché, afin que vous receviez les coordonnées des publics suivants.

| vitrine | audience |  
|:--- |:--- |  
| AppSource | clients professionnels |  
| Place de marché Azure | clients |  

Vos prospects constituent la base de vos campagnes suivantes, des méthodes de vente utilisées par vos commerciaux de terrain et des informations concernant les performances de votre application ou offre. 

| vitrine | campaign |  
|:--- |:--- |  
| AppSource | nurture |  
| Place de marché Azure | génération dans la demande |  

Utilisez les pratiques suivantes pour vos prospects.  
*   Qualifier les prospects et évaluer les prospects comme opportunités de vente
*   Renforcer votre relation avec les prospects pour les transformer en opportunités
*   Aligner vos efforts sur la stratégie globale de votre campagne marketing 

Vos prospects présentent un intérêt client dans l’annonce suivante. 

| vitrine | annonce |  
|:--- |:--- |  
| AppSource | <table> <tr><td>Application</td></tr> <tr><td>Service de conseil</td></tr> </table> |  
| Place de marché Azure | <table> <tr><td>Offre</td></tr> <tr><td>Technology</td></tr> </table> |  

Vous devez disposer d’un processus pour effectuer le suivi de vos prospects.  

| vitrine | prospects |  
|:--- |:--- |  
| AppSource | <table> <tr><td>votre CRM</td></tr> <tr><td>CRM vendeur sur le terrain Microsoft</td></tr> </table> |  
| Place de marché Azure | <table> <tr><td>Place de marché Azure</td></tr> <tr><td>CRM vendeur sur le terrain Microsoft</td></tr> </table> |  

Vos prospects dans les vitrines sur la Place de marché diffèrent des prospects qualifiés par le commercial Microsoft, qui font partie du programme de covente.  
*   Pour plus d’informations sur la façon d’accéder au programme de contente, consultez la page Promouvoir votre entreprise avec Microsoft à l’adresse [promote-your-business-with-microsoft](./promote-your-business-with-microsoft.md).
