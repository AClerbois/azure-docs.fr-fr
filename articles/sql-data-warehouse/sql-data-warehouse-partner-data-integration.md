# <a name="sql-data-warehouse-data-integration-partners"></a>Partenaires de SQL Data Warehouse en matière d’intégration de données
Pour créer votre solution d’entrepôt de données, vous pouvez choisir parmi un large éventail d’outils performants. Cet article met en évidence les sociétés partenaires de Microsoft qui proposent des solutions d’intégration de données officielles prenant en charge Azure SQL Data Warehouse.

## <a name="data-integration-partners"></a>Partenaires d’intégration de données
| Partenaire | Description | Site web/lien produit |
| ------- | ----------- | -------------------- |
| ![Alooma][7] |**Alooma**<br> Alooma est une solution ETL qui permet aux équipes en charge des données d’intégrer, d’enrichir et de diffuser en continu les données à partir de différents silos de données vers SQL Data Warehouse en temps réel. |[Page du produit][alooma_website] |
| ![Alteryx][1] |**Alteryx**<br> Alteryx Designer fournit un flux de travail répétitif pour l’analyse de données en libre-service qui offre de nouvelles perspectives en quelques heures, contrairement à quelques semaines pour les approches traditionnelles ! Alteryx Designer aide les analystes de données en combinant la préparation des données, le mélange des données et l’analyse (prédictive, statistique et spatiale) à l’aide de la même interface utilisateur intuitive. |[Page du produit][alteryx_designer_website]<br>[Place de marché Azure][alteryx_marketplace]<br>|
| ![Attunity][2] |**Attunity (CloudBeam)**<br>Attunity CloudBeam offre une solution automatisée de chargement de données dans SQL Data Warehouse. Il simplifie le chargement par lots et la réplication incrémentielle des données provenant de nombreuses sources (SQL Server, Oracle, DB2, Sybase, MySQL et bien plus encore). |[Page du produit][attunity_cloudbeam_website]<br>[Place de marché Azure][attunity_marketplace]<br> |
| ![Denodo][3] |**Denodo**<br>Denodo offre un accès en temps réel aux données présentes sur les différentes sources de données d’une organisation. Il utilise la virtualisation des données pour connecter des données sur plusieurs sources, sans réplication. Il offre un accès étendu aux données structurées et non structurées qui résident dans les sources cloud, d’entreprise et de Big Data, aussi bien en lots qu’en temps réel.|[Page du produit][denodo_website]<br>[Place de marché Azure][denodo_marketplace]<br> |
| ![Fivetran][8] |**Fivetran**<br>Fivetran vous permet de centraliser les données provenant de différentes sources. Il comprend un pipeline de données ne nécessitant aucune maintenance ou configuration ainsi qu’une liste croissante de connecteurs intégrés pour toutes les sources de données courantes. Il suffit de cinq minutes après l’authentification auprès des sources de données et de l’entrepôt de données cible pour terminer la configuration.|[Page du produit][fivetran_website]<br> |
| ![Informatica][4] |**1. Services Informatica Cloud pour Azure**<br> Informatica Cloud offre une solution de pointe en matière de capacités de migration, d’intégration et de gestion des données en libre-service. Les clients peuvent rapidement importer et exporter de manière fiable plusieurs pétaoctets de données vers Azure à partir de plusieurs sources différentes. Les services Informatica Cloud pour Azure offrent une connectivité native, à volume élevé et haute performance, à Azure SQL Data Warehouse, SQL Database, Stockage Blob, Data Lake Store et Azure Cosmos DB. <br><br> **2. Informatica PowerCenter** PowerCenter est une plate-forme d’intégration de données reposant sur des métadonnées qui renforce et accélère les projets d’intégration de données afin de fournir des données aux entreprises plus rapidement que si elles étaient codées à la main. Il sert de base pour vos investissements en intégration des données. |**Services Informatica Cloud pour Azure**<br>[Page du produit][informatica_Cloud_Services_website]<br>[Place de marché Azure][informatica_Cloud_Services_marketplace]<br><br> **Informatica PowerCenter**<br>[Page du produit][informatica_PowerCenter_website]<br> [Place de marché Azure][informatica_PowerCenter_byol]<br>|
| ![Générateurs d’informations][5] |**Générateurs d’informations (gestion des données Omni-Gen)**<br>La plateforme de gestion des données Omni-Gen du générateur d’informations fournit des solutions pour l’intégration de données, la qualité des données et la gestion des données de référence. Il facilite l’accès, le déplacement et la fusion de toutes les données, quel que soit leur format, leur emplacement, leur volume ou leur latence.|[Page du produit][omnifocus_website]<br> [Place de marché Azure][omnigen_marketplace] |
| ![Qubole][9] |**Qubole**<br>Qubole fournit une plateforme cloud native qui permet aux utilisateurs d’effectuer des opérations d’ETL, des analyses et des charges de travail IA/ML. Il prend en charge plusieurs moteurs open source : Apache Spark, TensorFlow, Presto, Airflow, Hadoop, Hive et bien plus encore. Il fournit des outils faciles à utiliser destinés aux utilisateurs finaux qui permettent de traiter des données, notamment des outils de requête SQL, des notebooks et des tableaux de bord utilisant des moteurs open source puissants.|[Page du produit][qubole_website]<br> [Place de marché Azure][qubole_marketplace] |
| ![Segment][10] |**Segment**<br>Segment est une solution de gestion des données et d’analytique qui vous permet de mieux comprendre les données client provenant de sources différentes. Il vous permet de connecter vos données à plus de 200 outils pour créer des décisions, des produits et des expériences de meilleure qualité. Segment transformera et chargera pour vous plusieurs sources de données dans votre entrepôt à l’aide de ses connecteurs de données intégrés.|[Page du produit][segment_website]<br> |
| ![Skyvia][11] |**Skyvia (intégration de données)**<br>Skyvia (intégration de données) fournit un Assistant qui automatise les importations de données. Cet Assistant vous permet de migrer des données entre plusieurs sources, notamment des CRM, des bases de données d’application, des fichiers CSV et bien plus encore. |[Page du produit][segment_website]<br> |
| ![SnapLogic][6] |**SnapLogic**<br>La plateforme SnapLogic permet aux clients de transférer rapidement des données à destination ou en provenance de Microsoft Azure SQL Data Warehouse. Elle offre la possibilité d’intégrer des centaines d’applications, services et scénarios IoT en une seule solution.|[Page du produit][snaplogic_website]<br>[Place de marché Azure][snaplogic_marketplace]<br> |
| ![Talend][12] |**Talend Cloud**<br>Talend Cloud est une plateforme d’intégration de données professionnelle permettant de se connecter à tous les types de données, d’y accéder et de les transformer sur le cloud et en local. Il s’agit d’une offre de platform-as-a-service d’intégration (iPaaS) qui propose une connectivité large, une qualité des données intégrée et une prise en charge native des dernières technologies cloud et Big Data. |[Page du produit][talend_website]<br> [Place de marché Azure][talend_marketplace] |
| ![Trifacta][13] |**Trifacta Wrangler**<br> Trifacta permet aux individus et aux organisations d’explorer des données différentes et de les fusionner afin de les analyser. Trifacta Wrangler est conçu pour gérer les charges de travail de data wrangling qui doivent prendre en charge des données à l’échelle et un grand nombre d’utilisateurs finaux.|[Page du produit][trifacta_website]<br> [Place de marché Azure][trifacta_marketplace] |
| ![WhereScape][14] |**WhereScape RED**<br> WhereScape RED est un environnement de développement intégré (IDE) qui fournit aux équipes des outils d’automatisation dédiés à la rationalisation des flux de travail ETL. L’IDE fournit des bonnes pratiques et du code natif optimisé destinés aux cibles de données courantes. Utilisez WhereScape RED pour réduire le temps de développement, de déploiement et de gestion de votre infrastructure de données.|[Page du produit][wherescape_website]<br> [Place de marché Azure][wherescape_marketplace] |
