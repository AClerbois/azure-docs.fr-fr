## <a name="versions-of-the-azure-cli"></a>Versions d’Azure CLI

Deux versions de l’interface de ligne de commande (CLI) Azure sont actuellement disponibles :

* [Azure CLI](../articles/storage/common/storage-azure-cli.md) : La version actuelle de l’interface CLI, pour une utilisation avec le modèle de déploiement Resource Manager.
* [Azure classic CLI](../articles/storage/common/storage-azure-cli-nodejs.md): Une version antérieure de l’interface CLI Azure, destiné uniquement avec le modèle de déploiement classique.
