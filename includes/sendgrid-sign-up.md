Les clients Azure peuvent débloquer 25 000 courriers électroniques gratuits chaque mois. Ces 25 000 courriers électroniques mensuels gratuits vous donnent accès aux rapports avancés, aux analyses et à toutes les API (Web, SMTP, Event, Parse, Sub-User). Pour plus d'informations sur les services supplémentaires proposés par SendGrid, consultez la page [Fonctionnalités de SendGrid][].

### Création d'un compte SendGrid

1. Connectez-vous au [portail de gestion Azure][].

2. Dans le volet inférieur du portail de gestion, cliquez sur **Nouveau**.

	![command-bar-new][command-bar-new]

3. Cliquez sur **Magasin**.

	![sendgrid-store][store-sendgrid]

4. Dans la boîte de dialogue **Choisir un module complémentaire**, sélectionnez **SendGrid** et cliquez sur la flèche vers la droite.

5. Dans la boîte de dialogue **Personnaliser un module complémentaire**, sélectionnez le plan **SendGrid** auquel vous souhaitez vous inscrire.

6. Entrez un nom pour identifier votre service **SendGrid** dans les paramètres Azure ou utilisez la valeur par défaut **SendGrid**. Les noms doivent contenir entre 1 et 100 caractères et ne peuvent inclure que des lettres, des chiffres, des tirets, des points et des traits de soulignement. Le nom doit être unique dans la liste des éléments de l'Azure Store auxquels vous êtes abonnés.

	![store-screen-2][store-screen-2]

7. Choisissez une valeur pour la région ; par exemple, Bretagne.

8. Cliquez sur la flèche droite.

9. Sous l'onglet **Revoir achat**, vérifiez le plan et les informations tarifaires, ainsi que les conditions juridiques. Si vous acceptez les conditions juridiques, cliquez sur la coche. Une fois que vous avez coché la case, votre compte SendGrid lance le [processus d'approvisionnement de SendGrid].

	![store-screen-3][store-screen-3]

10. Après la confirmation de votre achat, vous êtes redirigé vers le tableau de bord des modules complémentaires et le message **Achat de SendGrid** s'affiche.

	![sendgrid-purchasing-message][sendgrid-purchasing-message]

	Votre compte SendGrid est immédiatement approvisionné et le message **Achat du module complémentaire SendGrid réussi** s'affiche. Votre compte et vos informations d'identification sont créés. Vous pouvez désormais envoyer des courriers électroniques. 

	Pour modifier votre plan d'abonnement ou voir les paramètres de contact de SendGrid, cliquez sur le nom du service SendGrid pour ouvrir le tableau de bord des modules complémentaires de SendGrid. 

	![sendgrid-add-on-dashboard][sendgrid-add-on-dashboard]

	Pour envoyer un courrier électronique à l'aide de SendGrid, vous devez fournir les informations d'identification de votre compte (nom d'utilisateur et mot de passe).

### Recherche de vos informations d'identification SendGrid ###

1. Cliquez sur **Informations de connexion**.

	![sendgrid-connection-info-button][sendgrid-connection-info-button]

2. Dans la boîte de dialogue *Informations de connexion*, copiez le **mot de passe** et le nom d'utilisateur pour une utilisation ultérieure dans ce didacticiel.

	![sendgrid-connection-info][sendgrid-connection-info]

	Pour définir les paramètres de remise des courriers électroniques, cliquez sur le bouton **Gérer**. L'interface Web de Sendgrid.com s'ouvre et vous pouvez vous connecter et ouvrir le panneau de configuration de SendGrid. 

	![sendgrid-control-panel][sendgrid-control-panel]

	Pour plus d'informations sur la mise en route de SendGrid, consultez la page [Prise en main de SendGrid][].
