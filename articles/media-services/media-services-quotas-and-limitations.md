# <a name="quotas-and-limitations"></a>Quotas et limitations
Cet article décrit les quotas et les limitations associés à Microsoft Azure Media Services.

## <a name="open-a-support-ticket-to-request-changes-to-the-default-quotas"></a>Ouvrir un ticket de support pour demander à modifier les quotas par défaut
Pour demander à modifier les quotas par défaut indiqués, vous pouvez ouvrir un ticket de support. Merci d’inclure des informations détaillées dans la requête sur les modifications souhaitées pour les quotas, les scénarios de cas d’usage et les régions requises.

### <a name="how-to-open-a-support-ticket"></a>Ouverture d’un ticket de support
Pour ouvrir un ticket de support, procédez comme suit :

1. Cliquez sur [Obtenir un support](https://manage.windowsazure.com/?getsupport=true). Si vous n'êtes pas connecté, vous devrez entrer vos informations d'identification.
2. Sélectionnez votre abonnement.
3. Sous le type de support, sélectionnez « Technique ».
4. Cliquez sur « Créer un ticket ».
5. Sélectionnez « Azure Media Services » dans la liste de produits affichée sur la page suivante.
6. Sélectionnez un « type de problème » approprié pour votre problème.
7. Cliquez sur Continuer.
8. Suivez les instructions de la page suivante, puis entrez les détails relatifs à votre problème.
9. Cliquez sur Envoyer pour ouvrir le ticket.
