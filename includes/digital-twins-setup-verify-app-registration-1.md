Pour vérifier si les autorisations ont été correctement configurées, suivez [ce lien](https://portal.azure.com/#blade/Microsoft_AAD_IAM/ActiveDirectoryMenuBlade/RegisteredApps) pour accéder à la page de vue d’ensemble des inscriptions d’applications Azure AD dans le portail Azure. Cette page affiche toutes les inscriptions d’applications qui ont été créées dans votre abonnement.

Vous devriez voir l’inscription d’application que vous venez de créer dans la liste de la vue d’ensemble. Sélectionnez-la pour afficher ses détails.
