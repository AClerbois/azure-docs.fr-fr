# <a name="azure-policy-built-in-policy-definitions-for-azure-managed-applications"></a>Définitions de stratégie intégrées d’Azure Policy pour Applications managées Azure

Cette page constitue un index des définitions de stratégie intégrées d’[Azure Policy](../../governance/policy/overview.md) pour Applications managées Azure. Pour obtenir des éléments intégrés supplémentaires d’Azure Policy pour d’autres services, consultez [Définitions intégrées d’Azure Policy](../../governance/policy/samples/built-in-policies.md).

Le nom de chaque définition de stratégie intégrée est un lien vers la définition de la stratégie dans le portail Azure. Utilisez le lien de la colonne **GitHub** pour voir la source dans le [dépôt GitHub Azure Policy](https://github.com/Azure/azure-policy).
