# <a name="microsoft-authenticator-app"></a>Application Microsoft Authenticator

>[!Important]
>Ce contenu est destiné aux utilisateurs. Si vous êtes un administrateur, vous trouverez plus d’informations sur la méthode à suivre pour configurer et gérer votre environnement Azure Active Directory (Azure AD) dans le [documentation relative à Azure Active Directory](https://docs.microsoft.com/en-us/azure/active-directory).

L’application Microsoft Authenticator vous permet de sécuriser vos comptes en effectuant une vérification en deux étapes après avoir entré votre mot de passe. Mais saviez-vous qu’elle peut remplacer complètement le mot de passe de votre compte Microsoft personnel ? Découvrez comment configurer et utiliser l’application Microsoft Authenticator.
