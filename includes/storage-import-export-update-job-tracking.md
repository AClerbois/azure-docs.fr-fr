Après avoir expédié les disques, revenez à la page **Import/Export** sur le portail Azure. Si le numéro de suivi n’est pas mis à jour dans les 2 semaines qui suivent la création du travail, ce dernier expire. Pour mettre à jour le numéro de suivi, effectuez les étapes suivantes :
 
1. Sélectionnez le travail et cliquez dessus.
2. Cliquez sur **Mettez à jour l’état de la tâche et les informations de suivi une fois les lecteurs expédiés**. 
3. Activez la case à cocher en regard de **Marquer comme expédié**.
4. Indiquez le **Transporteur** et le **Numéro de suivi**.
5. Vous pouvez suivre l’état d’avancement de votre travail sur le tableau de bord du portail. Pour obtenir une description de chaque état du travail, consultez [Affichage de l’état de votre travail](#viewing-your-job-status).
