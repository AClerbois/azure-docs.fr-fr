# <a name="getting-started-with-the-cloud-partner-portal"></a>Prise en main du portail Microsoft Cloud Partner
Ce document décrit pas à pas les actions à effectuer sur le portail Microsoft Cloud Partner qui prend en charge la publication de produits sur la [Place de marché Microsoft Azure](https://azuremarketplace.microsoft.com/) et sur [AppSource](https://appsource.microsoft.com/). Si ce n’est déjà fait, jetez un coup de œil à la vue d’ensemble [Qu’est-ce que le portail Microsoft Cloud Partner](./cloud-partner-portal-what-is-the-cloud-partner-portal.md).

## <a name="portal-tour"></a>Visite guidée de portail
Le portail Microsoft Cloud Partner contient tout ce dont vous avez besoin pour proposer et exploiter une entreprise prospère dans le cloud. Voici un aperçu rapide des éléments de ce site avec lesquels vous interagirez probablement le plus souvent.

### <a name="left-navigation-bar"></a>Barre de navigation gauche
Lorsque vous accédez pour la première fois au portail, vous devez voir la barre de navigation amovible. Il s’agit de l’emplacement à partir lequel vous pouvez naviguer parmi les options de menu.

![Page du portail Microsoft Cloud Partner](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/cloud-partner-portal-page.png)

Apprenez-en davantage sur chaque élément ci-dessous :
1.    **Nouvelle offre** : lancez une nouvelle offre ici et vérifiez tous les [types de produits pris en charge](./Cloud-partner-portal-products-that-can-get-published-via-portal.md).
2.    **Toutes les offres** : vérifiez l’état et le statut de toutes vos offres en cours.
3.    **Utilisateurs** : [gérez et contrôlez l’accès pour votre société](./cloud-partner-portal-manage-users.md).
4.    **Apprendre** : démarrez votre apprentissage et tenez-vous informé des nouveautés sur le portail Microsoft Cloud Partner.
5.  **Informations** : recueillez des informations sur l’utilisation de vos produits et sur vos clients en visitant le portail Insights.

### <a name="offer-page"></a>Page de l’offre 
Dans la suite de la visite guidée, lorsque vous démarrez une nouvelle offre ou revenez en arrière pour la modifier, vous travaillez principalement dans la page de l’offre. C’est la que vous configurez tous les paramètres de l’offre et vérifiez son état. 

![Page de l’offre](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/offer-page.png )

1.    Ajoutez du contenu ou vérifiez le statut de votre offre en basculant entre les deux modes de niveau supérieur : **Éditeur** et **Statut**.
2.    Utilisez la **Barre d’action** pour gérer explicitement les modifications de votre offre en enregistrant et en ignorant votre entrée.
3.    Naviguez entre les **Formulaires** obligatoires pour chaque offre tandis que vous modifiez les paramètres de votre offre.

### <a name="menu-bar"></a>Barre de menus
Enfin, en haut de votre écran, vous pouvez toujours accéder à la barre de menus supérieure. Cette barre contient des actions interactives pour la prise en charge et les commentaires, ainsi que toutes les informations de votre compte d’éditeur.

![Barre de menus](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/menu-bar.png)

*    **Vous avez besoin d’aide ?** – Si vous vous sentez désorienté et avez besoin d’aide, créez une demande de support et recherchez des liens vers la documentation.
*    **Commentaires** : vous avez quelque chose à nous dire ? Faites-nous en part ici. Microsoft souhaite rendre la Place de marché Microsoft Azure, le processus de publication sur AppSource et l’expérience globale du portail Microsoft Cloud Partner aussi faciles et intuitifs que possible. Nous trions et passons en revue tous les commentaires, et nous respectons toujours votre vie privée (nous vous contactons uniquement si nous avons besoin d’informations supplémentaires). 
*    **Compte d’éditeur** : lorsque vous visitez le portail pour la première fois, vous voyez s’afficher le contrat d’éditeur par lequel vous pouvez accepter de devenir un éditeur. À partir de là, vous pouvez apporter des modifications à votre profil d’éditeur ou revoir le contrat. Il s’agit également de l’emplacement à partir duquel vous vous connectez à votre compte du Centre de développement afin de pouvoir vendre vos offres.

## <a name="next-steps"></a>Étapes suivantes    
À présent que vous avez suivi la visite guidée du portail Microsoft Cloud Partner, apprenez-en davantage sur tous les [types de produits pris en charge](./Cloud-partner-portal-products-that-can-get-published-via-portal.md) que vous pouvez publier.
