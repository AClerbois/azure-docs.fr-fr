# <a name="knowledge-exploration-service"></a>Service d’exploration de la base de connaissances

Bienvenue dans le service d’exploration des connaissances de Microsoft.  Le service d’exploration des connaissances (KES) offre un moyen rapide et efficace d’ajouter les fonctions de recherche interactive et de perfectionnement aux applications.  Avec le service KES, vous pouvez générer un index compressé à partir de données structurées, créer une grammaire qui interprète les requêtes en langage naturel et fournir une formulation de requête interactive avec des suggestions de saisie semi-automatique.

Consultez l’article [Getting Started](GettingStarted.md) (Prendre en main le service d’exploration des connaissances) pour voir un exemple de procédure pas à pas pour utiliser le service KES afin de créer une interface de recherche interactive pour un domaine de publications universitaires.

![AcademicAutoComplete](AutoComplete.png)

Le service KES a été déplacé de la préversion de Cognitive Services à Cognitive Services Labs. Nouvelle page d’accueil du projet : [https://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration](https://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration).
