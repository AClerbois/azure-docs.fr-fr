# <a name="saas-application-channel-info-tab"></a>Onglet « Channel Info » (Informations de canal) pour une application SaaS

Utilisez l’onglet « Channel Info » (Informations de canal) pour fournir des informations sur le canal à votre application, y compris des détails sur la commercialisation et les coordonnées.

![Onglet « Channel Info » (Informations de canal)](./media/saas-channel-info.png)

## <a name="to-configure-channel-information"></a>Pour configurer les informations de canal

1. Sous **Channel Info (Informations de canal) > GTM materials (Supports de commercialisation)**, entrez l’URL du site Web qui héberge vos supports de commercialisation.
2. Sous **Channel Manager Contact (Coordonnées du manager de canal)**, indiquez les coordonnées d’un manager auquel Microsoft peut faire appel en cas de problème lié à la prise en charge ou à l’entreprise. Renseignez les informations de ces champs : Nom, adresse e-mail et numéro de téléphone.
3. Sélectionnez **Enregistrer** pour terminer la configuration des informations de canal.
