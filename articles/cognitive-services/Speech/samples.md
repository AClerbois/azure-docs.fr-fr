# <a name="microsoft-speech-client-samples"></a>Exemples de client Microsoft Speech

Le service Microsoft Speech fournit des exemples de bout en bout expliquant comment utiliser l’API de reconnaissance vocale de Microsoft dans différentes situations, telles que la reconnaissance de commande, la reconnaissance continue et la détection des intentions. Tous les exemples sont disponibles sur GitHub et peuvent être téléchargés à l’aide des liens suivants : le fichier README.md disponible dans chaque référentiel ainsi que la page [Bibliothèques clientes](GetStarted/GetStartedClientLibraries.md) fournissent des informations expliquant comment générer et exécuter les exemples.

- [Exemples d’API REST](https://github.com/Azure-Samples/SpeechToText-REST)

- [Exemples JavaScript](https://github.com/Azure-Samples/SpeechToText-WebSockets-Javascript)

- [Exemples de client C# Desktop](https://github.com/Azure-Samples/Cognitive-Speech-STT-Windows)

- [Exemples de service C#](https://github.com/Azure-Samples/Cognitive-Speech-STT-ServiceLibrary)

- [Exemples Java sur Android](https://github.com/Azure-Samples/Cognitive-Speech-STT-Android)

- [Exemples ObjectiveC sur iOS](https://github.com/Azure-Samples/Cognitive-Speech-STT-iOS)

Tous les SDK Microsoft Cognitive Services et les exemples sont fournis sous licence MIT. Pour plus d’informations, consultez [LICENCE](https://github.com/Azure-Samples/SpeechToText-REST/blob/master/LICENSE).
