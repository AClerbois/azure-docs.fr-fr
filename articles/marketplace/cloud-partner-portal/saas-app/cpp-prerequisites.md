# <a name="saas-application-publishing-prerequisites"></a>Conditions préalables à la publication d’applications SaaS

Cet article décrit les conditions préalables à la publication d’une offre d’application SaaS (App) sur la Place de marché Azure.  Si vous ne le n'avez pas déjà fait, passez en revue la [Guide de publication proposent des applications SaaS](../../marketplace-saas-applications-technical-publishing-guide.md). 


## <a name="publishing-prerequisites"></a>Publication des conditions préalables

Pour publier une offre d’application SaaS, vous devez respecter les conditions préalables suivantes :
- Accéder au [Portail Cloud Partner](https://cloudpartner.azure.com/).  Le portail Cloud Partner fournit l’accès en fonction du rôle, ce qui permet plusieurs personnes à collaborer à la publication d’une offre. 
- Avant de pouvoir publier une offre pour le compte d’un compte de publication, une des personnes disposant du rôle Propriétaire doit s’engager à respecter les [conditions d’utilisation](https://azure.microsoft.com/support/legal/website-terms-of-use/), la [déclaration de confidentialité de Microsoft](https://www.microsoft.com/privacystatement/default.aspx) et le [contrat du programme Microsoft Azure Certified](https://azure.microsoft.com/support/legal/marketplace/certified-program-agreement/).
- Tout le contenu non anglais doit être accompagné d’une version en langue anglaise. Ce contenu comprend le texte de la vitrine, les documents, les captures d’écran, les conditions d’utilisation et la politique de confidentialité. Il est acceptable pour fournir une URL de lien utile vers une application qui n’est pas en anglais. 
- Avoir des métadonnées prêtes à être utilisées. La liste suivante (non exhaustive) présente un exemple de métadonnées :
  - Un titre
  - Une description (au format HTML)
  - Une image de logo (au format PNG), dans les tailles d’image fixes suivantes : 40 x 40 pixels, 90 x 90 pixels, 115 x 115 pixels et 255 x 115 pixels.
  - Un *conditions d’utilisation* et *politique de confidentialité* instructions
  - Documentation de l’application
  - Contacts du support


## <a name="next-steps"></a>Étapes suivantes

Une fois que vous remplissez ces conditions préalables, vous pouvez [créer votre offre SaaS](./cpp-create-offer.md).
