# <a name="azure-application-offer"></a>Offre d’application Azure

<table> <tr> <td>Cette section explique comment publier une nouvelle offre d’application Azure managée sur la <a href="https://azuremarketplace.microsoft.com/marketplace/">Place de marché Microsoft Azure</a>.  Chaque application gérée contient un modèle Azure Resource Manager qui définit toutes les ressources techniques utilisées par l'application, qui comprend généralement une ou plusieurs machines virtuelles et d'autres services de prise en charge Azure ou Web. </td> <td><img src="./media/cloud-partner-portal-publish-managed-app/azureapp-icon1.png"  alt="Azure application icon" /></td> </tr> </table>

## <a name="publishing-overview"></a>Présentation de la publication

La vidéo suivante, [Création de modèles de solutions et d’applications managées pour la Place de marché Azure](https://channel9.msdn.com/Events/Build/2018/BRK3603), est un aperçu de la façon de créer un modèle Azure Resource Manager pour définir une solution d'application Azure, puis de publier l'offre d'application sur la Place de marché Azure Marketplace.

> [!VIDEO https://channel9.msdn.com/Events/Build/2018/BRK3603/player]


## <a name="publishing-process"></a>Processus de publication

Les articles suivants vous guide dans le processus de création et de publication d’une application Azure managée :

1. [Inscrire un compte au centre pour développeurs Microsoft](./cloud-partner-portal-dev-center-accounts-registration.md) pour être habilité à publier sur la Place de marché Azure.
2. [Publier une nouvelle offre d’application Azure](./cloud-partner-portal-managed-app-publish.md) sur la Place de marché Azure.
3. [Faire votre offre en direct](./cloud-partner-portal-managed-app-offer-go-live.md)! 


## <a name="next-steps"></a>Étapes suivantes

Ensuite, vous devez vérifier les configurations requises pour ce type d’offre et commencer le processus de publication conformément à la description dans [Application managée par l’auteur](./cloud-partner-portal-managed-app-publish.md).
