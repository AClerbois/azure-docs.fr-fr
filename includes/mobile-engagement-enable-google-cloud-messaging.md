
### <a name="create-a-google-firebase-project-with-api-key"></a>Créer un projet Google Firebase avec une clé API
> [!NOTE]
> Pour effectuer cette procédure, vous devez disposer d’un compte Google avec une adresse de messagerie valide. Pour créer un compte Google, accédez à <a href="http://go.microsoft.com/fwlink/p/?LinkId=268302" target="_blank">accounts.google.com</a>.
> 
> 

1. Accédez à la [console Firebase](https://console.firebase.google.com/) et connectez-vous avec les informations d’identification de votre compte Google.
2. Cliquez sur le bouton **Créer un nouveau projet** pour créer un nouveau projet. Sinon, vous pouvez également cliquer sur **Importer un projet Google** pour importer un projet existant. 
3. Si vous avez opté pour la création d’un nouveau projet, indiquez un **nom de projet** et choisissez un **pays ou une région**.
4. Dans la page de votre projet, cliquez sur la roue dentée **Paramètres** située en haut à gauche. Dans le menu, cliquez sur **Paramètres du projet**.  
5. Cliquez sur l’onglet **Messagerie cloud** en haut. 
6. Notez la **clé serveur** et l’**ID de l’expéditeur** que vous utiliserez ultérieurement dans le *fichier de manifeste Android*.
