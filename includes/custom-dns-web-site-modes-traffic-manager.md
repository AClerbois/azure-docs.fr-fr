La définition d’un nom de domaine personnalisé sur une application web intégrée avec Traffic Manager est uniquement disponible pour le niveau tarifaire **Standard**.  

Pour plus d’informations sur les niveaux tarifaires d’App Service, notamment sur la modification du niveau tarifaire de votre application, consultez [Montée en puissance d’une application dans Azure](../articles/app-service/manage-scale-up.md).
