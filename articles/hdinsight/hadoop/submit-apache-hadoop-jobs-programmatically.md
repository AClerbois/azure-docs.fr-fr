# <a name="submit-apache-hadoop-jobs-in-hdinsight"></a>Envoi de tâches Apache Hadoop dans HDInsight

Vous pouvez envoyer des tâches Apache Hadoop dans Azure HDInsight à l’aide du Kit de développement logiciel (SDK) Microsoft .NET pour HDInsight, curl ou Azure PowerShell.

- **Kit de développement logiciel (SDK) .NET** :

  - [Créer des applications .NET d’authentification non interactives](../hdinsight-create-non-interactive-authentication-dotnet-applications.md)
  - [Exécuter des requêtes Apache Hive avec le kit SDK HDInsight .NET](apache-hadoop-use-hive-dotnet-sdk.md)
  - [Exécuter des tâches Apache Sqoop avec le Kit SDK .NET pour Apache Hadoop dans HDInsight](apache-hadoop-use-sqoop-dotnet-sdk.md)
  - [Exécuter des tâches MapReduce à l’aide du Kit de développement logiciel (SDK) .NET HDInsight](apache-hadoop-use-mapreduce-dotnet-sdk.md)

- **curl** :

  - [Exécuter des requêtes Apache Hive avec Apache Hadoop dans HDInsight à l’aide de curl](apache-hadoop-use-hive-curl.md)
  - [Exécuter des tâches Apache Sqoop avec Apache Hadoop dans HDInsight à l’aide de curl](apache-hadoop-use-sqoop-curl.md)
  - [Exécuter des tâches MapReduce avec Apache Hadoop dans HDInsight à l’aide de curl](apache-hadoop-use-mapreduce-curl.md)

- **PowerShell** :

  - [Exécuter des requêtes Apache Hive avec PowerShell](apache-hadoop-use-hive-powershell.md)
  - [Utiliser Apache Sqoop avec Apache Hadoop dans HDInsight](apache-hadoop-use-sqoop-powershell.md)
